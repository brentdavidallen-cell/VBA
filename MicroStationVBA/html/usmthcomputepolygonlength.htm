ComputePolygonLength Method

                          

Returns the length of the object's control polygon.

Syntax

Double = object.ComputePolygonLength

The ComputePolygonLength method syntax has these parts:

Part Description
object A valid object.

Remarks

For BsplineCurve objects, the returned value is the length of the control polygon, the piecewise linear curve formed by the Poles.
