CursorInformation Object

                                                     

Provides information on the cursor location.

Remarks

Use DataPointRaw to get the actual location of the data point without adjustment for snaps.

Use DataPointAdjusted to get the location of the last data point adjusted for snaps.

Use CurrentPointRaw to get the current location of the cursor.

Use DataPointView and CurrentView to get the View where the last data point was entered and the View where the cursor is currently located.

Use IsCurrentlyInView to determine if the current cursor location is in a View.

Use IsDataPointPrecision to determine if the last data point was entered via a key-in.

See Centering the View at the Cursor for an example of how to use CurrentView and CurrentPointRaw to determine what View the cursor is in and to center that View about the cursor.
