LatLongFromCartesian Method

                          

Gets the Geographic point (longitude, latitude, and elevation) corresponding to the given Cartesian Coordinates.

Syntax

GeoPoint3D = object.LatLongFromCartesian (Point)

The LatLongFromCartesian method syntax has these parts:

Part Description
object A valid object.
Point A Point3d expression. The cartesian point for which to calculate the longitude, latitude, and elevation.

Remarks

The Cartesian Coordinates are in the units of the GeographicCoordinateSystem.
