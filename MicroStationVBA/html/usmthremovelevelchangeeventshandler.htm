RemoveLevelChangeEventsHandler Method

                          

Removes a level-change-events event handler from the set of active handlers.

Syntax

object.RemoveLevelChangeEventsHandler EventHandler

The RemoveLevelChangeEventsHandler method syntax has these parts:

Part Description
object A valid object.
EventHandler An ILevelChangeEvents expression. An object that was previously passed to AddLevelChangeEventsHandler
