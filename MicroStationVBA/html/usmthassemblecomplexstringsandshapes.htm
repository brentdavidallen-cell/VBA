AssembleComplexStringsAndShapes Method

                          

Returns an ElementEnumerator of new, maximally connected Elements assembled from the given unordered, unconnected set of ChainableElements.

Syntax

Set ElementEnumerator = object.AssembleComplexStringsAndShapes (ChainableElements [, GapTolerance])

The AssembleComplexStringsAndShapes method syntax has these parts:

Part Description
object A valid object.
GapTolerance Optional. A Double expression. The absolute distance tolerance for chaining one ChainableElement to another. If negative or omitted, a default value of 1.0e-5 is used.
ChainableElements A ChainableElement () expression.

Remarks

Important: The geometric arguments are given in the coordinate system of the currently active model. Therefore, before you call this method, you should ensure that the model to which you intend to add the element is the active model. (You can make a model active by calling the Activate method of the appropriate ModelReference object.)

Input ChainableElements may be chained in reversed orientation to preserve continuity.

Output Elements consist of ChainableElements and/or ClosedElements.
