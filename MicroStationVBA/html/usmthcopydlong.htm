CopyDLong Method

                          

Copies a DLong to or from the DataBlock object.

Syntax

object.CopyDLong DLongReference, CopyToDataBlock

The CopyDLong method syntax has these parts:

Part Description
object A valid object.
DLongReference A DLong expression.
CopyToDataBlock A Boolean expression.

Remarks

If CopyToDataBlock is True, it copies 8 bytes from DLongReference to the data block. Otherwise, it copies 8 bytes from the data block to DLongReference. It either case, it increments Offset by 8, the number of bytes transferred.
