VertexList Object

                                                     

An element class implements the VertexList interface if an element of that type can be defined by a list of vertices.

Remarks

Use IsVertexList to determine if an element supports the VertexList element interface. To use an element's VertexList interface, either access the element using AsVertexList or assign the element reference to a variable declared to be VertexList.

Here is an example illustrating the first technique:

    Dim vertexIndex As Long
    If oEle.IsVertexList Then
        With oEle.AsVertexList
            vertexIndex = .GetClosestVertex(point)
            oEle.Redraw msdDrawingModeErase
            .RemoveVertex vertexIndex
            oEle.Redraw msdDrawingModeNormal
            oEle.Rewrite
        End With
    End If

Here is an example illustrating the second technique:

    Dim oVL As VertexList
    If oEle.IsVertexList Then
        Set oVL = oEle
        vertexIndex = oVL.GetClosestVertex(point)
        oEle.Redraw msdDrawingModeErase
        oVL.RemoveVertex vertexIndex
        oEle.Redraw msdDrawingModeNormal
        oEle.Rewrite
    End If

Both examples first determine if the element is a VertexList prior to trying to use it as a VertexList. If an element is not a VertexList, then trying to access it as a VertexList results in an error indicating that there is a type mismatch or that the class does not support the expected interface.
