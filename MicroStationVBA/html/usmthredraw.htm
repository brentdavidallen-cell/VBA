Redraw Method

                          

Redraws this object.

Syntax

For the TextNodeElement object, TextElement object, TagElement object, SmartSolidElement object, SharedCellElement object, SharedCellDefinitionElement object, ShapeElement object, SavedViewElement object, PointStringElement object, NamedGroupElement object, MultiLineElement object, LineElement object, EllipseElement object, Element object, DimensionElement object, CurveElement object, ConeElement object, ComplexStringElement object, ComplexShapeElement object, CellElement object, BsplineSurfaceElement object, BsplineCurveElement object, AuxiliaryCoordinateSystemElement object, Attachment object, ArcElement object and ApplicationElement object:

   object.Redraw [DrawMode]

For the View object:

   object.Redraw

The Redraw method syntax has these parts:

Part Description
object A valid object.
DrawMode Optional. An MsdDrawingMode expression. How the element is drawn. Its default value is msdNormalDraw.

Remarks

Element.Redraw redraws the element in all views where it is visible.

For an element, calling Redraw raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.

View.Redraw first clears the view, then redraws its contents.
