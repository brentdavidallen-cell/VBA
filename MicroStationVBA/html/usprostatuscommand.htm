StatusCommand Property

                          

Read/Write String.

Gets or Sets the command message in MicroStation's status area.

Syntax

object.StatusCommand [= Command]

The StatusCommand property syntax has these parts:

Part Description
object A valid object.
Command A String expression.

Remarks

This gets or sets the Command regardless of whether MessagesEnabled is True or False. The command is the string to the left of the > character.

A program that implements a command by implementing IPrimitiveCommandEvents or ILocateCommandEvents should set the command name using CommandState.CommandName because MicroStation saves CommandState.CommandName when it starts a view manipulation command and restores it when it resumes the primitive command at the end of the view manipulation command.
