AddArrayElements Method

                          

Add no. of empty array elements to the specified array property

Syntax

Boolean = object.AddArrayElements (AccessString, size)

The AddArrayElements method syntax has these parts:

Part Description
object A valid object.
AccessString A String expression. The access string to the array property (shall not contain brackets)
size A Long expression. The number of empty array elements to add

Remarks

AddArrayElements creates no. of empty array elements using a ItemTypePropertyHandler
