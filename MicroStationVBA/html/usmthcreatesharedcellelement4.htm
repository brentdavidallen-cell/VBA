CreateSharedCellElement4 Method

                          

Creates a SharedCellElement.

Syntax

Set SharedCellElement = object.CreateSharedCellElement4 (CellName, DestinationModel, TrueScale, SearchControl)

The CreateSharedCellElement4 method syntax has these parts:

Part Description
object A valid object.
CellName A String expression. Name of the cell to search for.
DestinationModel A ModelReference expression. This parameter is normally ActiveModelReference. If it is Nothing CreateSharedCellElement4 uses the active model reference.
SearchControl An MsdSharedCellSearch expression. A set of flags telling CreateSharedCellElement4 where to search for the shared cell definition. Values can be combined with Or as in msdSharedCellSearchDestinationModel Or msdSharedCellSearchLibraries. Typically this will be msdSharedCellSearchDefaultFlags.
TrueScale A Boolean expression.

Remarks

Unlike the other CreateSharedCellElement methods, CreateSharedCellElement4 lets the caller specify where to search for the shared cell defintion. The other CreateSharedCellElement methods use msdSharedCellSearchDestinationModel Or msdSharedCellSearchReferences Or msdSharedCellSearchLibraries as the SearchControl. Notice that set of flags does not include msdSharedCellSearchActiveLibary. msdSharedCellSearchActiveLibary is required to find the shared cell definition in the active library if the active library does not also include the normal cell of the same name.
