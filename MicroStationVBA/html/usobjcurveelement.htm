CurveElement Object

                                                     

A kind of Element that corresponds to a MicroStation curve element.

Remarks

If you have a generic Element object (as the result of a Scan, for example), you can determine whether it's a CurveElement by examining the element's IsCurveElement property. If this property returns True, you can extract the element's implementation of CurveElement by calling the element's AsCurveElement property. (You can accomplish the same thing by assigning the element to a variable of the type CurveElement. But Element.AsCurveElement saves you a step.)

You cannot use the New keyword to create a CurveElement object. Rather, use the global function CreateCurveElement1.

Remember that a newly-created element (of any type) is an "invisible orphan:" It does not appear on the screen, nor is it part of any design file.

  • To display an element on the screen (regardless of whether it's part of a design file), call the element's Redraw method.
  • To add the element to the current design file (regardless of whether it's displayed on the screen), call AddElement on the appropriate ModelReference object (typically, ActiveModelReference).

Example

    Dim oCurve As CurveElement
    Set oCurve = CreateCurveElement1(...) ' Click on the link above to
                                             ' see the arguments list for
                                             ' CreateCurveElement1.
    With oCurve
        ' ...Set other properties of oCurve, if you wish.
    End With
    ActiveModelReference.AddElement oCurve
    oCurve.Redraw
