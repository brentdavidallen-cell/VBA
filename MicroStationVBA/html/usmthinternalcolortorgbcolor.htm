InternalColorToRGBColor Method

                          

Syntax

Long = object.InternalColorToRGBColor (InternalColor)

The InternalColorToRGBColor method syntax has these parts:

Part Description
object A valid object.
InternalColor A Long expression.

Remarks

An internal color value is a number that contains 2 indices. The least significant byte is an index of the closest matching color in the color table. The remainder of the value is an index into the model's table of RGB values.

This method uses the index into the table of RGB values to retrieve the exact RGB value this index represents.
