ApplyVerticalScalingFixForEMF Method

                          

Applies the vertical scaling fix for enhanced metafile (EMF). When displaying EMF in Windows 2000, the size of an EMF image changes. It assumes "magic" screen dimension writing to an enhanced metafile is 1209x907 regardless of actually screen resolution. This causes an image to be displayed in the wrong scale if the actual resolution does not match 1209x907.

Syntax

Long = object.ApplyVerticalScalingFixForEMF (PixelCoordinate)

The ApplyVerticalScalingFixForEMF method syntax has these parts:

Part Description
object A valid object.
PixelCoordinate A Double expression. Y coordinate in points.

Version

08.01.00
