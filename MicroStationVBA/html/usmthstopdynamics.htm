StopDynamics Method

                          

A command should call StopDynamics if it has been displaying dynamics and no longer needs to display the dynamics.

Syntax

object.StopDynamics

The StopDynamics method syntax has these parts:

Part Description
object A valid object.

Remarks

StopDynamics is rarely used because dynamics is terminated every time a command is started. For example, calling StartPrimitive or StartLocate, or starting a MicroStation primitive or locate command automatically terminates dynamics.
