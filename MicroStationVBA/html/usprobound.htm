Bound Property

                          

Read/Write Point2d ().

Gets or sets the array of points comprising the boundary of this object at the given valid index.

Syntax

object.Bound (Index)[= Point2dArray]

The Bound property syntax has these parts:

Part Description
object A valid object.
Index A Long expression. For BsplineSurface objects, the zero-based index (between 0 and BoundsCount-1, inclusive).
Point2dArray A Point2d () expression.

Remarks

For BsplineSurface objects, closed piecewise linear boundaries may be specified to trim or puncture (cf. BoundsTrim) the B-spline surface. Each boundary is a 1-dimensional array of arbitrarily many Point2d points representing the vertices of the boundary in uv-parameter space. The first and last points of a boundary must be equal, and there must be at least four points. The object's boundaries are maintained as an unordered collection using the methods AddBound and RemoveBoundAt.

For BsplineSurface objects, the x-coordinate of a boundary point falls within the range of the object's knots in the u-parameter direction, and the y-coordinate falls within the range of its knots in the v-parameter direction (usually both ranges are between 0.0 and 1.0, inclusive).
