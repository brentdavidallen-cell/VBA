RemoveViewUpdateEventsHandler Method

                          

Removes a view-update-events event handler from the set of active handlers.

Syntax

object.RemoveViewUpdateEventsHandler EventHandler

The RemoveViewUpdateEventsHandler method syntax has these parts:

Part Description
object A valid object.
EventHandler An IViewUpdateEvents expression. An object that was previously passed to AddViewUpdateEventsHandler
