AxisOrigin Property

                          

Read/Write Double.

Gets or sets the origin angle (in radians) for axis lock.

Syntax

object.AxisOrigin [= Double]

The AxisOrigin property syntax has these parts:

Part Description
object A valid object.
Double A Double expression.

Remarks

For example, if AxisOrigin is Pi/4 (45 degrees) and AxisAngle is Pi/2 (90 degrees), the axis lock will lock at 45, 135, 225... degrees.
