ExtractBackClippingPlane Method

                          

Returns the back clipping plane of the View.

Syntax

Plane3d = object.ExtractBackClippingPlane

The ExtractBackClippingPlane method syntax has these parts:

Part Description
object A valid object.

Remarks

The returned Plane3d.Origin is a point at the bottom left corner of the back face of the view area. The returned Plane3d.Normal is the back face's normal vector, pointing into the view area.
