Add Method

                          

Syntax

For the Attachments object:

   Set Attachment = object.Add (FileSpecification, ModelName, LogicalName, Description, MasterOrigin, ReferenceOrigin [, TrueScale [, DisplayImmediately]])

For the ModelReferences object:

   Set ModelReference = object.Add (ModelTemplate [, Name [, Description [, ModelType [, Is3d]]]])

For the RasterClipMasks object:

   Boolean = object.Add (Element [, ReferenceSystem])

For the TagDefinitions object:

   Set TagDefinition = object.Add (Name, TagType)

For the TagSets object:

   Set TagSet = object.Add (Name)

For the TextStyles object:

   Set TextStyle = object.Add (TextStyleTemplate, TextStyleName)

For the ViewGroups object:

   object.Add Name, Description

The Add method syntax has these parts:

Part Description
object A valid object.
FileSpecification A String expression. For Attachments.Add, specifies the name of a reference file. The drive, directory name, and extension are all optional.
ModelName A String expression. For Attachments.Add, the name of the model to attach. If vbNullString is specified, attaches the default model.
LogicalName A String expression. The logical name for the attachment. The logical name must be unique.
MasterOrigin A Point3d expression. A point in the model that contains the attachments. Specified in master units.
ReferenceOrigin A Point3d expression. A point in the model that is being attached. Specified in UOR's, and not in master units.
TrueScale Optional. A Boolean expression. Specify True to generate a scale factor that maps the master units of the master model and the attached model. Specify False to produce a one-to-one mapping of UOR's.
DisplayImmediately Optional. A Boolean expression. Specify True to immediately display the attachment. Specify False if scale (see ScaleFactor) or rotate (see Rotate) operations take place prior to displaying the reference file. Use Redraw to subsequently display the attachment.
Element An Element expression. The MicroStation element from which to shape the clip mask.
ReferenceSystem Optional. An MsdReferenceSystem expression. MsdReferenceSystem describing the Elem's reference system. Usually msdReferenceSystemDgn.
TextStyleTemplate A TextStyle expression. Provides the initial values for the TextStyle's properties. If TextStyleTemplate is Nothing, Add copies the initial values from ActiveSettings.TextStyle.
TextStyleName A String expression. The name of the new TextStyle. TextStyleName cannot be vbNullString.
Description A String expression.
ModelTemplate A ModelReference expression.
Name Optional. A String expression.
ModelType Optional. An MsdModelType expression.
Is3d Optional. A Boolean expression.
TagType An MsdTagType expression.

Remarks

Attachments.Add attaches a reference file.

For Attachments, the format of the FileSpecification is configuration_variable:filename where the configuration_variable: part is optional. The configuration_variable part and the colon must appear together. Do not use one without using the other.

If FileSpecification specifies configuration_variable: then MicroStation records configuration_variable: as part of the Attachment. It then evaluates the configuration variable everytime MicroStation loads the Attachment.

If FileSpecification does not provide a full file specification, the master file path and the environment variables MS_RFDIR and MS_DEF are searched sequentially for the file.

ModelReferences.Add creates a new model, copying the settings from ModelTemplate. If the Name argument is vbNullString, Add generates a unique name. Otherwise, the name must be unique. The default value for ModelType is msdModelTypeDefault. The default value for Is3d is False.

RasterClipMasks.Add adds a new clip mask to the list creating the clip mask from an element. For example, 


    Dim oElem As Element
    Dim theClipMasks As RasterClipMasks

    ...

    If (theClipMasks.Add(oElem)) Then
        ...
    End If

TagDefinitions' Add method creates a new TagDefinition, adds it to the TagDefinitions collection, and saves it to the DesignFile. If the TagDefinitions collection already contains a TagDefinition with the requested name, Add generates an error.

TagSets' Add method creates a new TagSet, adds it to the TagSets collection, and saves it to the DesignFile. If the TagSets collection already contains a TagSet with the requested name, Add generates an error.

TextStyles.Add verifies that there is not already a text style with the name that the TextStyle parameter specifies. If it finds a TextStyle with that name, it raises an msdErrorNameNotUnique exception.
