CreateShapeElement3 Method

                          

Gets the CreateShapeElement3 object for Array of Elements which is passed as VARIANT

Syntax

Set ShapeElement = object.CreateShapeElement3 (Template, Vertices [, FillMode])

The CreateShapeElement3 method syntax has these parts:

Part Description
object A valid object.
Template An Element expression.
Vertices A VARIANT expression.
FillMode Optional. An MsdFillMode expression.
