MsdCommandResult Enum

        

Defines constants that represent common status codes set by MicroStation commands.

The MsdCommandResult enum has these members:

Name Description
msdCommandResultSuccess 0 (&H0) Successful processing.
msdCommandResultReferenceNotFound 7 (&H7) Could not find the reference
msdCommandResultCellLibraryNotFound 12 (&HC) Could not find the cell library
msdCommandResultNoFenceActive 15 (&HF) Operation requires a fence, but there is no active fence
msdCommandResultUnknownCommand 16 (&H10) Command not recognized
msdCommandResultViewNotFound 18 (&H12) Could not find the named view
msdCommandResultNoActiveCell 19 (&H13) Attempt to place cell with no active cell
msdCommandResultElementNotFound 21 (&H15) Tried to locate the element, but could not
msdCommandResultOffDesignPlane 22 (&H16) Data point put element off the design plane
msdCommandResultIllegalDefinition 23 (&H17) Data point resulted in an illegal element definition
msdCommandResultEmptyFence 27 (&H1B) No elements found within the fence
msdCommandResultNeedCharacters 27 (&H1B) Text data point before entering string
msdCommandResult3dOnly 39 (&H27) 2D file, 3D command
msdCommandResultCellNestingError 43 (&H2B) Cell nesting error (nests itself)
msdCommandResultCellNotFound 44 (&H2C) Cell not found
msdCommandResultBadCellName 47 (&H2F) Invalid cell name when creating a cell
msdCommandResult3dLibrary2dFile 50 (&H32) Attempt to create 3D cell in 2D library
msdCommandResultNoCellLibrary 54 (&H36) No cell library is attached
msdCommandResultCellExists 55 (&H37) Cell already in library
msdCommandResultNoOrigin 56 (&H38) No cell origin defined when creating cell
msdCommandResultCellDeleted 59 (&H3B) Cell successfully deleted
msdCommandResultAcceptQuery 68 (&H44) Element found, prompting "Accept/Query"
msdCommandResultFileReadOnly 287 (&H11F) Tried to write to read-only design file
msdCommandResultInvalidReferenceOperation 481 (&H1E1) Invalid reference file operation

Remarks

Use CommandState.CommandResult to retrieve the result of the most recently executed MicroStation command.
