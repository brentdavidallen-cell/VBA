ExtractSurfaceFromSolid Method

                          

This function extract one surface from the solid. The output surface is a bspline surface element.

Syntax

Set BsplineSurfaceElement = object.ExtractSurfaceFromSolid (closestPointOnFace)

The ExtractSurfaceFromSolid method syntax has these parts:

Part Description
object A valid object.
closestPointOnFace A Point3d expression. input point that lie on the face.
