IntersectPlane3d Method

                          

Returns the intersections between this object and the given Plane3d.

Syntax

Point3d () = object.IntersectPlane3d (Parameters, Plane)

The IntersectPlane3d method syntax has these parts:

Part Description
object A valid object.
Parameters A Double () expression. The parameter(s) of the point(s) of intersection on this object.
Plane A Plane3d expression. The plane to intersect with this object.

Remarks

For BsplineCurve objects, on a successful return, the arrays are empty if and only if no intersections were found.
