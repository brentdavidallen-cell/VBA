DisplaysDynamics Property

                          

Read/Write Boolean.

Note this property was deprecated after Microstation V8i Edition.

Gets or sets a value indicating whether the view displays dynamics.

Syntax

object.DisplaysDynamics [= Boolean]

The DisplaysDynamics property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

This value is equivalent to the Dynamics checkbox in the View Attributes dialog box.

If DisplaysDynamics is True, elements are dynamically updated as they are created or modified, letting you see an element or modification before it is actually placed in the design.
