IsUsed Property

                          

Read-only Boolean.

Gets a value indicating whether the object is used.

Syntax

object.IsUsed

The IsUsed property syntax has these parts:

Part Description
object A valid object.

Remarks

For a SharedCellDefinitionElement, it returns a value indicating whether the shared cell definition is used anyplace in the design file. The value indicates whether the definition is used by any of the models, not just the active model or the model used for scan.

Shared cell definitions may be used by shared cell instances, patterns, dimensions, etc.
