Range3dIsContainedInRange3d Method

                          

Test if the first range is contained in the second range.

Syntax

Boolean = object.Range3dIsContainedInRange3d (InnerRange, OuterRange)

The Range3dIsContainedInRange3d method syntax has these parts:

Part Description
object A valid object.
InnerRange A Range3d expression. candidate inner range.
OuterRange A Range3d expression. candidate outer range.
