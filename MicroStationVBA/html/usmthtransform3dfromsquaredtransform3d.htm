Transform3dFromSquaredTransform3d Method

                          

Construct a transform which preserves both a primary column directon and a secondary column plane. Scale all columns to length of primary axis.

Syntax

Transform3d = object.Transform3dFromSquaredTransform3d (Transform, PrimaryAxis, SecondaryAxis)

The Transform3dFromSquaredTransform3d method syntax has these parts:

Part Description
object A valid object.
Transform A Transform3d expression. original matrix.
PrimaryAxis A Long expression. axis to be retained.
SecondaryAxis A Long expression. axis defining plane to be maintained.
