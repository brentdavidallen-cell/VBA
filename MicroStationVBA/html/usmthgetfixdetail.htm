GetFixDetail Method

                          

ShowCheckerErrorWithFixOptions calls the checker's GetFixDetail method to get information to display in the bottom of the Standards Checker status dialog box.

Syntax

object.GetFixDetail Fixes, SelectedFix, FixPropertiesLabel, FixProperties

The GetFixDetail method syntax has these parts:

Part Description
object A valid object.
Fixes A String () expression. This is the Fixes array passed into ShowCheckerErrorWithFixOptions.
SelectedFix A Long expression. An index into Fixes. It specifies the row that the user has selected.
FixPropertiesLabel A String expression. Output. This determines what the StandardsCheckerController displays as the label of the list box at the bottom of the status dialog box.
FixProperties A String () expression. Output.

Remarks

See ShowCheckerErrorWithFixOptions for a sample implementation of GetFixDetail.
