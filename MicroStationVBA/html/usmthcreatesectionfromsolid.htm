CreateSectionFromSolid Method

                          

Create section elements from the solid intersecting with a surface.The output may be a array of Elements.

Syntax

Set ElementEnumerator = object.CreateSectionFromSolid (solid, toolSurface)

The CreateSectionFromSolid method syntax has these parts:

Part Description
object A valid object.
solid A SmartSolidElement expression. the solid which is trimmed. .
toolSurface An Element expression. input surface intersecting with the solid.
