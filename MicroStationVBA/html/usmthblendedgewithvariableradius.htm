BlendEdgeWithVariableRadius Method

                          

Blends one edge of a solid.

Syntax

Set SmartSolidElement = object.BlendEdgeWithVariableRadius (solid, closestPoint, radius, smoothEdges [, startRadius [, endRadius]])

The BlendEdgeWithVariableRadius method syntax has these parts:

Part Description
object A valid object.
solid A SmartSolidElement expression. the solid which edges are blended.
radius A Double expression. the radius of the blend.
smoothEdges A Boolean expression. If TRUE, edges that are tangentially continuous are blended in one operation.
startRadius Optional. A Double expression. the radius at the beginning of the blend.
endRadius Optional. A Double expression. the radius at the end of the blend.
closestPoint A Point3d expression.
