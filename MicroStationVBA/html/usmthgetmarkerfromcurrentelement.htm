GetMarkerFromCurrentElement Method

                          

Retrieves the value of the program-defined marker stored on the current element.

Syntax

Long = object.GetMarkerFromCurrentElement

The GetMarkerFromCurrentElement method syntax has these parts:

Part Description
object A valid object.

Remarks

A program can mark an element in a CellElement using MarkCurrentElement and can use MoveToMarkedElement to make a specific component the current element.

An element can only have 1 program-defined marker value, but more than 1 element of a cell can have a marker.

The marker is temporary. It is not saved back to the cache or to the design file.
