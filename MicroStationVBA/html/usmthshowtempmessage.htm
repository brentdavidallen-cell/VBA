ShowTempMessage Method

                          

Displays a message temporarily in the status bar.

Syntax

object.ShowTempMessage Area, Message [, Details]

The ShowTempMessage method syntax has these parts:

Part Description
object A valid object.
Area An MsdStatusBarArea expression. The area of the status bar in which the message is displayed. The possible values are as follows:

msdStatusBarAreaLeft denotes the Command/Prompt (left) area of the status bar.

msdStatusBarAreaMiddle denotes the Message (middle) area of the status bar.

Message A String expression. The message text. This text appears in both the status bar and the Message Center.
Details Optional. A String expression. Additional text associated with this message. This text appears only in the Message Center.

Remarks

The message appears only until another message is displayed in the same status bar area.
