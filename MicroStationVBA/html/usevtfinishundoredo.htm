FinishUndoRedo Event

                          

Event handler called after undo or redo command has completed. This means that all of the changes between the start and end marks have been undone or redone.

Use AddChangeTrackEventsHandler and RemoveChangeTrackEventsHandler to add and remove a Change Track events handler. A Change Track events handler is an instance of a class that implements the IChangeTrackEvents interface.

Syntax

object.FinishUndoRedo IsUndo

The FinishUndoRedo method syntax has these parts:

Part Description
object A valid object.
IsUndo A Boolean expression.
