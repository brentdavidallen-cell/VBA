SetPattern Method

                          

Associate a pattern with this object.

Syntax

object.SetPattern Pattern, Rotation

The SetPattern method syntax has these parts:

Part Description
object A valid object.
Pattern A Pattern expression. Provide an AreaPattern, CrossHatchPattern, or HatchPattern object.
Rotation A Matrix3d expression. Defines the orientation of the coordinate system from which the pattern angle is measured. For no rotation, specify Matrix3dIdentity.

Remarks

SetPattern uses the element's origin as the origin of the pattern. Use SetPatternWithOrigin to specify the origin while setting the pattern.

SetPattern examines the Pattern parameter to decide what type of pattern is desired. Depending on the Pattern parameter, SetPattern may hatch, cross hatch, or area pattern the element.

The program must Rewrite the element to save the pattern and Redraw the element to display the pattern. The following example illustrates this.


Sub DoAreaPattern(eleClosed As ClosedElement)
    Dim ele As Element
    Dim ptrn As AreaPattern

    Set ele = eleClosed ' To provide access to the methods of Element
    Set ptrn = CreateAreaPattern(0.1, 0.1, Radians(45), "SRM360", 0.1)
    eleClosed.SetPattern ptrn, Matrix3dIdentity
    ele.Rewrite
    ele.Redraw
End Sub
