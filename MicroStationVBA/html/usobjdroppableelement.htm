DroppableElement Object

                                                     

A class implements the DroppableElement interface if elements of that class can be decomposed into a list of smaller elements.

Remarks

Every class that implements the ComplexElement interface also implements the DroppableElement interface. If a class supports the DroppableElement interface but not the ComplexElement interface, then elements of that class do not consist of other elements, but they can be decomposed into a list of smaller elements.

Use IsDroppableElement to determine if an element supports the DroppableElement element interface. To use an element's DroppableElement interface, either access the element using AsDroppableElement or assign the element reference to a variable declared to be DroppableElement.

Prior to trying to use an element as a DroppableElement, test to see if the element actually is a DroppableElement. If an element is not a DroppableElement, then trying to access it as a DroppableElement results in a error indicating that there is a type mismatch or that the class does not support the expected interface.
