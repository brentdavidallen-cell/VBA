DepthLockEnabled Property

                          

Read/Write Boolean.

Gets or sets a value indicating whether the depth lock is turned on.

Syntax

object.DepthLockEnabled [= Boolean]

The DepthLockEnabled property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

When this value is True and the user snaps to an element in a 3D file, the input data point is projected onto the active depth for the input view before it is processed.

When this value is False, the input data point is the point on the element that was snapped to.
