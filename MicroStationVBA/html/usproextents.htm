Extents Property

                          

Read/Write Point3d.

Gets or sets the size of the view.

Syntax

object.Extents [= Point3d]

The Extents property syntax has these parts:

Part Description
object A valid object.
Point3d A Point3d expression.

Remarks

Extents is specified in master units. X specifies the horizontal range of what it displayed and Y specifies the vertical range of what is displayed. For a View that is displaying a 3D model, Extents.Z specifies the distance from the back clipping plane (the plane that contains Origin) to the front clipping plane.

When used to set the size of the view, Extents adjusts the coordinates to match the aspect ratio of the view. It increases the X or Y of the new extents as necessary to preserve the aspect ratio. Therefore, setting Extents specifies the minimum area that is displayed in the view. The portion of the model that is actually displayed in the view may be larger. Because of the logic to preserve the aspect ratio, both of the following examples double the range in both the X and Y directions:


'  This subroutine quadruples the range
'  in both the X and Y directions
Sub ChangeExtents(oView As View)
    Dim pnt As Point3d

    '  Extents doubles Y to preserve the
    '  aspect ratio
    pnt = oView.Extents
    pnt.X = pnt.X * 2
    oView.Extents = pnt

    '  Extents doubles X to preserve the
    '  aspect ratio
    pnt = oView.Extents
    pnt.Y = pnt.Y * 2
    oView.Extents = pnt
End Sub

Changing this value does not cause an immediate change in what is displayed in the view. To show the change in the view, call the View.Redraw method.

See View for information on the relationship of Origin, Extents, ActiveDepth, and Rotation.
