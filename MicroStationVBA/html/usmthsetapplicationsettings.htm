SetApplicationSettings Method

                          

Given an XML DOM, it creates an element and stores the XML nodes on the element. It saves the element to the settings DgnLib.

Syntax

object.SetApplicationSettings SettingsDOM

The SetApplicationSettings method syntax has these parts:

Part Description
object A valid object.
SettingsDOM An Object expression.

Remarks

If a checker can represent its settings in an XML DOM, then it can save its settings in the settings DgnLib file using SetApplicationSettings, can retrieve the settings using GetApplicationSettings, and can delete the settings using DeleteApplicationSettings. This approach is recommended but is not a requirement.

SetApplicationSettings saves the data in an XML element with linkage ID LINKAGEID_StandardsCheckerSettings (22256) and subtype STDSCHECKXML_VBAHandlerData (5). SetApplicationSettings tags the element with a string linkage formed by combining the application's IdentityString and the current SettingsName. Given an IdentityString of theIdentityString and SettingsName of theSettingsName, the string linkage would be theIdentityString[theSettingsName].
