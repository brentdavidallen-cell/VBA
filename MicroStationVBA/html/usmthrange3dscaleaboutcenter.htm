Range3dScaleAboutCenter Method

                          

scale a range about its center point.

Syntax

Range3d = object.Range3dScaleAboutCenter (RangeIn, Scale)

The Range3dScaleAboutCenter method syntax has these parts:

Part Description
object A valid object.
RangeIn A Range3d expression. original range
Scale A Double expression. scale factor
