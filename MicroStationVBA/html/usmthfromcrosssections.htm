FromCrossSections Method

                          

Initializes this object with a lofting surface that interpolates the given cross sectional curves.

Syntax

object.FromCrossSections Sections [, LoftDirection [, LoftOrder [, MakeCurvesCompatible [, DerivativeContinuityAtSeam]]]]

The FromCrossSections method syntax has these parts:

Part Description
object A valid object.
Sections A BsplineCurve () expression. The cross-sectional curves to be interpolated.
LoftDirection Optional. An MsdBsplineSurfaceDirection expression. The parameter direction for lofting. If omitted, a default direction of msdBsplineSurfaceV is used, e.g., the Section curves are curves in the u parameter, each at fixed values of v.
LoftOrder Optional. A Long expression. The order in the lofting direction. If omitted, a default order of 4 (cubic) is used.
MakeCurvesCompatible Optional. A Boolean expression. Indicates whether the curves in Sections are to be made compatible (by degree-raising to a common order and by knot insertion to a common knot vector). If omitted, a default value of True is used. If False, the curves are assumed to be compatible already.
DerivativeContinuityAtSeam Optional. A Boolean expression. Indicates whether, in the case of the first and last Section curves being equal, to enforce C1 continuity across this closure. If omitted, a default value of True is used. If False, C1 continuity across this closure seam is not enforced.
