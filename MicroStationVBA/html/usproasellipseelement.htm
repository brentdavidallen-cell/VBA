AsEllipseElement Property

                          

Read-only EllipseElement.

Gets this object's implementation of the EllipseElement interface.

Syntax

object.AsEllipseElement

The AsEllipseElement property syntax has these parts:

Part Description
object A valid object.

Remarks

An error is raised if this object does not support the EllipseElement interface. If you're in doubt, firstly check this object's the value of IsEllipseElement property.
