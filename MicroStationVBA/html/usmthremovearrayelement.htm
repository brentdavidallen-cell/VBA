RemoveArrayElement Method

                          

Remove specific array element from the specified array property

Syntax

Boolean = object.RemoveArrayElement (AccessString, index)

The RemoveArrayElement method syntax has these parts:

Part Description
object A valid object.
AccessString A String expression. The access string to the array property (shall not contain brackets)
index A Long expression. The index of the element to remove

Remarks

RemoveArrayElement removes the specified array element using a ItemTypePropertyHandler
