ConstructFillet Method

                          

Creates and returns a new ArcElement representing a fillet between this object and the given Element.

Syntax

Set ArcElement = object.ConstructFillet (Element, Template, Radius, PickPoint, Rotation)

The ConstructFillet method syntax has these parts:

Part Description
object A valid object.
Element A TraversableElement expression. An Element whose IsTraversableElement property is True.
Template An Element expression. An existing element whose settings are used to initialize the new element. If Nothing, the new element's settings are initialized from MicroStation's active settings.
Radius A Double expression. The radius of the fillet.
PickPoint A Point3d expression. A point near the desired fillet location between this object and Element. There are generally multiple fillets possible given a radius and two intersecting elements; the one closest to PickPoints is created and returned.
Rotation A Matrix3d expression. UNUSED PARAMETER.

Remarks

Important: The geometric arguments are given in the coordinate system of the currently active model. Therefore, before you call this method, you should ensure that the model to which you intend to add the element is the active model. (You can make a model active by calling the Activate method of the appropriate ModelReference object.)

For best results, the TraversableElement instance and argument should intersect, or at least extend to the points where the computed fillet touches these elements. These two TraversableElements are not modified (e.g., trimmed, extended) by this method.
