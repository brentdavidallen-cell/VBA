BaseElement Property

                          

Read-only Element.

Gets the Element that this element is attached to.

Syntax

object.BaseElement

The BaseElement property syntax has these parts:

Part Description
object A valid object.

Remarks

The TagElement is dependent on the base element. Deleting the base element automatically deletes all of the base element's tag elements. Moving the base element also moves the tag elements.
