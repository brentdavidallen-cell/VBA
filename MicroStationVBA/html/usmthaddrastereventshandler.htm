AddRasterEventsHandler Method

                          

Adds a raster events handler. This handler will be notified when a raster is opened, closed, modified, selected or plotted.

Syntax

object.AddRasterEventsHandler EventHandler

The AddRasterEventsHandler method syntax has these parts:

Part Description
object A valid object.
EventHandler An IRasterEvents expression. Raster events handler to add.
