DLongFromLong Method

                          

Returns the DLong representation of a Long expression.

Syntax

DLong = object.DLongFromLong (Value)

The DLongFromLong method syntax has these parts:

Part Description
object A valid object.
Value A Long expression.

Remarks

Example: 

    Dim N As DLong
    N = DLongFromLong(123456789)
                    ' Now N contains the value 123456789 (as a 64-bit value).
