ExpandConfigurationVariable Method

                          

Returns the result of expanding the nested configuration variable references.

Syntax

String = object.ExpandConfigurationVariable (Expression)

The ExpandConfigurationVariable method syntax has these parts:

Part Description
object A valid object.
Expression A String expression. May contain nested expression variable references.

Remarks

ConfigurationVariableValue and ExpandConfigurationVariable provide similar functionality. The difference is that the argument to ConfigurationVariableValue is the name of a configuration variable but the argument to ExpandConfigurationVariable is a string to expand. ConfigurationVariableValue retrieves the value of the named configuration variable and optionally expands it. ExpandConfigurationVariable expands the string that is passed into it.

An argument to ConfigurationVariableValue would be something like "MS_TAGOUTOUT". An argument to ExpandConfigurationVariable would be something like "$(_USTN_USER)$(_USTN_USERNAME).txt".
