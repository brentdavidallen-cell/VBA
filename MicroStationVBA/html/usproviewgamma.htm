ViewGamma Property

                          

Read/Write Double.

Gets or sets the view gamma. 


    Dim rendInf As RenderingInformation

    Set rendInf = RasterManager.Rasters.Item(1).RenderingInformation

    If (rendInf.ViewGamma > 4) Then
        rendInf.ViewGamma = 3.9
    End If

Syntax

object.ViewGamma [= Double]

The ViewGamma property syntax has these parts:

Part Description
object A valid object.
Double A Double expression. Gamma correction used when drawing the raster in the view.
