StackedFractionType Property

                          

Read/Write MsdDimStackedFractionType.

Gets this object's stacked fraction type.

Syntax

object.StackedFractionType [= MsdDimStackedFractionType]

The StackedFractionType property syntax has these parts:

Part Description
object A valid object.
MsdDimStackedFractionType An MsdDimStackedFractionType expression.

Remarks

See MsdDimStackedFractionType for possible valid values.
