TextJustification Property

                          

Read/Write MsdDimTextJustification.

Gets this object's text justification.

Syntax

object.TextJustification [= MsdDimTextJustification]

The TextJustification property syntax has these parts:

Part Description
object A valid object.
MsdDimTextJustification An MsdDimTextJustification expression.

Remarks

See MsdDimTextJustification for possible valid values.
