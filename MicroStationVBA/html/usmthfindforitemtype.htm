FindForItemType Method

                          

Searches DesignFile/ModelReference/Element for a ItemTypePropertyHandler for given ItemType. If found, returns the corresponding ItemTypePropertyHandler object; else returns Nothing.

Syntax

Set ItemTypePropertyHandler = object.FindForItemType (source)

The FindForItemType method syntax has these parts:

Part Description
object A valid object.
source An Object expression. ItemType which can be attched to DesignFile/ModelReference/Element Items.
