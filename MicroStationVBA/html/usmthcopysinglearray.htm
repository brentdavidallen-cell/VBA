CopySingleArray Method

                          

Copies an array of Single's to or from the DataBlock object.

Syntax

object.CopySingleArray SingleArray, CopyToDataBlock

The CopySingleArray method syntax has these parts:

Part Description
object A valid object.
SingleArray A Double () expression.
CopyToDataBlock A Boolean expression.

Remarks

CopySingleArray uses the size of the array to determine how many bytes to transfer. CopySingleArray only transfers the contents of the array. It does not save or retrieve the size of the array.

After transferring the data, it increments Offset by the count of bytes transferred.
