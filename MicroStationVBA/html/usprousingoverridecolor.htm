UsingOverrideColor Property

                          

Read/Write Boolean.

Syntax

object.UsingOverrideColor [= Boolean]

The UsingOverrideColor property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

MicroStation does not use a Level's OverrideColor unless the Level's UsingOverrideColor is True.

Whenever you change any aspect of a Level, you should subsequently call Levels.Rewrite to make the change permanent.
