Selection Set Example

This examples changes the color of all of the selected elements. This examples uses the GetSelectedElements method of the active ModelReference to retrieve an ElementEnumerator so it can step through the list of selected elements. Initially, the ElementEnumerator is positioned before the first element. It is necessary to invoke MoveNext before trying to retrieve an element. MoveNext returns False if there are no more elements. If there is another selected element, this example calls Current to get the element. After processing all of the elements, ChangeColor clears the selection set by calling UnselectAllElements. 


Sub ChangeColor()
    Dim oEnumerator As ElementEnumerator

    Set oEnumerator = ActiveModelReference.GetSelectedElements

    Do While oEnumerator.MoveNext
        Dim oElement As Element

        Set oElement = oEnumerator.Current

        oElement.Color = 1
        oElement.Redraw msdDrawingModeNormal
        oElement.Rewrite
    Loop

    ActiveModelReference.UnselectAllElements
End Sub
