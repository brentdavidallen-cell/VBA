EvaluatePointWeight Method

                          

Returns the point and weight at the given parameter(s) along this object.

Syntax

For the BsplineCurve object:

   Point3d = object.EvaluatePointWeight (Weight, Parameter)

For the BsplineSurface object:

   Point3d = object.EvaluatePointWeight (Weight, Parameters)

The EvaluatePointWeight method syntax has these parts:

Part Description
object A valid object.
Weight A Double expression. The returned weight of the point.
Parameter A Double expression. The valid parameter of the desired point.
Parameters A Point2d expression. The valid u- and v-parameters of the desired point.

Remarks

For BsplineSurface objects, any boundaries in the object are ignored. To detect whether or not Parameters defines a point on a trimmed/punctured surface, call ContainsParametricPoint.
