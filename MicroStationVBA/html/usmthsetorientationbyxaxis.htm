SetOrientationByXAxis Method

                          

Sets AccuDraw's drawing plane orientation (rotation) by specifying the X axis direction, the Y and Z axes are then derived from the user-defined default.

Syntax

object.SetOrientationByXAxis XVector

The SetOrientationByXAxis method syntax has these parts:

Part Description
object A valid object.
XVector A Point3d expression.
