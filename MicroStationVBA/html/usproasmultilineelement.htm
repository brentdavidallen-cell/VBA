AsMultiLineElement Property

                          

Read-only MultiLineElement.

Gets this object's implementation of the MultiLineElement interface.

Syntax

object.AsMultiLineElement

The AsMultiLineElement property syntax has these parts:

Part Description
object A valid object.

Remarks

An error is raised if this object does not support the IsMultiLineElement interface. If you're in doubt, firstly check this object's IsMultiLineElement property.
