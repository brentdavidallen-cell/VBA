DeleteSettings Method

                          

The StandardsCheckerController calls the DeleteSettings method to tell the Standards Checker to delete the settings element of a given name.

Syntax

object.DeleteSettings

The DeleteSettings method syntax has these parts:

Part Description
object A valid object.

Remarks

If the Standards Checker has settings that were saved by calling SetApplicationSettings, then it can call DeleteApplicationSettings to delete the settings from the standards DgnLib file.
