RasterCompressionMode Object

                                                     

This is the compression mode for a raster file format's color mode.

Remarks

Use the RasterCompressionModes object to get a RasterCompressionMode object as follows: 


    'Instantiation with an Item :

    Dim theRasterCompressionModes As CompressionModes
    Dim oRasterCompressionMode As RasterCompressionMode

    ...

    Set oRasterCompressionMode = theRasterCompressionModes.Item(1)


    'Instantiation with Find:

    Dim theRasterCompressionModes As CompressionModes
    Dim oRasterCompressionMode As RasterColorMode

    ...

    Set oRasterCompressionMode = theRasterCompressionModes.Find("~16 Colors")
