SecondaryRadius Property

                          

Read/Write Double for ArcElement, EllipseElement.

Read-only Double for EllipticalElement.

Gets the length of this object's secondary radius.

Syntax

object.SecondaryRadius [= Double]

The SecondaryRadius property syntax has these parts:

Part Description
object A valid object.
Double A Double expression.

Remarks

The value is measured in the coordinate system of the containing model.
