PointCellName Property

                          

Read/Write String.

Gets or sets the active point cell, identified by name.

Syntax

object.PointCellName [= String]

The PointCellName property syntax has these parts:

Part Description
object A valid object.
String A String expression.

Remarks

If an attempt is made to set PointCell to a cell not found in any of the cell libraries in the search path, the active point is not changed.
