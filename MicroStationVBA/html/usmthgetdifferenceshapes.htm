GetDifferenceShapes Method

                          

Constructs and returns a list of ComplexShapeElements that collectively enclose the area that represents the difference between this object and a given Element.

Syntax

Set ElementEnumerator = object.GetDifferenceShapes (ClosedElement)

The GetDifferenceShapes method syntax has these parts:

Part Description
object A valid object.
ClosedElement An Element expression.
