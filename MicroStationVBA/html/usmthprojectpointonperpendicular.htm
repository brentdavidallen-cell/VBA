ProjectPointOnPerpendicular Method

                          

Returns the endpoint of a projection from a given point to this object, along a perpendicular of this object.

Syntax

Point3d = object.ProjectPointOnPerpendicular (Point, Rotation)

The ProjectPointOnPerpendicular method syntax has these parts:

Part Description
object A valid object.
Point A Point3d expression. The point from which to project.
Rotation A Matrix3d expression. For no rotation use Matrix3dIdentity.

Remarks

The returned value is measured in the coordinate system of the containing model.
