GetReferenceToMasterTransform Method

                          

Syntax

Transform3d = object.GetReferenceToMasterTransform

The GetReferenceToMasterTransform method syntax has these parts:

Part Description
object A valid object.

Remarks

Use GetReferenceToMasterTransform to get a Transform3d to transform an element from the master's coordinate system to the Attachment's coordinate system. Use GetMasterToReferenceTransform to get a Transform3d to transform an element from the Attachment's coordinate system to the master's coordinate system. 

    Sub DisplayInAttachment(elementID As DLong, att As Attachment)
        Dim ele As Element
        Dim trns As Transform3d

        Set ele = ActiveModelReference.GetElementByID(elementID)
        trns = att.GetReferenceToMasterTransform

        ele.Transform trns
        ele.Redraw
    End Sub
