Affinity Property

                          

Read/Write Double.

Gets or sets the raster's affinity.

Syntax

object.Affinity [= Double]

The Affinity property syntax has these parts:

Part Description
object A valid object.
Double A Double expression. Affinity angle in radians.

Remarks

The affinity is expressed in radians [0...PI/2[, ]PI/2...3PI/2[, ]3PI/2...2PI[. Application.Radians can be used to convert the angle from degrees to radians. 


    Dim theGeo As GeoReferenceInformation

    Set theGeo = RasterManager.Rasters.Item(1). GeoReferenceInformation
    theGeo.Affinity = 0.7854
    MsgBox "New affinity is: " & theGeo.Affinity
