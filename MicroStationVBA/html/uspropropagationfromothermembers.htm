PropagationFromOtherMembers Property

                          

Read/Write MsdChangePropagation.

Gets/Sets a value that determines the behavior of a named group traversal of type msdMemberTraverseManipulate.

Syntax

object.PropagationFromOtherMembers [= MsdChangePropagation]

The PropagationFromOtherMembers property syntax has these parts:

Part Description
object A valid object.
MsdChangePropagation An MsdChangePropagation expression.

Remarks

If set to msdChangePropagationGroupLock, then the traversal propagates from other members only if the graphic group lock is enabled. If set to msdChangePropagationNever, then the traversal never propagates from other members to this member. If set to msdChangePropagationAlways, then it always propagates to this member from other members.
