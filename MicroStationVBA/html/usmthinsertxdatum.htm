InsertXDatum Method

                          

Inserts a new XDatum item into a specified position in an XDatum() array, increasing the size of the array by one.

Syntax

For the Application object:

   object.InsertXDatum XData, Index, Type, Value

For the XDataObject object:

   object.InsertXDatum Index, Type, Value

The InsertXDatum method syntax has these parts:

Part Description
object A valid object.
Index A Long expression. A zero-based index that must be in the range 0 to Count-1.
XData A XDatum () expression.
Type An MsdXDatumType expression.
Value A VARIANT expression.

Remarks

XDatum() arrays returned from GetXData are zero-based (i.e., the first item in the array is at index number 0).

This method also validates that the type of Value is consistent with the given Type. For example, the following statement will raise a type mismatch error, because there is no automatic no way to convert a String value ("Hello") into a Point3d value:

InsertXDatum xdata, 2, msdXDatumTypePoint, "Hello" ' ERROR
