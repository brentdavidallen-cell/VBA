Matrix3dIsXYRotation Method

                          

Analyze a matrix and determine if it is a rotation in the xy plane, with no other effects. If so, return true as the function value and store the rotation angle (in radians) as the XYRotationRadians parameter. A false return (with no meaningful angle) occurs when the matrix has any scale, skew, or mirror effects effects (in any direction) or has rotation effects other than around Z.

Syntax

Boolean = object.Matrix3dIsXYRotation (Matrix, XYRotationRadians)

The Matrix3dIsXYRotation method syntax has these parts:

Part Description
object A valid object.
Matrix A Matrix3d expression. matrix being analyzed.
XYRotationRadians A Double expression. signed angle of rotation around z.
