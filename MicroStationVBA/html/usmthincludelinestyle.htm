IncludeLineStyle Method

                          

Include elements of the specified line style in the forthcoming scan.

Syntax

object.IncludeLineStyle LineStyle

The IncludeLineStyle method syntax has these parts:

Part Description
object A valid object.
LineStyle A LineStyle expression.

Remarks

By default, elements of every line style are included in the scan. To limit the scan to specific line styles, first call ExcludeAllLineStyles, followed by a separate call to IncludeLineStyle for each line style to be included.

Calling this method automatically restricts the scan to only graphical type elements.
