Matrix3dIsXRotationYRotationZRotationScale Method

                          

Returns True if the matrix is constructed of rotation and uniform scale, False if non-uniform scale. Also reinterprets a matrix as a product of simple rotations about X, Y, and Z with uniform scaling:

Rotate(X,thetaX) * Rotate(Y,thetaY) * Rotate(Z, thetaZ) * uniformScale

where Rotate(axis,angle) is rotation matrix for specified axis and angle.

Syntax

Boolean = object.Matrix3dIsXRotationYRotationZRotationScale (Matrix, RadiansX, RadiansY, RadiansZ, Scale)

The Matrix3dIsXRotationYRotationZRotationScale method syntax has these parts:

Part Description
object A valid object.
Matrix A Matrix3d expression. matrix to analyze.
RadiansX A Double expression. Angle, in radians, for the X factor.
RadiansY A Double expression. Angle, in radians, for the Y factor.
RadiansZ A Double expression. Angle, in radians, for the Z factor.
Scale A Double expression. Scale factor.

Remarks

This decomposition is sometimes called "Euler angles".

Euler angles are have very limited use for computation.
