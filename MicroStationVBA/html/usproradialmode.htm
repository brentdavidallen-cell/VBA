RadialMode Property

                          

Read/Write MsdDimRadialMode.

Gets this object's radial tool mode.

Syntax

object.RadialMode [= MsdDimRadialMode]

The RadialMode property syntax has these parts:

Part Description
object A valid object.
MsdDimRadialMode An MsdDimRadialMode expression.

Remarks

See MsdDimRadialMode for possible valid values.
