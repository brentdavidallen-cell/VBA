ComplexElement Object

                                                     

Represents the aspects of a MicroStation element that pertain to only those element types that are complex. An element is complex if it can contain sub-elements.

Remarks

The ComplexElement interface is supported by these object types:

ComplexShapeElement

ComplexStringElement

TextNodeElement

CellElement

SharedCellElement

SharedCellDefinitionElement
