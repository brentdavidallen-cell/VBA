ProjectCurveOntoSolid Method

                          

This function takes as input a body and a curve and tries to imprint the curve on to the body

Syntax

Set SmartSolidElement = object.ProjectCurveOntoSolid (solid, curve, direction, tolerance)

The ProjectCurveOntoSolid method syntax has these parts:

Part Description
object A valid object.
solid A SmartSolidElement expression. input body on which curves are to be imprinted
curve An Element expression. input curve to imprint
direction A Vector3d expression. vector to project curve along
tolerance A Double expression. tolerance to be used for resolving resolution problems and for computation of parametric space curve approximations. The recommended value is 1E-05.
