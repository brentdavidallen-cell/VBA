ElementCacheContainingFilePosition Method

                          

Given an element's file position, returns the cache that contains that element. Also, optionally, provides the corresponding index within that cache.

Syntax

Set ElementCache = object.ElementCacheContainingFilePosition (FilePosition [, CacheIndex])

The ElementCacheContainingFilePosition method syntax has these parts:

Part Description
object A valid object.
FilePosition A Long expression.
CacheIndex Optional. A Long expression.

Remarks

This method is provided only to facilitate the conversion of legacy MicroStation BASIC applications to VBA. New VBA projects should not use file positions.
