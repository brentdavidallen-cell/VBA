ProjectPointOnPerpendicular1 Method

                          

Returns the endpoint of a projection from a given point to this object, along a perpendicular of this object. The BiasPoint is used to determine how far the projection travels.

Syntax

Point3d = object.ProjectPointOnPerpendicular1 (Point, BiasPoint, Rotation)

The ProjectPointOnPerpendicular1 method syntax has these parts:

Part Description
object A valid object.
Point A Point3d expression. The point from which to project.
BiasPoint A Point3d expression. The point closest to the end of the projection.
Rotation A Matrix3d expression. For no rotation use Matrix3dIdentity.

Remarks

The returned value is measured in the coordinate system of the containing model.
