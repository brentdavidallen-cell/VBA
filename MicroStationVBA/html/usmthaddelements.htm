AddElements Method

                          

Adds an arbitrary number of new elements to this object.

Syntax

object.AddElements Elements

The AddElements method syntax has these parts:

Part Description
object A valid object.
Elements A Element () expression. The elements to add.

Remarks

Important: You should add elements to only the model that was the active model when the elements were created. Otherwise, the added elements may appear at different locations, and/or have different sizes, than you intended.

If you want to add a small number of individual elements, it's probably easier to call the single-element version, AddElement, for each element.

Adding elements to a model does not automatically display the elements. To display all of the elements after you've added them, call the global function RedrawAllViews.
