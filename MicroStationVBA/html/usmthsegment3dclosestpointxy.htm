Segment3dClosestPointXY Method

                          

Compute the a point closest to a space point and on an unbounded line, using only xy coordinates in projection calculation.

Syntax

object.Segment3dClosestPointXY Segment, SpacePoint, ClosePoint, CloseFraction

The Segment3dClosestPointXY method syntax has these parts:

Part Description
object A valid object.
Segment A Segment3d expression. Segment of the unbounded line.
ClosePoint A Point3d expression. Point on unbounded line
CloseFraction A Double expression. the fractional position of the close point with respect to the segment.
SpacePoint A Point3d expression.
