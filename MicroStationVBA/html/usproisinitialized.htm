IsInitialized Property

                          

Read-only Boolean.

Gets a value indicating whether MicroStation has finished initializing.

Syntax

object.IsInitialized

The IsInitialized property syntax has these parts:

Part Description
object A valid object.

Remarks

This is intended for COM clients that start MicroStation as a server process. They can use this property to decide if MicroStation is completely initialized and therefore able to handle other requests. Theoretically, this should always be True for a COM client that starts MicroStation as a server process since MicroStation does not allow incoming COM requests until it is completely initialized. However, there is chance that some library or MDL application will allow incoming COM requests.
