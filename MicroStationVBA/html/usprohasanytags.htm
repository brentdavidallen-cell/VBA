HasAnyTags Property

                          

Read-only Boolean.

Gets a value indicating whether this object has any associated tags.

For an element, getting the value of HasAnyTags raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.

Syntax

object.HasAnyTags

The HasAnyTags property syntax has these parts:

Part Description
object A valid object.
