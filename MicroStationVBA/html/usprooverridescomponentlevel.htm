OverridesComponentLevel Property

                          

Read/Write Boolean.

Gets/sets whether is element is displayed with the level from of the SharedCellElement or the level of the component. This affects the display if any of the elements have by-level symbology, or if the View has DisplaysLevelSymbology enabled.

Syntax

object.OverridesComponentLevel [= Boolean]

The OverridesComponentLevel property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.
