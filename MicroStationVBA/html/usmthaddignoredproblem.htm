AddIgnoredProblem Method

                          

Adds a problem node to the StandardsCheckerController's report setting the Ignored attribute to '1'. Use AddProblem to add a problem node with the Ignored attribute set to '0'.

Syntax

Set StandardsCheckerProblem = object.AddIgnoredProblem (Description, ObjectType, IgnoredBy, IgnoredWhen)

The AddIgnoredProblem method syntax has these parts:

Part Description
object A valid object.
ObjectType A String expression. Describes the type of object that has the problem. Common values are "Level", "Element", "LineStyle", "TextStyle", and "Reference".
Description A String expression.
IgnoredBy A String expression.
IgnoredWhen A Date expression.

Remarks

See StandardsCheckerProblem for a description of the node.

AddIgnoredProblem returns a StandardsCheckerProblem object that the standards checker can use to add more detailed information to the report.

The typical call to this method looks like: 

    StandardsCheckerController.Report.AddIgnoredProblem "The problem description", "Level", UserName, Now
