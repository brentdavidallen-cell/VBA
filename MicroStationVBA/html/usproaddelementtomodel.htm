AddElementToModel Property

                          

Sets whether the new element is added to the model.

Syntax

object.AddElementToModel [= Boolean]

The AddElementToModel property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

Setting AddElementToModel to True causes CopyElement to add the new Element to the model. If AddElementToModel is False, CopyElement creates a copy of the element in memory but does not add it to the model. Regardless of the value of AddElementToModel, CopyElement copies all of the components that are required to add the new element to the model. It always copies all of the styles, levels, etc. to the destination model.

The default is True when a CopyContext is created.
