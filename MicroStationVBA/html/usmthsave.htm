Save Method

                          

Saves any changes that have been made to the currently open design file, and creates a new backup file.

Syntax

object.Save

The Save method syntax has these parts:

Part Description
object A valid object.

Remarks

Generally, for the active design file this method is useful only if the user preference to immediately save design changes is off.

Changes to a design file opened with OpenDesignFileForProgram are not automatically saved. When the program calls Rewrite, the changes are saved to the ElementCache but not to the actual file. The program has to call Save to have the changes saved to the file.
