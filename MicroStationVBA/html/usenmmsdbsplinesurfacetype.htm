MsdBsplineSurfaceType Enum

        

Defines constants that represent special types of B-spline surfaces, as stored in the read-only property BsplineSurface.SurfaceType.

The MsdBsplineSurfaceType enum has these members:

Name Description
msdBsplineSurfaceGeneral 0 (&H0) Represents an arbitrary B-spline surface.
msdBsplineSurfacePlane 1 (&H1) Represents a trimmed plane.
msdBsplineSurfaceRightCylinder 2 (&H2) Represents a right circular cylinder.
msdBsplineSurfaceCone 3 (&H3) Represents a right circular cone.
msdBsplineSurfaceSphere 4 (&H4) Represents a sphere.
msdBsplineSurfaceTorus 5 (&H5) Represents a torus.
msdBsplineSurfaceRevolution 6 (&H6) Represents a surface of revolution.
msdBsplineSurfaceTabCylinder 7 (&H7) Represents a B-spline surface composed entirely of tabulated cylinders (surfaces of projection).
msdBsplineSurfaceRuledSurface 8 (&H8) Represents a ruled surface.

Remarks

These values are generally not used and are preserved for historical reasons.
