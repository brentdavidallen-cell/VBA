CreateGCSFromKeyName Method

                          

Creates a GeographicCoordinateSystem given the key name.

Syntax

Set GeographicCoordinateSystem = object.CreateGCSFromKeyName (KeyName)

The CreateGCSFromKeyName method syntax has these parts:

Part Description
object A valid object.
KeyName A String expression. The name by which the GeographicCoordinateSystem is known in the Geographic Coordinate System Library.
