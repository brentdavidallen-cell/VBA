PointsToPixelsX Method

                          

Converts an X point coordinate to a pixel coordinate. Many of the VBA Width and Height properties are measured in points. A point is 1/72 of an inch.

Syntax

Long = object.PointsToPixelsX (PointCoordinate)

The PointsToPixelsX method syntax has these parts:

Part Description
object A valid object.
PointCoordinate A Double expression. X coordinate in points.
