MoveToNextElement Method

                          

Adjusts the internal marker to point to the next element.

Syntax

Boolean = object.MoveToNextElement [(StepThroughNestingChanges, NestingChange)]

The MoveToNextElement method syntax has these parts:

Part Description
object A valid object.
StepThroughNestingChanges Optional. A Boolean expression. True to cause MoveToNextElement to step into and out of nested cells.
NestingChange A Long expression. MoveToNextElement increments this when it steps into a nested cell, and decrements it when it steps out of a nested cell.

Remarks

MoveToNextElement steps into nested cells, but does not step into other complex elements. It treats any other type of complex element as a single component of the CellElement.

The program must call MoveToNextElement once to advance to the first component of the cell.

The following function shows how to use MoveToNextElement and related functions to list the ID's of the components of a cell. 


Sub ListCompents(eleCell As CellElement, bThroughNested As Boolean)
    Dim nestingLevel As Long
    Dim strSpaces As String

    eleCell.ResetElementEnumeration
    Do While eleCell.MoveToNextElement(bThroughNested, nestingLevel)
        Dim ele As Element
        Set ele = eleCell.CopyCurrentElement
        strSpaces = Space(4 * nestingLevel)
        Debug.Print strSpaces & "Procssing element " & DLongToString(ele.ID)
    Loop
End Sub
