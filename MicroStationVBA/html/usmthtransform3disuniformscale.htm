Transform3dIsUniformScale Method

                          

Returns true if the transform is a uniform scale with scale factor other than 1.0.

Syntax

Boolean = object.Transform3dIsUniformScale (Transform, FixedPoint, Scale)

The Transform3dIsUniformScale method syntax has these parts:

Part Description
object A valid object.
Transform A Transform3d expression. IN The transformation to test
FixedPoint A Point3d expression. OUT (If function result is true) the (one) point which remains in place in the transformation.
Scale A Double expression. OUT The scale factor. If the transform is not a scale, this is returned as 1.0.

Remarks

Returns true if the transformation is a uniform scale.
