EvaluatePointFrame Method

                          

Returns the point, Frenet frame, curvature and torsion at the given parameter along this object.

Syntax

Point3d = object.EvaluatePointFrame (Frame, Curvature, Torsion, Parameter, LinearNormal)

The EvaluatePointFrame method syntax has these parts:

Part Description
object A valid object.
Frame A Matrix3d expression. The returned Frenet frame, or local right-handed coordinate system at the point. This orthogonal matrix is presented with normalized local axes as rows (like a View.Rotation). Frame.RowX is the tangent vector. Frame.RowY is the main normal vector. Frame.RowZ is the binormal vector.
Curvature A Double expression. The returned curvature of this object at the point.
Torsion A Double expression. The returned torsion of this object at the point.
Parameter A Double expression. The valid parameter of the desired point.
LinearNormal A Point3d expression. The default main normal vector to use if this object is locally linear at Parameter. In this case, the binormal is computed from the cross product of the tangent and LinearNormal, and the main normal is computed from the cross product of this binormal and the tangent.

Remarks

For BsplineCurve objects, the Frenet Frame has several useful properties like the following. The main normal vector points to the osculating circle's center, which is located a (signed) distance of 1.0 / Curvature from the returned point. For planar objects, the binormal vector equals the normal vector of the plane. Curvature is the angular velocity of the tangent vector, while torsion is the negative of the angular velocity of the binormal vector; both are parametrization-independent and invariant under a rigid body transformation of the curve.
