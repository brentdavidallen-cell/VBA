GetPoint3dPropertyValue Method

                          

Returns Point3d object as property value.

Syntax

Point3d = object.GetPoint3dPropertyValue (AccessString)

The GetPoint3dPropertyValue method syntax has these parts:

Part Description
object A valid object.
AccessString A String expression. Property whose value to be retrived.

Remarks

Use GetPointPropertyValue to retrieve a property value if the property's type is Point3d.
