PrimaryAlternateFormat Property

                          

Read/Write MsdDimTextFormat.

Gets this object's format for displaying primary distance values using the alternate condition.

Syntax

object.PrimaryAlternateFormat [= MsdDimTextFormat]

The PrimaryAlternateFormat property syntax has these parts:

Part Description
object A valid object.
MsdDimTextFormat An MsdDimTextFormat expression.

Remarks

See MsdDimTextFormat for possible valid values.
