IsReadOnly Property

                          

Read-only Boolean.

Gets a value indicating whether this object is read-only (i.e., cannot be changed).

Syntax

object.IsReadOnly

The IsReadOnly property syntax has these parts:

Part Description
object A valid object.

Remarks

An object may be read-only because it has been locked, or because it was read from a read-only file.

RasterInformation.IsReadOnly gets whether the file is read-only.
