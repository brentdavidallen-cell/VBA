ChangeUnits Property

                          

Sets whether various units are converted to the new model's values.

Syntax

object.ChangeUnits [= Boolean]

The ChangeUnits property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

This is normally False. The VBA object model ordinarily converts all units when reading an element so this must be False to avoid having the units converted twice. A program may want to set this to True if copying an element generated with Application.MdlCreateElementFromElementDescrP.

This has no effect on the coordinates.

The default is False when a CopyContext is created or when using CopyElement without specifying a CopyContext.
