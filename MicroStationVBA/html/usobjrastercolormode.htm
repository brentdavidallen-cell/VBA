RasterColorMode Object

                                                     

This is a raster file format color mode.

Remarks

Use the RasterColorModes object to get a RasterColorMode object as follows: 


    'Instantiation with an Item :

    Dim theRasterColorModes As RasterColorModes
    Dim oRasterColorMode As RasterColorMode

        ...

    Set oRasterColorMode = theRasterColorModes.Item(1)


    'Instantiation with Find:

    Dim theRasterColorModes As RasterColorModes
    Dim oRasterColorMode As RasterColorMode

    ...

    Set oRasterColorMode = theRasterColorModes.Find("~16 Colors")
