ComputeChordLengthKnots Method

                          

Computes and returns normalized interior knots for a B-spline curve, or for a B-spline surface in one parameter direction, based on the distances between successive Poles.

Syntax

Double () = object.ComputeChordLengthKnots (Poles, Weights, Order, Closed)

The ComputeChordLengthKnots method syntax has these parts:

Part Description
object A valid object.
Poles A Point3d () expression. The 1-dimensional array of poles.
Weights A Double () expression. The 1-dimensional array of weights. Pass empty array if the B-spline curve/surface is not Rational.
Order A Long expression. The valid B-spline Order.
Closed A Boolean expression. Whether the full knot vector is periodically extended (for a closed curve/surface) or clamped open (to interpolate the first and last poles).

Remarks

If the knots returned by this method are applied to a BsplineCurve or BsplineSurface (e.g., with SetKnots), the B-spline curve/surface that results generally follows its poles better than if it had Uniform knots.

If Closed is True, then the first and last entries of Poles must not be equal, or a runtime error is thrown.
