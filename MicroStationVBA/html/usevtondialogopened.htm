OnDialogOpened Event

                          

MicroStation calls OnDialogOpened after a modal dialog box opens but before it waits for user input.

Syntax

object.OnDialogOpened DialogBoxName, DialogResult

The OnDialogOpened method syntax has these parts:

Part Description
object A valid object.
DialogBoxName A String expression.
DialogResult An MsdDialogBoxResult expression.

Remarks

After calling OnDialogOpened, MicroStation waits for user input unless one of the OnDialogOpened methods set DialogResult to a MsdDialogResult value. If a OnDialogOpened method does set DialogResult then MicroStation does not wait for user input. Instead, it behaves as if the user had pressed one of the buttons that closes the dialog box.
