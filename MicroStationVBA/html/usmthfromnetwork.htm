FromNetwork Method

                          

Initializes this object with a Gordon surface that interpolates the network of curves given in the u- and v-parameter directions.

Syntax

object.FromNetwork UCurves, VCurves, Rotation [, Tolerance]

The FromNetwork method syntax has these parts:

Part Description
object A valid object.
UCurves A BsplineCurve () expression. The curves in the u-parameter direction (each at a constant value of v). For best results, these curves should be oriented similarly.
VCurves A BsplineCurve () expression. The curves in the v-parameter direction (each at a constant value of u). For best results, these curves should be oriented similarly.
Rotation A Matrix3d expression. The rotation matrix (e.g., View.Rotation, Matrix3dRotationFromRowZ) specifying the plane in which intersections among the given curves are to be computed. Pass Matrix3dIdentity to find the intersections as viewed in the xy-plane. Pass Matrix3dZero to compute true curve intersections.
Tolerance Optional. A Double expression. The relative scale for curve extension and the absolute tolerance for patch degeneracy detection. If omitted or negative, a default tolerance of 1.0e-3 is used.

Remarks

For BsplineSurface objects, this method interpolates the intersections (actual, or apparent in the plane of the given Rotation) by creating Coons patches inside each intersection quadrant and concatenating them together. No effort is made to ensure these patches join with higher than C0 continuity (see FromNetworkWithIntersections for an alternative).
