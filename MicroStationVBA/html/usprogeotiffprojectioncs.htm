GeotiffProjectionCS Property

                          

Read/Write Long.

Obsolete Gets or sets the geotiff's Projected Coordinate System.

Syntax

object.GeotiffProjectionCS [= Long]

The GeotiffProjectionCS property syntax has these parts:

Part Description
object A valid object.
Long A Long expression. The geotiff's Projected Coordinate System identification number.

Remarks

See http://www.remotesensing.org/geotiff/geotiff.html for more information. 


    Dim oGeo As GeoReferenceInformation

    Set oGeo = RasterManager.Rasters.Item(7).GeoReferenceInformation
    oGeo.GeotiffPCSLinearUnit = 31028
    MsgBox oGeo.GeotiffProjectionCS
