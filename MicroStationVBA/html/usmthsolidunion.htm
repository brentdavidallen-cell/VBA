SolidUnion Method

                          

Used to unite two overlapping solids.

Syntax

Set SmartSolidElement = object.SolidUnion (body1, body2)

The SolidUnion method syntax has these parts:

Part Description
object A valid object.
body1 A SmartSolidElement expression. the first solid.
body2 A SmartSolidElement expression. the second solid.

Remarks

These two solids should be overlapping. if not, the method will failed.
