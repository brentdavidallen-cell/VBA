MsdAngleFormat Enum

        

Defines constants that determine what AngleFormat is used.

The MsdAngleFormat enum has these members:

Name Description
msdFormatDD_DDDD 0 (&H0) Angle Format is DD DDDD.
msdFormatDD_MM_SS 1 (&H1) Angle Format is DD MM SS.
msdFormatGradians 2 (&H2) Angle Format is Gradians.
msdFormatRadians 3 (&H3) Angle Format is Radians.
