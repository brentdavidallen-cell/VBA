MsdFileAccessMode Enum

        

Defines constants that indicate if the access to the rasters is in read/write mode or only in read-only mode.

The MsdFileAccessMode enum has these members:

Name Description
msdFileAccessModeRead 1 (&H1) Access in read-only mode.
msdFileAccessModeReadWrite 3 (&H3) Access in read/write mode.

Version

08.00.04
