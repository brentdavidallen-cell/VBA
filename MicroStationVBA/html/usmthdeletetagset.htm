DeleteTagSet Method

                          

Removes all of this object's associated tags that are defined in the specified tag set.

Syntax

object.DeleteTagSet TagSet

The DeleteTagSet method syntax has these parts:

Part Description
object A valid object.
TagSet A TagSet expression.

Remarks

The tags are also deleted from the model in which they reside.

For an element, calling DeleteTagSet raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
