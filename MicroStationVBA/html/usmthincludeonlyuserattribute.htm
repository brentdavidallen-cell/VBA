IncludeOnlyUserAttribute Method

                          

Include elements with the specified user attributes in the forthcoming scan.

Syntax

object.IncludeOnlyUserAttribute UserAttributeID

The IncludeOnlyUserAttribute method syntax has these parts:

Part Description
object A valid object.
UserAttributeID A Long expression. The unique ID number, also known as a linkage ID, for the user linkage you want to include in your scan.
