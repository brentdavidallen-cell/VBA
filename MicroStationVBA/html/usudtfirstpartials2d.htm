FirstPartials2d Type

        

Represents vectors in the u,v-parametric plane corresponding to the vectors of a FirstPartials3d structure.

The FirstPartials2d type has these members:

Name Description
Du A Point2d that represents the parametric direction corresponding to the FirstPartials3d Du vector.
Dv A Point2d that represents the parametric direction corresponding to the FirstPartials3d Dv vector.

Remarks

The method BsplineSurface.EvaluatePointCurvature returns a FirstPartials2d structure containing unit vectors in the parametric directions of the minimum and maximum curvatures at a non-umbilic point specified by parametric coordinates.
