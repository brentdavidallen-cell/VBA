ExtractPatch Method

                          

Creates and returns the portion of this object that lies between the given parameters along this object.

Syntax

Set BsplineSurface = object.ExtractPatch (UParameterSpan, VParameterSpan)

The ExtractPatch method syntax has these parts:

Part Description
object A valid object.
UParameterSpan A Point2d expression. The start and end u-parameters of the returned object, given in this object's u-parameter space.
VParameterSpan A Point2d expression. The start and end v-parameters of the returned object, given in this object's v-parameter space.
