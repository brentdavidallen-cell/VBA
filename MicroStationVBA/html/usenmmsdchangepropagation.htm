MsdChangePropagation Enum

        

Defines the circumstances under which a change to an object can propagate to other objects, particularly among members of NamedGroupElements.

The MsdChangePropagation enum has these members:

Name Description
msdChangePropagationGroupLock 0 (&H0)
msdChangePropagationNever 1 (&H1) Never propagate
msdChangePropagationAlways 2 (&H2) Always propagate
