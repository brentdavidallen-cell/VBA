SendKeyin Method

                          

Used to send a key-in to MicroStation. SendKeyin does not return until MicroStation has processed the key-in. Sending a key-in this way simulates a user entering a key-in in MicroStation's key-in area. Many of the methods of CadPointQueue take Point3d arguments. Most of these values are measured in the coordinate system of the active model.

Syntax

object.SendKeyin Keyin

The SendKeyin method syntax has these parts:

Part Description
object A valid object.
Keyin A String expression.
