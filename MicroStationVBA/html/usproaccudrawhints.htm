AccuDrawHints Property

                          

Read-only AccuDrawHints.

Gets the AccuDrawHints object that a program can use to control the state of AccuDraw.

Syntax

object.AccuDrawHints

The AccuDrawHints property syntax has these parts:

Part Description
object A valid object.

Remarks

There is only one AccuDrawHints object, so this property has the same value everytime it is used within the same session.
