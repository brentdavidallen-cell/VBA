DataEntryRegion Type

        

Represents a data entry region within a text element.

The DataEntryRegion type has these members:

Name Description
StartPosition A Long expression
Length A Long expression
Justification An MsdDataEntryRegionJustification expression

Remarks

See User Defined Types as Property Values for an explanation of why the following expression does not do anything: 

    Element.AsTextElement.DataEntryRegion(1) = msdDataEntryRegionJustificationRight
but the following code works as expected: 
    Dim rgn As DataEntryRegion
    rgn = Element.AsTextElement.DataEntryRegion(1)
    rgn.Justification = msdDataEntryRegionJustificationRight
    Element.AsTextElement.DataEntryRegion(1) = rgn
