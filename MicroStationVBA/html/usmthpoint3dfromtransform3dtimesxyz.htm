Point3dFromTransform3dTimesXYZ Method

                          

Returns the product of a matrix times a point, with the point specified as explicit x, y, and z values.

Syntax

Point3d = object.Point3dFromTransform3dTimesXYZ (Transform, X, Y, Z)

The Point3dFromTransform3dTimesXYZ method syntax has these parts:

Part Description
object A valid object.
Transform A Transform3d expression. The transformation to apply
X A Double expression. The x component of the point
Y A Double expression. The y component of the point
Z A Double expression. The z component of the point
