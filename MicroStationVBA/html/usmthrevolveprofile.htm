RevolveProfile Method

                          

Revolves a wire or sheet body about the axis passing through the point and along the axis.

Syntax

Set SmartSolidElement = object.RevolveProfile (profile, pointOnAxis, axis, angle [, thickness])

The RevolveProfile method syntax has these parts:

Part Description
object A valid object.
profile An Element expression. profile body which is swept along the path, it should not be a closed curve. .
pointOnAxis A Point3d expression. a point on the axis
axis A Vector3d expression. the direction of the axis of revolution .
angle A Double expression. the revolution angle.
thickness Optional. A Double expression. the shell thickness applied to the resulting solid. Pass 0.0 if no shell is required.
