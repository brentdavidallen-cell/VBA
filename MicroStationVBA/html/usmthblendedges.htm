BlendEdges Method

                          

Blends edges of a solid.

Syntax

Set SmartSolidElement = object.BlendEdges (solid, closestPoints, radius, smoothEdges)

The BlendEdges method syntax has these parts:

Part Description
object A valid object.
solid A SmartSolidElement expression. the solid which edges are blended.
closestPoints A Point3d () expression. the points that lie on the edges intend to be blended.
radius A Double expression. the radius of the blend.
smoothEdges A Boolean expression. If TRUE, edges that are tangentially continuous are blended in one operation.
