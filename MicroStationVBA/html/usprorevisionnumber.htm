RevisionNumber Property

                          

Read/Write String for Attachment.

Read-only String for DesignFile.

Syntax

object.RevisionNumber [= String]

The RevisionNumber property syntax has these parts:

Part Description
object A valid object.
String A String expression.

Remarks

Attachment.RevisionNumber is the revision, from the DGN history, that is attached.

RevisionNumber of a DesignFile is one of the standard Windows file properties. You can use the Windows Explorer to view this along with the other file properties. You can also use MicroStation's File Properties dialog box (available from the File pulldown) to view a file's revision number.

MicroStation updates this value automatically, incrementing it by 1 on every user-initiated save and when it closes a file that has changes. You cannot change it programmatically or via the user interface.
