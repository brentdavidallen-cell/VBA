MsdRasterWorldFile Enum

        

Format of the file used to store information concerning the position, rotation and scaling of the raster.

The MsdRasterWorldFile enum has these members:

Name Description
msdRasterWorldFileNone 0 (&H0) No file is used.
msdRasterWorldFileHgr 1 (&H1) Represents the ".hgr" format.
msdRasterWorldFileWorldFile 2 (&H2) Represents the tiff world (".twf") format.
