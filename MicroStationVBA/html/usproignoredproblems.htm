IgnoredProblems Property

                          

Read/Write Long.

Syntax

object.IgnoredProblems [= Long]

The IgnoredProblems property syntax has these parts:

Part Description
object A valid object.
Long A Long expression.

Remarks

A standards checker should increment FixedProblems for every ignored problem. This includes the problems encountered on the current run that the user elects to ignore. It also includes problems that were previously ignored.
