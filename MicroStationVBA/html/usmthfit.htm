Fit Method

                          

Adjusts the view magnification so the entire design is visible in the view.

Syntax

object.Fit FitRefs

The Fit method syntax has these parts:

Part Description
object A valid object.
FitRefs A Boolean expression.

Remarks

Fit does not redraw the view. To show the change in the view, call the View.Redraw method.
