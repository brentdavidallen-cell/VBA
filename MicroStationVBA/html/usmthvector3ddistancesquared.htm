Vector3dDistanceSquared Method

                          

Computes the squared distance between two vectors.

Syntax

Double = object.Vector3dDistanceSquared (Vector1, Vector2)

The Vector3dDistanceSquared method syntax has these parts:

Part Description
object A valid object.
Vector1 A Vector3d expression. The first vector.
Vector2 A Vector3d expression. The second vector.
