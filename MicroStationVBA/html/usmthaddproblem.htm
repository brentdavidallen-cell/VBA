AddProblem Method

                          

Adds a problem node to the StandardsCheckerController's report setting the Ignored attribute to '0'. Use AddIgnoredProblem to add a problem node with the Ignored attribute set to '1'.

Syntax

Set StandardsCheckerProblem = object.AddProblem (Description, ObjectType, FixedByChecker)

The AddProblem method syntax has these parts:

Part Description
object A valid object.
ObjectType A String expression. Describes the type of object that has the problem. Common values are "Level", "Element", "LineStyle", "TextStyle", and "Reference".
Description A String expression.
FixedByChecker A Boolean expression.

Remarks

See StandardsCheckerProblem for a description of a problem node.
