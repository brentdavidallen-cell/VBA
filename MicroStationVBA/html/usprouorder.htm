UOrder Property

                          

Read/Write Long.

Gets or sets the order of this object in the u-parameter direction.

Syntax

object.UOrder [= Long]

The UOrder property syntax has these parts:

Part Description
object A valid object.
Long A Long expression.

Remarks

For BsplineSurface objects, UOrder is an integer between 2 and 26, inclusive, that is one more than the polynomial degree of the B-spline surface in the u-parameter direction.
