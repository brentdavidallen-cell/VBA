AddLevelChangeEventsHandler Method

                          

Adds a level-change-events event handler to the set of active handlers.

Syntax

object.AddLevelChangeEventsHandler EventHandler

The AddLevelChangeEventsHandler method syntax has these parts:

Part Description
object A valid object.
EventHandler An ILevelChangeEvents expression.

Remarks

The event handler remains active until it is terminated with RemoveLevelChangeEventsHandler, or the object becomes invalid. If the event handler is implemented in VBA, the object becomes invalid if the project is reset manually or due to an error.
