ColorName Property

                          

Read/Write String.

Gets or sets a string representation of the active color.

Syntax

object.ColorName [= String]

The ColorName property syntax has these parts:

Part Description
object A valid object.
String A String expression.

Remarks

If the active color source is from each element's level or cell, the value is "FromLevel" or "FromCell", respectively.

Otherwise, if the active color has a name, the value is the name.

Otherwise, the value is a string representation of the color index (in the range 0 to 254).
