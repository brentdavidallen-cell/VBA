AddStandardsChecker Method

                          

Registers an object to serve as a standards checker.

Syntax

object.AddStandardsChecker StandardsChecker, Priority

The AddStandardsChecker method syntax has these parts:

Part Description
object A valid object.
Priority A Long expression. The priority of a standards checker determines the checker's order in the MicroStation Standards Order dialog box. A low value will cause the StandardsCheckerController to display the checker near the beginning of the list. A high value will cause the StandardsCheckerController to display the checker near the end of the list.
StandardsChecker An IStandardsChecker expression.

Remarks

An application that registers a standards checker must do so before the user runs MicroStation's Standards Checker.

An application can use RemoveStandardsChecker to unregister a handler.
