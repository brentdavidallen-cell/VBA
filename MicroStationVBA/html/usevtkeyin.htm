Keyin Event

                          

If the call to StartPrimitive specified that the command wants key-ins, MicroStation calls the Keyin event handler when the user enters a key-in in MicroStation's key-in area.

Syntax

object.Keyin Keyin

The Keyin method syntax has these parts:

Part Description
object A valid object.
Keyin A String expression.

Version

08.00.00
