RemovePattern Method

                          

Removes this object's pattern association, if it has one.

Syntax

object.RemovePattern

The RemovePattern method syntax has these parts:

Part Description
object A valid object.

Remarks

RemovePattern only removes associative patterns.

RemovePattern does not rewrite the element or redraw the element. The following example shows how to erase the element and its pattern, remove the pattern, save the change to the model, and then redisplay the element without the pattern.


Sub ClearPattern(ele As Element)
    If Not ele.IsClosedElement Then Exit Sub
    ' Erases the element and the pattern
    ele.Redraw msdDrawingModeErase

    With ele.AsClosedElement
        .RemovePattern
    End With

    ele.Rewrite
    ele.Redraw msdDrawingModeNormal
End Sub
