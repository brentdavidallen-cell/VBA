EvaluatePointCurvature Method

                          

Queries this object for point, normal and curvature data at the given u- and v-parameters.

Syntax

Point3d = object.EvaluatePointCurvature (GaussianCurvature, MeanCurvature, PrincipalCurvatures, PrincipalDirections, PrincipalParametricDirections, IsUmbilicPoint, Normal, Parameters)

The EvaluatePointCurvature method syntax has these parts:

Part Description
object A valid object.
GaussianCurvature A Double expression. Product of the principal curvatures at the point.
MeanCurvature A Double expression. Average of the principal curvatures at the point.
PrincipalCurvatures A Point2d expression. Minimum (X) and maximum (Y) values of the normal curvature at the point.
PrincipalDirections A FirstPartials3d expression. Unit vectors in the tangent plane of the given point, in the direction of min (Du) and max (Dv) principal curvature.
PrincipalParametricDirections A FirstPartials2d expression. Unit vectors in the parametric plane at the given point, in the parametric direction of min (Du) and max (Dv) principal curvature. If IsUmbilicPoint is True, PrincipalParametricDirections is invalid and should not be used.
IsUmbilicPoint A Boolean expression. Whether the point is an umbilic (spherical) point, for which the principal directions are undefined. If True, PrincipalParametricDirections is invalid, but PrincipalDirections.Du is set to the normalized first partial derivative with respect to u at the point, and PrincipalDirections.Dv is set to the normalized cross product of the Normal and the aforementioned partial.
Normal A Point3d expression. The returned (normalized) normal at the point.
Parameters A Point2d expression. The valid u- and v-parameters of the desired point.

Remarks

For BsplineSurface objects, any boundaries in the object are ignored. To detect whether or not Parameters defines a point on a trimmed/punctured surface, call ContainsParametricPoint.

For BsplineSurface objects, if the surface normal is undefined at Parameters, this method fails with Err.Number -2147221504 (zero normal).
