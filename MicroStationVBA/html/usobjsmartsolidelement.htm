SmartSolidElement Object

                                                     

A kind of Element that corresponds to a MicroStation SmartSolid/SmartSurface element.

SmartSolids/SmartSurfaces are a core MicroStation representation of b-rep solid bodies.These representations are used and created when the user does (solid) modeling operations on either existing SmartSolids, or (older style) MicroStation 3D primitives.

SmartSolid can be used only for closed volumetric objects, if the object is not closed we can obtain only a SmartSurface.
