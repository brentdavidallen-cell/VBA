VerticalTextOptionsForTool Property

                          

Read/Write MsdDimVerticalTextOptions.

Gets this object's options to align the text vertically for the specified tool

Syntax

object.VerticalTextOptionsForTool (DimensionCommand)[= MsdDimVerticalTextOptions]

The VerticalTextOptionsForTool property syntax has these parts:

Part Description
object A valid object.
DimensionCommand An MsdDimType expression.
MsdDimVerticalTextOptions An MsdDimVerticalTextOptions expression.

Remarks

See MsdDimType for possible dimension commands and MsdDimVerticalTextOptions for vertical options.
