UsesDisplaySet Property

                          

Read/Write Boolean.

Gets/Sets whether the view displays just the contents of the current display set.

Syntax

object.UsesDisplaySet [= Boolean]

The UsesDisplaySet property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

Changing this value does not cause an immediate change in what is displayed in the view. To show the change in the view, call the View.Redraw method.
