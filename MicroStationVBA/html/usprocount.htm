Count Property

                          

Read-only Long.

Syntax

object.Count

The Count property syntax has these parts:

Part Description
object A valid object.

Remarks

For an ElementCache object, gets the number of element entries in this cache. This includes both valid and invalid elements.

For a RasterBlockTypes collection, gets the number of RasterBlockType available.

For a RasterClipMasks collection, gets the number of ClipMask objects.

For a RasterColorModes collection, gets the number of RasterColorMode available.

For a RasterCompressionModes collection, gets the number of RasterCompressionMode available.

For a RasterFormats collection, gets the number of RasterFormat available.

For a Rasters collection, gets the number of Raster objects.
