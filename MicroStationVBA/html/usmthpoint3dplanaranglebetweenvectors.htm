Point3dPlanarAngleBetweenVectors Method

                          

Computes the signed angle between the projection of two vectors onto a plane with given normal.

Syntax

Double = object.Point3dPlanarAngleBetweenVectors (Vector1, Vector2, PlaneNormal)

The Point3dPlanarAngleBetweenVectors method syntax has these parts:

Part Description
object A valid object.
Vector1 A Point3d expression. The first vector
Vector2 A Point3d expression. The second vector
PlaneNormal A Point3d expression. The plane normal vector
