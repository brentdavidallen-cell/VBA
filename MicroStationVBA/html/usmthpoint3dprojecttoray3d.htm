Point3dProjectToRay3d Method

                          

Returns the projection of a point onto a line (optionally as perceived in a view).

Syntax

Point3d = object.Point3dProjectToRay3d (Parameter, Point, Ray [, ViewSpecifier [, UseAuxiliaryCoordinateSystem]])

The Point3dProjectToRay3d method syntax has these parts:

Part Description
object A valid object.
Parameter A Double expression. The (possibly negative) parameter along Ray at which the projection point is found: the returned projection point equals Point3dAddScaled(Ray.Origin, Ray.Direction, Parameter)
Ray A Ray3d expression. Provides a parametrization for the line.
ViewSpecifier Optional. A VARIANT expression. The view index (1 through 8) or View object for projection in a view, or zero for true projection.
UseAuxiliaryCoordinateSystem Optional. A Boolean expression. Whether to use the AccuDraw plane or the active ACS when projecting in a view.
Point A Point3d expression.

Remarks

If ViewSpecifier is nonzero, then Point is projected to Ray flattened to the plane with view normal and containing Point. If in addition UseAuxiliaryCoordinateSystem is True and camera is not on in the view, then the plane to flatten Ray is instead computed from the AccuDraw plane (if AccuDraw is active) or the active auxiliary coordinate system (if ACS plane lock is set). If the normal of the flattening plane is parallel to Ray, then the true (perpendicular) projection is returned.

Ray.Direction does not have to be normalized, but if it is, then Parameter represents the signed distance from Ray.Origin to the computed projection point (not the distance from Point to the projection point).
