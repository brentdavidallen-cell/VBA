NoteVerticalJustification Property

                          

Read/Write MsdDimNoteVerticalJustification.

Gets this object's note vertical justification.

Syntax

object.NoteVerticalJustification [= MsdDimNoteVerticalJustification]

The NoteVerticalJustification property syntax has these parts:

Part Description
object A valid object.
MsdDimNoteVerticalJustification An MsdDimNoteVerticalJustification expression.

Remarks

See MsdDimNoteVerticalJustification for possible valid values.
