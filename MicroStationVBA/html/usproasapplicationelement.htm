AsApplicationElement Property

                          

Read-only ApplicationElement.

Gets this object's implementation of the ApplicationElement interface.

Syntax

object.AsApplicationElement

The AsApplicationElement property syntax has these parts:

Part Description
object A valid object.

Remarks

An error is raised if this object does not support the IsApplicationElement interface. If you're in doubt, firstly check this object's IsApplicationElement.
