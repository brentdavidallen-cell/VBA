IsInUseWithinModel Method

                          

Gets a value indicating whether this level is in use (has any elements on it) within the given model.

Syntax

Boolean = object.IsInUseWithinModel (Model)

The IsInUseWithinModel method syntax has these parts:

Part Description
object A valid object.
Model A ModelReference expression.

Remarks

If you want to know whether the level is in use within any model in the design file, use IsInUse.
