RenderAccelerated Property

                          

Read/Write Boolean.

Note this property is deprecated after Microstation V8i Edition.

Gets or sets a value indicating whether graphics acceleration is enabled.

Syntax

object.RenderAccelerated [= Boolean]

The RenderAccelerated property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

This value is equivalent to the Graphics Acceleration checkbox in the Rendering View Attributes dialog box.

Changing the value of RenderAccelerated property takes effect immediately. It is not necessary to call the View.Redraw method after changing the RenderAccelerated property.
