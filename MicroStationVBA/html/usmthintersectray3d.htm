IntersectRay3d Method

                          

Returns True if the given ray intersects this object, and if so, also returns the coordinates and parameters of the first intersection point.

Syntax

Boolean = object.IntersectRay3d (IntersectionPoint, Parameters, Ray [, Tolerance])

The IntersectRay3d method syntax has these parts:

Part Description
object A valid object.
IntersectionPoint A Point3d expression. The intersection point.
Parameters A Point2d expression. The returned u- and v-parameters of the intersection point.
Ray A Ray3d expression. The ray to fire into this object.
Tolerance Optional. A Double expression. The error tolerance for the computation. If omitted or negative, a default value of 1.0e-5 times the largest coordinate difference of the unweighted Poles of the surface is used.

Remarks

For BsplineSurface objects, this method replaces the method ComputeBoresite, which was deprecated after MicroStation V8 2004 Edition.
