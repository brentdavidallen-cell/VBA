ScanForACSElements Method

                          

Gets an ElementEnumerator for stepping through a list of the active model's named auxiliary coordinate systems.

Syntax

Set ElementEnumerator = object.ScanForACSElements

The ScanForACSElements method syntax has these parts:

Part Description
object A valid object.

Remarks

For each named auxiliary coordinate system, there is a AuxiliaryCoordinateSystemElement in the ModelReference.ControlElementCache. This method scans for those elements. It is just a shortcut for the following method: 


    Function DoScanForACSElements() As ElementEnumerator
        Dim ee As ElementEnumerator
        Dim esc As New ElementScanCriteria

        esc.ExcludeAllTypes
        esc.ExcludeAllSubtypes

        esc.IncludeType msdElementTypeGroupData
        esc.IncludeSubtype msdElementSubtypeAuxiliaryCoordinateSystem

        Set DoScanForACSElements = ActiveModelReference.ControlElementCache.Scan(esc)
    End Function
