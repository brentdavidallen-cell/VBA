SendMessageToApplication Method

                          

The method SendMessageToApplication sends the String to the indicated task. The format of the message is defined by task.

Syntax

object.SendMessageToApplication MdlApplication, Message

The SendMessageToApplication method syntax has these parts:

Part Description
object A valid object.
MdlApplication A String expression. The name of an MDL application.
Message A String expression. This String is sent to the named MDL application.

Remarks

This statement is output by the macro recorder when the user's actions cannot be adequately described by commands and application variables. This ordinarily is not used except in code generated by the macro recorder.
