DLongToString Method

                          

Returns the String representation of a DLong expression.

Syntax

String = object.DLongToString (Value)

The DLongToString method syntax has these parts:

Part Description
object A valid object.
Value A DLong expression.

Remarks

Example: 

    Dim N As DLong
    Dim S As String
    N = DLongFromLong(123456789)
    S = DLongToString(X) ' Now S contains the string value "123456789".

DLongToString converts the value to a string of decimal digits. If you want to convert the value to a string of hexadecimal (base 16) digits, use DLongToHexString.
