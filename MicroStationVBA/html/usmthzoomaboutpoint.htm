ZoomAboutPoint Method

                          

Zooms this view by ZoomFactor about the point specified by Center.

Syntax

object.ZoomAboutPoint Center, ZoomFactor

The ZoomAboutPoint method syntax has these parts:

Part Description
object A valid object.
Center A Point3d expression.
ZoomFactor A Double expression.

Remarks

ZoomAboutPoint does not redraw the view. To show the change in the view, call the View.Redraw method.

ZoomFactor must be a positive value. Use a value less than 1 to zoom in. Use a value greater than 1 to zoom out. A ZoomFactor of 2 doubles both the X and Y coordinates of Extents. A ZoomFactor of .5 halves both the X and Y coordinates of Extents.
