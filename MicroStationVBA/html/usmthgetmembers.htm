GetMembers Method

                          

Syntax

NamedGroupMember () = object.GetMembers

The GetMembers method syntax has these parts:

Part Description
object A valid object.

Remarks

NamedGroupElement.GetMembers gets the members of a named group. Use this or GetMember to get a member to change the properties of the member or to remove the member. Use NamedGroupElement.GetElements or Element.GetRelatedElements to process related elements.

If the named group does not have any members, GetMembers returns a zero-length array.
