SecondaryFormat Property

                          

Read/Write MsdDimTextFormat.

Gets this object's format for displaying secondary distance values.

Syntax

object.SecondaryFormat [= MsdDimTextFormat]

The SecondaryFormat property syntax has these parts:

Part Description
object A valid object.
MsdDimTextFormat An MsdDimTextFormat expression.

Remarks

See MsdDimTextFormat for possible valid values.
