Delete Method

                          

Syntax

For the ModelReferences object:

   object.Delete Model

For the ViewGroup object and TextStyle object:

   object.Delete

The Delete method syntax has these parts:

Part Description
object A valid object.
Model A ModelReference expression.

Remarks

TextStyle.Delete raises an exception when trying to delete a TextStyle object returned from ActiveSettings.TextStyle or Element.TextStyle. TextStyle.Delete only works for text style objects returned from the TextStyles collection.
