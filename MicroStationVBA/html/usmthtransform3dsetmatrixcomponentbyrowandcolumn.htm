Transform3dSetMatrixComponentByRowAndColumn Method

                          

Sets a specified component of the matrix part of a a Transform3d.

Syntax

object.Transform3dSetMatrixComponentByRowAndColumn Transform, RowIndex, ColumnIndex, Value

The Transform3dSetMatrixComponentByRowAndColumn method syntax has these parts:

Part Description
object A valid object.
Transform A Transform3d expression. The Transform3d whose matrix component is set.
RowIndex A Long expression. The row (0, 1, 2) where the value is set.
ColumnIndex A Long expression. the column (0, 1, 2) where the value is set.
Value A Double expression. The new value for the component.
