ComputeKnotsCount Method

                          

Returns the number of interior knots for a B-spline curve, or for a B-spline surface in one parameter direction.

Syntax

Long = object.ComputeKnotsCount (PolesCount, Order, Closed)

The ComputeKnotsCount method syntax has these parts:

Part Description
object A valid object.
PolesCount A Long expression. The number of poles.
Order A Long expression. The valid B-spline Order.
Closed A Boolean expression. Whether the full knot vector is periodically extended (for a closed curve/surface) or clamped open (to interpolate the first and last poles).

Remarks

When the parameters are taken from the properties of a valid BsplineCurve object, the returned count is equal to BsplineCurve.KnotsCount. When the parameters are taken from the properties of a valid BsplineSurface in the u-parameter direction, the returned count is equal to BsplineSurface.UKnotsCount (similar for the v-parameter direction).
