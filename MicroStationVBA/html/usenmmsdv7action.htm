MsdV7Action Enum

        

Defines constants that represent what the OpenDesignFile method will do when opening a V7 design file. A V7 design file is a file that was saved in the previous version of MicroStation, or that was saved when operating in V7 mode.

The MsdV7Action enum has these members:

Name Description
msdV7ActionAskUser 0 (&H0) Opens the Alert Dialog asking the user if he wants to open it in Read-Only mode, or upgrade to V8. If the user selects Cancel then OpenDesignFile raises an msdErrorUserCanceledAction exception.
msdV7ActionUpgradeToV8 1 (&H1) Upgrades the design file to V8.
msdV7ActionWorkmode 3 (&H3) Opens a V7 design file in V7 Workmode. In this workmode, the user can modify a V7 design file, but many of the V8 capabilities are disabled.
