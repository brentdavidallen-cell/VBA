GetClosestSegment Method

                          

Returns the index of the segment of the element that is closest to the point.

Syntax

Long = object.GetClosestSegment (Point)

The GetClosestSegment method syntax has these parts:

Part Description
object A valid object.
Point A Point3d expression.

Remarks

The value that is returned is an index into the array of vertices returned by GetVertices. It is the index of the starting point of the segment. The next point is the ending point of the segment. GetClosestVertex always returns a point that is either the starting or ending point of the closest segment.

The input point is specified in the coordinate system of the containing model.
