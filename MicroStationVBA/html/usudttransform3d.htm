Transform3d Type

        

Represents the transform.

The Transform3d type has these members:

Name Description
RowX A Point3d expression
TranslationX A Double expression
RowY A Point3d expression
TranslationY A Double expression
RowZ A Point3d expression
TranslationZ A Double expression

Remarks

A Transform3d encompasses a set of Double values. However, you should not attempt to interpret these values directly. Rather, if you need to access the individual components of a Transform3d, use the global functions Transform3dGetMatrixComponentByRowAndColumn, Transform3dSetMatrixComponentByRowAndColumn, Transform3dGetPointComponent, and Transform3dSetPointComponent.

There are a number of additional global functions for managing Transform3d values:

  • Returns a transformation from a given matrix part and a zero translation part: Transform3dFromMatrix3d.
  • Returns a transformation from a given matrix part, and a translation part computed from the matrix and a given fixed point: Transform3dFromMatrix3dAndFixedPoint3d.
  • Returns a transformation from a matrix times a transformation: Transform3dFromMatrix3dTimesTransform3d.
  • Returns a transformation from a transformation times a matrix, treating the matrix as a transformation with zero as its translation components: Transform3dFromTransform3dTimesMatrix3d.
  • Returns a transformation from the given row values: Transform3dFromRowValues.
  • Returns a transformation from the product of two transformations: Transform3dFromTransform3dTimesTransform3d.
  • Returns a transformation from the product of three transformations: Transform3dFromTransform3dTimesTransform3dTimesTransform3d.
  • Returns a transformation of rotation about a specified line: Transform3dFromLineAndRotationAngle.
  • Returns a transformation with the identity matrix and translation given as X, Y and Z components: Transform3dFromXYZ.
  • Returns a transformation with the identity matrix part and a given translation part: Transform3dFromPoint3d.
  • Returns a transformation from given matrix and translation parts: Transform3dFromMatrix3dPoint3d.
  • Returns a transformation which mirrors about a plane: Transform3dFromMirrorPlane.
  • Returns the world-to-plane transform of the given Plane3d: Transform3dFromWorldToPlane3d.
  • Returns the plane-to-world transform of the given Plane3d: Transform3dFromPlane3dToWorld.
  • Returns the identity transformation: Transform3dIdentity.
  • Returns the zero transformation: Transform3dZero.
  • Returns the inverse of a transform: Transform3dInverse.
  • Returns the matrix part of a transformation: Matrix3dFromTransform3d.
  • Returns the product of a matrix times a point, with the point specified as explicit x, y, and z values: Point3dFromTransform3dTimesXYZ.
  • Returns the product of a transform times a point: Point3dFromTransform3dTimesPoint3d.
  • Returns the translation (point) part of a transformation: Point3dFromTransform3d.
  • Tests whether a transform has an inverse: Transform3dHasInverse.
  • Tests whether a transform is the identity transform: Transform3dIsIdentity.
  • Tests whether the matrix part of a transform is a rigid body rotation: Transform3dIsRigid.
  • Tests whether transformation effects are entirely within the plane having a given normal: Transform3dIsPlanar.
  • Tests whether a transform has a singular value decomposition (and return the components): Transform3dIsTranslateRotateScaleRotate.
  • Tests whether a transform is a simple translation (and return translation): Transform3dIsTranslate.
  • Tests whether a transform is a uniform, nontrivial scale (and return scale factor and fixed point): Transform3dIsUniformScale.
  • Tests whether a transform is a non-zero rotation about a line (and return angle and line): Transform3dIsRotateAroundLine.
  • Tests whether a transform is a mirror with respect to a plane (and return plane): Transform3dIsMirrorAboutPlane.
  • Factors a mirror transform into a mirror part and non-mirror part: Transform3dFactorMirror.
  • Tests whether a transform is a uniform scale and rotation about a line (and return scale factor, angle and line): Transform3dIsUniformScaleAndRotateAroundLine.
  • Tests whether two transforms are equal: Transform3dEqual.
