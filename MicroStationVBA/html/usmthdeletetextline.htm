DeleteTextLine Method

                          

DeleteTextLine deletes a line of displayed text at a specified line number.

Syntax

object.DeleteTextLine LineNumber

The DeleteTextLine method syntax has these parts:

Part Description
object A valid object.
LineNumber A Long expression. The position at which to delete the line. The first line number is 1.

Remarks

To delete all of the lines of displayed text, use DeleteAllTextLines.
