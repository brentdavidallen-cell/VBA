AttachNamed Method

                          

Attaches the named auxiliary coordinate system.

Syntax

object.AttachNamed ACSName, UseRotation, UseOrigin

The AttachNamed method syntax has these parts:

Part Description
object A valid object.
ACSName A String expression.
UseRotation A Boolean expression.
UseOrigin A Boolean expression.

Remarks

This is equivalent to selecting the Saved ACS in the Auxiliary Coordinate System dialog box and selecting Attach.

Note this UseRotation was deprecated after Microstation V8i edition, it will always be treat as ture.

Note this UseOrigin was deprecated after Microstation v8i edition, it will always be treat as true.
