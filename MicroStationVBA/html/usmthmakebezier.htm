MakeBezier Method

                          

Converts this object into a composite Bezier curve or surface.

Syntax

object.MakeBezier

The MakeBezier method syntax has these parts:

Part Description
object A valid object.

Remarks

For BsplineCurve objects, this method increases the multiplicity of each interior Knot to Order - 1, after which the i_th parameter interval (between unique Knot values) corresponds to the Bezier curve on the i_th set of Order successive Poles.

For BsplineSurface objects, this method increases the multiplicity of each interior UKnot to UOrder - 1 and each interior VKnot to VOrder - 1, after which the i_th u-parameter range (between unique UKnot values) and j_th v-parameter range (between unique VKnot values) corresponds to the Bezier patch on the (j * (1 + UKnotsCount/(UOrder - 1)) + i)_th VOrder x UOrder sub-array of the Pole array, counting in row-major order.
