Matrix3dFromRowValues Method

                          

Returns a matrix with the 9 specified coefficients given in "row major" order.

Syntax

Matrix3d = object.Matrix3dFromRowValues (X00, X01, X02, X10, X11, X12, X20, X21, X22)

The Matrix3dFromRowValues method syntax has these parts:

Part Description
object A valid object.
X00 A Double expression. The 00 entry
X01 A Double expression. The 01 entry
X02 A Double expression. The 02 entry
X10 A Double expression. The 10 entry
X11 A Double expression. The 11 entry
X12 A Double expression. The 12 entry
X20 A Double expression. The 20 entry
X21 A Double expression. The 21 entry
X22 A Double expression. The 22 entry
