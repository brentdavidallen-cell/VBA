CopyLongArray Method

                          

Copies an array of Long's to or from the DataBlock object.

Syntax

object.CopyLongArray LongArray, CopyToDataBlock

The CopyLongArray method syntax has these parts:

Part Description
object A valid object.
LongArray A Long () expression.
CopyToDataBlock A Boolean expression.

Remarks

CopyLongArray uses the size of the array to determine how many bytes to transfer. CopyLongArray only transfers the contents of the array. It does not save or retrieve the size of the array.

After transferring the data, it increments Offset by the count of bytes transferred.
