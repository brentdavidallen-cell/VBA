Offset3d Method

                          

Creates and returns an object that is an approximation of the offset of this object that passes through the given point(s).

Syntax

Set BsplineCurve = object.Offset3d (ThroughPoint0, ThroughPoint1 [, SamplesCount [, TurnsCount [, Tolerance]]])

The Offset3d method syntax has these parts:

Part Description
object A valid object.
ThroughPoint0 A Point3d expression. A point through which the returned offset must pass. If ThroughPoint0 does not equal ThroughPoint1, then the returned offset starts at this point; otherwise, this entire length of this object is offset.
ThroughPoint1 A Point3d expression. A point through which the returned offset must pass. If ThroughPoint1 does not equal ThroughPoint0, then the returned offset ends at this point; otherwise, the entire length of this object is offset.
SamplesCount Optional. A Long expression. The number of offset samples (per full/partial turn) to approximate. If nonpositive, then a default value of 10 times the PolesCount of this object is used (times the absolute value of TurnsCount if nonzero).
TurnsCount Optional. A Long expression. The signed number of full/partial turns of the resulting offset around this object. If negative, then the returned offset spins counter-clockwise in the direction of parametrization (a positive value results in clockwise spin). If zero, then the smallest possible turn angle is used; if plus/minus one, then either this angle or its 360-degree complement are used, depending on the spin direction dictated by sign; other values yield additional full turns in the returned offset. This value is only relevant if ThroughPoint0 and ThroughPoint1 are unique.
Tolerance Optional. A Double expression. The absolute distance tolerance for offset sample approximation, knot removal and point equality. If omitted or negative, a default tolerance of 1.0e-3 times the largest coordinate difference of the unweighted Poles of this object is used. If zero, the offset samples are interpolated, no knots are removed, and the aforementioned default tolerance is used for point equality.

Remarks

If this object is a BsplineCurve, it should be G1-continuous (e.g., no corners or cusps) for best results. If the curve is not G1-continuous, then the entire curve is offset through ThroughPoint0 without regard to ThroughPoint1, SamplesCount or TurnsCount.

Cusps and self-intersections may appear in the returned offset, especially if the offset distance anywhere exceeds the radius of curvature of this object.
