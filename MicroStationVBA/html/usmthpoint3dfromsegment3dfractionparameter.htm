Point3dFromSegment3dFractionParameter Method

                          

Returns a point located at a fractional parameter along a line segment.

Syntax

Point3d = object.Point3dFromSegment3dFractionParameter (Segment, Fraction)

The Point3dFromSegment3dFractionParameter method syntax has these parts:

Part Description
object A valid object.
Segment A Segment3d expression. segment to evaluate
Fraction A Double expression. fractional parameter along segment. Fraction 0 is start point, Fraction 1 is target point. Negative fractions are "before" the start point. Positive fractions are past the endpoint.
