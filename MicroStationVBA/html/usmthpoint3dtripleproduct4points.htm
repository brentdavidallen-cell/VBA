Point3dTripleProduct4Points Method

                          

Computes the triple product of vectors from a base point three target points.

Syntax

Double = object.Point3dTripleProduct4Points (Origin, Target1, Target2, Target3)

The Point3dTripleProduct4Points method syntax has these parts:

Part Description
object A valid object.
Origin A Point3d expression. The base point for all three vectors.
Target1 A Point3d expression. The target point for the first vector.
Target2 A Point3d expression. The target point for the second vector.
Target3 A Point3d expression. The target point for the third vector.
