InterpolationCurve Object

                                                     

Corresponds to the MDL B-spline interpolation curve structure, MSInterpolationCurve.

Remarks

As per MDL analogue, most of the manipulation of a BsplineCurveElement in VBA is performed on a BsplineCurve or InterpolationCurve object, and only at the last step is an element created (cf. CreateBsplineCurveElement1, CreateBsplineCurveElement2) or updated (cf. UpdateFromBsplineCurve, UpdateFromInterpolationCurve) from this object.

The properties of an InterpolationCurve object may be set directly, or they may be populated by various InterpolationCurve methods. When an InterpolationCurve property is changed, dependent properties are adjusted as necessary to maintain a valid object state. For example, InterpolationCurve.SetFitPoints automatically computes the interpolation parameters (obtainable via GetParameters) based on the new fit points.

To get a "blank" InterpolationCurve object, use the New keyword.
