EvaluatePointsAtEqualLengthChords Method

                          

Returns points (and parameters) which determine chords of this object with equal length.

Syntax

Point3d () = object.EvaluatePointsAtEqualLengthChords (Parameters, ChordsCount [, Tolerance])

The EvaluatePointsAtEqualLengthChords method syntax has these parts:

Part Description
object A valid object.
Parameters A Double () expression. The returned parameters associated with the returned points.
ChordsCount A Long expression. The number of chords to compute (the number of points returned equals ChordsCount + 1).
Tolerance Optional. A Double expression. The maximum chord length difference. If omitted or negative, a default tolerance of 1.0e-8, or the largest absolute Pole coordinate times 1.0e-8 (whichever is larger), is used.
