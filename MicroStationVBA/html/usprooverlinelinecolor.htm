OverlineLineColor Property

                          

Read/Write Long.

Gets or sets the color of the line (if any) that appears over this object's text.

Syntax

object.OverlineLineColor [= Long]

The OverlineLineColor property syntax has these parts:

Part Description
object A valid object.
Long A Long expression.

Remarks

The color is represented as an index into MicroStation's color table.

This property has no apparent effect unless OverlineUsesCustomLine is True.
