DrawToEnhancedMetafile Method

                          

Creates an in-memory image of the element in enhanced-metafile format and returns a Win32 handle to the memory containing the image.

Syntax

Long = object.DrawToEnhancedMetafile (Width, Height, DrawBackGround)

The DrawToEnhancedMetafile method syntax has these parts:

Part Description
object A valid object.
Width A Long expression.
Height A Long expression.
DrawBackGround A Boolean expression.

Remarks

DrawToEnhancedMetafile provides .NET programs with a functionality similar to what GetPicture provides to VB and VBA programs.

The following snippet of code shows how a C# program can use this. This code is assumed to execute in a class that has a member declared as System.Windows.Forms.PictureBox cellPreview;. 


    cellElement = app.CreateCellElement3 (cellName, ref zeroOrigin, true);

    System.IntPtr   metafileHandle = new IntPtr(cellElement.DrawToEnhancedMetafile (100, 100, false));
    System.Drawing.Imaging.Metafile metafile =
        new System.Drawing.Imaging.Metafile (metafileHandle, true);
    cellPreview.Image = metafile;
