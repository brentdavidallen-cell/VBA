DefineFromModelPoints Method

                          

Defines a fence.

Syntax

object.DefineFromModelPoints ViewSpecifier, Perimeter

The DefineFromModelPoints method syntax has these parts:

Part Description
object A valid object.
ViewSpecifier A VARIANT expression. May be a View object or a view number. Valid view numbers are 1 through 8.
Perimeter A Point3d () expression. Defines the perimeter of the fence. The points are in master units. This works the same regardless of whether the first point is repeated as the last point. Either way, DefineFromModelPoints interprets the points as a closed shape.

Remarks

This function creates a ShapeElement from the points in Perimeter and then calls DefineFromElement.
