FromTwoBoundaryCurves Method

                          

Initializes this object with a ruled surface that interpolates the given boundary curves.

Syntax

object.FromTwoBoundaryCurves Curve0, Curve1 [, LinearDirection]

The FromTwoBoundaryCurves method syntax has these parts:

Part Description
object A valid object.
Curve0 A BsplineCurve expression. One of two boundary curves to fit.
Curve1 A BsplineCurve expression. One of two boundary curves to fit.
LinearDirection Optional. An MsdBsplineSurfaceDirection expression. The parameter direction in which the resulting surface is linear (order 2). If omitted, a default direction of msdBsplineSurfaceV is used.
