Removing a Named Group, Removing a Named Group Member

This example shows how to remove a named group and a member of a named group. 


Sub RemoveMember(ele As Element, ng As NamedGroupElement)
    Dim member As NamedGroupMember

    On Error GoTo RemoveFailed
    Set member = ng.GetMember(ele)
    ng.RemoveMember member
    ng.Rewrite
    Exit Sub

RemoveFailed:
    MsgBox "Failed with " & Err.Description
    Exit Sub
End Sub

Sub RemoveNamedGroup(ng As NamedGroupElement)
    ng.ModelReference.RemoveElement ng
End Sub
