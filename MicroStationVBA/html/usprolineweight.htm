LineWeight Property

                          

Read/Write Long.

Gets or sets this object's line weight (thickness).

Syntax

object.LineWeight [= Long]

The LineWeight property syntax has these parts:

Part Description
object A valid object.
Long A Long expression.

Remarks

There is a predefined Long constant, ByLevelLineWeight, that represents the special line weight value, "by level." If the value of LineWeight is ByLevelLineWeight, then the element's line weight is derived from the level that the element resides on.

There is also a predefined Long constant, ByCellLineWeight, that represents the special line weight value, "by cell." If the value of LineWeight is ByCellLineWeight, then the element's line weight is derived from the cell that the element resides within.

For an element, getting or setting the value of LineWeight raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
