MsdTangentElementOutputType Enum

        

Defines constants that determine the type of the objects in the ElementEnumerator object returned by the global method ConstructCirclesTangentToThreeElements.

The MsdTangentElementOutputType enum has these members:

Name Description
msdTangentCircles 0 (&H0) Outputs unfilled, circular EllipseElement objects tangent to the three input Elements.
msdTangentArcs 1 (&H1) Outputs ArcElement objects tangent to the three input Elements. Each ArcElement's StartPoint lies on the first input Element and EndPoint lies on the third input Element.
msdTangentTriangles 2 (&H2) Outputs triangular ShapeElement objects, each containing a triplet of tangency points as vertices.

Remarks

Specifically, these values represent the ways in which each triplet of tangency points is represented on return.
