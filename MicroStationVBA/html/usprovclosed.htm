VClosed Property

                          

Read/Write Boolean.

Gets or sets a value indicating whether this object is closed in the v-parameter direction.

Syntax

object.VClosed [= Boolean]

The VClosed property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

For BsplineSurface objects, if this property is True, then the B-spline surface is physically closed (and may additionally be periodically defined) in the v-parameter direction; if this property is False, then the B-spline surface may still be physically closed (but never periodically defined) in the v-parameter direction.

For BsplineSurface objects, setting this property does not in general preserve the shape of the B-spline surface: the surface's Poles are preserved but its VKnots are recomputed. To change the closure of a BsplineSurface while maintaining its shape, use the methods Close and Open.
