HasXData Method

                          

Returns a value indicating whether there is any XData for a given application name on this object. The object may be any type of graphical element or a ModelReference.

Syntax

Boolean = object.HasXData (ApplicationName)

The HasXData method syntax has these parts:

Part Description
object A valid object.
ApplicationName A String expression.

Remarks

To determine whether the object has any XData, regardless of application name, use HasAnyXData.

Calling HasXData for an element type raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
