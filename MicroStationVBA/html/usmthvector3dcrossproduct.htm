Vector3dCrossProduct Method

                          

Returns the (vector) cross product of two vectors.

Syntax

Vector3d = object.Vector3dCrossProduct (Vector1, Vector2)

The Vector3dCrossProduct method syntax has these parts:

Part Description
object A valid object.
Vector1 A Vector3d expression. The first vector
Vector2 A Vector3d expression. The second vector
