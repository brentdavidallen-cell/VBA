CopyDouble Method

                          

Copies a Double to or from the DataBlock object.

Syntax

object.CopyDouble DoubleReference, CopyToDataBlock

The CopyDouble method syntax has these parts:

Part Description
object A valid object.
DoubleReference A Double expression.
CopyToDataBlock A Boolean expression.

Remarks

If CopyToDataBlock is True, it copies 8 bytes from DoubleReference to the data block. Otherwise, it copies 8 bytes from the data block to DoubleReference. It either case, it increments Offset by 8, the number of bytes transferred.
