GetDistance Method

                          

Gets the distance in Master Units, and the initial Azimuthal direction, between two points.

Syntax

object.GetDistance Distance, InitialAzimuth, StartPoint, EndPoint

The GetDistance method syntax has these parts:

Part Description
object A valid object.
StartPoint A GeoPoint3D expression. The starting longitude and latitude.
EndPoint A GeoPoint3D expression. The starting longitude and latitude.
Distance A Double expression.
InitialAzimuth A Double expression.
