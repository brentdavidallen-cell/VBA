DLongFromString Method

                          

Returns the DLong representation of a String expression.

Syntax

DLong = object.DLongFromString (Value)

The DLongFromString method syntax has these parts:

Part Description
object A valid object.
Value A String expression.

Remarks

Example: 

    Dim N As DLong
    N = DLongFromString("1234567890987654321")
                    ' Now N contains the value 1234567890987654321.

If the string cannot be converted to a DLong value, an "invalid argument" error (Error 5) occurs.

DLongFromString assumes that the string contains a decimal (base 10) representation of the value. If, instead, the string contains a hexadecimal (base 16) representation of the value, use DLongFromHexString.
