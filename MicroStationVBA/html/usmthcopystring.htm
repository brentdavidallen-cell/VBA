CopyString Method

                          

Copies a String to or from the DataBlock object.

Syntax

object.CopyString StringReference, CopyToDataBlock

The CopyString method syntax has these parts:

Part Description
object A valid object.
StringReference A String expression.
CopyToDataBlock A Boolean expression.

Remarks

If CopyToDataBlock is True, it copies StringReference to the data block. Otherwise, it copies from the data block to StringReference.

When transferring data to the DataBlock, CopyString first stores 2 bytes that give the length of the string in bytes. Next it transfers the string itself. Each character of the string occupies 2 bytes in the DataBlock.

When transferring date from the DataBlock, CopyString first copies the size of the string from the DataBlock and then it transfers the string itself. It creates a String that is the same length as the String that was saved in the DataBlock. If StringReference is a fixed length String, then Visual Basic adjusts the string as necessary.

To save space, you may want to convert the string to an array of bytes and use CopyByteArray to copy the Byte array to the data block.
