DefaultValue Property

                          

Read/Write VARIANT.

Syntax

object.DefaultValue [= Variant]

The DefaultValue property syntax has these parts:

Part Description
object A valid object.
Variant A VARIANT expression.

Remarks

DefaultValue of a TagDefinition is the initial value of any TagElement created from the TagDefinition. Setting the DefaultValue of a TagDefinition performs whatever type conversion is necessary to make the value you supply match the type that the TagDefinition expects. If an integer is expected, then setting DefaultValue to "Tag Number=" + Cstr(I) raises an error. If a string is expected, then setting it to "Tag Number=" + Cstr(I) works as expected. Setting it to a numeric value causes an automatic conversion to a string. When used to get the default value of a TagDefinition, DefaultValue does not convert the data. It returns the data as a Variant. Visual Basic then converts the data as appropriate depending on how the macro uses it.
