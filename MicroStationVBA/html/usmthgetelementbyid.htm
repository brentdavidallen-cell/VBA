GetElementByID Method

                          

Returns an Element object that corresponds to the element having the specified ID.

Syntax

For the ModelReference object and Attachment object:

   Set Element = object.GetElementByID (ElementId)

For the DesignFile object:

   Set Element = object.GetElementByID (ElementID)

The GetElementByID method syntax has these parts:

Part Description
object A valid object.
ElementId A DLong expression.
ElementID A DLong expression.

Remarks

DesignFile.GetElementByID only finds elements in the default model. In most cases, it is better to use ModelReference.GetElementID.
