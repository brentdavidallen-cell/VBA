GetVertices Method

                          

Returns an array of points. The array contains the vertices of the element.

Syntax

Point3d () = object.GetVertices

The GetVertices method syntax has these parts:

Part Description
object A valid object.

Remarks

Unlike the vertex manipulation methods (such as ModifyVertex), indexes into the array is zero-based, not one-based.

Item(0) is the start of the first segment. Item(1) is both the end of the first segment and the start of the second segment. And so on.

The number of segments is one less than the number of items in the array.
