DeleteAllXData Method

                          

Deletes all XData, for all application names, from this object. The object may be any graphical element type or a ModelReference.

Syntax

object.DeleteAllXData

The DeleteAllXData method syntax has these parts:

Part Description
object A valid object.

Remarks

To delete the XData for just a specific application name, use DeleteXData.

In most cases, a program should not call DeleteAllXData for a ModelReference since most models are not under the exclusive control of one program.
