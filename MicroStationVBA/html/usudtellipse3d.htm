Ellipse3d Type

        

Represents an ellipse (or elliptical arc) in three-dimensional space.

The Ellipse3d type has these members:

Name Description
Center A Point3d value that represents the point at the center of the ellipse.
Vector0 A Point3d value that represents the vector from Center to the 0-degree point of the ellipse.
Vector90 A Point3d value that represents the vector from Center to the 90-degree point of the ellipse.
Start A double value that represents the signed radian angle from the 0-degree point to the start of the ellipse.
Sweep A double value that represents the signed radian angle swept by the ellipse.

Remarks

For points in a ModelReference's geometry, the values are measured in the model's coordinate system, unless stated otherwise.

The vectors Vector0 and Vector90 do not have to be normalized; if they have equal length, the ellipse or elliptical arc is circular. These vectors do not have to be perpendicular.

The angles Start and Sweep are measured in the counter-clockwise direction if positive, or in the clockwise direction if negative. A Sweep of 2*Pi or more in absolute value represents a full ellipse.

A few functions for converting to and from an Ellipse3d value are listed below.

  • Creates an EllipticalElement from an ellipse: CreateEllipticalElement1.
  • Initializes an ellipse from an EllipticalElement: Ellipse3dFromEllipticalElement.
  • Initializes a B-spline curve from an ellipse: BsplineCurve.FromEllipse3d.
