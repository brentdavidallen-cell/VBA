Vector3dAngleBetweenVectors Method

                          

Returns the angle between two vectors. This angle is between 0 and pi. Rotating the first vector by this angle around the cross product between the vectors aligns it with the second vector.

Syntax

Double = object.Vector3dAngleBetweenVectors (Vector1, Vector2)

The Vector3dAngleBetweenVectors method syntax has these parts:

Part Description
object A valid object.
Vector1 A Vector3d expression. The first vector
Vector2 A Vector3d expression. The second vector
