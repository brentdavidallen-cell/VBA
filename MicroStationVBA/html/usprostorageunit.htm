StorageUnit Property

                          

Read/Write MeasurementUnit.

Get or sets this object's storage unit.

Syntax

object.StorageUnit [= MeasurementUnit]

The StorageUnit property syntax has these parts:

Part Description
object A valid object.
MeasurementUnit A MeasurementUnit expression.

Remarks

The storage unit defines the relationship between positional units and a universal base unit (usually meters). MicroStation does not store a direct relationship between positional units (uors) and sub units or master units, but instead uses the "storage unit" definition as a conversion factor between the base unit and positional units. This allows changing the units used within a model without changing the size of the elements in the model.
