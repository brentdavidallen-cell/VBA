ShowCheckerError Method

                          

Syntax

object.ShowCheckerError ReplacementChoice, ProblemDescription, ReplaceOptions, CurrentlyIgnored [, IgnoredBy [, IgnoredWhen]]

The ShowCheckerError method syntax has these parts:

Part Description
object A valid object.
ReplacementChoice An MsdStandardsCheckerReplaceChoice expression.
ProblemDescription A String expression.
ReplaceOptions An MsdStandardsCheckerReplaceOptions expression.
CurrentlyIgnored A Boolean expression.
IgnoredBy Optional. A String expression.
IgnoredWhen Optional. A Date expression.

Remarks

ShowCheckerError is similar to ShowCheckerErrorWithFixOptions except ShowCheckerError does not provide any information describing possible fixes. See ShowCheckerErrorWithFixOptions for more information.
