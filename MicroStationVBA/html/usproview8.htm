View8 Property

                          

Read/Write Boolean.

Gets or sets the display state for view 8. 


    Dim rendInf As RenderingInformation

    Set rendInf = RasterManager.Rasters.Item(1).RenderingInformation

    If (rendInf.View8 = True) Then
        rendInf.View8 = False
    End If

Syntax

object.View8 [= Boolean]

The View8 property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression. Is the raster displayed in the view 8?
