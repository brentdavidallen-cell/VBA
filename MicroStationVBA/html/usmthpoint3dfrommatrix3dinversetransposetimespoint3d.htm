Point3dFromMatrix3dInverseTransposeTimesPoint3d Method

                          

Return the product of a matrix inverse transpose and a point.

Syntax

Point3d = object.Point3dFromMatrix3dInverseTransposeTimesPoint3d (Matrix, Point)

The Point3dFromMatrix3dInverseTransposeTimesPoint3d method syntax has these parts:

Part Description
object A valid object.
Matrix A Matrix3d expression. The matrix whose transpose is inverted
Point A Point3d expression. The known point multiplied by the matrix inverse.

Remarks

Throws an exception if the matrix is singular (can't be inverted).
