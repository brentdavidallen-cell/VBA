TraversableElement Object

                                                     

Represents the aspects of a MicroStation element that pertain to only those element types that are traversable. An element is traversable if it is comprised of connected segments.

Remarks

Every class that implements the ChainableElement interface also implements the TraversableElement interface.

Use IsTraversableElement to determine if an element supports the TraversableElement element interface. To use an element's TraversableElement interface, either access the element using AsTraversableElement or assign the element reference to a variable declared to be TraversableElement.

Prior to trying to use an element as a TraversableElement, test to see if the element actually is a TraversableElement. If an element is not a TraversableElement, then trying to access it as a TraversableElement results in a error indicating that there is a type mismatch or that the class does not support the expected interface.
