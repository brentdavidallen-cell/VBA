Vector3dUnitPerpendicularXY Method

                          

Compute a unit vector perpendicular to the xy parts of given vector.

Syntax

Vector3d = object.Vector3dUnitPerpendicularXY (Vector)

The Vector3dUnitPerpendicularXY method syntax has these parts:

Part Description
object A valid object.
Vector A Vector3d expression. The source vector
