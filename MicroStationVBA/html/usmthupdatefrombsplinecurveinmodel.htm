UpdateFromBsplineCurveInModel Method

                          

Updates this object with the data from the given BsplineCurve object.

Syntax

object.UpdateFromBsplineCurveInModel Curve [, ReplaceInModel]

The UpdateFromBsplineCurveInModel method syntax has these parts:

Part Description
object A valid object.
Curve A BsplineCurve expression. The B-spline curve with which to update this object.
ReplaceInModel Optional. A Boolean expression. If omitted or True, this object's containing model is also updated. If False, or the element has not been added to a model, only this object is updated.
