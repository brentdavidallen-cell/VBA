SetKnots Method

                          

Sets the array of interior knots of this object.

Syntax

object.SetKnots Knots

The SetKnots method syntax has these parts:

Part Description
object A valid object.
Knots A Double () expression. The array of interior knots to set.

Remarks

For BsplineCurve objects, see the Knot property for further description of this array.

For BsplineCurve objects, passing an empty array makes the curve Uniform.

For BsplineCurve objects, the Poles and Order of this object must be set before calling this method, and the number of entries in the array must be equal to the count returned by Bspline.ComputeKnotsCount.
