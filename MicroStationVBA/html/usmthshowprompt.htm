ShowPrompt Method

                          

Displays a prompt in MicroStation's status area.

Syntax

object.ShowPrompt [Prompt]

The ShowPrompt method syntax has these parts:

Part Description
object A valid object.
Prompt Optional. A String expression. The prompt to be displayed.

Remarks

This displays Prompt regardless of whether MessagesEnabled is True or False. Set MessagesEnabled to False to prevent MicroStation from overwriting the prompt.
