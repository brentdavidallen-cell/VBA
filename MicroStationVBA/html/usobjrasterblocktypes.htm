RasterBlockTypes Object

                                                     

A collection of RasterBlockType object. It gives access to all block types available for a raster file compression mode. It is used like this: 


    'With a variable:
    Dim oCompression As RasterCompressionMode
    Dim theRasterBlockTypes As RasterBlockTypes

        ...

    Set theRasterBlockTypes = oCompression.BlockTypes
    oRasterBlockTypes.[method/property]

    'Without a variable:
    oCompression.RasterBlockTypes.[method/property]

Remarks

It is possible to step through the collection using the For Each statement as follows: 


    Dim oCompression As RasterCompressionMode
    Dim oRasterBlockType As RasterBlockType
    Dim theRasterBlockTypes As RasterBlockTypes

        ...

    Set theRasterBlockTypes = oCompression.BlockTypes

    For Each oRasterBlockType In theRasterBlockTypes
    oRasterBlockType.[method/property]
    Next
