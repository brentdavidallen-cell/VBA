InternalColorToIndexedColor Method

                          

Syntax

Long = object.InternalColorToIndexedColor (InternalColor)

The InternalColorToIndexedColor method syntax has these parts:

Part Description
object A valid object.
InternalColor A Long expression.

Remarks

This returns a standard V8 color. It returns either a color table index, or a special value for by-level or by-cell.
