WriteGCS Method

                          

Assignes the GeographicCoordinateSystem to the current Model.

Syntax

object.WriteGCS GCS, Primary, ReprojectData

The WriteGCS method syntax has these parts:

Part Description
object A valid object.
GCS A GeographicCoordinateSystem expression. The GeographicCoordinateSystem to assign to the model.
Primary A Boolean expression. Indicates that the method is to work on the Primary GeographicCoordinateSystem. Always pass True.
ReprojectData A Boolean expression. If True, and there is already a GeographicCoordinateSystem assigned to the model, all data within the model is reprojected from the old GCS to the new one.
