IsFixedSpacing Property

                          

Read/Write Boolean.

Gets or sets a valid indicating whether all of the characters in this object's text have the same width.

Syntax

object.IsFixedSpacing [= FixedSpacing]

The IsFixedSpacing property syntax has these parts:

Part Description
object A valid object.
FixedSpacing A Boolean expression.
Boolean A Boolean expression.
