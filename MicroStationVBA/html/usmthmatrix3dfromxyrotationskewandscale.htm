Matrix3dFromXYRotationSkewAndScale Method

                          

Constructs a matrix from xy rotation, skew, and scale factors. In this matrix, the first two columns represent the X and Y axes of a local coordinate system. These differ from global X and Y axes in both direction and length.

Syntax

Matrix3d = object.Matrix3dFromXYRotationSkewAndScale (XAxisAngle, YAxisSkewAngle, XScale, YScale, ZScale)

The Matrix3dFromXYRotationSkewAndScale method syntax has these parts:

Part Description
object A valid object.
XAxisAngle A Double expression. Angle (in radians) from global x axis to local x axis.
YAxisSkewAngle A Double expression. Angle (in radians) from a hypothetical y axis 90 CCW from the local axis to the actual y axis represented in the matrix.
XScale A Double expression. (signed) scale factor to apply to x direction
YScale A Double expression. (signed) scale factor to apply to y direction.
ZScale A Double expression. (signed) scale factor to apply to z direction.

Remarks

Returns Constructed matrix.

Various reflection (mirroring) effects can be introduced into the coordinate system via negative XScale, negative YScale, or by YSkewAngle between 90 and 270 degrees (or, between -90 and -270)., or by negative ZScale.
