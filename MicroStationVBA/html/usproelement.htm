Element Property

                          

Read-only Element.

A read-only and non-copyable reference to the element that is currently being printed.

Syntax

object.Element

The Element property syntax has these parts:

Part Description
object A valid object.

Remarks

It is not legal to add, remove, or modify elements within a model while inside the printing process.

It is not legal to add, remove, or modify elements within a model while inside the printing process. Note that in some cases (i.e. pattern components) this may be a different element than the one received in the BeginElement and EndElement events.
