ArePolesCollinear Method

                          

Returns a value indicating whether the object's Poles are ordered (without backtracking) along a straight line.

Syntax

Boolean = object.ArePolesCollinear (StartPoint, EndPoint [, Tolerance])

The ArePolesCollinear method syntax has these parts:

Part Description
object A valid object.
StartPoint A Point3d expression. The returned first Pole, if this method returns True.
EndPoint A Point3d expression. The returned last Pole, if this method returns True.
Tolerance Optional. A Double expression. The maximum deviation from 1.0 of the cosine of the angle between successive directed segments of this object's control polygon for a True return. If omitted, negative or greater than 1.0, a default tolerance of 0.005 is used.

Remarks

For BsplineCurve objects, if this method returns True, then the curve represented by the object traces a (possibly degenerate) line between the returned points.
