PropagationToOtherGroups Property

                          

Read/Write MsdChangePropagation.

Gets/Sets a value that determines the behavior of a named group traversal of type msdMemberTraverseManipulate.

Syntax

object.PropagationToOtherGroups [= MsdChangePropagation]

The PropagationToOtherGroups property syntax has these parts:

Part Description
object A valid object.
MsdChangePropagation An MsdChangePropagation expression.

Remarks

If set to msdChangePropagationGroupLock, then the traversal propagates from this member to other named groups that are members of the same group only if the graphic group lock is enabled. If set to msdChangePropagationNever, then the traversal never propagates this member to NamedGroup members of the same NamedGroup. If set to msdChangePropagationAlways, then it always propagates to other NamedGroupMembers.
