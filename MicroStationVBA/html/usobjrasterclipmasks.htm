RasterClipMasks Object

                                                     

A collection of RasterClipMask objects. It gives access to all clip masks attached to a raster. It is used like this: 


    'With a variable:
    Dim oRaster As Raster
    Dim theClipMasks As RasterClipMasks

        ...

    Set theClipMasks = oRaster.ClipMasks
    theClipMasks.[method/property]

    'Without a variable:
    oRaster.ClipMasks.[method/property]

Remarks

It is possible to step through the collection using the For Each statement as follows: 


    Dim oRaster As Raster
    Dim oClipMask As RasterClipMask
    Dim theClipMasks As RasterClipMasks

        ...

    Set theClipMasks = oRaster.ClipMasks

    For Each oClipMask In theClipMasks
    oClipMask.[method/property]
