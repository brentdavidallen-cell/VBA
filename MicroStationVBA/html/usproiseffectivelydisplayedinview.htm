IsEffectivelyDisplayedInView Property

                          

Read-only Boolean.

Gets a value yielded by the expression (theLevel.IsDisplayed And Not theLevel.IsFrozen And theLevel.IsDisplayedInView(theView))

Syntax

object.IsEffectivelyDisplayedInView (View)

The IsEffectivelyDisplayedInView property syntax has these parts:

Part Description
object A valid object.
View A View expression.
