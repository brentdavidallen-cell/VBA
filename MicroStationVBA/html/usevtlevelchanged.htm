LevelChanged Event

                          

Syntax

object.LevelChanged ChangeType, TheLevel, TheModel

The LevelChanged method syntax has these parts:

Part Description
object A valid object.
ChangeType An MsdLevelChangeType expression.
TheLevel A Level expression.
TheModel A ModelReference expression.

Remarks

MicroStation calls LevelChanged methods for each of the events named in MsdLevelChangeType.

MicroStation passes Nothing for TheLevel for some level change events.

When MicroStation calls a LevelChanged method to notify it of a change in an Attachment's levels, TheModel is actually an Attachment. The program can assign TheModel to a variable declared as Attachment. The program can also access the Attachment properties using an expression like TheModel.AsAttachment.LogicalName.

Most programs will only care about the msdLevelChangeAfterChangeActive and msdLevelChangeBeforeChangeActive events. For both events, the value in TheLevel is the same as ActiveSettings.Level. For msdLevelChangeBeforeChangeActive, it is the old active level. For msdLevelChangeAfterChangeActive, it is the new active level.
