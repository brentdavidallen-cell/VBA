SetCExpressionValue Method

                          

Assigns the value to the location designated by the expression.

Syntax

object.SetCExpressionValue CExpression, NewValue [, MdlApplicationName]

The SetCExpressionValue method syntax has these parts:

Part Description
object A valid object.
CExpression A String expression. Where the value is saved.
NewValue A VARIANT expression. The value to be stored.
MdlApplicationName Optional. A String expression. The MDL application that published the variable referenced by the CExpression argument. This is important only if more than 1 MDL application has published a variable with that name.

Remarks

SetCExpressionValue has two purposes. It provides access to variables published by MDL applications. It also provides access to MicroStation internal variables in the rare instances where there is no interface available through the object model. Effective use of SetCExpressionValue requires a good working knowledge of C and MDL (MicroStation Development Language). It should be used with extreme caution, since none of the safeguards provided by the Visual Basic interface are in place when it is used.

SetCExpressionValue first evaluates the expression to determine the required data type. Then it converts NewValue to the required type and stores it in the location. Since SetCExpressionValue performs the necessary conversions, the following expressions are equivalent. They both set "tcb->msToolSettings.igen.placeArcMode" to the integer value 1. 


        SetCExpression "tcb->msToolSettings.igen.placeArcMode", 1
        SetCExpression "tcb->msToolSettings.igen.placeArcMode", "1"
