SetPropertyValue Method

                          

Syntax

Boolean = object.SetPropertyValue (AccessString, propertyValue)

The SetPropertyValue method syntax has these parts:

Part Description
object A valid object.
AccessString A String expression. Name of Property whose value to be changed.
propertyValue A VARIANT expression. new value to be set to given property.

Remarks

Use SetPropertyValue to set a property value if the property's type is Boolean, Date, Double, String , Integer.

The ItemTypePropertyHandler object works on the file's copy of the object. SetPropertyValue always writes a change back to the file immediately. This is particularly significant when using a ItemTypePropertyHandler object to access an element properties.
