CreateCone Method

                          

Generate a Cone at origin with SmartSolidElement Type.

Syntax

Set SmartSolidElement = object.CreateCone (Template, topRadius, baseRadius, height)

The CreateCone method syntax has these parts:

Part Description
object A valid object.
topRadius A Double expression. radius of top plan.
baseRadius A Double expression. radius of base plan.
Template An Element expression.
height A Double expression.
