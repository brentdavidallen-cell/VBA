Description Property

                          

Read-only String for ACSManager, GeographicCoordinateSystem, IStandardsChecker, SavedViewElement.

Read/Write String for Attachment, AuxiliaryCoordinateSystemElement, CellElement, CellInformation, DimensionStyle, ExtendedInformation, Level, ModelReference, NamedGroupElement, SharedCellDefinitionElement, ViewGroup.

Gets or sets the description of this object.

Syntax

object.Description [= String]

The Description property syntax has these parts:

Part Description
object A valid object.
String A String expression. The raster's description.

Remarks

ACSManager.Description raises an error if there is no auxiliary coordinate system defined. A program can test for this using ACSManager.IsDefined.

The Description of an Attachment is actually the description of the model that is attached. It is the description that MicroStation displays in the Models dialog box. Use LogicalDescription to get/set the description that MicroStation displays in the References dialog box.

After setting Description of a CellElement or SharedCellDefinitionElement, use Rewrite to save the new description. Otherwise, the new description is not saved with the model.

Here's an example for ExtendedInformation: 


    Dim oDescription As String

    oDescription = RasterManager.Rasters.Item(1).ExtendedInformation.Description
    oDescription = "New " + oDescription

    If (Len(oDescription) < RasterManager.Rasters.Item(1).ExtendedInformation.MaxDescriptionLength) Then
        RasterManager.Rasters.Item(1).ExtendedInformation.Description = oDescription
    End If

The portion of the StandardsCheckerController's report that lists the Standards Checker applications lists the DialogString, VersionString, and Description of each Standards Checker application.

Whenever you change any aspect of a Level, you should subsequently call Levels.Rewrite to make the change permanent.

Whenever you change any aspect of a NamedGroupElement, you should subsequently call Rewrite to make the change permanent.
