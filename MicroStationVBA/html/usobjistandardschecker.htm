IStandardsChecker Object

                                                     

Defines the methods and properties a class must implement to be a standards checker. If a program implements a class that implements this interface, and the program calls AddStandardsChecker to register that object as a Standards Checker, then MicroStation's StandardsCheckerController calls the methods of the class when running a standards check. See StandardsCheckerController for more information.
