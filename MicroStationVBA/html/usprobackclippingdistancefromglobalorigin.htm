BackClippingDistanceFromGlobalOrigin Property

                          

Read-only Double.

Gets the distance from the Global Origin to the back clipping plane.

Syntax

object.BackClippingDistanceFromGlobalOrigin

The BackClippingDistanceFromGlobalOrigin property syntax has these parts:

Part Description
object A valid object.

Remarks

This is the first value that the MicroStation key-in DP=? displays.

See View for information on the relationship of Origin, Extents, ActiveDepth, and Rotation.
