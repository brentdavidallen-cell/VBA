CreateTransientElementContainer1 Method

                          

Syntax

Set TransientElementContainer = object.CreateTransientElementContainer1 (TransientElement, Flags, ViewMask, DrawMode)

The CreateTransientElementContainer1 method syntax has these parts:

Part Description
object A valid object.
TransientElement An Element expression. CreateTransientElementContainer1 makes a copy of the element and saves in in the container. May be Nothing
Flags An MsdTransientFlags expression.
ViewMask An MsdViewMask expression.
DrawMode An MsdDrawingMode expression.

Remarks

A program can use AppendCopyOfElement to add more elements to the container.
