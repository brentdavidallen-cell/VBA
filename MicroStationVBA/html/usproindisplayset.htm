InDisplaySet Property

                          

Read/Write Boolean.

Gets/Sets whether the Element is in the current display set.

Syntax

object.InDisplaySet [= Boolean]

The InDisplaySet property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

Setting InDisplaySet raises an error if the element is not already in a model.

Changes to the display set are not shown in a view until the next time the view is redrawn.
