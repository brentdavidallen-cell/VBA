CacheIndex Property

                          

Read-only Long.

Gets the current location of this object within the element cache in which it resides.

Syntax

object.CacheIndex

The CacheIndex property syntax has these parts:

Part Description
object A valid object.

Remarks

In practice, you'll probably have little reason to use an element's CacheIndex. In particular, since the value of CacheIndex can change without notice, you can't use it to identify an element over time. (If you want to identify an element over time, use the element's ID property.)
