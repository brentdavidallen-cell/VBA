GetXData Method

                          

Returns a copy of the XData (if any) for a given application name on this object. The object may be any graphical element type or ModelReference.

Syntax

XDatum () = object.GetXData (ApplicationName)

The GetXData method syntax has these parts:

Part Description
object A valid object.
ApplicationName A String expression.

Remarks

A .NET program that uses .NET COM interop to access the MicroStationDGN object model should use GetXData1 instead of GetXData. GetXData1 performs the same function as GetXData, but GetXData1 presents the XData as an XDataObject object while GetXData presents the XData as an array of XDatum UDTs. A .NET program should not use GetXData because .NET COM interop raises an exception if one of the XDatum entries in the array contains a point. See XDataObject for more information on this topic.

XData is represented as an array of XDatum values. If the object does not have any XData, the array is empty.

The returned array is zero-based (i.e., the first item in the array is at index number 0).

Calling GetXData for an element type raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
