ColorTable Object

                                                     

A ColorTable determines how Element colors are mapped into RGB values.

Use DesignFile.ExtractColorTable to extract a design file's color table. Use DesignFile.AttachColorTable to set a new color table for the design file.

A ColorTable contains a background color definition and an array of RGB values that define the element colors. Valid indices for the array are 0 through 254. Position 0 contains the RGB value for MicroStation color 0. Position 254 contains the RGB value for MicroStation color 254.

An RGB value is encoded in a long. Use the function RGB to create a combined RGB value from separate red, green, and blue values.
