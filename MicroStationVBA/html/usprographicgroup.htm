GraphicGroup Property

                          

Read/Write Long.

Gets or sets this object's graphic group number.

Syntax

object.GraphicGroup [= Long]

The GraphicGroup property syntax has these parts:

Part Description
object A valid object.
Long A Long expression.

Remarks

A value of 0 means that this object is not part of a graphic group.

For an element, getting or setting the value of GraphicGroup raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
