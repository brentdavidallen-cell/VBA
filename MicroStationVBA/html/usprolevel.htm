Level Property

                          

Read/Write Level.

Gets or sets the level that this object resides on.

Syntax

[Set] object.Level [= Level]

The Level property syntax has these parts:

Part Description
object A valid object.
Level A Level expression.

Remarks

If the Element has not yet been added to a model, the only value you can assign to the Level property is Nothing. Assigning an actual Level would mean actually putting the element onto that level, which cannot be done if the element is not a member of a model.

Conversely, you cannot assign Nothing if the Element is a member of a model.

Newly-created elements get their level from the Level property of Settings.

For an element, getting or setting the value of Level raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
