SetSheetDefinition Method

                          

Uses a SheetDefintion object to set a model's sheet definition.

Syntax

object.SetSheetDefinition NewDefinition

The SetSheetDefinition method syntax has these parts:

Part Description
object A valid object.
NewDefinition A SheetDefinition expression.

Remarks

A program can apply the sheet definition to any model. It does not have to use the same model reference for the calls to GetSheetDefinition and SetSheetDefinition. A program can take advantage of this to copy a sheet definition to multiple models.

Use PropagateAnnotationScale to propagate scale factor changes to annotation elements. Call PropagateAnnotationScale after calling SetSheetDefinition.
