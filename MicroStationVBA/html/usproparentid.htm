ParentID Property

                          

Read-only DLong.

Gets a value that uniquely identifies the parent of this element. Returns 0 if the element does not have a parent.

Syntax

object.ParentID

The ParentID property syntax has these parts:

Part Description
object A valid object.

Remarks

For an Element object, the value returned reflects the persistent state of the element. If the element has never been added to a model, this property raises an error.
