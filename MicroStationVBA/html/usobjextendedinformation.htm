ExtendedInformation Object

                                                     

An ExtendedInformation object provides extra information about a raster.

Remarks

Use a Raster object to get an ExtendedInformation object as follows: 


    'With a variable :

    Dim theRasters As Rasters
    Dim oExt As ExtendedInformation

    Set theRasters = RasterManager.Rasters
    Set oExt = theRasters.Item(1). ExtendedInformation
    oExt.[method/property]


    'Without a variable :

    Dim theRasters As Rasters

    Set theRasters = RasterManager.Rasters

    theRasters.Item(1).ExtendedInformation.[method/property]
