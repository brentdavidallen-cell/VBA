AddPropertyToStandard Method

                          

Adds a property to a standard. See AddCheckerToStandardsCheckerApps for a sample report that contains properties.

Syntax

Set Object = object.AddPropertyToStandard (StandardDOMNode, Name, Value)

The AddPropertyToStandard method syntax has these parts:

Part Description
object A valid object.
StandardDOMNode An Object expression. An XML node returned from AddStandardToLibrary.
Name A String expression.
Value A String expression.

Remarks

AddPropertyToStandard returns an IXMLDOMNode that the program can use to add more information. Typically standards checkers do not add any more information.
