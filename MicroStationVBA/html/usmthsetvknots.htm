SetVKnots Method

                          

Sets the array of interior knots of this object in the v-parameter direction.

Syntax

object.SetVKnots VKnots

The SetVKnots method syntax has these parts:

Part Description
object A valid object.
VKnots A Double () expression. The array of interior knots to set in the v-parameter direction.

Remarks

For BsplineSurface objects, see the VKnot property for further description of this array.

For BsplineSurface objects, passing an empty array makes the curve uniform in the v-parameter direction (cf. VUniform).

For BsplineSurface objects, the Poles and VOrder of this object must be set before calling this method, and the number of entries in the array must be equal to the count returned by Bspline.ComputeKnotsCount.
