AlignAngularExtensionLinesToChordForTool Property

                          

Read/Write Integer.

Gets this object's flag to align the angular extension lines along the chord instead of radially.

Syntax

object.AlignAngularExtensionLinesToChordForTool (DimensionCommand)[= Integer]

The AlignAngularExtensionLinesToChordForTool property syntax has these parts:

Part Description
object A valid object.
DimensionCommand An MsdDimType expression.
Integer An Integer expression.

Remarks

See MsdDimType for possible dimension commands. This is applicable only to arc size dimension commands that use length measure.
