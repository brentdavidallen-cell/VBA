OffsetFace Method

                          

This function offsets an input face

Syntax

Set SmartSolidElement = object.OffsetFace (solid, closestPointOnFace, distance)

The OffsetFace method syntax has these parts:

Part Description
object A valid object.
solid A SmartSolidElement expression. input body on which face is to be offset
closestPointOnFace A Point3d expression. input point that lie on the face to be offset
distance A Double expression. offset distance
