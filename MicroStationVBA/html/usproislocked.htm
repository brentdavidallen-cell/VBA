IsLocked Property

                          

Read/Write Boolean for ApplicationElement, ArcElement, Attachment, AuxiliaryCoordinateSystemElement, BsplineCurveElement, BsplineSurfaceElement, CellElement, ComplexShapeElement, ComplexStringElement, ConeElement, CurveElement, DimensionElement, Element, EllipseElement, Level, LineElement, ModelReference, MultiLineElement, NamedGroupElement, PointStringElement, SavedViewElement, ShapeElement, SharedCellDefinitionElement, SharedCellElement, SmartSolidElement, TagElement, TextElement, TextNodeElement.

Read-only Boolean for TextStyle.

Gets or sets a value indicating whether this object is locked.

Syntax

object.IsLocked [= Boolean]

The IsLocked property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

IsLocked for any element type controls whether or not the element can be modified. It is possible to copy an element that is locked, but it is not possible to delete or modify an element that is locked.

Setting Level.IsLocked tells MicroStation to treat all of the elements on the level like they were locked. Is a level is locked, then it is possible to add new elements to the level and it is possible to copy elements that are on that level but it is not possible to modify or delete elements on that level.

Setting Level.IsLocked to True is equivalent to setting Level.ElementAccess to msdLevelElementAccessLocked; setting IsLocked to False is equivalent to setting ElementAccess to msdLevelElementAccessAll.
