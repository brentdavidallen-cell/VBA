Point3d Type

        

Represents the Cartesian coordinates of a point or a vector in three-dimensional space.

The Point3d type has these members:

Name Description
X A double value that represents the X-axis coordinate of the point.
Y A double value that represents the Y-axis coordinate of the point.
Z A double value that represents the Z-axis coordinate of the point.

Remarks

For points in a ModelReference's geometry, the values are measured in the model's coordinate system, unless stated otherwise.

Often, a Point3d value is used to represent a point in two-dimensional space (a plane). In this case, the Z member is ignored.

There are a number of global utility functions for managing Point3d values:

  • Returns a negated (opposite) vector from a given vector: Point3dNegate.
  • Returns a point from a column of a matrix: Point3dFromMatrix3dColumn.
  • Returns a point from a matrix inverse transpose and a point: Point3dFromMatrix3dInverseTransposeTimesPoint3d.
  • Returns a point from a matrix times a point which is expressed as its XYZ components: Point3dFromMatrix3dTimesXYZ.
  • Returns a point from a matrix times a point which is specified by its X, Y and Z values: Point3dFromTransform3dTimesXYZ.
  • Returns a point from a matrix times a point: Point3dFromMatrix3dTimesPoint3d.
  • Returns a point from a matrix transpose times a point which is expressed as its XYZ components: Point3dFromMatrix3dTransposeTimesXYZ.
  • Returns a point from a matrix transpose times a point: Point3dFromMatrix3dTransposeTimesPoint3d.
  • Returns a point from a row in a matrix: Point3dFromMatrix3dRow.
  • Returns a point from a transform times a point: Point3dFromTransform3dTimesPoint3d.
  • Returns a point from its X and Y components (Z = 0): Point3dFromXY.
  • Returns a point from its X, Y and Z components: Point3dFromXYZ.
  • Returns a point from the product of a matrix inverse and a point: Point3dFromMatrix3dInverseTimesPoint3d.
  • Returns a point whose position is given by a fractional argument and two endpoints: Point3dInterpolate.
  • Returns a unit vector in the same direction as a given vector: Point3dNormalize.
  • Returns a vector that's the difference between two vectors or points: Point3dSubtract.
  • Returns a vector that's the result of rotating a given vector around the Z axis: Point3dRotateXY.
  • Returns a zero vector (all components zero): Point3dZero.
  • Returns a vector with all components one: Point3dOne.
  • Returns the (scalar) cross product of the XY parts of two vectors: Point3dCrossProduct3PointsXY.
  • Returns the (scalar) cross product of the XY parts of two vectors: Point3dCrossProductXY.
  • Returns the (scalar) dot product of the XY parts of two vectors which are relative to a given origin: Point3dDotProduct3PointsXY.
  • Returns the (scalar) dot product of the XY parts of two vectors: Point3dDotProductXY.
  • Returns the (scalar) dot product of two vectors which are relative to a given origin: Point3dDotProduct3Points.
  • Returns the (scalar) dot product of two vectors, one of which is relative to a given origin: Point3dDotDifference.
  • Returns the (scalar) dot product of two vectors: Point3dDotProduct.
  • Returns the (vector) cross product of two vectors: Point3dCrossProduct.
  • Returns the (vector) cross product of two vectors which are relative to a given origin: Point3dCrossProduct3Points.
  • Returns the angle between the projection of two vectors onto a plane with a given normal: Point3dPlanarAngleBetweenVectors.
  • Returns the angle between two vectors: Point3dAngleBetweenVectors.
  • Returns the angle from one vector to another, in the plane of the two vectors: Point3dSignedAngleBetweenVectors.
  • Returns the angle from two vectors using only their XY members: Point3dAngleBetweenVectorsXY.
  • Returns the distance distance between two points: Point3dDistance.
  • Returns the largest absolute value among the components of a vector: Point3dMaxAbs.
  • Returns the magnitude (length) of a vector: Point3dMagnitude.
  • Returns the product of a vector and a scale factor: Point3dScale.
  • Returns the squared distance between two points, using only the XY parts: Point3dDistanceSquaredXY.
  • Returns the squared distance between two points: Point3dDistanceSquared.
  • Returns the squared magnitude (length) of a vector: Point3dMagnitudeSquared.
  • Returns the sum of an origin and a scaled vector: Point3dAddScaled.
  • Returns the sum of an origin and three scaled vectors: Point3dAdd3Scaled.
  • Returns the sum of an origin and two scaled vectors: Point3dAdd2Scaled.
  • Returns the sum of two vectors: Point3dAdd.
  • Returns the triple product of three vectors: Point3dTripleProduct.
  • Returns the triple product of vectors from a base point three target points: Point3dTripleProduct4Points.
  • Returns a single component of a point: Point3dGetComponent.
  • Returns the projection of a point onto a plane (optionally as perceived in a view): Point3dProjectToPlane3d.
  • Returns the projection of a point onto a line (optionally as perceived in a view): Point3dProjectToRay3d.
  • Returns a value representing whether a point is on, inside, or outside a polygon, ignoring z-coordinates: Point3dInPolygonXY.
  • Tests whether a point is within a sector defined by an origin and two boundary points: Point3dIsPointInCCWSector.
  • Tests whether a vector formed from two points is within the smaller angle between two other vectors: Point3dIsPointInSmallerSector.
  • Tests whether a vector is between two other vectors, with the CCW direction resolved by a given up vector: Point3dIsVectorInCCWSector.
  • Tests whether a vector is between two vectors: Point3dIsVectorInSmallerSector.
  • Tests whether two vectors are equal within a given tolerance: Point3dEqualTolerance.
  • Tests whether two vectors are parallel: Point3dAreVectorsParallel.
  • Tests whether two vectors are perpendicular: Point3dAreVectorsPerpendicular.
  • Tests whether two vectors are equal: Point3dEqual.
