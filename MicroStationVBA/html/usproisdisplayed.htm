IsDisplayed Property

                          

Read/Write Boolean.

Gets or sets the value of Global Display.

Syntax

object.IsDisplayed [= Boolean]

The IsDisplayed property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

Whenever you change any aspect of a Level, you should subsequently call Levels.Rewrite to make the change permanent.

MicroStation will not display the elements on a Level unless Global Display is on, the view display is turned on, and the Level is not frozen.

Use IsDisplayedInView to determine if view display is turned on.

Use IsFrozen to determine if the Level is frozen.

A program can use IsEffectivelyDisplayedInView to determine if a Level is displayed in a View. It is equivalent to 

    theLevel.IsDisplayed And Not theLevel.IsFrozen And theLevel.IsDisplayedInView(theView)
