Offset Property

                          

Read/Write Long for DataBlock.

Read-only Point3d for TagElement.

Read/Write Point2d for TextStyle.

Gets or sets the position of this object's text within its region, relative to its normal position.

Syntax

object.Offset [= Long]

The Offset property syntax has these parts:

Part Description
object A valid object.
Long A Long expression.
Point2d A Point2d expression.

Remarks

For DataBlock, Offset tells where in the data block the next transfer starts. All of the Copy methods increment Offset by the count of bytes transferred.

A TagElement's offset is the distance between the tag element and the base element. The offset changes when the tag element is moved.
