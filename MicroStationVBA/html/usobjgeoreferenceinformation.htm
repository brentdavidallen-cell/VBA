GeoReferenceInformation Object

                                                     

This is the georeference information

Remarks

Use a Raster object to get a GeoReferenceInformation object as follows: 


    'With a variable :

    Dim theRasters As Rasters
    Dim theGeo As GeoReferenceInformation

    Set theRasters = RasterManager.Rasters
    Set theGeo = theRasters.Item(1). GeoReferenceInformation
    theGeo.[method/property]


    'Without a variable :

    Dim theRasters As Rasters

    Set theRasters = RasterManager.Rasters

    theRasters.Item(1).GeoReferenceInformation.[method/property]
