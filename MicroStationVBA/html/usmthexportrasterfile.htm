ExportRasterFile Method

                          

Writes a copy of the specified raster image file to the designated file name and format. If the file already exist, it is overwritten.

Syntax

object.ExportRasterFile SourceFilename, DestinationFilename [, NewFormatSpecifier [, NewColorModeSpecifier [, NewCompressionModeSpecifier [, NewBlockTypeSpecifier [, CompressionLevel [, MaxNumberOfColors [, InvertColors [, Gamma [, NewSizeInX [, NewSizeInY [, ResampleImage [, SisterFileType [, GeotiffProjectionCS [, GeotiffPCSLinearUnit]]]]]]]]]]]]]]

The ExportRasterFile method syntax has these parts:

Part Description
object A valid object.
SourceFilename A String expression. Full name (including path) of the file to be copied.
DestinationFilename A String expression. Full name (including path) of the copy to create.
NewFormatSpecifier Optional. A VARIANT expression. Format identification number or a RasterFormat object to use for the copy. Use values -1 or Nothing to make a copy using the same format as the original file.
NewColorModeSpecifier Optional. A VARIANT expression. Color mode identification number or a RasterColorMode object to use for the copy. Use values -1 or Nothing to make a copy using the same color mode as the original file. Make sure the color mode is supported by the copy's format.
NewCompressionModeSpecifier Optional. A VARIANT expression. Compression mode identification number or a RasterCompressionMode object to use for the copy. Use values -1 or Nothing to make a copy using the same compression mode as the original file. Make sure the compresssion mode is supported by the copy's color mode.
NewBlockTypeSpecifier Optional. A VARIANT expression. Block type identification number or a RasterBlockType object to use for the copy. Use values -1 or Nothing to make a copy using the same block type as the original file.
CompressionLevel Optional. A Byte expression. Level of compression when applicable. 0 min and 255 max.
MaxNumberOfColors Optional. A Long expression. Limits the number of colors used in the destination raster. Valid with palleted color mode only except 1bit. A value of 0 will set the limit to the maximum number of colors available.
InvertColors Optional. A Boolean expression. If True the raster's colors will be inverted.
Gamma Optional. A Double expression. Gamma correction to apply. 1.0 means no gamma correction applied.
NewSizeInX Optional. A Long expression. Desired size in X of the destination raster. Use -1, to keep the same size as the original file.
NewSizeInY Optional. A Long expression. Desired size in Y of the destination raster. Use -1, to keep the same size as the original file.
ResampleImage Optional. A Boolean expression. If True the raster will be resampled.
SisterFileType Optional. An MsdGeoReferenceSisterFileType expression. Sister file type to use for information relative to georeferencing. Ignored if no support for sister file available.
GeotiffProjectionCS Optional. A Long expression. is now obsolete, Specify the geotiff Projected Coordinate System. See http://www.remotesensing.org/geotiff/geotiff.html for more information.
GeotiffPCSLinearUnit Optional. A Long expression. is now obsolete, Specify the geotiff Projected Coordinate Linear Units. See http://www.remotesensing.org/geotiff/geotiff.html for more information.
