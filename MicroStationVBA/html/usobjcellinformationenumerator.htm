CellInformationEnumerator Object

                                                     

Use a CellInformationEnumerator to determine what cells are available to be placed.

Remarks

Use GetCellInformationEnumerator to get a CellInformationEnumerator. Then use the enumerator to step through the descriptions of all of the available cells.

Prior to calling CellInformationEnumerator.Current for the first time, the program must call CellInformationEnumerator.MoveNext to advance to the first entry. Each time the program calls MoveNext, the CellInformationEnumerator advances 1. MoveNext returns False when it cannot advance because the enumerator is already at the last element

If there is a CellLibrary attached, CellInformationEnumerator return information on all of the cell definitions in that library. If the call to GetCellInformationEnumerator specified True for the IncludeSharedCells argument, the enumerator also returns information about all of the shared cell definitions. If the call to GetCellInformationEnumerator specified True for the IncludeFullPath argument, then the enumerator also returns information about all of the cells from the files named by the MS_CELLLIST and the MS_BLOCKLIST configuration variables.
