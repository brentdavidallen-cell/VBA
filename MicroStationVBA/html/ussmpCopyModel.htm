Copying a Model From an Attachment

This examples illustrates copying a model from an attachment. 


Sub CopyModelFromAttachment(att As Attachment, strName As String)
    Dim mdlNew As ModelReference
    Dim strDescription As String

    '  Since an Attachment is a type of ModelReference, it is possible to
    '  use an Attachment object as a parameter for Copy
    strDescription = "Copied from " & att.DesignFile.Name & " (" & att.Name & ")"
    Set mdlNew = ActiveDesignFile.Models.Copy(att, strName, strDescription)
    mdlNew.Activate
End Sub
