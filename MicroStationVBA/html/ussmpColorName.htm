ColorName Property Example

This example illustrates a technique for setting the active color to "BYLEVEL". It uses the ColorName property of ActiveSettings to get the name of the current active color and to set the active color to ByLevel. 


Sub SetByLevel()
    With ActiveSettings
        ShowStatus "Replacing " & .ColorName & " with ByLevel"
        .ColorName = "ByLevel"
    End With
End Sub
