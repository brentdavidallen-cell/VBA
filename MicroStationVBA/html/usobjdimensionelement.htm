DimensionElement Object

                                                     

A kind of Element that corresponds to a MicroStation dimension element.

Remarks

Majority of a DimensionElement's settings are accessed by extracting the dimension's style object (which is of type DimensionStyle), changing it and putting it back on the DimensionElement. The remaining settings are accessed directly. When a particular setting on a DimensionElement does not match its parent DimensionStyle, an override flag will be set for that setting. To find out more information about the current overrides on a DimensionElement, use the methods PropertyOverridden and AnyPropertyOverridden.
