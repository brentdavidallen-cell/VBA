Point3dAreVectorsPerpendicular Method

                          

Tests if two vectors are perpendicular.

Syntax

Boolean = object.Point3dAreVectorsPerpendicular (Vector1, Vector2)

The Point3dAreVectorsPerpendicular method syntax has these parts:

Part Description
object A valid object.
Vector1 A Point3d expression. The first vector
Vector2 A Point3d expression. The second vector
