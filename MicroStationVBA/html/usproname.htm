Name Property

                          

Read-only String for ACSManager, Application, DesignFile, Font, GeographicCoordinateSystem, LineStyle, RasterInformation, SavedViewElement, SharedCellDefinitionElement, SharedCellElement.

Read/Write String for Attachment, AuxiliaryCoordinateSystemElement, CellElement, CellInformation, ColorTable, DimensionStyle, Level, ModelReference, NamedGroupElement, TagDefinition, TagSet, TextStyle, ViewGroup.

Gets or sets the name of this object.

Syntax

object.Name [= String]

The Name property syntax has these parts:

Part Description
object A valid object.
String A String expression.

Remarks

ACSManager.Named raises an error if there is no auxiliary coordinate system defined. A program can test for this using ACSManager.IsDefined.

For Application, gets the base name (without path or extension) of the file.

For an Attachment, Name is the name of the model. It is the name that MicroStation's Models dialog box displays. Use LogicalName to get/set the name that MicroStation's References dialog box displays.

Whenever you change any aspect of a Level, you should subsequently call Levels.Rewrite to make the change permanent.

Whenever you change any aspect of a NamedGroupElement, you should subsequently call Rewrite to make the change permanent.

TextStyle.Name raises an exception when trying to rename a TextStyle object returned from ActiveSettings.TextStyle or Element.TextStyle. Setting TextStyle.Name only works for text style objects returned from the TextStyles collection.
