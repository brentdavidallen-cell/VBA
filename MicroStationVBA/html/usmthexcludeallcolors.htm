ExcludeAllColors Method

                          

Excludes elements of any color from the forthcoming scan.

Syntax

object.ExcludeAllColors

The ExcludeAllColors method syntax has these parts:

Part Description
object A valid object.

Remarks

It is presumed that, after you call this method, you will make one or more calls to IncludeColor. Otherwise, your scan results will be empty.

Calling this method automatically restricts the scan to only graphical type elements.
