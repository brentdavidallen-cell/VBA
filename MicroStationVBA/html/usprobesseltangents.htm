BesselTangents Property

                          

Read/Write Boolean.

Gets or sets a value indicating whether this object uses the Bessel end condition to compute and/or scale its StartTangent and EndTangent.

Syntax

object.BesselTangents [= Boolean]

The BesselTangents property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

For InterpolationCurve objects whose property Periodic is False, the StartTangent and the EndTangent are used in addition to the fit points to compute the curve. This property determines how each tangent is computed (if it is stored as the zero vector), and how each is scaled. If BesselTangents is True, then the Bessel end condition (based on a parabola fit) is used for both computation and scaling. If False, then the natural end condition (based on a linear fit) is used for computation, and scaling is based on chord length; this method produces a poorer shape at the end points.

For InterpolationCurve objects with either BesselTangents or Periodic equal to True, there is a caveat. A persistent BsplineCurveElement created from such a curve exports to DWG format as a pole-based (rather than fit point-based) open B-spline curve; it does have identical shape, though.
