IsChainableElement Property

                          

Read-only Boolean.

Gets a value indicating whether this object supports the ChainableElement interface.

Syntax

object.IsChainableElement

The IsChainableElement property syntax has these parts:

Part Description
object A valid object.

Remarks

If IsChainableElement is True, AsChainableElement gets this object's implementation of the ChainableElement interface.
