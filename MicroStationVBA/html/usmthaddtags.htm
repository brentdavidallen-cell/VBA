AddTags Method

                          

Creates a TagElement for each tag definition in the tag set and associates those new elements with the object used to call AddTags.

Syntax

TagElement () = object.AddTags (TagSet)

The AddTags method syntax has these parts:

Part Description
object A valid object.
TagSet A TagSet expression.

Remarks

AddTags adds the new created tag elements to the active model. If the element already has a TagElement of the same name, then the new tag element replaces it. Adding tags also modifies the original element and rewrites it.

AddTags does not display the new elements. The program can display the new tag elements after positioning them.

For an element, calling AddTags raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
