AddLibraryToCheckerApp Method

                          

Adds a Library node to the list of libraries for a standards checker. See AddCheckerToStandardsCheckerApps for a sample report that contains a Library node.

Syntax

Set Object = object.AddLibraryToCheckerApp (CheckerAppNode, DgnLib, LibraryID)

The AddLibraryToCheckerApp method syntax has these parts:

Part Description
object A valid object.
CheckerAppNode An Object expression. The IXMLNode object that AddCheckerToStandardsCheckerApps returned.
DgnLib A DesignFile expression. The design file used to hold the settings.
LibraryID A Long expression. Output. The value that the standards checker should use to refer to the library in subsequent report entries This may be used as an argument to AddStandard.

Remarks

AddLibraryToCheckerApp returns an object that can be used as input to AddStandardToLibrary.
