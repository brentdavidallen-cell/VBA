AddConfigurationVariable Method

                          

Adds a definition for a new configuration variable.

Syntax

object.AddConfigurationVariable ConfigurationVariableName, ConfigurationVariableValue [, AlsoAddToUserConfigurationFile]

The AddConfigurationVariable method syntax has these parts:

Part Description
object A valid object.
ConfigurationVariableName A String expression. The name of the new configuration variable.
ConfigurationVariableValue A String expression. The value of the new configuration variable.
AlsoAddToUserConfigurationFile Optional. A Boolean expression. Denotes whether the variable and value should also be added (or updated) in the user's configuration file. The default value is False, denoting that the change lasts only until MicroStation exits.

Remarks

If the configuration variable is not already defined, AddConfigurationVariable creates the variable and sets the definition. If the configuration is already defined, it just updates the definition.

The value is saved as as User value. MicroStation's Configuration dialog labels each configuration variable as System, User, Project, etc. By defining the value as User, AddConfigurationVariable replaces an existing User. It overrides an existing System, System Env, or Project value.
