GetUnionShapes Method

                          

Constructs and returns a list of ComplexShapeElements that collectively enclose the area that represents the union of this object and a given Element.

Syntax

Set ElementEnumerator = object.GetUnionShapes (ClosedElement)

The GetUnionShapes method syntax has these parts:

Part Description
object A valid object.
ClosedElement An Element expression.
