RemoveModelActivateEventsHandler Method

                          

Removes a model-activate-events event handler from the set of active handlers.

Syntax

object.RemoveModelActivateEventsHandler EventHandler

The RemoveModelActivateEventsHandler method syntax has these parts:

Part Description
object A valid object.
EventHandler An IModelActivateEvents expression. An object that was previously passed to AddModelActivateEventsHandler
