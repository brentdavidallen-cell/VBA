PrimaryIsReferencePerSegment Property

                          

Read/Write Boolean.

Gets this object's segment-specific flag to indicate whether the primary dimension text is a reference.

Syntax

object.PrimaryIsReferencePerSegment (iSegment)[= val]

The PrimaryIsReferencePerSegment property syntax has these parts:

Part Description
object A valid object.
iSegment A Long expression.
val A Boolean expression.
