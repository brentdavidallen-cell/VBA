AppendCurve Method

                          

Appends the given BsplineCurve object to the end of this object.

Syntax

object.AppendCurve Curve [, ForceContinuity [, Reparametrize]]

The AppendCurve method syntax has these parts:

Part Description
object A valid object.
Curve A BsplineCurve expression. The B-spline curve to append to this object.
ForceContinuity Optional. A Boolean expression. Indicates whether to force Curve to be appended continuously to this object (by effectively replacing the first Pole and Weight of Curve with the last Pole and Weight of this object). If omitted, a default value of True is used. If ForceContinuity is False and Curve does not start where this object ends, then this method produces a discontinuous curve.
Reparametrize Optional. A Boolean expression. Indicates whether the Knots contributed by Curve occupy a portion of the composite curve's parameter space proportional to Curve.ComputePolygonLength. If omitted, a default value of True is used. If Reparametrize is False, then the Knots contributed by Curve occupy the latter half of the composite curve's parameter space.

Remarks

For BsplineCurve objects, no attempt is made to enforce higher order continuity across the junction.
