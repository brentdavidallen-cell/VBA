InsertTextLine Method

                          

Inserts an additional line of displayed text at a specified line number.

Syntax

object.InsertTextLine LineNumber, NewTextLine

The InsertTextLine method syntax has these parts:

Part Description
object A valid object.
LineNumber A Long expression. The position at which to insert the new line. The first line number is 1.
NewTextLine A String expression. The text of the line to be inserted.

Remarks

Example: If oElem is a TextNodeElement...

oElem.InsertTextLine(1, "Hello") inserts "Hello" as the first displayed line.

oElem.InsertTextLine(oElem.TextLinesCount + 1, "Hello") is the same as oElem.AddTextLine("Hello").
