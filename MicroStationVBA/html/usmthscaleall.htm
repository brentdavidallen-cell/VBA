ScaleAll Method

                          

Scales this object.

Syntax

object.ScaleAll Origin, XFactor, YFactor, ZFactor

The ScaleAll method syntax has these parts:

Part Description
object A valid object.
Origin A Point3d expression.
XFactor A Double expression.
YFactor A Double expression.
ZFactor A Double expression.

Remarks

ScaleAll of any graphical element scales the element about the origin by creating the appropriate Transform3d and calling Transform,

For an element, calling ScaleAll raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
