SetExpression Method

                          

Sets expression and failure value to to item type property.

Syntax

Boolean = object.SetExpression (expression, failureValue, failureMessage)

The SetExpression method syntax has these parts:

Part Description
object A valid object.
expression A String expression. string value to set expression to property.
failureValue A String expression. string value if expression is failed to execute then failure value set to property.
failureMessage A String expression. string value if expression syntax is wrong then failure message is returned.
