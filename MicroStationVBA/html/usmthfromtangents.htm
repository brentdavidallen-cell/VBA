FromTangents Method

                          

Initializes this object to a B-spline curve that interpolates the specified points and tangents.

Syntax

object.FromTangents Tangents [, Cubic [, FitLengths [, Parametrization [, Closed [, RemoveDuplicates [, Tolerance]]]]]]

The FromTangents method syntax has these parts:

Part Description
object A valid object.
Tangents A Ray3d () expression. The array of points (Tangents(i).Origin) and tangents (Tangents(i).Direction) to interpolate.
Cubic Optional. A Boolean expression. Indicates whether the order of the initialized object is cubic. If omitted or False, the order is quadratic.
FitLengths Optional. A Boolean expression. Indicates whether to interpolate tangent directions and lengths. If omitted or False, only tangent directions are interpolated.
Parametrization Optional. An MsdBsplineParametrizationType expression. The knot parametrization of the initialized object. Only the values msdBsplineParametrizationUniform, msdBsplineParametrizationChordLength and msdBsplineParametrizationCentripetal are valid. If omitted or invalid, msdBsplineParametrizationChordLength is used.
Closed Optional. A Boolean expression. Indicates whether the initialized object is Closed. If omitted, the object is open.
RemoveDuplicates Optional. A Boolean expression. Indicates whether to remove duplicate tangents from the given array. If omitted, duplicates are removed.
Tolerance Optional. A Double expression. The tolerance for duplicate tangent removal. If negative or omitted, a default tolerance of 1.0e-5 is used.
