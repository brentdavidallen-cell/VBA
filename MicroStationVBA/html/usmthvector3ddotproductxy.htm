Vector3dDotProductXY Method

                          

Returns the (scalar) dot product of xy parts of two vectors.

Syntax

Double = object.Vector3dDotProductXY (Vector1, Vector2)

The Vector3dDotProductXY method syntax has these parts:

Part Description
object A valid object.
Vector1 A Vector3d expression. The first vector
Vector2 A Vector3d expression. The second vector
