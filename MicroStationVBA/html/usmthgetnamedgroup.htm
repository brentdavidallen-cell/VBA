GetNamedGroup Method

                          

Gets a NamedGroupElement object representing the named group.

Syntax

Set NamedGroupElement = object.GetNamedGroup (GroupName)

The GetNamedGroup method syntax has these parts:

Part Description
object A valid object.
GroupName A String expression. The name must match exactly, including case.

Remarks

GetNamedGroup returns Nothing if the model does not contain the named group. It does not raise an error in this case.

It is also possible to get NamedGroupElements by scanning the model's ControlElementCache.
