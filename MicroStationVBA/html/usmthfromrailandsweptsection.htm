FromRailAndSweptSection Method

                          

Initializes this object by sweeping the given Section along the given Rail path, orienting each cross section with Rail's Frenet frame (cf. EvaluatePointFrame).

Syntax

object.FromRailAndSweptSection Rail, Section [, Tolerance [, ArcTolerance]]

The FromRailAndSweptSection method syntax has these parts:

Part Description
object A valid object.
Rail A BsplineCurve expression. The path along which Section is swept.
Section A BsplineCurve expression. The cross-sectional profile to sweep along Rail. For best results, Section should be planar and in position at the start of Rail.
Tolerance Optional. A Double expression. The absolute maxiumum section-surface deviation for portions of this object swept over nonlinear, noncircular G1-continuous sections of Rail. If omitted or nonpositive, a default tolerance of 1.0e-3 times the largest coordinate difference of the unweighted Poles of Section is used.
ArcTolerance Optional. A Double expression. The maximum deviation of a circular section of Rail, as an absolute tolerance on the section's radius computation. If omitted or nonpositive, a default tolerance of 1.0e-2 times Tolerance is used.

Remarks

For BsplineSurface objects, if the Rail and Section curves' start tangents are not perpendicular or if Rail does not start in Section's initial osculating plane, then Section is rotated internally so that its start binormal is parallel to Rail's start tangent, and it is translated so that its start point lies on Rail's start point if these points were initially close, or if not, so that the projection of Rail's start point onto Section's initial osculating plane lies on Rail's start point (cf. EvaluatePointFrame). If Rail starts at a point along the (transformed) Section, then both curves are preserved in the resulting surface as isoparametric curves in the u- and v-parameter directions, respectively; otherwise, only Section is so preserved.

For BsplineSurface objects, if Rail is G1-continuous (continuity of the normalized tangent), a method with more options and greater rail approximation accuracy is FromRailsAndSweptSections.
