Range3dUnionXYZ Method

                          

Extends the coordinates of the range cube points in Range so as to include the single additional point given as xyz coordinates.

Syntax

Range3d = object.Range3dUnionXYZ (Range, X, Y, Z)

The Range3dUnionXYZ method syntax has these parts:

Part Description
object A valid object.
Range A Range3d expression. initial range.
X A Double expression. x coordinate of additional point.
Y A Double expression. y coordinate of additional point.
Z A Double expression. z coordinate of additional point.
