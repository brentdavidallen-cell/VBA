GetCellInformationEnumerator Method

                          

Returns a CellInformationEnumerator. The program can then use that enumerator to get information about all of the cells in the attached cell library. The enumerator may also return information on all shared cell definitions and definitions of cells in cell libraries specified by the MS_CELLLIST and MS_BLOCKLIST configuration variables.

Syntax

Set CellInformationEnumerator = object.GetCellInformationEnumerator (IncludeSharedCells, IncludeFullPath)

The GetCellInformationEnumerator method syntax has these parts:

Part Description
object A valid object.
IncludeSharedCells A Boolean expression. Indicates whether the enumerator returns information about shared cell definitions.
IncludeFullPath A Boolean expression. Indicates whether the enumerator returns information about the cell definitions contained in the cell libraries named by the MS_CELLLIST MS_BLOCKLIST configuration variables.
