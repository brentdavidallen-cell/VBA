Transform3dGetPointComponent Method

                          

Returns a value from a specified component of the point (translation) part of the transformation.

Syntax

Double = object.Transform3dGetPointComponent (Transform, Row)

The Transform3dGetPointComponent method syntax has these parts:

Part Description
object A valid object.
Transform A Transform3d expression. The input transformation.
Row A Long expression. The index of point component to read. Indices are 0, 1, 2 for x, y, z
