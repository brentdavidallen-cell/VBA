SetPatternWithOrigin Method

                          

Associate a pattern with this object.

Syntax

object.SetPatternWithOrigin Pattern, PatternOrigin, Rotation

The SetPatternWithOrigin method syntax has these parts:

Part Description
object A valid object.
Pattern A Pattern expression. Provide an AreaPattern, CrossHatchPattern, or HatchPattern object.
PatternOrigin A Point3d expression. The origin point of the pattern. This point's values are measured in the coordinate system of the containing model.
Rotation A Matrix3d expression. Defines the orientation of the coordinate system from which the pattern angle is measured. For no rotation, specify Matrix3dIdentity.

Remarks

SetPatternWithOrigin lets the program speficy the origin. Otherwise, SetPatternWithOrigin is identical to SetPattern.
