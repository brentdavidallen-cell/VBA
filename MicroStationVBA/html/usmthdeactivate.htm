Deactivate Method

                          

Deactivates AccuDraw.

Syntax

object.Deactivate

The Deactivate method syntax has these parts:

Part Description
object A valid object.

Remarks

Calling Deactivate is not equivalent to disabling AccuDraw from the user interface. Disabling AccuDraw from the user interface hides the AccuDraw dialog box and prevents subsequent commands from using AccuDraw. Deactivating AccuDraw merely stops AccuDraw from displaying the compass and other aids. It does not disable AccuDraw and does not affect subsequent commands.

Call Deactivate at the start of a command if AccuDraw does not make sense for the command.
