RightTerminatorForTool Property

                          

Read/Write Integer.

Gets this object's right terminator type for the specified tool

Syntax

object.RightTerminatorForTool (DimensionCommand)[= Integer]

The RightTerminatorForTool property syntax has these parts:

Part Description
object A valid object.
DimensionCommand An MsdDimType expression.
Integer An Integer expression.

Remarks

See MsdDimType for possible dimension commands
