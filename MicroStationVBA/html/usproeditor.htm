Editor Property

                          

Read-only String.

Retrieves the name of the last application used to edit this file.

Syntax

object.Editor

The Editor property syntax has these parts:

Part Description
object A valid object.

Remarks

Editor of a DesignFile is one of the standard Windows file properties. It tells what application was used the last time the file was edited. You can use the Windows Explorer to view this along with the other file properties. The Application field of MicroStation's File->Properties dialog box also displays the value of this property.
