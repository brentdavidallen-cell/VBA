GetMessage Method

                          

Returns a MessageCenterMessage value that contains information extracted from the indexed Message Center message.

Syntax

MessageCenterMessage = object.GetMessage (Index)

The GetMessage method syntax has these parts:

Part Description
object A valid object.
Index A Long expression. The row number, from the Message Center, of the desired message object. This is a 0-based index where valid values are 0 to MessageCount-1 inclusive. 0 refers to the most recently added message.
