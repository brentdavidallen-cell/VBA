AngleMode Property

                          

Read/Write MsdAngleMode.

Gets or sets the Angle Mode.

Syntax

object.AngleMode [= Value]

The AngleMode property syntax has these parts:

Part Description
object A valid object.
Value An MsdAngleMode expression.

Remarks

The Settings properties AngleFormat, AngleMode, and AngleAccuracy correspond to the values in the Coordinate Readout Angles section of MicroStation's DGN File Settings dialog box.
