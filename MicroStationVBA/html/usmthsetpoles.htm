SetPoles Method

                          

Sets the array of poles of this object.

Syntax

object.SetPoles Poles

The SetPoles method syntax has these parts:

Part Description
object A valid object.
Poles A Point3d () expression. The array of poles to set.

Remarks

For BsplineCurve and BsplineSurface objects, see the Pole property for further description of this array. In particular, note that this array has zero-based indexing and is 1-dimensional for BsplineCurve objects, and 2-dimensional for BsplineSurface objects.
