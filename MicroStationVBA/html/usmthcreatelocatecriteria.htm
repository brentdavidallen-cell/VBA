CreateLocateCriteria Method

                          

Creates and initializes a new LocateCriteria object.

Syntax

Set LocateCriteria = object.CreateLocateCriteria (WillModifyElements)

The CreateLocateCriteria method syntax has these parts:

Part Description
object A valid object.
WillModifyElements A Boolean expression. Determines the initial state of the new LocateCriteria object.

Remarks

Commands that copy elements or extract information from the elements but do not modify the original elements should specify False for WillModifyElements. Commands that do modify the located elements should specify True for WillModifyElement.

If WillModifyElement is True, then CreateLocateCriteria initializes the LocateCriteria object to only allow elements that are in the ActiveModelReference and are unlocked. If WillModifyElement is False, then CreateLocateCriteria initializes the LocateCriteria object to allow both locked and unlocked elements from any model.

The initial values for the LocateCriteria do not exclude elements based on any other criteria. Use the properties of LocateCriteria to establish additional exclusions.
