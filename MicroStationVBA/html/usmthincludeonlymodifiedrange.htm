IncludeOnlyModifiedRange Method

                          

Include elements within the specified Minimum and Maximum modified date.

Syntax

object.IncludeOnlyModifiedRange Min [, Max]

The IncludeOnlyModifiedRange method syntax has these parts:

Part Description
object A valid object.
Min A Date expression. The starting date.
Max Optional. A Date expression. The ending date.

Remarks

The Min and Max parameter values are inclusive. If the Max date is not specified, it defaults to the current date and time.

Since VBA automatically converts strings to dates when necessary, it is possible to specify the dates as String's. Consult the VB or VBA documentation for permissible formats and conversion rules.

Sample


    oSC.IncludeOnlyModifiedRange "9/15/03 12:21:42"

    or

    oSC.IncludeOnlyModifiedRange "9/15/03 12:21:42", "9/16/03"
