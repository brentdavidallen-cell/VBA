GetPoint3d Method

                          

Syntax

Point3d = object.GetPoint3d (Index)

The GetPoint3d method syntax has these parts:

Part Description
object A valid object.
Index A Long expression. A zero-based index that must be in the range 0 to Count-1.

Remarks

A .NET program using COM interop can use GetXDatum to retrieve any XDatum that is not a point, but must use GetPoint3d to process an XDatum that contains a Point3d.
