BorderAndBackgroundVisible Property

                          

Read/Write Boolean.

Gets or sets a value indicating whether this object's border and background are visible.

Syntax

object.BorderAndBackgroundVisible [= Boolean]

The BorderAndBackgroundVisible property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

Note that the background fill color is visible in a View only if BorderAndBackgroundVisible is True and the View's DisplaysFill property is True.
