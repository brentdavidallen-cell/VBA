SetCameraProperties Method

                          

Sets the camera properties to the values described in the Parameters parameter.

Syntax

object.SetCameraProperties Parameters

The SetCameraProperties method syntax has these parts:

Part Description
object A valid object.
Parameters A ViewCameraParameters expression.

Remarks

SetCameraProperties inspects each property of the ViewCameraParameters object. If the property has been initialized, then SetCameraProperties sets the corresponding view camera property to the specified value. If it has not been initialized, then SetCameraProperties computes a reasonable default value for the property.
