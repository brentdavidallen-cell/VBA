Ellipse3dFromEllipticalElement Method

                          

Initializes an Ellipse3d from an EllipticalElement.

Syntax

Ellipse3d = object.Ellipse3dFromEllipticalElement (Element)

The Ellipse3dFromEllipticalElement method syntax has these parts:

Part Description
object A valid object.
Element An EllipticalElement expression.
