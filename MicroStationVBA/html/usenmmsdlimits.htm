MsdLimits Enum

        

These constants correspond to various capacity limits within MicroStation.

The MsdLimits enum has these members:

Name Description
msdLimitsMaxVertices 5000 (&H1388) The maximum number of vertices that can be contained within a VertexList object (i.e., an object that supports the VertexList interface).
msdLimitsMaxViews 8 (&H8) The maximum number of onscreen views; the highest index that can be given to a Views collection. (The lowest index is 1.)
