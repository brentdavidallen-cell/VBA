IsEllipticalElement Property

                          

Read-only Boolean.

Gets a value indicating whether this object supports the EllipticalElement interface.

Syntax

object.IsEllipticalElement

The IsEllipticalElement property syntax has these parts:

Part Description
object A valid object.

Remarks

If IsEllipticalElement is True, AsEllipticalElement gets this object's implementation of the EllipticalElement interface.
