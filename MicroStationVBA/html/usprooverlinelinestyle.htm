OverlineLineStyle Property

                          

Read/Write LineStyle.

Gets or sets the style of the line (if any) that appears over this object's text.

Syntax

[Set] object.OverlineLineStyle [= LineStyle]

The OverlineLineStyle property syntax has these parts:

Part Description
object A valid object.
LineStyle A LineStyle expression.

Remarks

This property has no apparent effect unless OverlineUsesCustomLine is True.
