Mirror Method

                          

Mirrors the object about the line in the XY plane.

Syntax

object.Mirror Point1, Point2

The Mirror method syntax has these parts:

Part Description
object A valid object.
Point1 A Point3d expression.
Point2 A Point3d expression.

Remarks

Mirror of any graphical element is equivalent to: 


    pnt1 = Point1
    pnt2 = Point2
    pnt3 = Point1

    ' Force the place to be perpendicular to the XY plane
    pnt1.Z = 0
    pnt2.Z = 0
    pnt3.Z = 1

    ele.Mirror3d pnt1, pnt2, pnt3

For an element, calling Mirror raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
