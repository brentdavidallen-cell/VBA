Mirror3d Method

                          

Mirrors the object about the plane defined by the 3 points.

Syntax

object.Mirror3d PlanePoint1, PlanePoint2, PlanePoint3

The Mirror3d method syntax has these parts:

Part Description
object A valid object.
PlanePoint1 A Point3d expression.
PlanePoint2 A Point3d expression.
PlanePoint3 A Point3d expression.

Remarks

For an element, calling Mirror3d raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
