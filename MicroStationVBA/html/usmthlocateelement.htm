LocateElement Method

                          

Finds the element at the position indicated by a data point within a view.

Syntax

Set Element = object.LocateElement (DataPoint, View, RestartLocate)

The LocateElement method syntax has these parts:

Part Description
object A valid object.
DataPoint A Point3d expression.
View A View expression.
RestartLocate A Boolean expression.

Remarks

LocateElement looks for elements in the range specified by the LocateTolerance. The View argument is required because the locate tolerance is expressed in terms of pixels, and the number of UOR's per pixel varies depending on the View.

Passing True for RestartLocate causes LocateElement to assemble a list of all of the elements that fall within the "locate tolerance". Then RestartLocate returns the first element from the list. Each additional call to LocateElement specifying False for RestartLocate causes LocateElement to advance to the next element in the list. It then returns that element. If LocateElement does not find any elements, or if RestartLocate is False and LocateElement has already returned all of the elements, then LocateElement returns Nothing.

See Interaction of HitPoint, LocateElement, and GetLocatedElement for an example that illustrates how to use these methods together.
