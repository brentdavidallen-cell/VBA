Plane3dIntersectsPlane3d Method

                          

Returns True if the given planes intersect, and if so, also returns a ray representing the line of intersection.

Syntax

Boolean = object.Plane3dIntersectsPlane3d (IntersectionRay, Plane0, Plane1)

The Plane3dIntersectsPlane3d method syntax has these parts:

Part Description
object A valid object.
IntersectionRay A Ray3d expression. The intersection line, represented as a ray.
Plane0 A Plane3d expression. One of the two planes to intersect.
Plane1 A Plane3d expression. One of the two planes to intersect.

Remarks

This method replaces the method Plane3dIntersectPlane3d, which was deprecated after MicroStation V8 2004 Edition.
