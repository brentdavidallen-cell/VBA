ParentStyle Property

                          

Read/Write TextStyle.

Gets or sets the parent style of this object.

Syntax

[Set] object.ParentStyle [= TextStyle]

The ParentStyle property syntax has these parts:

Part Description
object A valid object.
TextStyle A TextStyle expression.

Remarks

By default, this object's properties have the same values as the properties of its parent style. The exceptions are those properties to which new values have been assigned. Those properties will retain their latest assigned values, regardless of changes to the parent style.
