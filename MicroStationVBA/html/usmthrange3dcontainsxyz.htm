Range3dContainsXYZ Method

                          

Test if a point given as x,y,z is contained in a range.

Syntax

Boolean = object.Range3dContainsXYZ (Range, X, Y, Z)

The Range3dContainsXYZ method syntax has these parts:

Part Description
object A valid object.
Range A Range3d expression. candidate containing range.
X A Double expression. x coordinate
Y A Double expression. y coordinate
Z A Double expression. z coordinate
