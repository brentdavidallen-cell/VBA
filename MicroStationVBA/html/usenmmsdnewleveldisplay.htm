MsdNewLevelDisplay Enum

        

Defines constants that represent all of the possible values of the property, Attachment.NewLevelDisplay.

The MsdNewLevelDisplay enum has these members:

Name Description
msdNewLevelDisplayFromConfig 0 (&H0) New Level display is set by the MS_REF_NEWLEVELDISPLAY configuration variable.
msdNewLevelDisplayAlways 1 (&H1) New Levels are always displayed.
msdNewLevelDisplayNever 2 (&H2) New Levels are never displayed.
