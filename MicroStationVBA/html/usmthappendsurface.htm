AppendSurface Method

                          

Appends the given BsplineSurface object at the end of this object in the given parameter Direction.

Syntax

object.AppendSurface Surface, Direction [, ForceContinuity [, Reparametrize]]

The AppendSurface method syntax has these parts:

Part Description
object A valid object.
Surface A BsplineSurface expression. The B-spline surface to append to this object.
Direction An MsdBsplineSurfaceDirection expression. The parameter direction of the edge along which Surface is appended. For example, if Direction is msdBsplineSurfaceV, then the last isoparametric curve in the v-parameter direction of this object is joined to the first such curve of Surface (cf. VRulesCount); in other words, the UKnots of Surface are appended to the UKnot vector of this object and the columns of the Pole array of Surface (suitably refined) are appended as columns to the Pole array of this object.
ForceContinuity Optional. A Boolean expression. Indicates whether to force Surface to be appended continuously to this object (by effectively replacing the first Poles and Weights of Surface corresponding to the given parameter Direction with the last Poles and Weights of this object in the same Direction). If omitted, a default value of True is used. If ForceContinuity is False and Surface does not start (in the given parameter Direction) where this object ends, then this method produces a discontinuous surface.
Reparametrize Optional. A Boolean expression. Indicates whether the UKnots (or VKnots, as per Direction) contributed by Surface occupy a portion of the composite surface's u-parameter (v-parameter) space proportional to the length of the control polygon of the first isoparametric curve of Surface in the u-parameter (v-parameter) direction. If omitted, a default value of True is used. If Reparametrize is False, then the UKnots (VKnots) contributed by Surface occupy the latter half of the composite surface's u-parameter (v-parameter) space.

Remarks

For BsplineSurface objects, no attempt is made to enforce higher order continuity across the junction.

For BsplineSurface objects, any boundaries in the object are ignored.
