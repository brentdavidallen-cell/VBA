ApproximateWithArcs Method

                          

Returns an ArcElement, a LineElement or a ComplexStringElement composed of ArcElements/LineElements that approximates this planar object.

Syntax

Set Element = object.ApproximateWithArcs (Parameters [, FitPointsCount [, StartTangent [, EndTangent]]])

The ApproximateWithArcs method syntax has these parts:

Part Description
object A valid object.
Parameters A Double () expression. The optional parameters that determine the fit points along this object. If this array is empty, then FitPointsCount fit points are computed by this method.
FitPointsCount Optional. A Long expression. The optional number of fit points to compute. If this parameter is positive, then the contents of the Parameters array are ignored, and fit points are computed first via EvaluatePointsAtEqualDeviationChords, EvaluatePointsAtEqualLengthChords, or EvaluatePointsAtEqualParametricLengthChords, whichever method succeeds first.
StartTangent Optional. An MsdTangentInterpolationType expression. How the start tangent of the approximation is computed.
EndTangent Optional. An MsdTangentInterpolationType expression. How the end tangent of the approximation is computed.

Remarks

For BsplineCurve objects, the returned Element uses two ArcElements to span each pair of successive fit points; if three successive fit points are collinear, then two LineElements are used to span those fit points.

For nonplanar BsplineCurve objects, the computations are computed in a plane that approximates the fit points, and the original fit points are preserved in the output Element. This method works best with planar input.
