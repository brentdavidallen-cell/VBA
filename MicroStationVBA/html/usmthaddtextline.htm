AddTextLine Method

                          

Adds an additional line of displayed text to this object.

Syntax

object.AddTextLine NewTextLine

The AddTextLine method syntax has these parts:

Part Description
object A valid object.
NewTextLine A String expression. The text of the line to be added.

Remarks

AddTextLine always adds to the end of the existing set of lines. If you want to insert a new line at a specific line number, use InsertTextLine.
