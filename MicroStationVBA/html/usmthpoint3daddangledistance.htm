Point3dAddAngleDistance Method

                          

Add an offset vector to a given starting point or vector, with the offset specified in XY polar form, i.e. as angle and distance.

Syntax

Point3d = object.Point3dAddAngleDistance (Point1, AngleRadians, DistanceXY, Dz)

The Point3dAddAngleDistance method syntax has these parts:

Part Description
object A valid object.
Point1 A Point3d expression. The the first point or vector
AngleRadians A Double expression. angle from x axis to offset vector.
DistanceXY A Double expression. distance in xy plane.
Dz A Double expression. z part of offset.
