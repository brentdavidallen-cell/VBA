ElementCache Object

                                                     

Represents a set of elements stored contiguously in memory.

Remarks

Virtually every aspect of a design file is stored in elements. Whenever a design file is in use, its elements are managed in a special array called an element cache.

Actually, there are several element caches, each containing elements that serve a common purpose:

  • Each design file has a NonModelElementCache, which holds the elements that pertain to the overall design file.
  • Each model has a GraphicalElementCache, which holds the model's graphical elements.
  • Each model also has a ControlElementCache, which holds the model's control (non-graphical) elements.

In practice, you'll probably have little reason to use the element caches. But they're here if you need them.
