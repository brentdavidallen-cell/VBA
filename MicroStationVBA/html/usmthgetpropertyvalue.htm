GetPropertyValue Method

                          

Returns the property value as a variant.

Syntax

VARIANT = object.GetPropertyValue (AccessString)

The GetPropertyValue method syntax has these parts:

Part Description
object A valid object.
AccessString A String expression. Property whose value to be retrived.

Remarks

Use GetPropertyValue to retrieve a property value if the property's type is Boolean, Date, Double, String , Integer or Point3d.
