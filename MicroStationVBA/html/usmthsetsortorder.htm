SetSortOrder Method

                          

Specifies the print order for this element. Valid only when printing in non-rasterized mode.

Syntax

object.SetSortOrder sortOrder

The SetSortOrder method syntax has these parts:

Part Description
object A valid object.
sortOrder A Long expression.

Remarks

This element sorting technique is valid only when printing in non-rasterized mode, but supports both 2D and 3D data. This call has no effect unless PrintManager.EnableSortOrder is first called inside BeginPrintPage. It is not recommended to set the sort order for every element, as doing so increases the amount of memory used during printing. All unsorted elements print before any sorted elements. To change the priority of 2D elements when printing in rasterized mode, the IPrintEvents.Resymbolize.DisplayParams.Priority method must be used.
