Scale Property

                          

Read/Write Double for AreaPattern.

Read-only Point3d for CellElement, SharedCellDefinitionElement, SharedCellElement.

Read/Write Point3d for Settings.

Gets or sets this object's scale.

Syntax

object.Scale [= Double]

The Scale property syntax has these parts:

Part Description
object A valid object.
Double A Double expression.
Point3d A Point3d expression.

Remarks

See User Defined Types as Property Values for a description of a common mistake programmers make when trying to change just one coordinate of a Point3d scale.
