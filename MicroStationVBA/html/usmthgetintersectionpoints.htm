GetIntersectionPoints Method

                          

Returns a list of points where the IntersectableElement object intersects with another element, called the intersector. If the list of points returned has an upper limit of -1, there are no intersections.

Syntax

Point3d () = object.GetIntersectionPoints (Intersector, Rotation)

The GetIntersectionPoints method syntax has these parts:

Part Description
object A valid object.
Intersector An Element expression. The intersector element.
Rotation A Matrix3d expression. The coordinate system in which the apparent intersections are calculated.

Remarks

For three dimensional elements, GetIntersectionPoints returns apparent intersections, ignoring differences along the Z-axis.

Because all rotation matrices produce the same result when working in two dimensions, Rotation has no meaning for two dimensions. Specify Matrix3dIdentity in such cases.
