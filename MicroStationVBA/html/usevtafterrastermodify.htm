AfterRasterModify Event

                          

The AfterRasterModify event occurs when a raster's information was modified.

Syntax

object.AfterRasterModify RasterID, ModificationType

The AfterRasterModify method syntax has these parts:

Part Description
object A valid object.
RasterID An Int64 expression. Identification number of the raster that is about to be or was modified.
ModificationType An MsdRasterModificationType expression. Type of modification that is to be or was applied to the raster.
