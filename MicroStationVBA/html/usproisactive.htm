IsActive Property

                          

Read-only Boolean for Attachment, DesignFile, ModelReference, ViewGroup.

Read/Write Boolean for Level.

Gets a value indicating whether this object is the object that's currently being worked on.

Syntax

object.IsActive [= Boolean]

The IsActive property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

Whenever you change any aspect of a Level, you should subsequently call Levels.Rewrite to make the change permanent.
