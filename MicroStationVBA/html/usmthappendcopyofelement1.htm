AppendCopyOfElement1 Method

                          

Makes a copy of the element and appends it to the list of elements the container holds. If the DrawNow parameter is True, it immediately draws all of the elements that the TransientElementContainer contains.

Syntax

object.AppendCopyOfElement1 ElementToCopy, DrawNow

The AppendCopyOfElement1 method syntax has these parts:

Part Description
object A valid object.
ElementToCopy An Element expression.
DrawNow A Boolean expression.

Remarks

If the program is adding a lot of elements to the TransientElementContainer, AppendCopyOfElement can be very inefficient because it draws all of the elements everytime the program adds an element to the container. One solution to this problem is to package all of the elements into a cell and then add that to the TransientElementContainer. Another solution is to call AppendCopyOfElement1 with DrawNow set to False for every element except the last.
