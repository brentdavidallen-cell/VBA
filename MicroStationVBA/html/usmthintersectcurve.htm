IntersectCurve Method

                          

Returns the intersections between this object and the given BsplineCurve.

Syntax

Point3d () = object.IntersectCurve (Parameters, IntersectorParameters, Intersector, Rotation [, Tolerance [, UseSubdivision]])

The IntersectCurve method syntax has these parts:

Part Description
object A valid object.
Parameters A Double () expression. The parameter(s) of the point(s) of intersection on this object.
IntersectorParameters A Double () expression. The parameter(s) of the point(s) of intersection on the Intersector object.
Intersector A BsplineCurve expression. The B-spline curve to intersect with this object.
Rotation A Matrix3d expression. The rotation matrix (e.g., View.Rotation, Matrix3dRotationFromRowZ) specifying the plane in which intersections are apparent. Pass Matrix3dIdentity to find the intersections as viewed in the xy-plane. Pass Matrix3dZero to find true intersections.
Tolerance Optional. A Double expression. The error tolerance for the computation. If omitted or negative, a default tolerance of 1.0e-8 times the larger of the largest coordinate differences of the unweighted Poles of this object and the Intersector is used.
UseSubdivision Optional. A Boolean expression. Indicates whether the intersection computation subdivides the objects to Tolerance. If omitted or False, and the Orders of the objects are sufficiently small and Rotation is nonzero, then an exact computation in the plane of Rotation is performed.

Remarks

For BsplineCurve objects, on a successful return, the arrays are empty if and only if no intersections were found.
