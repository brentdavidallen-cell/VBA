MsdMessageCenterPriority Enum

        

Defines constants that represent all of the possible values of the Priority argument of the method, AddMessage.

The MsdMessageCenterPriority enum has these members:

Name Description
msdMessageCenterPriorityInfo 12 (&HC) Denotes a message that conveys incidental information.
msdMessageCenterPriorityWarning 11 (&HB) Denotes a message that you want to bring to the user's attention.
msdMessageCenterPriorityError 10 (&HA) Denotes a message that reports an error or unexpected condition within your application (not the user's fault).
msdMessageCenterPriorityDebug 13 (&HD) Denotes a debugging message that is primarily of benefit of you, the application developer. (Debug messages are displayed only if the user turns on "Display Debug Messages" in the Message Center Properties dialog. You can open this dialog by right-clicking in Message Center dialog or in the message bar.)
msdMessageCenterPriorityNone 14 (&HE) Denotes a message without an associated icon.
