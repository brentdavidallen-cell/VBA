GPSPositionData Type

        

Contains the data retrieved from GetLastPositionData

The GPSPositionData type has these members:

Name Description
Longitude Degrees, negative for west, WGS84 datum. Invalid if fixDimension < 2
Latitude Degrees, negative for south, WGS84 datum. Invalid if fixDimension < 2
Elevation Meters, invalid if fixDimension < 3
EventTime Time in Variant time format; can be assigned to Date (invalid if < 0)
FixDimension 0, 2, or 3.
PDOP Composite position dilution of precision (invalid if < 0)
HDOP Horizontal dilution of precision (invalid if < 0)
VDOP Vertical dilution of precision (invalid if < 0)
Heading Heading clockwise from true north, degrees. (invalid if < 0)
Velocity Speed in direction
