ThinShell Method

                          

Offsets faces of the input body by the offset distance provided to created a thin shell body.

Syntax

Set SmartSolidElement = object.ThinShell (solid, generalOffset)

The ThinShell method syntax has these parts:

Part Description
object A valid object.
solid A SmartSolidElement expression. solid to be shelled .
generalOffset A Double expression. default offset value or shell thickness.

Remarks

The input solid should not be a SmartSurface. For a SmartSurface, you can use OffsetFace mdthod.
