StatusPrompt Property

                          

Read/Write String.

Gets or Sets the prompt message in MicroStation's status area.

Syntax

object.StatusPrompt [= Prompt]

The StatusPrompt property syntax has these parts:

Part Description
object A valid object.
Prompt A String expression.

Remarks

This gets or sets the Prompt regardless of whether MessagesEnabled is True or False. The prompt is the string to the right of the > character.
