RasterColorModes Object

                                                     

A collection of RasterColorMode objects. It gives access to all color modes available for a raster file format. It is used like this: 


    'With a variable:
    Dim oFormat As RasterFormat
    Dim theRasterColorModes As RasterColorModes

        ...

    Set theRasterColorModes = oFormat.ColorModes
    theRasterColorModes.[method/property]

    'Without a variable:
    oFormat.RasterColorModes.[method/property]

Remarks

It is possible to step through the collection using the For Each statement as follows: 


    Dim oFormat As RasterFormat
    Dim oRasterColorMode As RasterColorMode
    Dim theRasterColorModes As RasterColorModes

        ...

    Set theRasterColorModes = oFormat.ColorModes

    For Each oRasterColorMode In theRasterColorModes
        oRasterColorMode.[method/property]
    Next
