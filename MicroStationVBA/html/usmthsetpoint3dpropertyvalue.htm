SetPoint3dPropertyValue Method

                          

Sets given Point3d object to sepcified point type property.

Syntax

Boolean = object.SetPoint3dPropertyValue (AccessString, point)

The SetPoint3dPropertyValue method syntax has these parts:

Part Description
object A valid object.
AccessString A String expression. Name of Property whose value to be changed.
point A Point3d expression. New value to be set to given property.

Remarks

SetPointTypeProperty always writes a change back to the file immediately. This is particularly significant when using a ItemTypePropertyHandler object to access a point type properties value.
