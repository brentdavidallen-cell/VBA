IPrintEvents Object

                                                     

Defines the methods MicroStation uses to notify a program of print/preview events.

Remarks

To declare that a class is an event handler for print events, use the statement Implements IPrintEvents. To activate the event handler, the program must call PrintManager.AddPrintEventsHandler passing in an instance of the class that implements IPrintEvents. To deactivate the event handler, the program must call PrintManager.RemovePrintEventsHandler, passing in the same object that was passed to AddPrintEventsHandler.

The event handler remains active until the program deactivates it using RemovePrintEventsHandler, or until the the project is unloaded or is reset.

Example of auto-starting print event handler: 


        Option Explicit

        Dim g_printEventHandler As MyPrintEvents

        Sub OnProjectLoad()
            Set g_printEventHandler = New MyPrintEvents
            Application.PrintManager.AddPrintEventsHandler g_printEventHandler
        End Sub

        Sub OnProjectLoadNonGraphics()
            'This method is required for event handler to load in Print Organizer non-graphics worker process.
            Set g_printEventHandler = New MyPrintEvents
            Application.PrintManager.AddPrintEventsHandler g_printEventHandler
        End Sub
