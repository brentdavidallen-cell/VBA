BeforeRedraw Event

                          

Syntax

object.BeforeRedraw TheViews, TheModels, DrawMode

The BeforeRedraw method syntax has these parts:

Part Description
object A valid object.
TheViews A View () expression.
TheModels A ModelReference () expression.
DrawMode An MsdDrawingMode expression.

Remarks

See IViewUpdateEvents for a description of the arguments and information on the relationship of AfterRedraw and BeforeRedraw.

See Overview of MicroStation Events for more information about MicroStation events. for more information.

Warning

Using the IViewUpdateEvents to draw elements that have not been added to a model does not work well in views that have Graphics Acceleration enabled.
