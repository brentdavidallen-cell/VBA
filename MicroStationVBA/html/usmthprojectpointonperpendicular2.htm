ProjectPointOnPerpendicular2 Method

                          

Returns the endpoint of a projection from a given point to this object, along a perpendicular of this object.

Syntax

Point3d = object.ProjectPointOnPerpendicular2 (Point, Rotation)

The ProjectPointOnPerpendicular2 method syntax has these parts:

Part Description
object A valid object.
Point A Point3d expression. The point from which to project.
Rotation A Matrix3d expression. For no rotation use Matrix3dIdentity.

Remarks

ProjectPointOnPerpendicular always generates a projection that is perpendicular to the element. ProjectPointOnPerpendicular1 lets the program specify a bias point. That is important when there are multiple solutions. ProjectPointOnPerpendicular2 always generates a projection that intersects the element. ProjectPointOnPerpendicular2 projects to the end point of the element if the elements do no intersect.
