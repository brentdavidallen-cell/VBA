RasterEnumerator Object

                                                     

A RasterEnumerator is a CollectionEnumerator specialized to work exclusively with sets of Raster.

Remarks

The following is an example of a RasterEnumerator: 


    Dim oEnum As RasterEnumerator

    Set oEnum = RasterManager.Rasters.GetSelectedRasters

    While oEnum.MoveNext
        MsgBox oEnum.Current.RasterInformation.FileName
    Wend
