Vector3dFromXYAngleAndMagnitude Method

                          

Sets a vector from given angle and distance in xy plane. Z part is set to zero.

Syntax

Vector3d = object.Vector3dFromXYAngleAndMagnitude (Theta, Magnitude)

The Vector3dFromXYAngleAndMagnitude method syntax has these parts:

Part Description
object A valid object.
Theta A Double expression. Angle from X axis to the vector, in the xy plane.
Magnitude A Double expression. Vector magitude
