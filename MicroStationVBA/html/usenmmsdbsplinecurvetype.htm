MsdBsplineCurveType Enum

        

Defines constants that represent special types of B-spline curves, as stored in the read-only property BsplineCurve.CurveType.

The MsdBsplineCurveType enum has these members:

Name Description
msdBsplineCurveGeneral 0 (&H0) Represents an arbitrary B-spline curve.
msdBsplineCurveLine 1 (&H1) Represents a B-spline curve composed entirely of line segments.
msdBsplineCurveCircularArc 2 (&H2) Represents part of a circle.
msdBsplineCurveCircle 3 (&H3) Represents a circle.
msdBsplineCurveEllipticalArc 4 (&H4) Represents part of a noncircular ellipse.
msdBsplineCurveEllipse 5 (&H5) Represents a noncircular ellipse.
msdBsplineCurveParabolicArc 6 (&H6) Represents part of a parabola.
msdBsplineCurveHyperbolicArc 7 (&H7) Represents part of a hyperbola.

Remarks

These values are generally not used and are preserved for historical reasons.
