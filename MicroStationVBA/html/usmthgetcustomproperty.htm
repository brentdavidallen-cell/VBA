GetCustomProperty Method

                          

Gets the value of a custom file property.

Syntax

VARIANT = object.GetCustomProperty (Name)

The GetCustomProperty method syntax has these parts:

Part Description
object A valid object.
Name A String expression.

Remarks

GetCustomProperty extracts the data type from what is in the file and stores that in the variant value it returns. Then Visual Basic (or some other Automation client) can perform any necessary conversions such as converting a number or date to a string, or converting a string to a number or date.

GetCustomProperty raises an error if the property is not defined. Use CustomPropertyExists to determine if the property exists.

You can also examine or modify custom properties using a tool like Windows Explorer.
