RasterCompressionModes Object

                                                     

A collection of RasterCompressionMode. It gives access to all compression modes available for a raster file format's color mode. It is used like this: 


    'With a variable:
    Dim oRasterColorMode As RasterColorMode
    Dim theRasterCompressionModes As RasterCompressionModes

        ...

    Set theRasterCompressionModes = oRasterColorModes.CompressionModes
    theRasterColorModes.[method/property]

    'Without a variable:
    Dim oRasterColorMode As RasterColorMode

    ...

    aRasterColorMode.RasterCompressionModes.[method/property]

Remarks

It is possible to step through the collection using the For Each statement as follows: 


    Dim oRasterColorMode As RasterColorMode
    Dim oRasterCompressionMode As RasterCompressionMode
    Dim theRasterCompressionModes As RasterCompressionModes

    ...

    Set theRasterCompressionModes = oRasterColorMode.CompressionModes

    For Each oRasterCompressionMode In theRasterCompressionModes
        oRasterCompressionMode.[method/property]
    Next
