ConvertToSmartSolidElement Method

                          

Used to convert normal element to SmartSolidElement.

Syntax

Set ElementEnumerator = object.ConvertToSmartSolidElement (elem)

The ConvertToSmartSolidElement method syntax has these parts:

Part Description
object A valid object.
elem An Element expression. the input element.

Remarks

Not all type of element can be converted to SmartSolidElement.
