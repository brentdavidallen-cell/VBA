CreateComplexShapeElement4 Method

                          

Gets the CreateComplexShapeElement4 object for Array of Elements which is passed as VARIANT

Syntax

Set ComplexShapeElement = object.CreateComplexShapeElement4 (ChainableElements [, FillMode [, GapTolerance]])

The CreateComplexShapeElement4 method syntax has these parts:

Part Description
object A valid object.
ChainableElements A VARIANT expression.
FillMode Optional. An MsdFillMode expression.
GapTolerance Optional. A Double expression.
