AreaPattern Object

                                                     

An AreaPattern object holds parameters needed to perform area patterning.

Remarks

Use GetPattern to retrieve the patterning parameters of an element.

Use SetPattern to set the patterning parameters of an element.

Use HasPattern to determine if an element has a pattern.

Use RemovePattern to remove an element's pattern.

Use CreateAreaPattern to create a new AreaPattern object.

Only closed elements can have patterns.
