Vector3d Type

        

Represents the Cartesian coordinates of a vector in three-dimensional space.

The Vector3d type has these members:

Name Description
X A double value that represents the X-axis component of the vector.
Y A double value that represents the Y-axis component of the vector.
Z A double value that represents the Z-axis component of the vector.

Remarks

  • Direct constructions
    • Return a vector with given X, Y and Z components: Vector3dFromXYZ.
    • Return a vector with given X and Y components, with Z component of zero: Vector3dFromXY.
    • Return a vector with X, Y, Z components copied from a Point3d: Vector3dFromPoint3d.
    • Return a vector with X, Y, and Z all zeros: Vector3dZero.
    • Return a vector with X, Y, and Z all ones: Vector3dOne.
    • Return an X, Y, or Z component of a vector, selected by an integer 0, 1, or 2: Vector3dGetComponent.
  • Magnitude and distance queries
    • Return the magnitude of a vector: Vector3dMagnitude.
    • Get the squared magnitude of a vector: Vector3dMagnitudeSquared.
    • Return the magnitude of the XY parts of a vector: Vector3dMagnitudeXY.
    • Return the squared magnitude of the XY parts of a vector: Vector3dMagnitudeSquaredXY.
    • Return the distance between two vectors (considering them both to be based at a common point): Vector3dDistance.
    • Return the squared distance between two vectors: Vector3dDistanceSquared.
    • Return the squared distance between the XY parts of two vectors: Vector3dDistanceSquaredXY.
    • Return the distance between the XY parts of two vectors: Vector3dDistanceXY.
    • Return the largest absolute value among the components of a vector: Vector3dMaxAbs.
    • Return the largest absolute value among the components of the difference of two vectors: Vector3dMaxAbsDifference.
  • Addition, subtraction, and scaling
    • Interpolate between two vectors: Vector3dInterpolate.
    • Add two vectors: Vector3dAdd.
    • Add a vector and a scaled second vector: Vector3dAddScaled.
    • Add a vector and two scaled vectors: Vector3dAdd2Scaled.
    • Add a vector and three scaled vectors: Vector3dAdd3Scaled.
    • Subtract two vectors: Vector3dSubtract.
    • Subtract two points (i.e. get the vector from second to first): Vector3dSubtractPoint3dPoint3d.
    • Return a scaled vector: Vector3dScale.
    • Return a negated vector: Vector3dNegate.
    • Return a unit vector in given direction: Vector3dNormalize.
    • Return a unit vector perpendicular to the XY parts of a given vector: Vector3dUnitPerpendicularXY.
  • Vector3d and Matrix3d
    • Return a vector from a matrix times a vector which is expressed as its XYZ components: Vector3dFromMatrix3dTimesXYZ.
    • Return a vector from a matrix times a vector: Vector3dFromMatrix3dTimesVector3d.
    • Return a vector from a matrix transpose times a vector which is expressed as its XYZ components: Vector3dFromMatrix3dTransposeTimesXYZ
    • Return a vector from a matrix transpose times a vector: Vector3dFromMatrix3dTransposeTimesVector3d.
    • Return a vector from a column of a Matrix3d: Vector3dFromMatrix3dColumn.
    • Return a vector from a row of a Matrix3d: Vector3dFromMatrix3dRow.
  • Perpendicular, parallel, and equality tests
    • Test if two vectors are parallel: Vector3dAreVectorsParallel.
    • Test if two vectors are perpendicular: Vector3dAreVectorsPerpendicular.
  • Vector products
    • Return the (scalar) dot product of two vectors: Vector3dDotProduct.
    • Return the (scalar) dot product of the XY parts of two vectors: Vector3dDotProductXY.
    • Return the (scalar) dot product of a Vector3d and a second vector given as X,Y, and Z values: Vector3dDotProductXYZ.
    • Return the (vector) cross product of two vectors: Vector3dCrossProduct.
    • Return the (vector) cross product of vectors from a base point to each of two target points: Vector3dCrossProduct3Points.
    • Return the (scalar) cross product of the XY parts of two vectors: Vector3dCrossProductXY.
    • Return the (scalar) triple product of three points: Vector3dTripleProduct.
  • Angles
    • Return the angle between two vectors: Vector3dAngleBetweenVectors.
    • Return the smaller of two angles between unoriented vectors: Vector3dSmallerAngleBetweenUnorientedVectors.
    • Return the counterclockwise angle between vectors, using only XY parts: Vector3dAngleBetweenVectorsXY.
    • Return the smaller of two angles between XY parts of unoriented vectors: Vector3dSmallerAngleBetweenUnorientedVectorsXY.
    • Return a vector by rotating only the XY parts of a given vector: Vector3dRotateXY.
    • Return the angle between two vectors, using a third vector to point to the positive side of the plane defined by the two vectors: Vector3dSignedAngleBetweenVectors.
    • Test if a vector lies between two other vectors, considering the smaller of two possible sweeps from first bounding vector to second bounding vector: Vector3dIsVectorInSmallerSector.
    • Test if a vector lies between two other vectors, considering only XY parts and counterclockwise sweep from first bounding vector to second bounding vector: Vector3dIsVectorInCCWSector.
    • Return the angle between projections of two vectors to a plane with given normal: Vector3dPlanarAngleBetweenVectors.
    • Return the angle from the X axis to the XY parts of a vector: Vector3dPolarAngle.
  • Equality tests
    • Test for exact (bitwise) equality between components of two vectors: Vector3dEqual.
    • Test : Vector3dEqualTolerance.
