ChamferEdge Method

                          

Chamfer one or more edges of a solid.

Syntax

Set SmartSolidElement = object.ChamferEdge (solid, closestPoint, distance1, distance2, smoothEdges)

The ChamferEdge method syntax has these parts:

Part Description
object A valid object.
solid A SmartSolidElement expression. the solid which edge is belonging to. .
closestPoint A Point3d expression. input point that lie on the face.
distance1 A Double expression. the first chamfer range.
distance2 A Double expression. the second chamfer range.
smoothEdges A Boolean expression. If TRUE, edges that are tangentially continuous are blended in one operation.
