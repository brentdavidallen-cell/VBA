CopyDesignFile Method

                          

Creates a design file by making a copy of an existing design file.

Syntax

object.CopyDesignFile ExistingDesignFileName, NewDesignFileName [, Overwrite]

The CopyDesignFile method syntax has these parts:

Part Description
object A valid object.
ExistingDesignFileName A String expression. The name of the design file from which the copy is made. If ExistingDesignFileName does not include a path, CopyDesignFile searches for this file in the standard location for design files. If ExistingDesignFileName does not include an extension, CopyDesignFile presumes that the extension is ".dgn".
NewDesignFileName A String expression. The name of the new design file. If NewDesignFileName does not include a path, MicroStation places the new file into its standard location for new design files. If NewDesignFileName does not include an extension, the new file's extension is ".dgn".
Overwrite Optional. A Boolean expression. Relevant only if the new design file already exists. How CopyDesignFile behaves in this case. If True, the new design file is overwritten with the copy of the existing design file. If False, CopyDesignFile raises an error.

Remarks

If, instead, you want to create a design file from scratch, use CreateDesignFile.

If you merely want to open an existing design file, use OpenDesignFile.
