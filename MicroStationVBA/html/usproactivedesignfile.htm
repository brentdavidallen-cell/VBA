ActiveDesignFile Property

                          

Read-only DesignFile.

Gets the design file that contains the active model.

Syntax

object.ActiveDesignFile

The ActiveDesignFile property syntax has these parts:

Part Description
object A valid object.

Remarks

Equivalent to ActiveModelReference.DesignFile.

An error is raised if you try to read this value when no design file is active. If you're in doubt, firstly check the value of HasActiveDesignFile.
