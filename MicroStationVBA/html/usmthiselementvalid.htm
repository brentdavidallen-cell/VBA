IsElementValid Method

                          

Returns a value indicating whether the given location contains a valid (non-deleted) element.

Syntax

Boolean = object.IsElementValid (Index)

The IsElementValid method syntax has these parts:

Part Description
object A valid object.
Index A Long expression.

Remarks

When an element is deleted, it continues to occupy a slot in the cache until the design file is closed.
