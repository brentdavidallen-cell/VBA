RemoveBoundAt Method

                          

Removes the boundary at the given valid index from this object.

Syntax

object.RemoveBoundAt Index

The RemoveBoundAt method syntax has these parts:

Part Description
object A valid object.
Index A Long expression. For BsplineSurface objects, the zero-based index (between 0 and BoundsCount-1, inclusive).

Remarks

For BsplineSurface objects, see the Bound property for further description of boundaries.
