CurrentPointRaw Property

                          

Read-only Point3d.

Gets the current location of the cursor expressed in master units.

Syntax

object.CurrentPointRaw

The CurrentPointRaw property syntax has these parts:

Part Description
object A valid object.

Remarks

Use DataPointRaw to get the location of the cursor at the time of the last data point.

Use DataPointAdjusted to get the location of the last data point adjusted for snaps.

Returns (0,0,0) if the cursor is not in a View. Use IsCurrentlyInView to test whether the cursor is currently in a View.
