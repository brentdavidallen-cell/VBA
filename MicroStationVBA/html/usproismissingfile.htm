IsMissingFile Property

                          

Read-only Boolean.

Syntax

object.IsMissingFile

The IsMissingFile property syntax has these parts:

Part Description
object A valid object.

Remarks

IsMissingFile is True if MicroStation could not find the file that contains the model to be attached.

If the file is missing, many of the methods and properties of Attachment raise errors. However, Reattach, LogicalDescription, LogicalName, AttachName, and Name work regardless of whether the file is available.
