DLong Type

        

Represents a 64-bit integer.

The DLong type has these members:

Name Description
Low A Long expression
High A Long expression

Remarks

VBA and Visual Basic 6 do not have a native 64-bit integer type. (An Integer is 16 bits and a Long is 32 bits.) Since some of the MicroStation features, such as element IDs, require 64-bit integers, the DLong type serves this purpose.

There are a number of global functions for managing DLong values:

  • Converts a DLong to a Long: DLongToLong.
  • Converts a Long to a DLong: DLongFromLong.
  • Converts a DLong to a String: DLongToString.
  • Converts a String to a DLong: DLongFromString.
  • Converts a DLong to a String in hexadecimal format: DLongToHexString.
  • Converts a String in hexadecimal format to a DLong: DLongFromHexString.
  • Converts a Double to a DLong: DLongFromDouble.
  • Compares two DLongs: DLongComp.
  • Calculates the sum of two DLongs: DLongAdd.
  • Calculates the difference between two DLongs: DLongSubtract.
  • Calculates the product of two DLongs: DLongMultiply.
  • Calculates the quotient of two DLongs: DLongDivide.
  • Calculates the modulus (remainder after division) of two DLongs: DLongMod.
  • Reverses the sign of a DLong: DLongNegate.
  • Calculates the absolute value of a DLong: DLongAbs.

The following functions cannot be used in VBA or Visual Basic 6:

  • Converts a DLong to a 64-bit integer: DLongToInt64.
  • Converts a 64-bit integer to a DLong: DLongFromInt64.
