AddedCheckerToStandardsCheckerApps Method

                          

Called after the Standards Checker adds a description of the standards checker report. When the StandardsCheckerController wants a standard checker to perform its checks, the StandardsCheckerController first adds a description to the report, then it calls the checker's AddedCheckerToStandardsCheckerApps method to let it add more detailed information to the report. Then it calls the checker's RunCheck method.

Syntax

object.AddedCheckerToStandardsCheckerApps ApplicationXMLNode

The AddedCheckerToStandardsCheckerApps method syntax has these parts:

Part Description
object A valid object.
ApplicationXMLNode An Object expression.

Remarks

If the user does not request a report, then the StandardsCheckerController does not call the checker's AddedCheckerToStandardsCheckerApps method.

The StandardsCheckerController adds the minimum amount of information to the report. The AddedCheckerToStandardsCheckerApps may use AddLibraryToCheckerApp, AddStandardToLibrary, and AddPropertyToStandard to add detailed information describing the standards checker.

The following XML sample shows the StandardsCheckerApp XML code that is added to the report adds to the XML file. The StandardsCheckerController adds the first line prior to calling AddedCheckerToStandardsCheckerApps. AddedCheckerToStandardsCheckerApps has to call AddStandardToLibrary and AddPropertyToStandard to add the other lines to the report. 

<StandardsCheckerApp name="VBA ByLevel Checker" alias="bentley.com.ByLevel" version="Version 1.0.0.0" description="Verifies that elements use ByLevel symbology">
   <Libraries>
      <Library id="1" name="d:\ustation\dgn\testsc2d.dgnlib" time="2003/10/29 09:38:23" size="36864" format="DGNV8" lastauthor="John.Gooding">
         <Standards>
            <Standard name="Line" type="Element">
               <Properties>
                  <Property name="Color" value="ByLevel"/>
                  <Property name="Style" value="ByLevel"/>
               </Properties>
            </Standard>
            <Standard name="LineString" type="Element">
               <Properties>
                  <Property name="Color" value="ByLevel"/>
               </Properties>
            </Standard>
            <Standard name="Shape" type="Element">
               <Properties>
                  <Property name="Color" value="ByLevel"/>
                  <Property name="Weight" value="ByLevel"/>
                  <Property name="Style" value="ByLevel"/>
               </Properties>
            </Standard>
         </Standards>
      </Library>
   </Libraries>
</StandardsCheckerApp>
