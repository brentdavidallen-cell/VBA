RasterFormat Object

                                                     

This is a raster file format.

Remarks

Use the RasterFormats object to get a RasterFormat object as follows: 


    'Instantiation with an Item :

    Dim oFormat As RasterFormat

    Set oFormat = RasterManager.RasterFormats.Item(1)


    'Instantiation with Find:

    Dim oFormat As Raster

    Set oFormat = RasterManager.RasterFormats.Find("TI~FF")
