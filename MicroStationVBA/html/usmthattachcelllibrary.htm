AttachCellLibrary Method

                          

Attaches the named file as a cell library.

Syntax

object.AttachCellLibrary CellLibraryName [, ConvertFromV7]

The AttachCellLibrary method syntax has these parts:

Part Description
object A valid object.
CellLibraryName A String expression.
ConvertFromV7 Optional. An MsdConversionMode expression.

Remarks

The active cell does not have to come from the attached cell library. When adding cells, MicroStation searches the attached cell library plus all of the cell libraries named by the MS_CELLLIST and MS_BLOCKLIST configuration variables.

A program cannot save a cell definition unless there is an attached cell library. MicroStation saves new definition in the attached cell library.

MicroStation does not permit attaching a cell library when the active model is read-only.

The CellLibraryName may be a full file path, or just part of the name. If it does not contain a file extension, MicroStation searches for files with the ".cel" extension. AttachCellLibrary searches all of the directories specified by the MS_CELL configuration variable.

If there is already an attached cell library, AttachCellLibrary detaches it. It is not necessary to detach a cell library prior to calling AttachCellLibrary.

The specified cell library may be a MicroStation V7 cell library, a MicroStation V8 DGN file, or a .dwg file.
