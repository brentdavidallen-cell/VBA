EditSettings Method

                          

Tells the Standards Checker application to display a dialog box to let the user view or edit the settings for that specific handler.

Syntax

object.EditSettings IsReadOnly

The EditSettings method syntax has these parts:

Part Description
object A valid object.
IsReadOnly A Boolean expression.

Remarks

If the standards checker's property HasSettings returns False then the StandardsCheckerController will not call the standards checker's EditSettings method.
