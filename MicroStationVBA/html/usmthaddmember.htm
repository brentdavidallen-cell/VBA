AddMember Method

                          

Syntax

Set NamedGroupMember = object.AddMember (NewMember)

The AddMember method syntax has these parts:

Part Description
object A valid object.
NewMember An Element expression. Must be a graphical element (IsGraphical) or a NamedGroupElement.

Remarks

This method raises an error if the NewMember element has never been written to a file. AddMember does not let a program add an element that is not part of a model.

This method raises an error if the group is closed, if adding the member would create a cycle, or if the element is in one model, the named group is in another model, and far references are not allowed.

The new member gets the NamedGroupElement's default member propagation flags.

Whenever you change any aspect of a NamedGroupElement, you should subsequently call Rewrite to make the change permanent.
