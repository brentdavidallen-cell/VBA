SubstituteText Method

                          

Specifies an alternate text string to be printed. Valid only for text and tag elements.

Syntax

object.SubstituteText newString

The SubstituteText method syntax has these parts:

Part Description
object A valid object.
newString A String expression.

Remarks

This is the preferred technique for print text substitution, as it does not requiring cloning the original element.
