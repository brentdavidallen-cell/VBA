IsHidden Property

                          

Read/Write Boolean.

Gets/Sets whether the object is hidden or displayed.

Syntax

object.IsHidden [= Boolean]

The IsHidden property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

Any graphical element can be hidden by setting the IsHidden property to True.

IsHidden of TagDefinition determines whether the IsHidden property of a TagElement created from that TagDefinition is displayed. MicroStation's Attach Tags command lets the user override this property.

When dealing with Model and Attachment Objects the results will be slightly different. When the Model isHidden the Model name is no longer visible in the Models dialog. When the Attachment isHidden the Display attribute is turned off and the Attachment Elements are no longer visible.
