Changing Logical Name and Logical Description

This example shows how to change the LogicalName and LogicalDescription of an attachment. 


Sub ChangeAttachment()
    Dim att As Attachment

    Set att = ActiveModelReference.Attachments(1)
    att.LogicalDescription = "Update at " & Time
    att.LogicalName = "NewName"
    '  The rewrite saves it to the file and updates the References
    '  dialog box.
    att.Rewrite
End Sub
