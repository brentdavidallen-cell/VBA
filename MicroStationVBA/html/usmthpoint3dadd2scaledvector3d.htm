Point3dAdd2ScaledVector3d Method

                          

Adds an origin and two scaled vectors.

Syntax

Point3d = object.Point3dAdd2ScaledVector3d (Origin, Vector1, Scale1, Vector2, Scale2)

The Point3dAdd2ScaledVector3d method syntax has these parts:

Part Description
object A valid object.
Origin A Point3d expression. The origin. May be null.
Vector1 A Vector3d expression. The first direction vector
Scale1 A Double expression. The first scale factor
Vector2 A Vector3d expression. The second direction vector
Scale2 A Double expression. The second scale factor
