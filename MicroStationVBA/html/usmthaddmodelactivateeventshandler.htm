AddModelActivateEventsHandler Method

                          

Adds a model-activate-events event handler to the set of active handlers.

Syntax

object.AddModelActivateEventsHandler EventHandler

The AddModelActivateEventsHandler method syntax has these parts:

Part Description
object A valid object.
EventHandler An IModelActivateEvents expression.

Remarks

The event handler remains active until it is terminated with RemoveModelActivateEventsHandler, or the object becomes invalid. If the event handler is implemented in VBA, the object becomes invalid if the project is reset manually or due to an error.
