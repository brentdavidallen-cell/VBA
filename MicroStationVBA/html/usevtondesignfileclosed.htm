OnDesignFileClosed Event

                          

The OnDesignFileClosed event occurs when a design file is closed.

Syntax

object.OnDesignFileClosed DesignFileName

The OnDesignFileClosed method syntax has these parts:

Part Description
object A valid object.
DesignFileName A String expression. The name of the file that was closed.

Remarks

See OnDesignFileOpened for an example that shows how to set up OnDesignFileOpened and OnDesignFileClosed event handlers.
