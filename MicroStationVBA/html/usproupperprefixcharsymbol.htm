UpperPrefixCharSymbol Property

                          

Read/Write String.

Gets this object's upper prefix character symbol.

Syntax

object.UpperPrefixCharSymbol [= Char]

The UpperPrefixCharSymbol property syntax has these parts:

Part Description
object A valid object.
Char A String expression.

Remarks

This is used only for dual dimensions.
