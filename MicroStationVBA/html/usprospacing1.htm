Spacing1 Property

                          

Read/Write Double.

Gets or sets this object's primary spacing.

Syntax

object.Spacing1 [= Double]

The Spacing1 property syntax has these parts:

Part Description
object A valid object.
Double A Double expression.

Remarks

Depending on the object type, its "primary" spacing may be its only spacing.

For AreaPattern, HatchPattern and CrossHatchPattern, Spacing1 is the distance between pattern components in the X direction of a coordinate system rotated by Angle1.
