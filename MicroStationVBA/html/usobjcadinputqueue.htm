CadInputQueue Object

                                                     

The CadInputQueue object is used to send and receive MicroStation key-ins, commands, and resets. A program can send input to MicroStation using SendCommand, SendDataPoint, SendDragPoints, SendKeyin, SendLastInput, SendReset, or SendTentativePoint. A program can retrieve input from MicroStation using GetInput.

It is possible to use the methods of CadInputQueue to implement procedures that sequence MicroStation input, but the best way to implement a command that behaves like standard MicroStation commands is to implement a class that implements either IPrimitiveCommandEvents or ILocateCommandEvents. Use IPrimitiveCommandEvents if the new command is an element-creation command. Use IPrimitiveLocateCommand if the new command is an element-modification command.

MicroStation's macro recorder records key-ins, datapoints, etc. to be replayed using the methods of CadInputQueue.
