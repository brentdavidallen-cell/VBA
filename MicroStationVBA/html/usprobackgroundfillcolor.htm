BackgroundFillColor Property

                          

Read/Write Long.

Gets or sets the fill color of this object's background.

Syntax

object.BackgroundFillColor [= Long]

The BackgroundFillColor property syntax has these parts:

Part Description
object A valid object.
Long A Long expression.

Remarks

Note that the background fill color is visible in a View only if (1) this object's BorderAndBackgroundVisible is True, and (2) the View's DisplaysFill property is True.
