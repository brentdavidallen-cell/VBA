EvaluatePointsAtFixedLengthChords Method

                          

Returns points (and parameters) which determine chords of this planar object with fixed length.

Syntax

Point3d () = object.EvaluatePointsAtFixedLengthChords (Parameters, ChordLength [, Tolerance])

The EvaluatePointsAtFixedLengthChords method syntax has these parts:

Part Description
object A valid object.
Parameters A Double () expression. The returned parameters associated with the returned points.
ChordLength A Double expression. The length of the chords to compute (last chord may be smaller).
Tolerance Optional. A Double expression. The curve-curve intersection tolerance. If omitted or negative, a default tolerance of 1.0e-8, or the largest absolute Pole coordinate times 1.0e-8 (whichever is larger), is used.

Remarks

For nonplanar BsplineCurve objects, the stroke parameters are computed using a planar approximating curve, and the points are mapped back to the input curve. This method works best with planar input.
