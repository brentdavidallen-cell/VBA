Vector3dDotProduct Method

                          

Returns the (scalar) dot product of two vectors.

Syntax

Double = object.Vector3dDotProduct (Vector1, Vector2)

The Vector3dDotProduct method syntax has these parts:

Part Description
object A valid object.
Vector1 A Vector3d expression. The first vector
Vector2 A Vector3d expression. The second vector
