UnweightPoles Method

                          

Scale the array of poles in place by the reciprocals of the values in the array of weights.

Syntax

object.UnweightPoles Poles, Weights

The UnweightPoles method syntax has these parts:

Part Description
object A valid object.
Poles A Point3d () expression. The 1- or 2-dimensional array of poles to unweight.
Weights A Double () expression. The 1- or 2-dimensional array of weights.

Remarks

Dimensions and counts of input arrays must match.

This method is useful for implementing algorithms involving Rational B-spline curves/surfaces, where it is often more efficient to process weighted (homogeneous) poles. Use this method to return to unweighted (Cartesian) poles.
