EvaluatePointDerivatives1 Method

                          

Returns the point, first partial derivatives, and the normal at the given u- and v-parameters along this object.

Syntax

Point3d = object.EvaluatePointDerivatives1 (Tangents, Normal, Parameters)

The EvaluatePointDerivatives1 method syntax has these parts:

Part Description
object A valid object.
Tangents A FirstPartials3d expression. The returned first partial derivatives with respect to u and v at the point.
Normal A Point3d expression. The returned (normalized) normal at the point.
Parameters A Point2d expression. The valid u- and v-parameters of the desired point.

Remarks

For BsplineSurface objects, any boundaries in the object are ignored. To detect whether or not Parameters defines a point on a trimmed/punctured surface, call ContainsParametricPoint.
