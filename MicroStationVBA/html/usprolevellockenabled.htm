LevelLockEnabled Property

                          

Read/Write Boolean.

Gets or sets a value indicating whether the lock lock is turned on.

Syntax

object.LevelLockEnabled [= Boolean]

The LevelLockEnabled property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

When this value is True, the user can locate for manipulation only elements on the active level.
