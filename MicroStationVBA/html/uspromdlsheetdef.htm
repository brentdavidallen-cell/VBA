MdlSheetDef Property

                          

Read-only Int64.

Retrieves the associated SheetDef pointer that a program can use as an argument to MDL mdlSheetDef functions.

Syntax

object.MdlSheetDef

The MdlSheetDef property syntax has these parts:

Part Description
object A valid object.

Remarks

Most VB/VBA programs will not use this method. This is intended only for programs that use the MDL API. See Calling DLL Functions for more information.
