Color Property

                          

Read/Write Long.

Gets or sets this object's color.

Syntax

object.Color [= Long]

The Color property syntax has these parts:

Part Description
object A valid object.
Long A Long expression.

Remarks

The color is represented as an index into MicroStation's ColorTable.

There is a predefined Long constant, ByLevelColor, that represents the special color value, "by level." If the value of Color is ByLevelColor, then the element's color is derived from the level that the element resides on.

There is also a predefined Long constant, ByCellColor, that represents the special color value, "by cell." If the value of Color is ByCellColor, then the element's color is derived from the cell that the element resides within.

For an element, getting or setting the value of Color raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.

For a TextStyle object, setting this value also sets UsesColor to True.
