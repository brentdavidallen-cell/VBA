HasItems Method

                          

Searches DesignFile/ModelReference/Element for a ItemTypePropertyHandler whose ItemTypelibrary name and Itemtype name matches a given pattern. If found, returns True ; else returns False.

Syntax

Boolean = object.HasItems (Libname [, ItemTypename])

The HasItems method syntax has these parts:

Part Description
object A valid object.
Libname A String expression. The name of the ItemTypeLibrary(s) to find ItemTypePropertyHandler of its Items. This name follows the same rules as a pattern given to Visual Basic's Like operator. (Exception: The pattern match is always case-insensitive, regardless of the Option Compare setting.)
ItemTypename Optional. A String expression. name of ItemType(s) to find ItemTypePropertyHandler for its Items, This name follows the same rules as a pattern given to Visual Basic's Like operator. (Exception: The pattern match is always case-insensitive, regardless of the Option Compare setting.)

Remarks

You can Check one specific attached Items by simply giving ItemTypeLibrary name and Itemtype name, without any wildcard characters.

You can successively check all attached Items by giving the ItemTypeLibrary name and Itemtype name , "*".
