CapMode Property

                          

Read/Write Integer.

Gets or sets the active cap mode.

Syntax

object.CapMode [= Integer]

The CapMode property syntax has these parts:

Part Description
object A valid object.
Integer An Integer expression.

Remarks

If this value is 0, surfaces are uncapped (treated as open surfaces). If this value is 1, surfaces are capped (treated as solids).
