SendAdjustedDataPoint Method

                          

Sends a data point to MicroStation. If there is an outstanding tentative point or snap at the time of the call, it first replaces the value of the DataPoint argument with the point established by the tentative point or snap. DataPoint does not return until MicroStation has processed the datapoint.

Syntax

object.SendAdjustedDataPoint DataPoint [, ViewSpecifier [, Qualifier]]

The SendAdjustedDataPoint method syntax has these parts:

Part Description
object A valid object.
ViewSpecifier Optional. A VARIANT expression. May be either a view index (1 through 8) or a View object.
DataPoint A Point3d expression.
Qualifier Optional. A Long expression.

Remarks

Sending a datapoint this way simulates a user entering a datapoint in one of MicroStation's views.

When there is an active tentative point, SendAdjustedDataPoint sends the point selected by the tentative point snap. When there is no active tentative point, SendAdjustedDataPoint sends the point that is passed in as the argument.

In most cases, a program is either sending an exact point or is waiting for a data point from the user and then sending that data point to MicroStation. SendDataPoint is sufficient in both those cases. SendAdjustedDataPoint is only needed when a program sends a tentative point using SendTentativePoint, and then sends a data point whose value is to be determined by the tentative point.

This method was first added in 08.09.04.73. It is not available in earlier versions of 08.09.04.
