DLongToLong Method

                          

Returns the Long representation of a DLong expression.

Syntax

Long = object.DLongToLong (Value)

The DLongToLong method syntax has these parts:

Part Description
object A valid object.
Value A DLong expression.

Remarks

Example: 

    Dim N As DLong
    Dim I As Long
    N = DLongFromString("123456789")
                    ' Now N contains 123456789 (as a 64-bit value).
    I = DLongToLong(N)
                    ' Now I contains 123456789 (as a Long value).

If the DLong expression exceeds the range of a Long (2147483647), an overflow error (Error 6) occurs.
