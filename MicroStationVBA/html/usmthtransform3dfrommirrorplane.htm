Transform3dFromMirrorPlane Method

                          

Returns a transformation which mirrors about a plane

Syntax

Transform3d = object.Transform3dFromMirrorPlane (Origin, Normal)

The Transform3dFromMirrorPlane method syntax has these parts:

Part Description
object A valid object.
Origin A Point3d expression. Any point on the plane.
Normal A Point3d expression. Vector perpendicular to the plane.

Remarks

If the normal vector is all zeros, an identity transform is returned.
