Transform3dFromMatrix3dAndFixedPoint3d Method

                          

Returns a transformation with given matrix part, and translation part computed from the matrix and a given fixed point. This translation part is generally different from the fixed point itself. The resulting transformation will leave the fixed point unchanged and apply whatever effects are contained in the matrix as if the fixed point is the origin.

Syntax

Transform3d = object.Transform3dFromMatrix3dAndFixedPoint3d (Matrix, Origin)

The Transform3dFromMatrix3dAndFixedPoint3d method syntax has these parts:

Part Description
object A valid object.
Matrix A Matrix3d expression. The matrix part
Origin A Point3d expression. The point that is to remain fixed when multiplied by the transformation.
