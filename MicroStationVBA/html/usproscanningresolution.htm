ScanningResolution Property

                          

Read/Write Point2d.

Allows to obtain or to adjust the resolution of a raster in dots per inch (DPI).

Syntax

object.ScanningResolution [= Point2d]

The ScanningResolution property syntax has these parts:

Part Description
object A valid object.
Point2d A Point2d expression. The resolution of the raster in dots per inch (DPI).

Remarks

Scanning resolution (X and Y directions). 


    Dim theGeo As GeoReferenceInformation

    Set theGeo = RasterManager.Rasters.Item(1). GeoReferenceInformation
    theGeo.Affinity = 0.7854
    MsgBox "New scanning resolution is: " & theGeo.ScanningResolution
