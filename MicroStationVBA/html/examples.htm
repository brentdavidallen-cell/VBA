Overview of Examples

The Examples table-of-contents lists examples illustrating the most important concepts in using MicroStation's object model. Pertinent examples are also accessible from the Examples lists of the documentation pages of the types, objects, events, methods, and objects. A page's examples list may refer to examples that appear in the Examples table-of-contents. They may also refer to shorter examples that are not mentioned elsewhere.

In addition to the examples in the help file, there are examples in projects distributed with MicroStation. You can find these in the WorkSpace\Example\Standards\Macros directory. Here is a brief description of those examples.

Project Description
CopyElementExample This example implements a simple copy command.
DataBaseExamples This connects to a ODBC database. Then it starts a locate command to allow the user to select an element. Once the user has selected the element, it gets the element's database linkages, retrieves the records corresponding to the linkages, and displays the data in a ListBox. The module modDBCommands has the code for starting the command. The module modDatabaseAccess has some general purpose subroutines and functions for working with databases. The form frmDBConnect has the code for connecting to the database. The form frmDBLinkInfo has the code for extracting the linkages and populating the ListBox. The class clsDBReviewCommand has the implementation of the locate command.
ElementPlacementExample This project has 7 sample commands that show illustrate different techniques for creating elements. The module modPlaceShapeCommands has the entry points for each of the commands.
LineStylePicker This project illustrates a technique for implementing a line style picker.
MdlExamples The examples in this project illustrate how to call into MicroStation's MDL functions. Many of the declarations in this section are copied directly out of the MDL documentation which now includes possible VBA declarations for each of the MDL functions that is available to VBA. See the section "Calling DLL functions from VBA" in this help file for more information.
mdl2vba Illustrates how a VBA macro can use GetCExpressionValue and SetCExpressionValue to access MDL functions and data. This example requires the vba2mdl MDL example.
NativeCodeUtilities This example contains a number of utility procedures that perform tasks that are commonly required by projects that call into native code. Many of the examples in MdlExamples use these utilities.
PictureExample This example shows how to display an element in a form or control. This example uses GetPicture to get the image, and a control's Picture property to display it.
ReadElementExample This example reads all of the TextElement, EllipseElement, and LineElement elements from the active model and changes the color of each one.
TextExamples This example has a command that converts all tags, text elements, and text nodes to upper case or lower case. It also has a command to rotate and scale text elements and text nodes. Both commands have the ability to process all of the elements in a fence, all of the selected elements, or all of the elements in the model. The module modTextCommands has both command entry points plus a function GetElementEnumerator that determines whether the command processes the elements from a selection set, a fence, or all of the element in the model. The form frmChangeCase has the code for changing the case of the elements. The form frmTransformText has the code for scaling and rotating text elements.
SmartSolid This example shows how to use SmartSolid API for VBA. We provide a new interface: SmartSolid. The interface contains many solid modeling tools, such as generating primitive solid, boolean operation,etc.And also there is a new element type: SmartSolidElement, which represents Microstation SmartSolid/SmartSurface element.
