SelectByAccessString Method

                          

Selects a property based on the access string.

Syntax

Boolean = object.SelectByAccessString (AccessString)

The SelectByAccessString method syntax has these parts:

Part Description
object A valid object.
AccessString A String expression.

Remarks

A program must select a property before using any of the Get/Set methods.

SelectByAccessString does not raise an error if it fails to find the property. Instead, it just returns False. The Get/Set methods all raise an error if no property is selected.
