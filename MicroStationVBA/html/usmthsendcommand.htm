SendCommand Method

                          

This statement sends the string to MicroStation's input queue. The string is to be interpreted as a command.

Syntax

object.SendCommand Command [, TreatLikeKeyboardInput]

The SendCommand method syntax has these parts:

Part Description
object A valid object.
Command A String expression. A key-in that invokes a MicroStation command.
TreatLikeKeyboardInput Optional. A Boolean expression. Indicates whether to highlight the corresponding icon in the MicroStation user interface.
