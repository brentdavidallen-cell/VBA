CreateWedge Method

                          

Generate a Wedge Solid at origin with SmartSolidElement Type.

Syntax

Set SmartSolidElement = object.CreateWedge (Template, radius, height, angle)

The CreateWedge method syntax has these parts:

Part Description
object A valid object.
radius A Double expression. radius of the wedge (distance from circular edge to center)
height A Double expression. height of the wedge
angle A Double expression. sweep angle (in degrees) of the wedge
Template An Element expression.
