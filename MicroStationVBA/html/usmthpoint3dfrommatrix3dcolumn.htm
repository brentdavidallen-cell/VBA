Point3dFromMatrix3dColumn Method

                          

Returns a point taken from a column of a matrix.

Syntax

Point3d = object.Point3dFromMatrix3dColumn (Matrix, Col)

The Point3dFromMatrix3dColumn method syntax has these parts:

Part Description
object A valid object.
Matrix A Matrix3d expression. The input matrix
Col A Long expression. The index of column to extract. Column indices are 0, 1, 2.
