SendDragPoints Method

                          

This method sends data point information to the active command as if the user pressed and held the data button while moving the mouse (or pointer). The points are measured in the coordinate system of the active model.

Syntax

object.SendDragPoints DownPoint, UpPoint [, ViewSpecifier [, Qualifier]]

The SendDragPoints method syntax has these parts:

Part Description
object A valid object.
DownPoint A Point3d expression. The coordinates where the data button was pressed down.
UpPoint A Point3d expression. The coordinates where the data button was released.
ViewSpecifier Optional. A VARIANT expression. The view to which the data point information is directed. May be an index (1-8) or a View object.
Qualifier Optional. A Long expression. The qualifiers (if any) associated with the data point information. The Selection Tool makes use of the qualifiers. A value of 1 causes the Selection Tool to toggle the selection of the elements enclosed by the dynamic rectangle. A value of 3 causes the Selection Tool to toggle the selection of the elements enclosed by and overlapping the rectangle.
