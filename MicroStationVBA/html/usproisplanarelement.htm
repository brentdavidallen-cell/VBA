IsPlanarElement Property

                          

Read-only Boolean.

Gets a value indicating whether this object supports the PlanarElement interface.

Syntax

object.IsPlanarElement

The IsPlanarElement property syntax has these parts:

Part Description
object A valid object.

Remarks

If IsPlanarElement is True, AsPlanarElement gets this object's implementation of the PlanarElement interface.
