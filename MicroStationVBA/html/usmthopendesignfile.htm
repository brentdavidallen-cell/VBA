OpenDesignFile Method

                          

Opens an existing design file after closing the currently open design file.

Syntax

Set DesignFile = object.OpenDesignFile (DesignFileName [, ReadOnly [, V7Action]])

The OpenDesignFile method syntax has these parts:

Part Description
object A valid object.
DesignFileName A String expression. The name of the design file to open. If DesignFileName does not include a path, OpenDesignFile searches for the file in the standard location for design files. If DesignFileName does not include an extension, the extension is presumed to be ".dgn".
ReadOnly Optional. A Boolean expression. Indicates whether the file is opened in read-only mode (such that it cannot be modified). The default value is False
V7Action Optional. An MsdV7Action expression. Indicates how a V7 design file will be opened. This parameter, V7Action, is relevant only if the design file is V7 and ReadOnly is False. V7 refers to a file saved in the previous version of MicroStation. The default value is msdV7ActionAskUser.

Remarks

OpenDesignFile returns the newly-opened DesignFile object. This is the same value as ActiveDesignFile.

If an error is raised during this method, the original design file is left open.

If you want to create a design file, rather than opening an existing one, use CreateDesignFile or CopyDesignFile.

Example

    Sub main()
        Dim fileName As String

        fileName = "D:\Bentley\Workspace\Projects\Examples\General\dgn\FlowTags.dgn"

        On Error GoTo errorTrap
        OpenDesignFile fileName, False, msdV7ActionWorkmode
        On Error GoTo 0

        Exit Sub

        errorTrap:
            MsgBox Err.Description
            Resume Next
    End Sub
