GetElements Method

                          

Gets a collection of Element's in this group.

Syntax

Set ElementEnumerator = object.GetElements (Locked [, TraverseType [, NewTraversal]])

The GetElements method syntax has these parts:

Part Description
object A valid object.
Locked A Boolean expression. It is common to pass the value of ActiveSettings.GraphicGroupLockEnabled for this.
TraverseType Optional. An MsdMemberTraverseType expression. Normally msdMemberTraverseManipulate if the elements will be modified; msdMemberTraverseCopy otherwise
NewTraversal Optional. A Boolean expression.

Remarks

Every time MicroStation traverses the named groups, it marks each element that it visits. When MicroStation finds an element that is already marked, it does not process it. Normally, this is used to avoid processing an element more than once in a traversal, but it can be used to exclude a processed element from a subsequent traversal. Passing True for NewTraversal tells GetElements to reset all of the traversal flags before starting, so every element is a candidate to be included in the current traversal.

Element.GetRelatedElements always puts the original element into the set of returned elements, even if Locked is False. NamedGroupElement.GetElements may return an ElementEnumerator with no elements if Locked is False
