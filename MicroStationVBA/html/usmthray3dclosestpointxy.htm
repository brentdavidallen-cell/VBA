Ray3dClosestPointXY Method

                          

Compute the a point closest to a space point and on an unbounded line, using only xy coordinates in projection calculation.

Syntax

object.Ray3dClosestPointXY Ray, SpacePoint, ClosePoint, CloseFraction

The Ray3dClosestPointXY method syntax has these parts:

Part Description
object A valid object.
Ray A Ray3d expression. Ray of the unbounded line.
ClosePoint A Point3d expression. Point on unbounded line
CloseFraction A Double expression. the fractional position of the close point with respect to the ray.
SpacePoint A Point3d expression.
