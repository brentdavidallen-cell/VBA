CopyPoint3dArray Method

                          

Copies an array of Point3d's to or from the DataBlock object.

Syntax

object.CopyPoint3dArray Point3dArray, CopyToDataBlock

The CopyPoint3dArray method syntax has these parts:

Part Description
object A valid object.
Point3dArray A Point3d () expression.
CopyToDataBlock A Boolean expression.

Remarks

CopyPoint3dArray uses the size of the array to determine how many bytes to transfer. CopyPoint3dArray only transfers the contents of the array. It does not save or retrieve the size of the array.

After transferring the data, it increments Offset by the count of bytes transferred.
