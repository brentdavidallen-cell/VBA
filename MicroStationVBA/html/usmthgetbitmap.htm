GetBitmap Method

                          

Gets the bitmap representing the portion of the Raster delimited by the origin and the extent.

Syntax

Set RasterBitmap = object.GetBitmap (BitmapSize, SrcOrigin, SrcExtent)

The GetBitmap method syntax has these parts:

Part Description
object A valid object.
BitmapSize A Point2d expression. Size in pixels of the bitmap to create.
SrcOrigin A Point2d expression. Origin of the portion from which to extract the bitmap. Its coordinates are measured in Raster's pixels.
SrcExtent A Point2d expression. Extent of the portion from which to extract the bitmap. It is measured in Raster's pixels. It does not need to be the same as the BitmapSize.
