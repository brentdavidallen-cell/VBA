Point3dDotProduct3Points Method

                          

Returns the (scalar) dot product of two vectors. The vectors are computed from the Origin to Target1 and Target2.

Syntax

Double = object.Point3dDotProduct3Points (Origin, Target1, Target2)

The Point3dDotProduct3Points method syntax has these parts:

Part Description
object A valid object.
Origin A Point3d expression. The base point for computing vectors.
Target1 A Point3d expression. The target point for the first vector.
Target2 A Point3d expression. The target point for the second vector.
