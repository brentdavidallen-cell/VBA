Point3dInterpolate Method

                          

Computes a point whose position is given by a fractional argument and two endpoints.

Syntax

Point3d = object.Point3dInterpolate (Point0, FractionParameter, Point1)

The Point3dInterpolate method syntax has these parts:

Part Description
object A valid object.
Point0 A Point3d expression. The point corresponding to fractionParameter of 0.
FractionParameter A Double expression. The fractional parametric coordinate. 0.0 is the start of the segment, 1.0 is the end, 0.5 is midpoint.
Point1 A Point3d expression. The point corresponding to fractionParameter of 1.
