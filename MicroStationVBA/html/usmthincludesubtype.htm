IncludeSubtype Method

                          

Include elements of the specified subtype in the forthcoming scan.

Syntax

object.IncludeSubtype Long

The IncludeSubtype method syntax has these parts:

Part Description
object A valid object.
Long A Long expression.

Remarks

By default, elements of every type and subtype are included in the scan. To limit the scan to specific subtypes, first call ExcludeAllSubtypes, followed by a separate call to IncludeSubtype for each type to be included.

Calling this method automatically restricts the scan to only non-graphical type elements.
