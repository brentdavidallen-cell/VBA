SetClipBoundary Method

                          

Sets the current raster clip boundary shape to that of the parameter element.

Syntax

object.SetClipBoundary Element [, System]

The SetClipBoundary method syntax has these parts:

Part Description
object A valid object.
Element An Element expression. Defines the shape of the clip boundary.
System Optional. An MsdReferenceSystem expression. Specifies Elem's coordinate system.
