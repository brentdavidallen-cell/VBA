ExtractIsoparametricCurve Method

                          

Returns an array of B-spline curves that constitute an isoparametric curve at the desired parameter.

Syntax

BsplineCurve () = object.ExtractIsoparametricCurve (ParameterSpans, Parameter, Direction)

The ExtractIsoparametricCurve method syntax has these parts:

Part Description
object A valid object.
ParameterSpans A Point2d () expression. The returned array of parameter spans of the segments of the returned isoparametric curve. Each span consists of a start parameter and an end parameter for that segment. Multiple segments/spans are returned for isoparametric curves that cross surface boundaries (cf. Bound).
Parameter A Double expression. The valid parameter at which to extract the isoparametric curve. Parameter direction is given by Direction.
Direction An MsdBsplineSurfaceDirection expression. The direction of Parameter. If msdBsplineSurfaceU, this method returns an isoparametric curve in the v-parameter direction at u = Parameter, and ParameterSpans are given in the v parameter. If msdBsplineSurfaceV, this method returns an isoparametric curve in the u-parameter direction at v = Parameter, and ParameterSpans are given in the u parameter.

Remarks

For BsplineSurface objects, this method replaces the method ExtractIsoCurve, which was deprecated after MicroStation V8 2004 Edition.

For BsplineSurface objects, the returned arrays do not include (curves with) parameter spans of length less than 1.0e-4.
