BooleanDisjoint Method

                          

This function accepts as input a set of bodies and booleans them.

Syntax

Set ElementEnumerator = object.BooleanDisjoint (target, DistjointSolids, booleanType)

The BooleanDisjoint method syntax has these parts:

Part Description
object A valid object.
target A SmartSolidElement expression. target body to boolean the tool bodies with.
DistjointSolids A SmartSolidElement () expression. tool bodies to boolean with target body .
booleanType A Long expression. boolean operation. This can be one of MODELER_BOOLEAN_unite = 0 - Unites tool bodies with target body MODELER_BOOLEAN_intersect=1 - Intersects target and tool bodies MODELER_BOOLEAN_difference=2 - Subtracts tool bodies from target body .
