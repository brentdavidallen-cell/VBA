Range3d Type

        

Represents an axis-aligned box in three dimensional space.

The Range3d type has these members:

Name Description
Low A Point3d value that represents the corner of the box with the smallest coordinates.
High A Point3d value that represents the corner of the box with the largest coordinates.

Remarks

For points in a ModelReference's geometry, the values are measured in the model's coordinate system, unless stated otherwise.

For a non-null Range3d, each coordinate (X,Y,Z) of Low is less than or equal to the corresponding coordinate of High.

The Point3ds of a null Range3d are initialized by Range3dInit to large magnitude values, with Low greater than High.

There are a number of global utility functions for managing Point3d values:

  • Initializes a range: Range3dInit
  • Returns the squared length of the diagonal of a range: Range3dExtentSquared
  • Returns a range from a point: Range3dFromPoint3d
  • Returns a range from a point specified by its coordinates: Range3dFromXYZ
  • Returns a range from two points: Range3dFromPoint3dPoint3d
  • Returns a range from two points specified by their coordinates: Range3dFromXYZXYZ
  • Returns a range from three points: Range3dFromPoint3dPoint3dPoint3d
  • Returns a range from a range expanded by a margin: Range3dFromRange3dMargin
  • Returns a range from a range extended by a point: Range3dUnionPoint3d
  • Returns a range from a range extended by a point specified by its coordinates: Range3dUnionXYZ
  • Returns a range from the union of two ranges: Range3dUnion
  • Returns a range from the intersection of two ranges: Range3dIntersect
  • Returns a range from a range scaled about its center point: Range3dScaleAboutCenter
  • Tests whether a range has just been initialized: Range3dIsNull
  • Tests whether a range is contained in another range: Range3dIsContainedInRange3d
  • Tests whether a range contains a point: Range3dContainsPoint3d
  • Tests whether a range contains a point specified by its coordinates: Range3dContainsXYZ
  • Tests whether two ranges are exactly equal: Range3dEqual
  • Tests whether two ranges are equal within tolerance: Range3dEqualTolerance
