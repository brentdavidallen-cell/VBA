SpinFace Method

                          

This function spins one face belonging a solid body around the input axis .

Syntax

Set SmartSolidElement = object.SpinFace (solid, closestPointOnFace, origin, axis, angle)

The SpinFace method syntax has these parts:

Part Description
object A valid object.
solid A SmartSolidElement expression. the solid contain input face.
closestPointOnFace A Point3d expression. input point that lie on the face.
origin A Point3d expression. origin of axis about which the face need to be swept .
axis A Vector3d expression. direction of axis .
angle A Double expression. angle of sweep (in radians) .
