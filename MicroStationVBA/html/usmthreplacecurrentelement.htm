ReplaceCurrentElement Method

                          

Replaces the current element with the element that the argument specifies.

Syntax

object.ReplaceCurrentElement Replacement

The ReplaceCurrentElement method syntax has these parts:

Part Description
object A valid object.
Replacement An Element expression.

Remarks

This example shows how to find all of the TextElements in a cell and change their text to upper case: 


Sub UpperAllText(eleCell As CellElement)
    eleCell.Redraw msdDrawingModeErase

    eleCell.ResetElementEnumeration
    Do While eleCell.MoveToNextElement(True)
        Dim eleTemp As Element

        Set eleTemp = eleCell.CopyCurrentElement
        If eleTemp.IsTextElement Then
            With eleTemp.AsTextElement
                .Text = StrConv(.Text, vbUpperCase)
            End With
            eleCell.ReplaceCurrentElement eleTemp
        End If
    Loop
    eleCell.Redraw
    eleCell.Rewrite
End Sub
