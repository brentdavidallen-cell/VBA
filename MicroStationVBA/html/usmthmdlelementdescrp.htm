MdlElementDescrP Method

                          

Retrieves the associated element descriptor (an MDL MSElementDescr). Most VB/VBA programs will not use this method. This is intended only for programs that use the MDL API. See Calling DLL Functions for more information.

Syntax

Int64 = object.MdlElementDescrP [(Detach)]

The MdlElementDescrP method syntax has these parts:

Part Description
object A valid object.
Detach Optional. A Boolean expression. Set to True if the program is taking control of the element descriptor; False to leave the element descriptor under control of the VBA object.

Remarks

MdlMSElementDescrP returns a pointer to dynamically allocated memory. If Detach is True then the program is taking responsiblity for freeing the element descriptor. If it is False, then the element descriptor will be freed as soon as there are no more references to the VBA object used to retrieve the element descriptor.

If a program tries to access any property or method of the element after calling MdlElementDescrP (True), then that method/property will raise an error saying something like "Element has not been initialized".

Use MdlCreateElementFromElementDescrP to create an object model Element from an MSElementDescr.
