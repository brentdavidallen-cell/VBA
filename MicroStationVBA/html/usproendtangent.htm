EndTangent Property

                          

Read/Write Point3d.

Gets or sets the end tangent vector of this object.

Syntax

object.EndTangent [= Point3d]

The EndTangent property syntax has these parts:

Part Description
object A valid object.
Point3d A Point3d expression.

Remarks

For InterpolationCurve objects, this property sets the negative of the tangent direction interpolated at the end of a nonPeriodic curve. If set to the zero vector, then the end tangent is computed from the FitPoints (cf. BesselTangents) but not stored in this property. For Periodic objects, this property returns the zero vector. Nonzero vectors returned by this property are normalized and point into the curve from the last FitPoint.
