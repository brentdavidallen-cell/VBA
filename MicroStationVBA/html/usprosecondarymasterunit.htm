SecondaryMasterUnit Property

                          

Read/Write MeasurementUnit.

Gets this object's secondary master unit.

Syntax

object.SecondaryMasterUnit [= MasterUnit]

The SecondaryMasterUnit property syntax has these parts:

Part Description
object A valid object.
MasterUnit A MeasurementUnit expression.

Remarks

Use of this function is generally not recommended. Instead set the master and sub units together. Master units that are smaller than the sub units are rejected.
