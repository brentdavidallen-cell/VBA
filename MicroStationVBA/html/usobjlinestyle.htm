LineStyle Object

                                                     

Represents an Element's line style.

Remarks

Use DesignFile.LineStyles to extract a collection of all of the line styles available in a design file.

Upgrade Alert

In versions of MicroStation V8 prior to 8.1, element line styles were represented as integers, not as LineStyle objects. For a discussion of upgrading existing VBA code, please refer to the LineStyle property.
