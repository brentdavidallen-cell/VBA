SetColors Method

                          

Sets the RGB values of the color table.

Syntax

object.SetColors Colors

The SetColors method syntax has these parts:

Part Description
object A valid object.
Colors A Long () expression.

Remarks

This only affects the color table object. Use AttachColorTable to apply the changes to MicroStation's color table.

Entry index in the array contains an RGB value of MicroStation color index.
