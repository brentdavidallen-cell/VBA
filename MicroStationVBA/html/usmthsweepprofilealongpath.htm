SweepProfileAlongPath Method

                          

Sweeps an input wire or sheet body along a path.

Syntax

Set SmartSolidElement = object.SweepProfileAlongPath (profile, path)

The SweepProfileAlongPath method syntax has these parts:

Part Description
object A valid object.
profile An Element expression. profile body which is swept along the path. .
path An Element expression. path along which profile is swept.

Remarks

If the input profile is open curve, the method will return a SmartSurface.
