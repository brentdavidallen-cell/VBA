NewTextNodes Property

                          

Sets whether new text node numbers are assigned.

Syntax

object.NewTextNodes [= Boolean]

The NewTextNodes property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

Set to True to cause CopyElement to assign new text node numbers to the new elements; False to retain the existing text node numbers. The default is True when a CopyContext is created.
