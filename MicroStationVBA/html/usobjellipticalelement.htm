EllipticalElement Object

                                                     

Represents the aspects of a MicroStation element that pertain to only those element types that are elliptical (including circular).

Remarks

A program does not explicitly create an EllipticalElement object. EllipticalElement is an interface that some of the element types implement.

You can use IsEllipticalElement to determine if an element is elliptical. You can use AsEllipticalElement to reference an element as an elliptical element. Alternatively, you can access the element as an elliptical element by assigning the Element reference to a variable that is declared as EllipticalElement.
