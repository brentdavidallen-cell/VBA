Vector3dIsVectorInCCWSector Method

                          

Test if the test vector vector is "between" vector0 and vector1, with CCW direction resolved by an up vector. The cross product of vector0 and vector1 is considered the positive plane normal if its dot product with the up vector is positive.

Syntax

Boolean = object.Vector3dIsVectorInCCWSector (TestVector, Vector0, Vector1, UpVector)

The Vector3dIsVectorInCCWSector method syntax has these parts:

Part Description
object A valid object.
TestVector A Vector3d expression. The vector to test
Vector0 A Vector3d expression. The boundary vector.
Vector1 A Vector3d expression. The boundary vector.
UpVector A Vector3d expression. The out of plane vector. If null, the z vector is used.
