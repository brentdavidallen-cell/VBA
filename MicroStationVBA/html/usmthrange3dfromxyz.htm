Range3dFromXYZ Method

                          

Initialize the range from a single point given by components.

Syntax

Range3d = object.Range3dFromXYZ (X, Y, Z)

The Range3dFromXYZ method syntax has these parts:

Part Description
object A valid object.
X A Double expression. x coordinate
Y A Double expression. y coordinate
Z A Double expression. z coordinate
