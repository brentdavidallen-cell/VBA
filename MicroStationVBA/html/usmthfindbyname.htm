FindByName Method

                          

Return already created ItemTypeLibrary object.

Syntax

Set ItemTypeLibrary = object.FindByName (name)

The FindByName method syntax has these parts:

Part Description
object A valid object.
name A String expression. Name of ItemtypeLibrary to search.

Remarks

Important:Name that is used in the argument is name of ItemTypeLibrary to be searched. Therefore before calling this method, you should ensure that the required ItemTypeLibrary is exist in DgnFile.
