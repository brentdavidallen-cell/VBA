ComputeCurveLength Method

                          

Returns the length of (a portion of) this object.

Syntax

Double = object.ComputeCurveLength [(StartParameter [, EndParameter [, IgnoreZ]])]

The ComputeCurveLength method syntax has these parts:

Part Description
object A valid object.
StartParameter Optional. A Double expression. The start parameter of this object from which to start the length computation. If omitted, 0.0 is used.
EndParameter Optional. A Double expression. The end parameter of this object at which to end the length computation. If omitted, 1.0 is used.
IgnoreZ Optional. A Boolean expression. Indicates whether the computation ignores z-coordinates. If True, the returned value is the length of the curve projected into the xy-plane (or as seen from the Top View). If omitted, a default value of FALSE is used, and true length is returned.

Remarks

For BsplineCurve objects, the returned length is nonnegative and accurate to machine precision.

For BsplineCurve objects, this method replaces the method ComputeLength, which was deprecated after MicroStation V8 2004 Edition.
