CreateAreaPattern Method

                          

Creates a new AreaPattern object.

Syntax

Set AreaPattern = object.CreateAreaPattern (RowSpacing, ColSpacing, Angle, CellName, Scale)

The CreateAreaPattern method syntax has these parts:

Part Description
object A valid object.
RowSpacing A Double expression.
ColSpacing A Double expression.
Angle A Double expression.
CellName A String expression.
Scale A Double expression.

Remarks

CreateAreaPattern does not verify that it can find the cell. It just saves the value passed in via CellName. A subsequent call to SetPattern will fail if the cell is not available.
