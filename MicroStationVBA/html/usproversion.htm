Version Property

                          

Read-only String.

Gets the version of the installed MicroStation.

Syntax

object.Version

The Version property syntax has these parts:

Part Description
object A valid object.

Remarks

For Application, the String returned by this property has the format "Version RR.MM.mm.ss Windows x86", where

  • RR is the release version number,
  • MM is the major version number,
  • mm is the minor version number and
  • ss is the sub-minor version number.
All four version numbers are numbers from 00-99, zero-padded on the left.

For Application, the following code converts the value of Version to a number that can be used to test for ranges of versions of MicroStation for which the object model contains a known incompatibility (cf. LineStyle). For example, the first release of MicroStation V8 2004 Edition has version number 08.05.00.64, which is returned as 8050064. 


    Function GetComparableVersion() As Long
        ' Assume Version string's second space-delimited token consists of 4 period-delimited numbers
        Dim aTokens() As String
        aTokens = Split(Split(Application.Version)(1), ".")
        GetComparableVersion = CLng(aTokens(3)) + 100 * CLng(aTokens(2)) + 10000 * CLng(aTokens(1)) + 1000000 * CLng(aTokens(0))
    End Function
