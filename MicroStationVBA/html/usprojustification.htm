Justification Property

                          

Read/Write MsdTextJustification for TagElement, TextStyle.

Read-only MsdTextJustification for TextNodeElement.

Gets or sets the justification of this object's text within its region.

Syntax

object.Justification [= Justification]

The Justification property syntax has these parts:

Part Description
object A valid object.
Justification An MsdTextJustification expression.
MsdTextJustification An MsdTextJustification expression.
