DateLastSaved Property

                          

Read-only Date.

Syntax

object.DateLastSaved

The DateLastSaved property syntax has these parts:

Part Description
object A valid object.

Remarks

DateLastSaved of a DesignFile is one of the standard Windows file properties. You can use the Windows Explorer to view this along with the other file properties. You can also use MicroStation's File Properties dialog box (available from the File pulldown) to view a file's last-saved-date.

MicroStation updates a DesignFile's DateLastSaved on every user-initiated save and when it closes a file that has changes.
