GetDatabaseLinks Method

                          

Returns an array of DatabaseLink objects that represent this object's database linkages.

Syntax

DatabaseLink () = object.GetDatabaseLinks [(DatabaseType [, EntityNumber])]

The GetDatabaseLinks method syntax has these parts:

Part Description
object A valid object.
DatabaseType Optional. An MsdDatabaseLinkage expression. If given, limits the search to only these linkage types.
EntityNumber Optional. A Long expression. If given, limits the search to only this entity number.

For an element, calling GetDatabaseLinks raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
