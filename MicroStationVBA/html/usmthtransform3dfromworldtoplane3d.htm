Transform3dFromWorldToPlane3d Method

                          

Returns the world-to-plane transform of the given plane.

Syntax

Transform3d = object.Transform3dFromWorldToPlane3d (Plane)

The Transform3dFromWorldToPlane3d method syntax has these parts:

Part Description
object A valid object.
Plane A Plane3d expression. Plane.Normal does not have to be normalized; if it is zero, the returned Transform3d has identity matrix part.

Remarks

In the following, let Plane.Origin and (normalized) Plane.Normal be given by o and n, respectively. Let the matrix and translation parts of the returned Transform3d T be given by R and t, respectively.

T represents a change of basis from world to plane coordinates, where the rows of R are the plane's basis vectors in world coordinates (in particular, R.RowZ = n) and t = R(-o).

R is orthogonal: its inverse is its transpose.

If o + x is a vector in the given plane then the product T(o + x) = Rx is a vector in the xy-plane.
