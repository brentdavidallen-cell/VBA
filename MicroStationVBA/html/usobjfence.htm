Fence Object

                                                     

Represents a MicroStation fence.

Remarks

You cannot use the New keyword to create a Fence object. However, you can use the property, DesignFile.Fence, to get a design file's associated fence object.

Rather than representing a specific fence the Fence object is a manager object that a program can use to define or use the current fence, or to get a list of named fences.
