Matrix3dSetComponentByRowAndColumn Method

                          

Sets a specified component of a Matrix3d.

Syntax

object.Matrix3dSetComponentByRowAndColumn Matrix, RowIndex, ColumnIndex, Value

The Matrix3dSetComponentByRowAndColumn method syntax has these parts:

Part Description
object A valid object.
Matrix A Matrix3d expression. The Matrix3d whose component is set.
RowIndex A Long expression. The row (0, 1, 2) where the value is set.
ColumnIndex A Long expression. the column (0, 1, 2) where the value is set.
Value A Double expression. The new value for the component.
