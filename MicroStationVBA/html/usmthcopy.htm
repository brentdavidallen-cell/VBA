Copy Method

                          

Returns a new object that is a copy of this object.

Syntax

For the Attachment object:

   Set Attachment = object.Copy

For the BsplineCurve object:

   Set BsplineCurve = object.Copy

For the BsplineSurface object:

   Set BsplineSurface = object.Copy

For the InterpolationCurve object:

   Set InterpolationCurve = object.Copy

For the ModelReferences object:

   Set ModelReference = object.Copy (SourceModel, Name [, Description])

The Copy method syntax has these parts:

Part Description
object A valid object.
SourceModel A ModelReference expression. Specifies the model to copy. SourceModel may be from a different design file.
Name A String expression. Specifies the name of the new model. If there is already a model with that name, Copy raises an error. If ModelName is vbNullString, then Copy generates a unique name.
Description Optional. A String expression. vbNullString is allowed.

Remarks

Attachments.Copy creates an identical attachment in the same collection of attachments.

ModelReferences.Copy copies levels, styles, and other components as necessary for elements in the model.
