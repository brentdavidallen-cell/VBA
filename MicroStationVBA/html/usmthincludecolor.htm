IncludeColor Method

                          

Include elements of the specified color in the forthcoming scan.

Syntax

object.IncludeColor ColorIndex

The IncludeColor method syntax has these parts:

Part Description
object A valid object.
ColorIndex A Long expression.

Remarks

By default, elements of every color are included in the scan. To limit the scan to specific colors, first call ExcludeAllColors, followed by a separate call to IncludeColor for each color to be included.

Calling this method automatically restricts the scan to only graphical type elements.
