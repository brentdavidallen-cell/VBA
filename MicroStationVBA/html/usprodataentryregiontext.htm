DataEntryRegionText Property

                          

Read/Write String.

Gets or sets the text of the DataEntryRegion at the specified index. The first index is 1.

Syntax

object.DataEntryRegionText (DataEntryRegionIndex)[= String]

The DataEntryRegionText property syntax has these parts:

Part Description
object A valid object.
DataEntryRegionIndex A Long expression.
String A String expression.
