RenderingInformation Object

                                                     

This is the rendering information.

Remarks

Use a Raster object to get a RenderingInformation object as follows: 


    'With a variable :

    Dim theRasters As Rasters
    Dim rendInf As RenderingInformation

    Set theRasters = RasterManager.Rasters
    Set rendInf = theRasters.Item(1). RenderingInformation
    rendInf.[method/property]


    'Without a variable :

    Dim theRasters As Rasters

    Set theRasters = RasterManager.Rasters

    theRasters.Item(1).RenderingInformation.[method/property]
