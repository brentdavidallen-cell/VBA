MsdBsplineParametrizationType Enum

        

Defines constants that represent different parametrizations for a BsplineCurve or BsplineSurface object.

The MsdBsplineParametrizationType enum has these members:

Name Description
msdBsplineParametrizationUniform 0 (&H0) Represents uniform parametrization: equally spaced Knots.
msdBsplineParametrizationChordLength 1 (&H1) Represents chord length parametrization: Knots spaced proportional to the distances between successive Poles.
msdBsplineParametrizationCentripetal 2 (&H2) Represents centripetal parametrization: Knots spaced proportional to the square roots of the distances between successive Poles.
msdBsplineParametrizationInherited 3 (&H3) Represents a parametrization that matches that of another BsplineCurve or BsplineSurface.

Remarks

These constants are used in methods such as BsplineCurve.Rebuild to determine the configuration of the Knots of the BsplineCurve object to be constructed.
