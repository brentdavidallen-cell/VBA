MsdDevelopableElementOutputType Enum

        

Defines constants that determine the type of the objects in the ElementEnumerator object returned by the method BsplineCurve.ConstructDevelopableRulingsToCurve.

The MsdDevelopableElementOutputType enum has these members:

Name Description
msdDevelopableRuleLines 0 (&H0) Outputs LineElement objects that represent rule lines of the developable surface.
msdDevelopableRuleLinesPlanar 1 (&H1) Outputs LineElement objects that represent rule lines of the unrolled developable surface.
msdDevelopableShapes 2 (&H2) Outputs ShapeElement objects that represent quadrilaterals approximating the conical surface patches of the developable surface.
msdDevelopableShapesPlanar 3 (&H3) Outputs ShapeElement objects that represent planar quadrilaterals approximating the unrolled conical surface patches of the unrolled developable surface.
msdDevelopableCones 4 (&H4) Outputs ConeElement objects that represent the cones tangent to the developable surface at each rule line (one cone per rule line).
msdDevelopableConesPlanar 5 (&H5) Outputs ConeElement objects that represent the cones tangent to the unrolled developable surface at each rule line (one cone per rule line).

Remarks

Specifically, these values represent the different output representations of the developable surface approximately spanned by the two BsplineCurve objects. Planar output representations are in the coordinates of the developable surface as unrolled into the xy-plane containing a specified seed point.
