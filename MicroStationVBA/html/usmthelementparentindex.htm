ElementParentIndex Method

                          

If the given index refers to a sub-element, returns the index of the corresponding outer (containing) element.

Otherwise (if the given index refers to a top-level element), returns 0.

Syntax

Long = object.ElementParentIndex (Index)

The ElementParentIndex method syntax has these parts:

Part Description
object A valid object.
Index A Long expression.

Remarks

For example, if the given index refers to a sub-element within a cell, this method returns the index of the corresponding cell header element.
