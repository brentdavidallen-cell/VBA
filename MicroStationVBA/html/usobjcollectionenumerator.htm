CollectionEnumerator Object

                                                     

A method that needs to return a subset of a collection, returns a CollectionEnumerator. The CollectionEnumerator can be used to access all of the subset's elements. MoveNext returns False if it was unable to advance, i.e., if there are no more elements. A program can use Current to retrieve the current element and Reset to restart the process.

Remarks

Before calling the Current method on a newly created or reset CollectionEnumerator, MoveNext must be called to position the CollectionEnumerator to the first element of the subset. The return value of MoveNext should also be tested to check for empty subsets.
