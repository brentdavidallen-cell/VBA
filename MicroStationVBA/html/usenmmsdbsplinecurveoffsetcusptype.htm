MsdBsplineCurveOffsetCuspType Enum

        

Defines constants that determine the treatment of cusps (corners) in a BsplineCurve object on which the BsplineCurve.Offset method is called.

The MsdBsplineCurveOffsetCuspType enum has these members:

Name Description
msdBsplineCurveOffsetCuspJump 0 (&H0) Cusps are not offset (there is a discontinuity in the offset).
msdBsplineCurveOffsetCuspChamfer 1 (&H1) Cusps are offset by a line segment.
msdBsplineCurveOffsetCuspPoint 2 (&H2) Cusps are offset by two line segments which converge at the intersection of adjacent tangents.
msdBsplineCurveOffsetCuspParabola 3 (&H3) Cusps are offset by a parabolic arc.
msdBsplineCurveOffsetCuspArc 4 (&H4) Cusps are offset by a circular arc.

Remarks

In particular, these values represent the ways in which the portions of the offset on either side of a cusp are joined.
