MsdRenderingMode Enum

        

Defines the possible values of the property, Attachment.Presentation.

The MsdRenderingMode enum has these members:

Name Description
msdRenderingModeWireFrame 0 (&H0)
msdRenderingModeCrossSection 1 (&H1)
msdRenderingModeWireMesh 2 (&H2)
msdRenderingModeHiddenLine 3 (&H3)
msdRenderingModeSolidFill 4 (&H4)
msdRenderingModeConstantShade 5 (&H5)
msdRenderingModeSmoothShade 6 (&H6)
msdRenderingModePhong 7 (&H7)
msdRenderingModeRayTrace 8 (&H8)
msdRenderingModeRenderWireFrame 9 (&H9)
msdRenderingModeRadiosity 10 (&HA)
msdRenderingModeParticleTrace 11 (&HB)

Remarks

Support for msdRenderingModeConstantShade and msdRenderingModeWireMesh was dropped in MicroStation XM. These options remain in the enumeration so that existing code continues to compile.
