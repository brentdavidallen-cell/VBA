MsdSharedCellSearch Enum

        

Flags to control search for shared cell.

The MsdSharedCellSearch enum has these members:

Name Description
msdSharedCellSearchDestinationModel 1 (&H1) Allowed to search destination model
msdSharedCellSearchReferences 2 (&H2) Allowed to search reference files
msdSharedCellSearchLibraries 4 (&H4) Allowed to search attached libs and MS_CELLLIST for non-shared cells to turn into shared cells.
msdSharedCellSearchActiveLibary 8 (&H8) Allowed to search attached library for shared cells. Note that msdSharedCellSearchLibraries also causes MicroStation to look in the attached library. However, msdSharedCellSearchActiveLibary causes MicroStation to look for shared cell definitions that do not have a corresponding normal cell definition. If msdSharedCellSearchActiveLibary is not specified, MicroStation will not find a shared cell definition that does not have a corresponding normal cell definition.
msdSharedCellSearchActiveModel 16 (&H10) Allowed to search active model
msdSharedCellSearchActiveRefs 32 (&H20) Allowed to search references of active model
msdSharedCellSearchDefaultFlags 15 (&HF) combines msdSharedCellSearchDestinationModel, msdSharedCellSearchReferences, msdSharedCellSearchLibraries, and msdSharedCellSearchActiveLibary
