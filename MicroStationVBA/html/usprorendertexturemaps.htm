RenderTextureMaps Property

                          

Read/Write Boolean.

Gets or sets a value indicating whether texture maps are rendered in this view.

Syntax

object.RenderTextureMaps [= Boolean]

The RenderTextureMaps property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

This value is equivalent to the Pattern/Bump Maps checkbox in the Rendering View Attributes dialog box.

Changing this value does not cause an immediate change in what is displayed in the view. To show the change in the view, call the View.Redraw method.
