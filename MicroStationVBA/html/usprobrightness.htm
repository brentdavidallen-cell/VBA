Brightness Property

                          

Read/Write Long.

Gets or sets the brightness. 


    Dim rendInf As RenderingInformation

    Set rendInf = RasterManager.Rasters.Item(1).RenderingInformation

    If (rendInf.Brightness = 0) Then
        rendInf.Brightness = 30
    End If

Syntax

object.Brightness [= Long]

The Brightness property syntax has these parts:

Part Description
object A valid object.
Long A Long expression. Intensity of the raster's brightness.
