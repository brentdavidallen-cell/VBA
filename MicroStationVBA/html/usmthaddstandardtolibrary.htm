AddStandardToLibrary Method

                          

Adds a Standard node to the list of standards for a given library.

Syntax

Set Object = object.AddStandardToLibrary (LibraryDOMNode, NameOfStandard, ObjectType)

The AddStandardToLibrary method syntax has these parts:

Part Description
object A valid object.
LibraryDOMNode An Object expression. The IXMLNode object that AddLibraryToCheckerApp returned.
NameOfStandard A String expression. The name of the standard.
ObjectType A String expression. Describes the type of object that has the problem. Common values are "Level", "Element", "LineStyle", "TextStyle", and "Reference".

Remarks

AddStandardToLibrary returns an object that can be used as input to AddPropertyToStandard.
