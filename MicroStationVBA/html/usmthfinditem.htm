FindItem Method

                          

It will return Item, which was created by Attaching ItemType to CustomItemHost.

Syntax

Set ItemTypePropertyHandler = object.FindItem (source)

The FindItem method syntax has these parts:

Part Description
object A valid object.
source An Object expression.

Remarks

Source may be any type of Element object, a DesignFile object, a ModelReference object.
