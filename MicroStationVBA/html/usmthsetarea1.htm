SetArea1 Method

                          

Sets the View's Origin, Extents, Rotation, and ActiveDepth.

Syntax

object.SetArea1 Boundary1, Boundary2, PointOnBackClippingPlane, ZDelta, NewRotation, NewActiveDepth

The SetArea1 method syntax has these parts:

Part Description
object A valid object.
Boundary1 A Point3d expression.
Boundary2 A Point3d expression.
PointOnBackClippingPlane A Point3d expression.
ZDelta A Double expression.
NewRotation A Matrix3d expression.
NewActiveDepth A Double expression.

Remarks

SetArea1 sets the View's Rotation to NewRotation. Then it uses the rotation to rotate Boundary1 and Boundary2. It uses the minimum X and Y values of the rotated points as the X and Y of the Origin, and uses the maximum X and Y values of the rotated points to compute the X and Y of Extents. It computes the plane that is parallel to the view and contains the point PointOnBackClippingPlane. It sets Origin.Z to the location where that plane crosses the View's Z-axis. Finally, it sets Extents.Z to ZDelta and sets ActiveDepth to NewActiveDepth.

See View for information on the relationship of Origin, Extents, ActiveDepth, and Rotation.
