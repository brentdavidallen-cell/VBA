FilePositionFromIndex Method

                          

Given a file position within this cache, returns the equivalent index.

Syntax

Long = object.FilePositionFromIndex (Index)

The FilePositionFromIndex method syntax has these parts:

Part Description
object A valid object.
Index A Long expression.

Remarks

This method is provided only to facilitate the conversion of legacy MicroStation BASIC applications to VBA. New VBA projects should not use file positions.
