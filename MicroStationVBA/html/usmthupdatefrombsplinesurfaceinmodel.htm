UpdateFromBsplineSurfaceInModel Method

                          

Updates this object with the data from the given BsplineSurface object.

Syntax

object.UpdateFromBsplineSurfaceInModel Surface [, ReplaceInModel]

The UpdateFromBsplineSurfaceInModel method syntax has these parts:

Part Description
object A valid object.
Surface A BsplineSurface expression. The B-spline surface with which to update this object.
ReplaceInModel Optional. A Boolean expression. If omitted or True, this object's containing model is also updated. If False, or the element has not been added to a model, only this object is updated.
