MdlModelRefP Method

                          

Retrieves the associated DGN model pointer (an MDL DgnModelRefP).

Syntax

Int64 = object.MdlModelRefP

The MdlModelRefP method syntax has these parts:

Part Description
object A valid object.

Remarks

Most VB/VBA programs will not use this method. This is intended only for programs that use the MDL API. See Calling DLL Functions for more information.

Use MdlGetModelReferenceFromModelRefP to get the ModelReference object corresponding to a DgnModelRefP.
