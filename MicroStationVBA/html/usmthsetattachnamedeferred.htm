SetAttachNameDeferred Method

                          

Sets the value of AttachName. See AttachName, Reattach, and SetAttachNameDeferred for more information.

Syntax

object.SetAttachNameDeferred FileSpecification

The SetAttachNameDeferred method syntax has these parts:

Part Description
object A valid object.
FileSpecification A String expression.

Remarks

The format of FileSpecification is configuration_variable:filename where the configuration_variable: part is optional. The configuration_variable part and the colon must appear together. Do not use one without using the other.

If FileSpecification specifies configuration_variable: then MicroStation records configuration_variable: as part of the Attachment. It then evaluates the configuration variable everytime MicroStation loads the Attachment.

If FileSpecification does not provide a full file path, then MicroStation searches first the master file path and then the paths defined by the environment variables MS_RFDIR and MS_DEF when it loads the file.
