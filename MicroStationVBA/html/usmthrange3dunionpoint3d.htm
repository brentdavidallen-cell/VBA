Range3dUnionPoint3d Method

                          

Extends the coordinates of the range cube points in Range so as to include the single additional point.

Syntax

Range3d = object.Range3dUnionPoint3d (Range, Point)

The Range3dUnionPoint3d method syntax has these parts:

Part Description
object A valid object.
Range A Range3d expression. initial range.
Point A Point3d expression. new point to be included in the range.
