AddTag Method

                          

Creates a TagElement and associates it with this object.

Syntax

Set TagElement = object.AddTag (DgnTagDefinition)

The AddTag method syntax has these parts:

Part Description
object A valid object.
DgnTagDefinition A TagDefinition expression.

Remarks

Adding a tag creates a TagElement and adds to the active model. If the element already has a TagElement of the same name, then the new tag element replaces it. Adding a tag also modifies the original element and rewrites it.

AddTag does not display the new element. The program can display the new tag element after positioning it.

For an element, calling AddTag raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
