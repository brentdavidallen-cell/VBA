ITimerEvent Object

                                                     

Defines the interface MicroStation uses to notify a program of an ITimerEvent event.

Remarks

To receive these events, a program must have a class that implements ITimerEvent. The program must create an instance of the class, and register it with MicroStation by calling AddTimerEventHandler.

MicroStation only dispatches expired timers when it is idle. Once the specified duration expires, the timer will be dispatched the next time MicroStation is idle.

After calling the handler's TimerExpired method, MicroStation removes the timer unless the method sets RestartTimer parameter of TimerExpired to True. to True.
