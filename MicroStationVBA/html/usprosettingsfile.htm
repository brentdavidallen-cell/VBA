SettingsFile Property

                          

Read-only DesignFile.

Gets the DesignFile that contains the settings.

Syntax

object.SettingsFile

The SettingsFile property syntax has these parts:

Part Description
object A valid object.

Remarks

A standards checker may use SettingsFile as an argument to AddLibraryToCheckerApp.

A standards checker that uses an ApplicationElement instead of SetApplicationSettings, GetApplicationSettings, and DeleteApplicationSettings to manage its settings may use SettingsFile to get the DesignFile that contains the settings information.
