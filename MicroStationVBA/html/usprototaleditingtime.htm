TotalEditingTime Property

                          

Read-only Long.

Syntax

object.TotalEditingTime

The TotalEditingTime property syntax has these parts:

Part Description
object A valid object.

Remarks

TotalEditingTime of a DesignFile is one of the standard Windows file properties. You can use the Windows Explorer to view this along with the other file properties. You can also use MicroStation's File Properties dialog box (available from the File pulldown) to view a file's total editing time.

TotalEditingTime is a cumulative count of seconds giving the amount of time that the design file has been open in sessions where the file was changed. MicroStation updates a DesignFile's TotalEditingTime on every user-initiated save and when it closes a file that has changes.
