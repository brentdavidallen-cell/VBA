SetWeights Method

                          

Sets the array of weights of this object.

Syntax

object.SetWeights Weights

The SetWeights method syntax has these parts:

Part Description
object A valid object.
Weights A Double () expression. The array of weights to set.

Remarks

For BsplineCurve and BsplineSurface objects, see the Pole property for further description of this array. In particular, note that this array has zero-based indexing and is 1-dimensional for BsplineCurve objects, and 2-dimensional for BsplineSurface objects.

For BsplineCurve and BsplineSurface objects, passing an empty array or an array filled with identical values makes the object nonRational, otherwise the object becomes Rational. The Poles of this object must be set before calling this method, and the number of entries in the array must be equal to PolesCount (BsplineCurve objects) or UPolesCount * VPolesCount (BsplineSurface objects).
