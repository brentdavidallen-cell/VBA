Point2dAdd2Scaled Method

                          

Adds an origin and two scaled vectors.

Syntax

Point2d = object.Point2dAdd2Scaled (Origin, Vector1, Scale1, Vector2, Scale2)

The Point2dAdd2Scaled method syntax has these parts:

Part Description
object A valid object.
Origin A Point2d expression. origin . May be null.
Vector1 A Point2d expression. direction vector
Scale1 A Double expression. scale factor
Vector2 A Point2d expression. direction vector
Scale2 A Double expression. scale factor
