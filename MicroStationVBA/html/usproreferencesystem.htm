ReferenceSystem Property

                          

Read/Write MsdReferenceSystem.

Coordinate and unit system used for the clip element vertices.

Syntax

object.ReferenceSystem [= System]

The ReferenceSystem property syntax has these parts:

Part Description
object A valid object.
System An MsdReferenceSystem expression. Coordinate and unit system to use for the clip element vertices.
