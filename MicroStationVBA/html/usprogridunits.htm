GridUnits Property

                          

Read/Write Double.

Gets or sets the spacing between grid points.

Syntax

object.GridUnits [= Double]

The GridUnits property syntax has these parts:

Part Description
object A valid object.
Double A Double expression.

Remarks

For example, if the value is 10.0, there are 10 master units between grid points.

When the value is changed, the grid is redrawn in all views where it is turned on.
