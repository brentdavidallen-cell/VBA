ClosedElement Object

                                                     

Represents the aspects of a MicroStation element that pertain to only those element types that are always closed (and necessarily planar).

Remarks

The ClosedElement interface extends the PlanarElement interface. This means that any object that supports the ClosedElement interface also supports the PlanarElement interface, and that all of the methods and properties of the PlanarElement interface are also methods and properties of the ClosedElement
