RemoveRasterEventsHandler Method

                          

Removes the raster events handler.

Syntax

object.RemoveRasterEventsHandler EventHandler

The RemoveRasterEventsHandler method syntax has these parts:

Part Description
object A valid object.
EventHandler An IRasterEvents expression. Raster events handler to remove.

Remarks

EventHandler must be the same object that was passed into AddRasterEventsHandler.

It is not essential to call RemoveRasterEventsHandler. MicroStation will automatically remove the attachment events handler if the project is unloaded, or it is reset.
