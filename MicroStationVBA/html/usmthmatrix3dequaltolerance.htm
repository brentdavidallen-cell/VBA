Matrix3dEqualTolerance Method

                          

Tests for equality between two matrices.

Syntax

Boolean = object.Matrix3dEqualTolerance (Matrix1, Matrix2, Tolerance)

The Matrix3dEqualTolerance method syntax has these parts:

Part Description
object A valid object.
Matrix1 A Matrix3d expression. The first matrix
Matrix2 A Matrix3d expression. The second matrix
Tolerance A Double expression. The relative error tolerance. Absolute tolerance is this (relative) tolerance times the largest absolute value in Matrix1.
