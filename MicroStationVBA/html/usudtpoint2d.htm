Point2d Type

        

Represents the Cartesian coordinates of a point or a vector in a plane.

The Point2d type has these members:

Name Description
X A double value that represents the X-axis coordinate of the point.
Y A double value that represents the Y-axis coordinate of the point.

Remarks

For points in a ModelReference's geometry, the values are measured in the model's coordinate system, unless stated otherwise.

There are a number of global utility functions for managing Point2d values:

  • Returns a point from given X and Y components: Point2dFromXY.
  • Returns a point with all components zero: Point2dZero.
  • Returns a vector with all components one: Point2dOne.
  • Returns a unit vector in the same direction as a given vector: Point2dNormalize.
  • Returns an X, Y or Z component of a point: Point2dGetComponent.
  • Returns the (scalar) cross product of two vectors relative to a given origin: Point2dCrossProduct3Points.
  • Returns the (scalar) cross product of two vectors: Point2dCrossProduct.
  • Returns the angle between two vectors: Point2dSignedAngleBetweenVectors.
  • Returns the difference of two vectors: Point2dSubtract.
  • Returns the distance between two points: Point2dDistance.
  • Returns the dot product of a two vectors, one of which is relative to a given origin: Point2dDotDifference.
  • Returns the dot product of two vectors relative to a given origin: Point2dDotProduct3Points.
  • Returns the dot product of two vectors: Point2dDotProduct.
  • Returns the largest absolute value among the components of a vector: Point2dMaxAbs.
  • Returns the magnitude (length) of a vector: Point2dMagnitude.
  • Returns the negative of a vector: Point2dNegate.
  • Returns the point at an interpolated position between a start and an end point: Point2dInterpolate.
  • Returns the squared distance between two vectors: Point2dDistanceSquared.
  • Returns the squared magnitude of a vector: Point2dMagnitudeSquared.
  • Returns the sum of an origin and a scaled vector: Point2dAddScaled.
  • Returns the sum of an origin and three scaled vectors: Point2dAdd3Scaled.
  • Returns the sum of an origin and two scaled vectors: Point2dAdd2Scaled.
  • Returns the sum of two vectors: Point2dAdd.
  • Tests whether two vectors are equal within a given tolerance: Point2dEqualTolerance.
  • Tests whether two vectors are equal: Point2dEqual.
  • Tests whether two vectors are parallel: Point2dAreVectorsParallel.
  • Tests whether two vectors are perpendicular: Point2dAreVectorsPerpendicular.
