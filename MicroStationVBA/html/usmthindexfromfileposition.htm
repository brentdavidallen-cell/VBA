IndexFromFilePosition Method

                          

Given an index within this cache, returns the equivalent file position.

Syntax

Long = object.IndexFromFilePosition (FilePosition)

The IndexFromFilePosition method syntax has these parts:

Part Description
object A valid object.
FilePosition A Long expression.

Remarks

This method is provided only to facilitate the conversion of legacy MicroStation BASIC applications to VBA. New VBA projects should not use file positions.
