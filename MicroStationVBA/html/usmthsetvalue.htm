SetValue Method

                          

Syntax

object.SetValue NewValue

The SetValue method syntax has these parts:

Part Description
object A valid object.
NewValue A VARIANT expression.

Remarks

Use SetValue to set a property value if the property's type is Boolean, Date, Double, String or Long. Use SetValueAsPoint3d to set the value of a Point3d property. Use SetValueAsDLong to set the value of a DLong property.

The PropertyHandler object works on the file's copy of the object. SetValue always writes a change back to the file immediately. This is particularly significant when using a PropertyHandler object to access an element properties. When a program calls CreatePropertyHandler passing in an element object, CreatePropertyHandler extracts the element ID and model information from the element passed in and uses that to access the element from the file. If the Element object has changes that have not been rewritten, the PropertyHandler object does not see them. If the program uses the PropertyHandler object to change any properties, those changes are not automatically reflected in the Element object. The program has to load the object again (perhaps by using GetElementByID) to get the changes.

Raises an error if no property is selected. Use SelectByAccessString to select a property.
