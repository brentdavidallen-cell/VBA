Raster Object

                                                     

This is a raster.

Remarks

Use the Rasters object method to get a Raster object as follows: 


    ' Instantiation with Item:

    Dim oRaster As Raster

    Set oRaster = RasterManager.Rasters.Item(1)


    'Instantiation with Find:

    Dim oRaster As Raster

    Set oRaster = RasterManager.Raster.Find("C:\…\note.bmp")


    'Instantiation with FindRasterInView:

    Dim oPoint As Point3d
    Dim oRaster As Raster

    oPoint.X = 4
    oPoint.Y = 5

    Set oRaster =RasterManager.Rasters.FindRasterInView(oPoint,4)


    'Instantiation with Current:

    Dim oRaster As Raster
    Dim oEnum As CollectionEnumerator

    For Each oRaster In theRasters
        oRaster.ExtendedInformation.Selected = True
    Next

    Set oEnum = RasterManager.Rasters.GetSelectedRasters

    Do While oEnum.MoveNext
        Set oRaster = oEnum.Current
        oRaster.GeoReferenceInformation.RotationAboutZ = 45
    Loop
