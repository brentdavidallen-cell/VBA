SetUKnots Method

                          

Sets the array of interior knots of this object in the u-parameter direction.

Syntax

object.SetUKnots UKnots

The SetUKnots method syntax has these parts:

Part Description
object A valid object.
UKnots A Double () expression. The array of interior knots to set in the u-parameter direction.

Remarks

For BsplineSurface objects, see the UKnot property for further description of this array.

For BsplineSurface objects, passing an empty array makes the curve uniform in the u-parameter direction (cf. UUniform).

For BsplineSurface objects, the Poles and UOrder of this object must be set before calling this method, and the number of entries in the array must be equal to the count returned by Bspline.ComputeKnotsCount.
