CurrentGraphicGroup Property

                          

Read-only Long for Application.

Read/Write Long for Settings.

Gets or sets the current graphic group.

Syntax

object.CurrentGraphicGroup [= Long]

The CurrentGraphicGroup property syntax has these parts:

Part Description
object A valid object.
Long A Long expression.

Remarks

To assign a graphic group number to an element's GraphicGroup property, use the value returned by Application.CurrentGraphicGroup or the value returned by Application.UpdateGraphicGroupNumber. Application.UpdateGraphicGroupNumber and Application.CurrentGraphicGroup return the same value, but Application.UpdateGraphicGroupNumber also increments the graphic group number and saves it to the design file. MicroStation commands that create graphic groups use the value these properties return.

Settings.CurrentGraphicGroup gets or sets the graphic number that MicroStation assigns to newly created elements. Only programs that use the CadInputQueue to sequence MicroStation commands should use this property. This property is normally 0. These programs that change it must reset it to 0 prior to exiting.
