RasterInformation Object

                                                     

This is the raster file information.

Remarks

Use a Raster object to get a RasterInformation object as follows: 


    'With a variable :

    Dim theRasters As Rasters
    Dim rasInf As RasterInformation

    Set theRasters = RasterManager.Rasters
    Set rasInf = theRasters.Item(1). RasterInformation
    rasInf.[method/property]


    'Without a variable :

    Dim theRasters As Rasters

    Set theRasters = RasterManager.Rasters

    theRasters.Item(1). RasterInformation.[method/property]
