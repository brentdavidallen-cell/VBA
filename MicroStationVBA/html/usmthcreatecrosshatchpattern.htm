CreateCrossHatchPattern Method

                          

Creates a new CrossHatchPattern object.

Syntax

Set CrossHatchPattern = object.CreateCrossHatchPattern (Space1, Space2, Angle1, Angle2)

The CreateCrossHatchPattern method syntax has these parts:

Part Description
object A valid object.
Space1 A Double expression.
Space2 A Double expression.
Angle1 A Double expression.
Angle2 A Double expression.

Remarks

CreateCrossHatchPattern initializes LineWeight, LineStyle, and Color from the active settings. It initializes the other properties from the parameters passed to CreateCrossHatchPattern.
