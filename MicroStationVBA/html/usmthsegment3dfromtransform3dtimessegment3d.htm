Segment3dFromTransform3dTimesSegment3d Method

                          

Returns a segment resulting from applying a transform to an input segment.

Syntax

Segment3d = object.Segment3dFromTransform3dTimesSegment3d (Transform, Segment)

The Segment3dFromTransform3dTimesSegment3d method syntax has these parts:

Part Description
object A valid object.
Transform A Transform3d expression. Transformation to apply
Segment A Segment3d expression. input segment
