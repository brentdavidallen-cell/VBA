GetPixelPosition Method

                          

Calculates a screen position for an input point.

Syntax

Point2d = object.GetPixelPosition (MasterUnitsPoint, RelativeToView)

The GetPixelPosition method syntax has these parts:

Part Description
object A valid object.
MasterUnitsPoint A Point3d expression. The point for which the screen coordinates are desired.
RelativeToView A Boolean expression.

Remarks

If RelativeToView is True, the returned point is relative to the upper left corner of the view's window. If it is False, the returned point is relative to the upper left corner of the screen. In both cases, x increases to the right and y increases down the window.

The point returned may be outside the range that can be displayed in the indicated view, depending on the input and the area visible in the view.
