FillColor Property

                          

Read/Write Long.

Gets or sets the object's fill color.

Syntax

object.FillColor [= Long]

The FillColor property syntax has these parts:

Part Description
object A valid object.
Long A Long expression.

Remarks

The color is represented as an index into MicroStation's color table.

There is a predefined Long constant, ByLevelColor, that represents the special color value, "by level." If the value of FillColor is ByLevelColor, then the element's fill color is derived from the level that the element resides on.

There is also a predefined Long constant, ByCellColor, that represents the special color value, "by cell." If the value of FillColor is ByCellColor, then the element's fill color is derived from the cell that the element resides within.

FillColor of a graphical element gets or sets the element's fill color. It must be a color table index in the range 0 to 254.

FillColor of Settings gets or sets the active fill color. This determines the fill color of newly created elements. It must be a color table index, or -1 to indicate that the active color is also to be used as the active fill color.
