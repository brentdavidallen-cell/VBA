VKnot Property

                          

Read/Write Double.

Gets or sets the interior knot of this object at the given valid index in the v-parameter direction.

Syntax

object.VKnot (Index)[= Double]

The VKnot property syntax has these parts:

Part Description
object A valid object.
Index A Long expression. For BsplineSurface objects, the zero-based index (between 0 and VKnotsCount-1, inclusive).
Double A Double expression.

Remarks

For BsplineCurve and BsplineSurface objects, the knots characterize the parametrization of the curve and demarcate the local influence each Pole has over the B-spline curve or surface. Repeated knots decrease continuity at the corresponding point on the B-spline curve or the corresponding isoparametric curve on the B-spline surface.

For BsplineSurface objects, the interior knots (in either parameter direction) are almost always between 0.0 and 1.0, and must be nondecreasing. The full knot vector in the v-parameter direction consists of the interior knots in the v-parameter direction sandwiched between 2 * VOrder exterior knots, which are computed from other properties and are hidden from the object model.

For BsplineSurface objects, to set an interior knot in the v-parameter direction at an index beyond the valid index range, you must use SetVKnots, which resets VKnotsCount.
