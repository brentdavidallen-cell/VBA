DLongFromHexString Method

                          

Returns the DLong representation of a String expression that's in hexadecimal (base 16) form.

Syntax

DLong = object.DLongFromHexString (Value)

The DLongFromHexString method syntax has these parts:

Part Description
object A valid object.
Value A String expression.

Remarks

Example: 

    Dim N As DLong
    N = DLongFromHexString("112210F4B16C1CB1")
                    ' Now N contains the value 1234567890987654321.
                    ' (&H112210F4B16C1CB1 = 1234567890987654321)

If the string cannot be converted to a DLong value, an "invalid argument" error (Error 5) occurs.

If the string contains a decimal (base 10) representation of the number, rather than a hexadecimal representation, use DLongFromString
