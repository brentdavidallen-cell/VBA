BsplineSurfaceElement Object

                                                     

A kind of Element that corresponds to a MicroStation B-spline surface element.

Remarks

If you have a generic Element object (as the result of a Scan, for example), you can determine whether it's a BsplineSurfaceElement by examining the element's IsBsplineSurfaceElement property. If this property returns True, you can extract the element's implementation of BsplineSurfaceElement by calling the element's AsBsplineSurfaceElement property. (You can accomplish the same thing by assigning the element to a variable of the type BsplineSurfaceElement. But Element.AsBsplineSurfaceElement saves you a step.)

You cannot use the New keyword to create a BsplineSurfaceElement object. Rather, use New to create a BsplineSurface object, populate its properties, then create the BsplineSurfaceElement using CreateBsplineSurfaceElement1.

The new element will not become part of the design file until it is added to a model (using AddElement on the appropriate ModelReference object). 


    Dim oBsplineSurface As New BsplineSurface
    With oBsplineSurface
        ' Set properties of oBsplineSurface here...
    End With

    ' Create the element
    Dim oBsplineSurfaceElement As BsplineSurfaceElement
    Set oBsplineSurfaceElement = CreateBsplineSurfaceElement1(Nothing, oBsplineSurface)

    ' Add the element to the active model (whether or not it's displayed)
    ActiveModelReference.AddElement oBsplineSurfaceElement

    ' Modify and update the element in its model
    With oBsplineSurface
        ' Modify properties of oBsplineSurface here...
    End With
    oBsplineSurfaceElement.UpdateFromBsplineSurface oBsplineSurface

    ' Display the element (whether or not it's in a model)
    oBsplineSurfaceElement.Redraw
