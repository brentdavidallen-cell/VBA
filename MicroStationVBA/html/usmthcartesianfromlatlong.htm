CartesianFromLatLong Method

                          

Gets the Cartesian Coordinates, in the units of the GeographicCoordinateSystem, corresponding to the given longitude, latitude, and elevation.

Syntax

Point3d = object.CartesianFromLatLong (Point)

The CartesianFromLatLong method syntax has these parts:

Part Description
object A valid object.
Point A GeoPoint3D expression. The geographic point (longitude, latitude, and elevation) for which to calculate the Cartesian coordinates.
