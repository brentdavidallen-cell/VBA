MasterUnit Property

                          

Read/Write MeasurementUnit.

Gets or sets this object's master unit.

Syntax

object.MasterUnit [= MeasurementUnit]

The MasterUnit property syntax has these parts:

Part Description
object A valid object.
MeasurementUnit A MeasurementUnit expression.

Remarks

Whenever you set a model's MasterUnit, its SubUnit is set to the same value as the MasterUnit. Therefore, if you want to set both the MasterUnit and the SubUnit, you must set the MasterUnit first, followed by the SubUnit.
