AccuDrawHints Object

                                                     

A program can use the AccuDrawHints object to control the state of AccuDraw.

Remarks

AccuDrawHints are only useful if you are implementing your own primitive command instead of driving another command. Primitive commands control AccuDraw. If you have a procedure that sends a command and then sends datapoints, then that command will control AccuDraw. If you have your primitive command started via StartPrimitive or StartLocate, then your command can control AccuDraw.

By default, AccuDraw is activated after the first data point and the compass is updated on each subsequent datapoint.

Like most things that are accessed from the CommandState object, the effects of the hints are temporary. They last only until a new primitive command is started.

If AccuDraw is not active or the "Context Sensitivity" setting in MicroStation's AccuDraw Settings dialog box is turned off, operations on the AccuDraw object have no effect. Furthermore, the user can use AccuDraw's settings and shortcut commands to override the effect of the hints.
