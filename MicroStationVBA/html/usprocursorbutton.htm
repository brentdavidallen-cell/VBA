CursorButton Property

                          

Read-only Long.

Syntax

object.CursorButton

The CursorButton property syntax has these parts:

Part Description
object A valid object.

Remarks

When called with a CadInputMessage object that represents an unassigned cursor button, CursorButton represents the cursor button that the user entered. Do not use this property of a CadInputMessage object unless the InputType indicates that the user entered an unassigned cursor button.
