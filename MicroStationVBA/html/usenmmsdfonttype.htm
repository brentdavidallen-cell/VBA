MsdFontType Enum

        

Defines constants that represent the types of Fonts that can reside in a DesignFile's Fonts table.

The MsdFontType enum has these members:

Name Description
msdFontTypeMicroStation 0 (&H0) A native MicroStation font.
msdFontTypeSHX 1 (&H1) An AutoCAD SHX font.
msdFontTypeWindowsTrueType 2 (&H2) A Windows TrueType font.
msdFontTypeUnknown 3 (&H3) An unknown font type. (In Fonts.Find, msdFontTypeUnknown means "any font type.")
