Transform3dFactorMirror Method

                          

Factor a combination of a mirror part and a non-mirror part, attempting to use a common fixed point for the two parts. Equationally, the old transform T0 becomes 

            T0 = T1 * M
where M is the mirror, and T1 is a mirror-free transform. The mirror transform is returned in both matrix form and as plane point with normal. In an order of operations view, a point X is transformed as 
            T0 * X = T1 * M * X
That is, X mirrored first, followed by rotation and scaling in the residual transform.

Syntax

Boolean = object.Transform3dFactorMirror (Transform, ResidualTransform, MirrorTransform, FixedPoint, PlaneNormal)

The Transform3dFactorMirror method syntax has these parts:

Part Description
object A valid object.
Transform A Transform3d expression. IN The transformation to test
ResidualTransform A Transform3d expression. OUT the residual transform.
MirrorTransform A Transform3d expression. OUT the mirror transform.
FixedPoint A Point3d expression. OUT A fixed point of the mirror transform.
PlaneNormal A Point3d expression. OUT Unit normal for the mirror plane.

Remarks

Returns false if the transform has no mirror effects. In this case the mirror transform is the identity, the residual transform is the original transform, and the fixed point and normal are both zero.
