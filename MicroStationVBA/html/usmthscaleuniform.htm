ScaleUniform Method

                          

Uniformly scales this object.

Syntax

For the TextNodeElement object, TextElement object, TagElement object, SmartSolidElement object, SharedCellElement object, SharedCellDefinitionElement object, ShapeElement object, SavedViewElement object, PointStringElement object, NamedGroupElement object, MultiLineElement object, LineElement object, EllipseElement object, Element object, DimensionElement object, CurveElement object, ConeElement object, ComplexStringElement object, ComplexShapeElement object, CellElement object, BsplineSurfaceElement object, BsplineCurveElement object, AuxiliaryCoordinateSystemElement object, ArcElement object and ApplicationElement object:

   object.ScaleUniform Origin, ScaleFactor

For the Attachment object:

   object.ScaleUniform Origin, ScaleFactor, ApplyToClipElement

The ScaleUniform method syntax has these parts:

Part Description
object A valid object.
Origin A Point3d expression.
ScaleFactor A Double expression.
ApplyToClipElement A Boolean expression.

Remarks

ScaleUniform of any graphical element scales the element about the origin by calling ScaleAll specifying ScaleFactor as the X, Y, and Z scaling factors.

For an element, calling ScaleUniform raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.

Use Redraw to erase the object before scaling it and to display it again after scaling it. Use Rewrite to make the change permanent. Attachment and the element types all support Redraw and Rewrite.
