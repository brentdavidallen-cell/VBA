DeleteXDatum Method

                          

Removes and deletes an XDatum item at a specified position in an XDatum() array, decreasing the size of the array by one.

Syntax

object.DeleteXDatum XData, Index

The DeleteXDatum method syntax has these parts:

Part Description
object A valid object.
XData A XDatum () expression.
Index A Long expression.

Remarks

XDatum() arrays returned from GetXData are zero-based (i.e., the first item in the array is at index number 0).
