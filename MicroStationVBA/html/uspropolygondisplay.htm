PolygonDisplay Property

                          

Read/Write Boolean.

Gets or sets a value indicating whether this object's control polygon should be displayed.

Syntax

object.PolygonDisplay [= Boolean]

The PolygonDisplay property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

For BsplineCurve objects, if PolygonDisplay is True, then line segments between successive pairs of Poles are displayed whenever this object is displayed.

For BsplineSurface objects, if PolygonDisplay is True, then line segments between successive pairs of Poles (in both u- and v-parameter directions) are displayed whenever this object is displayed.

For InterpolationCurve objects, if PolygonDisplay is True, then line segments between successive pairs of FitPoints are displayed whenever the object is displayed, as is a line segment joining the first (last) FitPoint to a point along the StartTangent (EndTangent).
