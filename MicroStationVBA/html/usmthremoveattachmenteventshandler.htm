RemoveAttachmentEventsHandler Method

                          

Removes the attachment events handler.

Syntax

object.RemoveAttachmentEventsHandler EventHandler

The RemoveAttachmentEventsHandler method syntax has these parts:

Part Description
object A valid object.
EventHandler An IAttachmentEvents expression.

Remarks

EventHandler must be the same object that was passed into AddAttachmentEventsHandler.

It is not essential to call RemoveAttachmentEventsHandler. MicroStation will automatically remove the attachment events handler if the project is unloaded, or it is reset.
