LocateCriteria Object

                                                     

Use a LocateCriteria object to restrict the set of elements found by MicroStation's locate logic.

Remarks

The term "MicroStation's locate logic" refers to the logic MicroStation uses when it tries to find an element corresponding to a data point. Any MicroStation command that requires the user to select an element uses the locate logic. The CommandState object uses the locate logic when selecting elements to pass to the LocateFilter and Accept methods.

Use CreateLocateCriteria to create a LocateCriteria object. Use SetLocateCriteria to set MicroStation's locate criteria.

MicroStation resets all of its locate criteria everytime a new primitive command or locate command starts. Therefore, a class that implements ILocateCommandEvents or IPrimitiveCommandEvents may want to set the locate criteria in the class's Start method. A procedure that sequences MicroStation input may want to set the locate criteria immediately after calling SendCommand.

See CreateLocateCriteria for a description of the initial state of a LocateCriteria object.
