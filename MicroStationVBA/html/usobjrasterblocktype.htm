RasterBlockType Object

                                                     

This is a raster file compression mode's block type.

Remarks

Use the RasterBlockTypes object to get a RasterBlockType object as follows: 


    'Instantiation with an Item :

    Dim theRasterBlockTypes As RasterBlockTypes
    Dim oRasterBlockType As RasterBlockType

        ...

    Set oRasterBlockType = theRasterBlockTypes.Item(1)


    'Instantiation with Find:

    Dim theRasterBlockTypes As RasterBlockTypes
    Dim oRasterBlockType As RasterBlockType

    ...

    Set oRasterBlockType = theRasterBlockTypes.Find("~16 Colors")
