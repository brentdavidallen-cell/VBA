IsEquivalent Method

                          

Returns True if this GeographicCoordinateSystem is equivalent to the given GeographicCoordinateSystem

Syntax

Boolean = object.IsEquivalent (Other)

The IsEquivalent method syntax has these parts:

Part Description
object A valid object.
Other A GeographicCoordinateSystem expression. The GeographicCoordinateSystem to test for equivalence.
