AsSharedCellDefinitionElement Property

                          

Read-only SharedCellDefinitionElement.

Gets this object's implementation of the SharedCellDefinitionElement interface.

Syntax

object.AsSharedCellDefinitionElement

The AsSharedCellDefinitionElement property syntax has these parts:

Part Description
object A valid object.

Remarks

An error is raised if this object does not support the SharedCellDefinitionElement interface. If you're in doubt, firstly check this object's IsSharedCellDefinitionElement property.
