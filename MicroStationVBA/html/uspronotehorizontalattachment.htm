NoteHorizontalAttachment Property

                          

Read/Write MsdDimNoteHorizontalAttachment.

Gets this object's horizontal attachment of inline leader note.

Syntax

object.NoteHorizontalAttachment [= MsdDimNoteHorizontalAttachment]

The NoteHorizontalAttachment property syntax has these parts:

Part Description
object A valid object.
MsdDimNoteHorizontalAttachment An MsdDimNoteHorizontalAttachment expression.

Remarks

See MsdDimNoteHorizontalAttachment for possible valid values.
