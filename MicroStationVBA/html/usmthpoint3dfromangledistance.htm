Point3dFromAngleDistance Method

                          

Initialize a point from polar angle and distance from origin.

Syntax

Point3d = object.Point3dFromAngleDistance (AngleRadians, DistanceXY, Z)

The Point3dFromAngleDistance method syntax has these parts:

Part Description
object A valid object.
AngleRadians A Double expression. angle from x axis
DistanceXY A Double expression. xy plane distance from origin to result point. 8
Z A Double expression. z part of point.
