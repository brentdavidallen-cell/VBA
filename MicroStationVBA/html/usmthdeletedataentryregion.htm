DeleteDataEntryRegion Method

                          

Deletes a DataEntryRegion at a specified index.

Syntax

object.DeleteDataEntryRegion Index

The DeleteDataEntryRegion method syntax has these parts:

Part Description
object A valid object.
Index A Long expression. The position at which to delete the line. The first index is 1.

Remarks

To delete all of the data entry regions associated with this object, use DeleteAllDataEntryRegions.
