SendTentativePoint Method

                          

This method sends a tentative point to the active command.

Syntax

object.SendTentativePoint DataPoint, ViewSpecifier

The SendTentativePoint method syntax has these parts:

Part Description
object A valid object.
DataPoint A Point3d expression. The tentative point s measured in the coordinate system of the active model.
ViewSpecifier A VARIANT expression. The view to which the data point information is directed. May be an index (1-8) or a View object.
