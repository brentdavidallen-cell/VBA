DisplaysGrid Property

                          

Read/Write Boolean.

Gets or sets a value indicating whether the view displays the grid points.

Syntax

object.DisplaysGrid [= Boolean]

The DisplaysGrid property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

This value is equivalent to the Grid checkbox in the View Attributes dialog box. Changing this value does not cause an immediate change in what is displayed in the view. To show the change in the view, call the View.Redraw method.
