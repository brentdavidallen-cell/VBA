AddMessage Method

                          

Displays a message in MicroStation's status bar and Message Center.

Syntax

object.AddMessage Message [, Details [, Priority [, OpenAlertDialog]]]

The AddMessage method syntax has these parts:

Part Description
object A valid object.
Message A String expression. The message text. This text appears in both the status bar and the Message Center.
Details Optional. A String expression. Additional text associated with this message. This text appears only in the Message Center Details section.
Priority Optional. An MsdMessageCenterPriority expression. Determines which icon is associated with the message in the Message Center's list box. The default value is msdMessageCenterPriorityInfo.
OpenAlertDialog Optional. A Boolean expression. If True, MicroStation will open an alert dialog displaying the brief message.
