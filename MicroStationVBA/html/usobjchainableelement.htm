ChainableElement Object

                                                     

Represents the aspects of a MicroStation element that pertain to only those element types that are chainable. An element is chainable if it is an open element that can be a component of a complex string or a complex shape.

Remarks

Use IsChainableElement to determine if an element supports the ChainableElement element interface. To use an element's ChainableElement interface, either access the element using AsChainableElement or assign the element reference to a variable declared to be ChainableElement.

Here is an example illustrating the first technique:

    If oElement.IsChainableElement Then
        With oElement.AsChainableElement
            theStartPoint = .StartPoint
            theEndPoint = .EndPoint
        End With
    End If

Here is an example illustrating the second technique:

    Dim oChainable as ChainableElement
    If oElement.IsChainableElement Then
        Set oChainable = oElement
        theStartPoint = oChainable.StartPoint
        theEndPoint = oChainable.EndPoint
    End If

Both examples first determine if the element is a ChainableElement prior to trying to use it as a ChainableElement. If an element is not a ChainableElement, then trying to access it as a ChainableElement results in an error indicating that there is a type mismatch or that the class does not support the expected interface.
