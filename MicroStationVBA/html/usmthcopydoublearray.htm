CopyDoubleArray Method

                          

Copies an array of Double's to or from the DataBlock object.

Syntax

object.CopyDoubleArray DoubleArray, CopyToDataBlock

The CopyDoubleArray method syntax has these parts:

Part Description
object A valid object.
DoubleArray A Double () expression.
CopyToDataBlock A Boolean expression.

Remarks

CopyDoubleArray uses the size of the array to determine how many bytes to transfer. CopyDoubleArray only transfers the contents of the array. It does not save or retrieve the size of the array.

After transferring the data, it increments Offset by the count of bytes transferred.
