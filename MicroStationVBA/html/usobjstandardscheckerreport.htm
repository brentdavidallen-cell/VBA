StandardsCheckerReport Object

                                                     

Used to add to the StandardsCheckerController's report.

Remarks

A standards checker can use the methods of StandardsCheckerReport regardless of whether the user has requested a report. If the user has not requested a report, then methods of StandardsCheckerReport do not do anything. The Options parameter of RunCheck does not have the msdStandardsCheckerCheckOptionGenerateReport bit set if the user has not requested a report.

A standards checker will add to the report when the StandardsCheckerController calls its AddedCheckerToStandardsCheckerApps method and when it has a problem to report.

The StandardsCheckerController calls a standards checker's AddedCheckerToStandardsCheckerApps method after adding a minimal description of the standards checker to the report. The standards checker's AddedCheckerToStandardsCheckerApps method can use the AddLibraryToCheckerApp, AddStandardToLibrary, and AddPropertyToStandard methods to add more detail.

When adding a problem to the report, a standards checker normally calls AddProblem or AddIgnoredProblem. Both of these methods return a StandardsCheckerProblem object. The standards checker can use that object to add a more detailed description of the problem.
