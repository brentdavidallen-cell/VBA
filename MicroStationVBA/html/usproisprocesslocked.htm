IsProcessLocked Property

                          

Read/Write Boolean.

Setting IsProcessLocked to True prevents the MicroStation process from timing out. This code has no effect on an interactive MicroStation session or a MicroStation session that has Visible set to true since a MicroStation session displaying the user interface does not time out.

Syntax

object.IsProcessLocked [= Boolean]

The IsProcessLocked property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

If a MicroStation session started as an Automation server from another process does not have the user interface displayed, it may time out after 5 seconds of no activity in the object model. Setting this property to True prevents this from happening.
