Helix Method

                          

Initializes this object to an approximation of a helix.

Syntax

object.Helix RadiusStart, RadiusEnd, StartPoint, Axis, Pitch [, PitchAsTurnsCount [, Tolerance]]

The Helix method syntax has these parts:

Part Description
object A valid object.
RadiusStart A Double expression. The nonnegative start radius of the helix.
RadiusEnd A Double expression. The nonnegative end radius of the helix.
StartPoint A Point3d expression. A point which, together with Axis.StartPoint, determines the helix's local x-axis. The helix starts along its local x-axis at a distance of RadiusStart from Axis.StartPoint. StartPoint should not equal Axis.StartPoint. The local x-axis does not have to be perpendicular to Axis.
Axis A Segment3d expression. The extent and direction of the helix along its z-axis. The length of Axis may be zero, (e.g., for a planar spiral), in which case, the curve lies in a plane containing the local x-axis.
Pitch A Double expression. Either the helix's pitch (perpendicular height between turns) or its (fractional) number of turns, depending on the value of PitchAsTurnsCount. The helix is right-handed if Pitch is positive, left-handed if Pitch is negative.
PitchAsTurnsCount Optional. A Boolean expression. Indicates whether Pitch actually stores the number of turns of the helix. If omitted or False, Pitch stores the pitch of the helix.
Tolerance Optional. A Double expression. The geometric approximation tolerance (used only if radii are different). If omitted or negative, a default tolerance of 1.0e-5 is used.

Remarks

For BsplineCurve objects, if the given radii are different, this method initializes the object to an approximation of a conical spiral; this takes substantially more computation time than the helix case.
