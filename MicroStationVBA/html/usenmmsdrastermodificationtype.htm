MsdRasterModificationType Enum

        

Defines constants that represent the type of modification applied to a raster.

The MsdRasterModificationType enum has these members:

Name Description
msdRasterModificationType_ExtendedInformation 0 (&H0)
msdRasterModificationType_GeoReferenceInformation 1 (&H1)
msdRasterModificationType_RenderingInformation 2 (&H2)
msdRasterModificationType_RasterInformation 3 (&H3)
msdRasterModificationType_ClipMask 4 (&H4) Represents a modification to the raster's RasterClipMasks collection or a RasterClipMask from that collection.
msdRasterModificationType_ClipBoundary 5 (&H5) Represents a modification to the raster's RasterClipBoundary.
msdRasterModificationType_Reload 6 (&H6) Represents a raster being reloaded from its file.
