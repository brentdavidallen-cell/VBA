LockAngle Method

                          

Locks the angle to the specified offset from the origin.

Syntax

object.LockAngle Angle

The LockAngle method syntax has these parts:

Part Description
object A valid object.
Angle A Double expression. Measured in radians.

Remarks

Since this method has meaning only in polar coordinates, AccuDraw's compass is automatically set to this mode.
