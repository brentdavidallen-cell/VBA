DeleteXData Method

                          

Delete all of the XData (if any) for a given application name on this object. The object may be any type of graphical element or a ModelReference.

Syntax

object.DeleteXData ApplicationName

The DeleteXData method syntax has these parts:

Part Description
object A valid object.
ApplicationName A String expression.

Remarks

To delete an object's XData for all application names, use DeleteAllXData.

Calling DeleteXData for an element type raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
