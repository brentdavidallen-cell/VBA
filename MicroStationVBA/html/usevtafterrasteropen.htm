AfterRasterOpen Event

                          

The AfterRasterOpen event occurs when a raster was opened.

Syntax

object.AfterRasterOpen RasterID

The AfterRasterOpen method syntax has these parts:

Part Description
object A valid object.
RasterID An Int64 expression. Identification number of the raster that was opened.

Remarks

This event is not raised when opening a design even if it contains rasters.
