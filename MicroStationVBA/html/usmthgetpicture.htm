GetPicture Method

                          

Returns a reference to the IPicture interface which gives a preview of the line style or element.

Syntax

Set Object = object.GetPicture (Width, Height [, DrawBackGround])

The GetPicture method syntax has these parts:

Part Description
object A valid object.
Width A Long expression. The width of the rectangle in number of pixels.
Height A Long expression. The height of the rectangle in number of pixels.
DrawBackGround Optional. A Boolean expression. Draw back ground.

Remarks

VBA forms and many of the VBA controls have a Picture property. Setting the property to an object that implements IPicture causes the control or form to display the picture.

Element.GetPicture draws the element in the given rectangle and returns an IPicture interface.

For an element, calling GetPicture method raises an error if the element is not graphical. If you're in doubt, first check the element's IsGraphical property.

PictureExample.mvba in the WorkSpace\Example\Standards\Macros demonstrates how to use GetPicture.

Developers using the object model from .NET will need to use Element.DrawToEnhancedMetafile instead of Element.GetPicture.
