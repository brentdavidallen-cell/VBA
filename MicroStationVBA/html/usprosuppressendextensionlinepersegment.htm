SuppressEndExtensionLinePerSegment Property

                          

Read/Write Boolean.

Gets this object's segment-specific suppress end extension line flag.

Syntax

object.SuppressEndExtensionLinePerSegment (iSegment)[= val]

The SuppressEndExtensionLinePerSegment property syntax has these parts:

Part Description
object A valid object.
iSegment A Long expression.
val A Boolean expression.
