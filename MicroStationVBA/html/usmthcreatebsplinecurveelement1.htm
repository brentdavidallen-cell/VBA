CreateBsplineCurveElement1 Method

                          

Creates and returns a new BsplineCurveElement object from a BsplineCurve object.

Syntax

Set BsplineCurveElement = object.CreateBsplineCurveElement1 (Template, Curve)

The CreateBsplineCurveElement1 method syntax has these parts:

Part Description
object A valid object.
Template An Element expression. An existing element whose settings are used to initialize the new element. If Nothing, the new element's settings are initialized from MicroStation's active settings.
Curve A BsplineCurve expression.

Remarks

Important: The geometric data in the Curve argument is given in the coordinate system of the currently active model. Therefore, before you call this method, you should ensure that the model to which you intend to add the element is the active model. (You can make a model active by calling the Activate method of the appropriate ModelReference object.)
