IncludeLineWeight Method

                          

Include elements of the specified line weight in the forthcoming scan.

Syntax

object.IncludeLineWeight LineWeight

The IncludeLineWeight method syntax has these parts:

Part Description
object A valid object.
LineWeight A Long expression.

Remarks

By default, elements of every line weight are included in the scan. To limit the scan to specific line weights, first call ExcludeAllLineWeights, followed by a separate call to IncludeLineWeight for each line weight to be included.

Calling this method automatically restricts the scan to only graphical type elements.
