IsMissingModel Property

                          

Read-only Boolean.

Syntax

object.IsMissingModel

The IsMissingModel property syntax has these parts:

Part Description
object A valid object.

Remarks

IsMissingModel may be True because MicroStation could not find the file that contains the Attachment, or because the file no longer contains the model. Use IsMissingFile to determine if MicroStation could not find the file.
