Vertex Property

                          

Read/Write Point3d.

Gets or sets the vertex at the specified position within this object's list of vertices.

Syntax

object.Vertex (Index)[= Point3d]

The Vertex property syntax has these parts:

Part Description
object A valid object.
Index A Long expression. The relative position within the list of vertices. The index of the first vertex is 1.
Point3d A Point3d expression.

Remarks

Each vertex's value is measured in the coordinate system of the containing model.
