InternalColorToString Method

                          

Syntax

String = object.InternalColorToString (InternalColor, IncludeBookName, BookNameFirst)

The InternalColorToString method syntax has these parts:

Part Description
object A valid object.
InternalColor A Long expression.
IncludeBookName A Boolean expression.
BookNameFirst A Boolean expression.

Remarks

If the color is a color table index, InternalColorToString returns a string representing that index. If the color represents a color book value, it returns the name of the value. Otherwise, it returns the RGB value.
