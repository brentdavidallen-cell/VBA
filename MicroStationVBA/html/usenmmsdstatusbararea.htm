MsdStatusBarArea Enum

        

Defines constants that represent all of the possible values of the Area argument of the method, ShowTempMessage.

The MsdStatusBarArea enum has these members:

Name Description
msdStatusBarAreaMiddle 15 (&HF) The Message (middle) area of the status bar.

msdStatusBarAreaLeft 16 (&H10) The Command/Prompt (left) area of the status bar.
