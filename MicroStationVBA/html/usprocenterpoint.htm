CenterPoint Property

                          

Read/Write Point3d for ArcElement, EllipseElement.

Read-only Point3d for EllipticalElement.

Gets (or sets) this object's center point.

Syntax

object.CenterPoint [= Point3d]

The CenterPoint property syntax has these parts:

Part Description
object A valid object.
Point3d A Point3d expression.
