FromHelicalRailAndSweptSection Method

                          

Initializes this object by sweeping the given Section along an approximation of a helical rail in standard position.

Syntax

object.FromHelicalRailAndSweptSection Section, SectionFrame, Radius, Height, Pitch [, ConstantPitch [, RightHelix [, RemoveKnots [, Tolerance]]]]

The FromHelicalRailAndSweptSection method syntax has these parts:

Part Description
object A valid object.
Section A BsplineCurve expression. The cross-sectional profile to sweep along the helical rail. For best results, Section should be planar.
SectionFrame A Transform3d expression. The section-to-world transformation. Section x-axis and y-axis vectors and Section origin should be given (in world coordinates) in the first and second matrix columns and the translation part of SectionFrame, respectively. The third matrix column of SectionFrame is not used; the Section z-axis (normal) is inferred from the cross product of its x- and y-axes.
Radius A Double expression. The (constant) radius of the helical rail.
Height A Double expression. The extent of the helical rail along its z-axis.
Pitch A Double expression. The perpendicular height between turns of the helical rail.
ConstantPitch Optional. A Boolean expression. Indicates whether the Pitch of the helical rail is constant. If omitted, a default value of True is used. If False, the Pitch varies parabolically (turns i and i + 1 are separated by a distance proportional to 2 * i + 1).
RightHelix Optional. A Boolean expression. Indicates whether the helical rail is right-handed. If omitted, a default value of True is used.
RemoveKnots Optional. A Boolean expression. Indicates whether to thin out the knots of the resulting surface in the sweep direction. If omitted, a default value of True is used.
Tolerance Optional. A Double expression. The absolute geometric error tolerance for knot removal, used only if RemoveKnots is True. If omitted, a default tolerance of 1.0e-5 is used.

Remarks

For BsplineSurface objects, the rail curve of the resulting surface is based on a helix in standard position: helix axis is Vector3dFromXYZ(0,0,1) and the starting point is Point3dFromXYZ(Radius, 0, 0). Each Section is positioned at its origin on the helical rail and oriented such that its local x-axis is parallel to the world xy-plane, its local y-axis is parallel to the world z-axis, and its local z-axis lies along the helix tangent.
