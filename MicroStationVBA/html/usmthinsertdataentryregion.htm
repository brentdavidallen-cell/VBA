InsertDataEntryRegion Method

                          

Inserts an additional data entry region at a specified index.

Syntax

object.InsertDataEntryRegion Index, NewDataEntryRegion

The InsertDataEntryRegion method syntax has these parts:

Part Description
object A valid object.
Index A Long expression. The position at which to insert the new data entry region. The first index is 1.
NewDataEntryRegion A DataEntryRegion expression. The criteria of the field to be inserted.

Remarks

If Index is equal to DataEntryRegionsCount + 1, this method behaves the same as AddDataEntryRegion.
