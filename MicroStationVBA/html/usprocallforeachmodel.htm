CallForEachModel Property

                          

Read-only Boolean.

The StandardsCheckerController uses the checker's CallForEachModel property to determine if it has to call the checker once for each model in the DesignFile being checked, or just once for the entire DesignFile.

Syntax

object.CallForEachModel

The CallForEachModel property syntax has these parts:

Part Description
object A valid object.
