Ray3dLengthSquared Method

                          

Return the squared length of a Ray3d, treating the start and target points as limits of the measurement.

Syntax

Double = object.Ray3dLengthSquared (Ray)

The Ray3dLengthSquared method syntax has these parts:

Part Description
object A valid object.
Ray A Ray3d expression. Ray to measure.
