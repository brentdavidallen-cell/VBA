Point3dFromMatrix3dTransposeTimesXYZ Method

                          

Returns the product P = [x,y,z]*M where M is the input matrix and P is the product point.

Syntax

Point3d = object.Point3dFromMatrix3dTransposeTimesXYZ (Matrix, X, Y, Z)

The Point3dFromMatrix3dTransposeTimesXYZ method syntax has these parts:

Part Description
object A valid object.
Matrix A Matrix3d expression. The matrix to apply
X A Double expression. The x component
Y A Double expression. The y component
Z A Double expression. The z component
