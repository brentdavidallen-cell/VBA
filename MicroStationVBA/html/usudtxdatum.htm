XDatum Type

        

Represents an item of XData that is attached to an Element.

The XDatum type has these members:

Name Description
Type An MsdXDatumType value that specifies the meaning of the data contained in the Value field.
Value A Variant value that contains the data itself. The type of this data depends upon the value of the Type field. See the Remarks section for more information.

Remarks

See XDataObject for a discussion of why the MicroStationDGN object model supports two techniques for accessing XData. This information is particularly important for .NET programs using .NET COM interop to access the MicroStationDGN object model.

Any graphical element can have one or more occurrences of XData attached to it. XData is represented as an array of XDatum values.
