ComputeMinimumDistance Method

                          

Returns the minimum distance to this object from a given point.

Syntax

For the BsplineCurve object:

   Double = object.ComputeMinimumDistance (ClosestPoint, ClosestParameter, FromPoint, Rotation [, Tolerance])

For the BsplineSurface object:

   Double = object.ComputeMinimumDistance (ClosestPoint, ClosestParameters, FromPoint [, Tolerance])

The ComputeMinimumDistance method syntax has these parts:

Part Description
object A valid object.
ClosestPoint A Point3d expression. The returned closest point on this object.
ClosestParameter A Double expression. The returned parameter of the closest point on this object.
FromPoint A Point3d expression. The point from which to compute the minimum distance to this object.
Rotation A Matrix3d expression. The rotation matrix (e.g., View.Rotation, Matrix3dRotationFromRowZ) specifying the plane in which to find the minimum distance. If Rotation is Matrix3dIdentity, then an xy-distance is computed (ignoring z-coordinates of this object and FromPoint); if Rotation is not Matrix3dIdentity, then both FromPoint and this object are first rotated by Rotation, and then an xy-distance is computed. ClosestPoint is always returned on the original object. Pass Matrix3dZero to find the true minimum distance.
Tolerance Optional. A Double expression. The error tolerance for the computation. If omitted or negative, a default tolerance of 1.0e-8 times the largest coordinate difference of the unweighted Poles is used.
ClosestParameters A Point2d expression. The returned u- and v-parameters of the closest point on this object.

Remarks

For BsplineCurve objects, the Tolerance parameter is unused.

For BsplineSurface objects, any boundaries in the object are ignored. To detect whether or not ClosestParameters defines a point on a trimmed/punctured surface, call ContainsParametricPoint.
