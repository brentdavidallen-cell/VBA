RemoveProperty Method

                          

Removes the property with the specified ID from given ItemType.

Syntax

Boolean = object.RemoveProperty (propid)

The RemoveProperty method syntax has these parts:

Part Description
object A valid object.
propid A Long expression. Id of ItemTypeProperty to be removed.

Remarks

Important: This method will return true if Property is removed from given ItemType.
