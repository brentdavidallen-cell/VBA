OnDesignFileOpened Event

                          

The OnDesignFileOpened event occurs when a design file is opened.

Syntax

object.OnDesignFileOpened DesignFileName

The OnDesignFileOpened method syntax has these parts:

Part Description
object A valid object.
DesignFileName A String expression. The name of the file that was opened.

Remarks

This is an example of a class that implements OnDesignFileOpened and OnDesignFileClosed event handlers. Including WithEvents in the declaration tells VBA that instances of this class should receive Application's OnDesignFileOpened and OnDesignFileClosed events. The method Class_Initialize sets hooks to Application to tie this instance to the current Application object.

    Dim WithEvents hooks As Application

    Private Sub Class_Initialize()
        Set hooks = Application
    End Sub
    Private Sub hooks_OnDesignFileClosed(ByVal DesignFileName As String)
        Debug.Print "Closed design file " & DesignFileName
    End Sub

    Private Sub hooks_OnDesignFileOpened(ByVal DesignFileName As String)
        Debug.Print "Opened design file " & DesignFileName
    End Sub
The following code is from a Code Module. It simply creates an instance of the class that has the WithEvents declaration. 
    Dim oOpenClose as clsOpenClose
    Sub SetupHooks()
        Set oOpenClose = New clsOpenClose
    End Sub
