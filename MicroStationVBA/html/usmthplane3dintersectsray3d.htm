Plane3dIntersectsRay3d Method

                          

Returns True if the given ray intersects the given plane, and if so, also returns the coordinates and parameter of the intersection point.

Syntax

Boolean = object.Plane3dIntersectsRay3d (IntersectionPoint, Parameter, Plane, Ray)

The Plane3dIntersectsRay3d method syntax has these parts:

Part Description
object A valid object.
IntersectionPoint A Point3d expression. The intersection point, which equals Point3dAddScaled(Ray.Origin, Ray.Direction, Parameter)
Parameter A Double expression. The parameter along Ray at which the line represented by Ray intersects Plane.
Plane A Plane3d expression. The plane to intersect.
Ray A Ray3d expression. The line to intersect.

Remarks

This method replaces the method Plane3dIntersectRay3d, which was deprecated after MicroStation V8 2004 Edition.
