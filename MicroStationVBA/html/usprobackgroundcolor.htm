BackgroundColor Property

                          

Read/Write Long.

Gets or sets the background color: 


    Dim rendInf As RenderingInformation

    Set rendInf = RasterManager.Rasters.Item(1).RenderingInformation

    If (rendInf.BackgroundColor >10) Then
    rendInf.BackgroundColor = 9
    End If

Syntax

object.BackgroundColor [= Long]

The BackgroundColor property syntax has these parts:

Part Description
object A valid object.
Long A Long expression. Index of the background color in the MicroStation color table.
