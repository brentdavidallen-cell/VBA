Ray3dFromTransform3dTimesRay3d Method

                          

Returns a ray resulting from applying a transform to an input ray.

Syntax

Ray3d = object.Ray3dFromTransform3dTimesRay3d (Transform, Ray)

The Ray3dFromTransform3dTimesRay3d method syntax has these parts:

Part Description
object A valid object.
Transform A Transform3d expression. Transformation to apply
Ray A Ray3d expression. input ray
