DeleteAllTags Method

                          

Delete all of this object's associated tags.

Syntax

object.DeleteAllTags

The DeleteAllTags method syntax has these parts:

Part Description
object A valid object.

Remarks

The tags are also deleted from the model in which they reside.

For an element, calling DeleteAllTags raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
