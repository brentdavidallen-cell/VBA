IsUpdateViewAutomatic Property

                          

Read/Write Boolean.

Are all views updated each time an operation is applied to a raster? 


    If (RasterManager.IsUpdateViewAutomatic) Then
        RasterManager.IsUpdateViewAutomatic = False
    End If

Syntax

object.IsUpdateViewAutomatic [= Boolean]

The IsUpdateViewAutomatic property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression. Must the views be updated each time an operation is applied to a raster?
