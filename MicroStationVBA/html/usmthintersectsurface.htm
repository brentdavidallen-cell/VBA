IntersectSurface Method

                          

Returns the intersections between this object and the given BsplineSurface.

Syntax

Point3d () = object.IntersectSurface (Parameters, SurfaceParameters, Surface [, Tolerance])

The IntersectSurface method syntax has these parts:

Part Description
object A valid object.
Parameters A Double () expression. The parameter(s) of the point(s) of intersection on this object.
SurfaceParameters A Point2d () expression. The parameters of the point(s) of intersection on the Surface object.
Surface A BsplineSurface expression. The B-spline surface to intersect with this object.
Tolerance Optional. A Double expression. The distance tolerance for surface point inversion. If omitted or negative, a default tolerance of 1.0e-8 times the largest coordinate difference of the unweighted Poles is used.

Remarks

For BsplineCurve objects, on a successful return, the arrays are empty if and only if no intersections were found. Multiple intersections in the parallel arrays are ordered by increasing B-spline curve parameter.
