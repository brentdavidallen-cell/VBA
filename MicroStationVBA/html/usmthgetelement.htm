GetElement Method

                          

Returns an Element object corresponding to the element that resides at the specified location within this object.

Syntax

For the ElementCache object:

   Set Element = object.GetElement (Index)

For the NamedGroupMember object:

   Set Element = object.GetElement

The GetElement method syntax has these parts:

Part Description
object A valid object.
Index A Long expression. Values range from 1 to Count, inclusive.

Remarks

ElementCache.GetElement method raises an error if the element has been deleted. (When an element is deleted, it continues to occupy a slot in the cache until the design file is closed.) If you're in doubt, you should firstly call IsElementValid.

A NamedGroupMember object defines how a specific element behaves in a specific named group. NamedGroupMember.GetElement returns an Element object for the element that the NamedGroupMember describes.
