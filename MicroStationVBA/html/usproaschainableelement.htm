AsChainableElement Property

                          

Read-only ChainableElement.

Gets this object's implementation of the ChainableElement interface.

Syntax

object.AsChainableElement

The AsChainableElement property syntax has these parts:

Part Description
object A valid object.

Remarks

An error is raised if this object does not support the ChainableElement interface. If you're in doubt, firstly check this object's the value of IsChainableElement property.
