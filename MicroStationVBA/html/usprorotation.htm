Rotation Property

                          

Read-only Matrix3d for ACSManager, Attachment, AuxiliaryCoordinateSystemElement, CellElement, EllipticalElement, SharedCellDefinitionElement, SharedCellElement, TagElement, TextNodeElement.

Read/Write Matrix3d for ArcElement, ConeElement, DimensionElement, EllipseElement, TextElement, View.

Read/Write Double for SheetDefinition.

Gets this object's rotation matrix.

Syntax

object.Rotation [= Matrix3d]

The Rotation property syntax has these parts:

Part Description
object A valid object.
Matrix3d A Matrix3d expression.
Double A Double expression.

Remarks

ACSManager.Rotation raises an error if there is no auxiliary coordinate system defined. A program can test for this using ACSManager.IsDefined.

Setting View.Rotation rotates the view about its center point. The View.Extents are unchanged. Changing this value does not cause an immediate change in what is displayed in the view. To show the change in the view, call the View.Redraw method.

The rows of a View rotation matrix are the View's axes in world coordinates, with the Z-axis (oView.Rotation.RowZ) pointing out of the screen at the viewer.

See View for information on the relationship of Origin, Extents, ActiveDepth, and Rotation.
