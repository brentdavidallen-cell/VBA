HasAnyDatabaseLinks Method

                          

Returns a value indicating whether the element has any database links.

Syntax

Boolean = object.HasAnyDatabaseLinks [(DatabaseType [, EntityNumber])]

The HasAnyDatabaseLinks method syntax has these parts:

Part Description
object A valid object.
DatabaseType Optional. An MsdDatabaseLinkage expression. If given, limits the search to only these linkage types. (Use the Or operator combine multiple linkage types.)
EntityNumber Optional. A Long expression. If given, limits the search to only this entity number.

For an element, calling HasAnyDatabaseLinks raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
