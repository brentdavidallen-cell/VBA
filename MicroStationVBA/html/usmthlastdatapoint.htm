LastDataPoint Method

                          

Gets the last point the user clicked in a view.

Syntax

Point3d = object.LastDataPoint

The LastDataPoint method syntax has these parts:

Part Description
object A valid object.

Remarks

LastDataPoint returns the data point that the user last entered during the current command state. If an element creation command such as PLACE LINE is active and the user has entered a data point for that command, then LastDataPoint returns the point the user entered. If a view control command such as WINDOW AREA is active and the user has entered a data point for that command, then LastDataPoint returns the point the user entered. If the user starts an element placement command, enters a data point, starts a view control command, enters a data point, and then exits the view control command resuming the element placement command, then LastDataPoint returns the data point that was entered during the element placement command. It does not return the data point entered during the view control command, even though that was the most recent data point.

CursorInformation has more information on the location of the cursor.
