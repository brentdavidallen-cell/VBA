Close Method

                          

Closes the object.

Syntax

For the Raster object, DesignFile object and BsplineCurve object:

   object.Close

For the BsplineSurface object:

   object.Close Direction

The Close method syntax has these parts:

Part Description
object A valid object.
Direction An MsdBsplineSurfaceDirection expression. The parameter direction in which to close this object.

Remarks

For BsplineCurve objects whose Closed property is False and whose first and last Poles are the same, this method sets Closed to True and adjusts the Knot and/or Pole arrays to preserve the curve's shape. The resulting curve is periodically defined if possible; otherwise, the Knots are shifted in such a manner that when the curve is Opened at parameter 0.0, the original curve is restored. This method does nothing if the object's Closed property is already True. Note that setting Closed to True does not preserve the curve's shape.

For BsplineSurface objects, this method behaves analogously to its behavior on BsplineCurve objects, with the addition that the surface is closed only in the given parameter direction.

The behavior of DesignFile.Close depends on the DesignFile object. If it corresponds to the active design file, then this method is equivalent to the user selecting Close from the File menu. If it is a design file opened via OpenDesignFileForProgram, then this method closes the design file. If it is a design file for an attachment, then Close raises an error. After Close closes a design file, objects representing data contained in the design file are no longer valid. Any attempt to use those objects raises an error.
