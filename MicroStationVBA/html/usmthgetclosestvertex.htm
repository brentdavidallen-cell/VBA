GetClosestVertex Method

                          

Returns the index of the vertex of the element that is closest to the point.

Syntax

Long = object.GetClosestVertex (Point)

The GetClosestVertex method syntax has these parts:

Part Description
object A valid object.
Point A Point3d expression.

Remarks

The value that is returned is an index into the array of vertices returned by GetVertices.

The input point is specified in in the coordinate system of the containing model.
