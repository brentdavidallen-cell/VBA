VUniform Property

                          

Read-only Boolean.

Gets a value indicating whether this object is uniform in the v-parameter direction.

Syntax

object.VUniform

The VUniform property syntax has these parts:

Part Description
object A valid object.

Remarks

For BsplineSurface objects, this property is True if and only if the distance between each successive VKnot is the same.
