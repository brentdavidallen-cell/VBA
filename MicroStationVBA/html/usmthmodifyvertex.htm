ModifyVertex Method

                          

Replaces the value of the vertex at the specified position.

Syntax

object.ModifyVertex Index, Point

The ModifyVertex method syntax has these parts:

Part Description
object A valid object.
Index A Long expression. The position in the list of vertices at which to perform the replacement. Must be in the range 0 (the first vertex) to VerticesCount - 1 (the last vertex).
Point A Point3d expression. The point to insert into the vertex list.

Remarks

In this object library, most index values are one-based. However, the index values that you give to InsertVertex, ModifyVertex and RemoveVertex are zero-based.
