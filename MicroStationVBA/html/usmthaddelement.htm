AddElement Method

                          

Adds a new element to this object.

Syntax

object.AddElement Element

The AddElement method syntax has these parts:

Part Description
object A valid object.
Element An Element expression. The element to be added. The expression may refer to an object that is an instance of any of the various element types.

Remarks

Important: You should add each element to only the model that was the active model when the element was created. Otherwise, each added element may appear at a different location, and/or have a different size, than you intended.

If you want to add many elements at one time, it may be easier to use the array version, AddElements.

Adding an element to a model does not automatically display the element. To display an element after you've added it, call the element's Redraw method. Alternately, you can add a group of elements, then then make one call to the global function RedrawAllViews.

Use CopyElement for copying elements. By using a CopyContext with AddElementToModel set to True, you can cause CopyElement to copy the element and add it to the model.
