MsdCoordinateFormat Enum

        

Defines constants that determine what CoordinateFormat is used.

The MsdCoordinateFormat enum has these members:

Name Description
msdSubUnits 0 (&H0) Coordinate Format is Sub Units.
msdMasterUnits 1 (&H1) Coordinate Format is Master Units.
msdWorkingUnits 2 (&H2) Coordinate Format is Working Units.
