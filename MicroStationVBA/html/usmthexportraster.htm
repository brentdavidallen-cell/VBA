ExportRaster Method

                          

Creates a new raster along with its file from the Bitmap. If the file already exist, it is overwritten.

Syntax

object.ExportRaster Fullname, NewFormatSpecifier, NewColorModeSpecifier, NewCompressionModeSpecifier, NewBlockTypeSpecifier [, CompressionLevel [, MaxNumberOfColors [, InvertColors [, Gamma]]]]

The ExportRaster method syntax has these parts:

Part Description
object A valid object.
Fullname A String expression. Full name (including path) of the raster file to create.
NewFormatSpecifier A VARIANT expression. Format identification number or a RasterFormat object to use for the copy. Use values -1 or Nothing to make a copy using the same format as the original file.
NewColorModeSpecifier A VARIANT expression. Color mode identification number or a RasterColorMode object to use for the copy. Use values -1 or Nothing to make a copy using the same color mode as the original file. Make sure the color mode is supported by the copy's format.
NewCompressionModeSpecifier A VARIANT expression. Compression mode identification number or a RasterCompressionMode object to use for the copy. Use values -1 or Nothing to make a copy using the same compression mode as the original file. Make sure the compresssion mode is supported by the copy's color mode.
NewBlockTypeSpecifier A VARIANT expression. Block type identification number or a RasterBlockType object to use for the copy. Use values -1 or Nothing to make a copy using the same block type as the original file.
CompressionLevel Optional. A Byte expression. Level of compression when applicable. 0 min and 255 max.
MaxNumberOfColors Optional. A Long expression. Limits the number of colors used in the destination raster. Valid with palleted color mode only except 1bit. A value of 0 will set the limit to the maximum number of colors available.
InvertColors Optional. A Boolean expression. If True the raster's colors will be inverted.
Gamma Optional. A Double expression. Gamma correction to apply. 1.0 means no gamma correction applied. support for sister file available.
