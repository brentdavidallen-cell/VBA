ActiveDepthDistanceFromGlobalOrigin Property

                          

Read/Write Double.

Gets/Sets the value for the Active Depth measured as a distance from the Global Origin.

Syntax

object.ActiveDepthDistanceFromGlobalOrigin [= Double]

The ActiveDepthDistanceFromGlobalOrigin property syntax has these parts:

Part Description
object A valid object.
Double A Double expression.

Remarks

ActiveDepthDistanceFromGlobalOrigin refers to the value that MicroStation's Active Depth commands set and display. For example, it gets the same value that AZ=? displays. This value is the distance along the view's Z axis from the Global Origin to the active depth point.

See View for information on the relationship of Origin, Extents, ActiveDepth, and Rotation.
