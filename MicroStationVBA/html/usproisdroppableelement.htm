IsDroppableElement Property

                          

Read-only Boolean.

Gets a value indicating whether this object supports the DroppableElement interface.

Syntax

object.IsDroppableElement

The IsDroppableElement property syntax has these parts:

Part Description
object A valid object.

Remarks

If IsDroppableElement is True, AsDroppableElement gets this object's implementation of the DroppableElement interface.
