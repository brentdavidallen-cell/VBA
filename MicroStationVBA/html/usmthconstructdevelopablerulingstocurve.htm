ConstructDevelopableRulingsToCurve Method

                          

Creates and returns an ElementEnumerator of new Elements that represent a developable surface approximately bounded by this object and the given BsplineCurve.

Syntax

Set ElementEnumerator = object.ConstructDevelopableRulingsToCurve (Curve, SeedPoint [, OutputType [, SamplesCount]])

The ConstructDevelopableRulingsToCurve method syntax has these parts:

Part Description
object A valid object.
Curve A BsplineCurve expression. The second of two B-spline curves (the first being the instance object) that approximately bound the computed developable surface.
SeedPoint A Point3d expression. For planar OutputTypes, determines both the height of the xy-plane of the unrolled developable surface and the starting point of the first rule line, which is placed parallel to the global x-axis.
OutputType Optional. An MsdDevelopableElementOutputType expression. Determines both the Type (LineElement, ShapeElement or ConeElement) of the Elements in the returned ElementEnumerator object, and the coordinate system (spatial or planar) of the developable surface. If omitted, a default value of msdDevelopableRuleLines is used.
SamplesCount Optional. A Long expression. Determines how many developable rule lines originate within each Bézier segment of this object. These rule lines originate at points uniformly sampled within the local parameter space of each segment. If omitted, a default value of 10 is used.

Remarks

For BsplineCurve objects, if OutputType equals msdDevelopableShapes then nonplanar ShapeElements will generally result; such ShapeElements may not render as expected.
