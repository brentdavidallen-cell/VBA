CreateSettings Method

                          

The StandardsCheckerController calls the CreateSettings method to tell the Standards Checker to create a new settings element of a given name, using default settings.

Syntax

object.CreateSettings

The CreateSettings method syntax has these parts:

Part Description
object A valid object.

Remarks

The standards checker should save the settings to the settings DgnLib file. If it can represent the settings in an XML DOM, then it can call SetApplicationSettings to save the settings to the settings DgnLib file.
