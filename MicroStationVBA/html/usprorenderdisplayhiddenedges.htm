RenderDisplayHiddenEdges Property

                          

Read/Write Boolean.

Gets or sets the Render Display Hidden Edges.

Syntax

object.RenderDisplayHiddenEdges [= DisplayHiddenEdges]

The RenderDisplayHiddenEdges property syntax has these parts:

Part Description
object A valid object.
DisplayHiddenEdges A Boolean expression.

Remarks

This value is equivalent to the Display Hidden Edges checkbox in the Set View Display Mode dialog box.

Changing this value does not cause an immediate change in what is displayed in the view. To show the change in the view, call the View.Redraw method.
