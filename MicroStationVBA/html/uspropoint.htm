Point Property

                          

Read-only Point3d.

Gets this object's point3d at a specified index.

Syntax

object.Point (iPoint)

The Point property syntax has these parts:

Part Description
object A valid object.
iPoint A Long expression.

Remarks

When called with a CadInputMessage object that represents a data point or unassigned cursor button, Point represents the point entered by the user. Do not use this property of a CadInputMessage object unless the InputType indicates that the user entered a data point or unassigned cursor button. If so, it is also possible to use both the View and ScreenPoint properties.

The value is measured in the coordinate system of the active model.
