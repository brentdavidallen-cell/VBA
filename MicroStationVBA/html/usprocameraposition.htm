CameraPosition Property

                          

Read/Write Point3d.

Gets or sets the camera position.

Syntax

object.CameraPosition [= Point3d]

The CameraPosition property syntax has these parts:

Part Description
object A valid object.
Point3d A Point3d expression.

Remarks

The camera direction is always from CameraPosition towards CameraTarget, which coincides with the negative Z-Axis for the view. MicroStation's image plane is always coincident with the camera target. Therefore, the target appears the same size in a camera (perspective) or non-camera (orthographic) view.
