DefineFromViewPoints Method

                          

Defines a fence.

Syntax

object.DefineFromViewPoints ViewSpecifier, Perimeter

The DefineFromViewPoints method syntax has these parts:

Part Description
object A valid object.
ViewSpecifier A VARIANT expression. May be a View object or a view number. Valid view numbers are 1 through 8.
Perimeter A Point2d () expression. Defines the perimeter of the fence. The points are in master units and are offsets from the lower left corner of the view. This works the same regardless of whether the first point is repeated as the last point. Either way, DefineFromViewPoints interprets the points as a closed shape.

Remarks

The following code creates a fence that is the same size as the view: 


Sub DoViewFence(vw As View)
    Dim pts(1 To 4) As Point2d
    Dim size As Point3d

    size = vw.Extents
    pts(2).Y = size.Y
    pts(3).X = size.X
    pts(3).Y = size.Y
    pts(4).X = size.X
    With ActiveDesignFile.Fence
        .DefineFromViewPoints vw, pts
        .Draw
    End With
End Sub
