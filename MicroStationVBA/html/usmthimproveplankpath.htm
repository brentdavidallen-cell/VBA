ImprovePlankPath Method

                          

Modifies or fills the given arrays of points and parameters along this object to better approximate a plank path from the first to the last given points/parameters on this object.

Syntax

object.ImprovePlankPath Parameters, Points

The ImprovePlankPath method syntax has these parts:

Part Description
object A valid object.
Parameters A Point2d () expression. On input, the array of parameters of points determining the initial plank path, or an empty array. On output, the array containing parameters of points along the improved plank path. If empty on input, the Points array must be nonempty (computations are less efficient). If nonempty on input, the Points array is ignored as input.
Points A Point3d () expression. On input, the array of points determining the initial plank path, or an empty array. On output, the array containing points along the improved plank path. If empty on input, the Parameters array must be nonempty (computations are more efficient).

Remarks

For BsplineSurface objects, a plank path is a path along the object consisting of sections (planks) which may undergo pitch and roll (twist) deformations to allow the plank to conform to the object, but not yaw.

For BsplineSurface objects, the size of the Parameters (or Points, if Parameters is empty) array equals the size of the output arrays. The first and last input parameters (points) are preserved on output.

For BsplineSurface objects, any boundaries in the object are ignored. To detect whether or not Parameters defines points on a trimmed/punctured surface, call ContainsParametricPoint.
