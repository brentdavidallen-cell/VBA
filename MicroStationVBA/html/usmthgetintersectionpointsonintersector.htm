GetIntersectionPointsOnIntersector Method

                          

Returns a list of points where the Intersector object intersects with the IntersectableElement element.

Syntax

Point3d () = object.GetIntersectionPointsOnIntersector (Intersector, Matrix)

The GetIntersectionPointsOnIntersector method syntax has these parts:

Part Description
object A valid object.
Intersector An Element expression. The point that intersects with the element used to call this method.
Matrix A Matrix3d expression.

Remarks

The only difference between this method and GetIntersectionPoints is that this method returns points that are on the intersector. GetIntersectionPoints returns points that are on the element used to call the method.
