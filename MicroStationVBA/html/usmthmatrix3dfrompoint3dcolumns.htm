Matrix3dFromPoint3dColumns Method

                          

Returns a matrix with 3 points copied to respective columns.

Syntax

Matrix3d = object.Matrix3dFromPoint3dColumns (XVector, YVector, ZVector)

The Matrix3dFromPoint3dColumns method syntax has these parts:

Part Description
object A valid object.
XVector A Point3d expression. The vector to insert in column 0
YVector A Point3d expression. The vector to insert in column 1
ZVector A Point3d expression. The vector to insert in column 2
