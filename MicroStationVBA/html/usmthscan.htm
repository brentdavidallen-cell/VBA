Scan Method

                          

Used to generate an ElementEnumerator that can be used to scan through the elements in this object.

Syntax

Set ElementEnumerator = object.Scan [(ScanCriteria)]

The Scan method syntax has these parts:

Part Description
object A valid object.
ScanCriteria Optional. An ElementScanCriteria expression. Limits which elements are returned by the ElementEnumerator object that Scan returns.

The default value is Nothing, in which case the resultant ElementEnumerator includes all graphical elements.


Remarks

For an ElementCache object, the scan is restricted to only those elements that reside within that cache.
