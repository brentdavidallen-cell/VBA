Vector3dDotProductXYZ Method

                          

Computes the dot product of one vector given as a vector structure and another given as xyz components.

Syntax

Double = object.Vector3dDotProductXYZ (Vector, Ax, Ay, Az)

The Vector3dDotProductXYZ method syntax has these parts:

Part Description
object A valid object.
Vector A Vector3d expression. The first vector.
Ax A Double expression. The x component of second vector.
Ay A Double expression. The y component of second vector.
Az A Double expression. The z component of second vector.
