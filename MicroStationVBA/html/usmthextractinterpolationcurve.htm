ExtractInterpolationCurve Method

                          

Creates and returns a new InterpolationCurve object from this object.

Syntax

Set InterpolationCurve = object.ExtractInterpolationCurve

The ExtractInterpolationCurve method syntax has these parts:

Part Description
object A valid object.

Remarks

For this method to succeed, the object's IsInterpolationCurve property must be True.
