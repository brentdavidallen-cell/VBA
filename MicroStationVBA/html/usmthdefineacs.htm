DefineACS Method

                          

Defines a auxiliary coordinate system and makes it the active ACS.

Syntax

object.DefineACS Origin, Rotation, ACSType

The DefineACS method syntax has these parts:

Part Description
object A valid object.
Origin A Point3d expression.
Rotation A Matrix3d expression.
ACSType An MsdACSType expression.

Remarks

This sets the name and description blank. To update a named auxiliary coordinate system, a program must first use this method and then use SaveActive.
