ArcElement Object

                                                     

A kind of Element that corresponds to a MicroStation arc element.

Remarks

You cannot use the New keyword to create an ArcElement object. Rather, use one of the global functions like CreateArcElement1 and its variants or ConstructFillet.

The new element will not become part of the design file until it is added to a model (using AddElement on the appropriate ModelReference object).

    Dim oArc As ArcElement
    Set oArc = CreateArcElement1(...)
    '
    '  ...Set other properties of oArc, if you wish...
    '
    ActiveModelReference.AddElement oArc
    oArc.Redraw ' Display the newly-added element.
