ComplexShapeElement Object

                                                     

A kind of Element that corresponds to a MicroStation complex shape element.

Remarks

The subelements of a ComplexShapeElement are all ChainableElement's. Therefore, every element generated from a ComplexShapeElement's GetSubElements is a ChainableElement. A program can take advantage of this in a procedure like: 

    Sub DisplayStartPoints(eleComplex As ComplexShapeElement)
        Dim ee As ElementEnumerator
        Dim ele As element

        Set ee = eleComplex.GetSubElements
        Do While ee.MoveNext
            Set ele = ee.Current
            Debug.Print "Element ID " & DLongToString(ele.ID);
            '  Since the element has to be a ChainableElement,
            '  it is okay to access the StartPoint.
            With ele.AsChainableElement.StartPoint
                Debug.Print " at " & .X; ", " & .Y & ", " & .Z
            End With
        Loop
    End Sub
