MasterUnitsFromLatLong Method

                          

Gets the Master Unit coordinates corresponding to the given longitude, latitude, and elevation.

Syntax

Point3d = object.MasterUnitsFromLatLong (Point)

The MasterUnitsFromLatLong method syntax has these parts:

Part Description
object A valid object.
Point A GeoPoint3D expression. The geographic point (longitude, latitude, and elevation) for which to calculate the Master Unit coordinates.
