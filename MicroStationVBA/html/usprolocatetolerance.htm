LocateTolerance Property

                          

Read/Write Integer.

Gets or sets MicroStation's "location tolerance".

Syntax

object.LocateTolerance [= Integer]

The LocateTolerance property syntax has these parts:

Part Description
object A valid object.
Integer An Integer expression.

Remarks

The location tolerance is the diameter, in screen pixels, of a circle surrounding the screen pointer location. When a MicroStation command asks the user to locate an element, an element that has any part drawn inside the location tolerance circle is considered for location.

The location tolerance is also used to determine when two points are "close enough" to be considered the same for MicroStation. Since the location tolerance is based on screen coordinates, this means that two points may be "close enough" when looking at a zoomed out view, but not "close enough" when looking at a magnified view. Occasionally this causes problems for procedures that sequence commands like PLACE SHAPE, which assume the user meant to close a shape when the input point is "close enough" to the starting point. The problems surface in a zoomed out view, and can generally be prevented by setting LocateTolerance temporarily to zero and then setting it back to its original value when the operation is over.
