SetLocateCriteria Method

                          

Sets criteria that limit what elements MicroStation's locate logic will find.

Syntax

object.SetLocateCriteria LocateCriteria

The SetLocateCriteria method syntax has these parts:

Part Description
object A valid object.
LocateCriteria A LocateCriteria expression.

Remarks

MicroStation extracts information from the LocateCriteria object and uses that information for the duration of the current command.

Since MicroStation extracts the information from the LocateCriteria object instead of retaining a reference to the object, subsequent changes to the object will not affect MicroStation unless the program calls SetLocateCriteria again.

MicroStation resets all of its locate criteria everytime a new locate or primitive command starts. Therefore, a program may have to repeat calls to SetLocateCriteria.
