RemoveVertex Method

                          

Removes the specified vertex.

Syntax

object.RemoveVertex Index

The RemoveVertex method syntax has these parts:

Part Description
object A valid object.
Index A Long expression. A position in the list of vertices. Must be in the range 0 (the first vertex) to VerticesCount - 1 (the last vertex).

Remarks

In this object library, most index values are one-based. However, the index values that you give to InsertVertex, ModifyVertex and RemoveVertex are zero-based.
