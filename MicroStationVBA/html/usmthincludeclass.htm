IncludeClass Method

                          

Include elements of the specified class in the forthcoming scan.

Syntax

object.IncludeClass ElemClass

The IncludeClass method syntax has these parts:

Part Description
object A valid object.
ElemClass An MsdElementClass expression.

Remarks

By default, elements of every class are included in the scan. To limit the scan to a few element classes, first call ExcludeAllClasses, followed by a separate call to IncludeClass for each class to be included.

Calling this method automatically restricts the scan to only graphical type elements.
