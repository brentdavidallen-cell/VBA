SendDataPointForLocate Method

                          

Selects an element for the current command to operate on.

Syntax

object.SendDataPointForLocate ElementToLocate, DataPoint [, ViewSpecifier [, Qualifier]]

The SendDataPointForLocate method syntax has these parts:

Part Description
object A valid object.
ElementToLocate An Element expression. Must refer to an element that is already in the model. It cannot refer to a newly created element that has not been added to the model.
DataPoint A Point3d expression. The data point to send
ViewSpecifier Optional. A VARIANT expression.
Qualifier Optional. A Long expression.

Remarks

SendDataPointForLocate sends the data point, forcing it to select the specified element.

ElementToLocate may refer to a simple element, a complex element, or a member of the complex element.

Often the value of DataPoint is immaterial because the data point is not be used to select the element. It is often reasonable to specify Point3dZero for the data point. The value of the data point is only important if the command requires a reference point. For example, SCALE ORIGINAL uses the data point as the point to scale about.
