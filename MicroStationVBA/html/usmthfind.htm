Find Method

                          

Searches the collection for items that satisfy a specific criteria; particularly, items whose names match a specified pattern. If found, returns the corresponding item; otherwise returns Nothing.

Syntax

For the DimensionStyles object:

   Set DimensionStyle = object.Find (NamePattern [, PreviouslyFoundDimensionStyle])

For the Fonts object:

   Set Font = object.Find (Type, NamePattern [, PreviouslyFoundFont])

For the ItemType object:

   Set ItemTypeProperty = object.Find (NamePattern [, PreviouslyFoundItemTypeProperty])

For the ItemTypeLibraries object:

   Set ItemTypeLibrary = object.Find (NamePattern [, PreviouslyFoundItemTypeLibrary])

For the ItemTypeLibrary object:

   Set ItemType = object.Find (NamePattern [, PreviouslyFoundItemType])

For the Items object:

   Set ItemTypePropertyHandler = object.Find (Libname, ItemTypename [, PreviouslyFoundItem])

For the Levels object:

   Set Level = object.Find (NamePattern [, PreviouslyFoundLevel])

For the LineStyles object:

   Set LineStyle = object.Find (NamePattern [, PreviouslyFoundLineStyle])

For the RasterColorModes object:

   Set RasterColorMode = object.Find (RasterColorModePattern [, StartingRasterColorMode])

For the RasterCompressionModes object:

   Set RasterCompressionMode = object.Find (RasterCompressionModePattern [, StartingRasterCompressionMode])

For the RasterFormats object:

   Set RasterFormat = object.Find (FormatPattern [, StartingFormat])

For the Rasters object:

   Set Raster = object.Find (Path [, StartingRaster])

For the TagDefinitions object:

   Set TagDefinition = object.Find (NamePattern [, PreviouslyFoundTagDefinition])

For the TextStyles object:

   Set TextStyle = object.Find (NamePattern [, PreviouslyFoundTextStyle])

The Find method syntax has these parts:

Part Description
object A valid object.
Type An MsdFontType expression. The type of fonts to find. If you want to find all fonts regardless of the type, specify msdFontTypeUnknown.
PreviouslyFoundFont Optional. A Font expression. If Nothing (the default), returns the first item matching the given criteria. Otherwise, returns the next item found after the given item. Thus, you can call Find to find successive items that match the criteria.
NamePattern A String expression. The name of the ItemTypeProperty(s) to find. This name follows the same rules as a pattern given to Visual Basic's Like operator. (Exception: The pattern match is always case-insensitive, regardless of the Option Compare setting.)
PreviouslyFoundItemTypeProperty Optional. An ItemTypeProperty expression. If Nothing (the default), Find returns the first ItemTypeProperty whose name matches the given pattern. Otherwise, Find returns the next ItemTypeProperty found after this one. Thus, you can call Find successively to find successive ItemTypePropertys whose names match the pattern.
PreviouslyFoundItemTypeLibrary Optional. An ItemTypeLibrary expression. If Nothing (the default), Find returns the first ItemTypeLibrary whose name matches the given pattern. Otherwise, Find returns the next ItemTypeLibrary found after this one. Thus, you can call Find successively to find successive ItemTypeLibrarys whose names match the pattern.
PreviouslyFoundItemType Optional. An ItemType expression. If Nothing (the default), Find returns the first ItemType whose name matches the given pattern. Otherwise, Find returns the next ItemType found after this one. Thus, you can call Find successively to find successive ItemTypes whose names match the pattern.
Libname A String expression. The name of the ItemTypeLibrary(s) to find ItemTypePropertyHandler of its Items. This name follows the same rules as a pattern given to Visual Basic's Like operator. (Exception: The pattern match is always case-insensitive, regardless of the Option Compare setting.)
ItemTypename A String expression. name of ItemType(s) to find ItemTypePropertyHandler for its Items, This name follows the same rules as a pattern given to Visual Basic's Like operator. (Exception: The pattern match is always case-insensitive, regardless of the Option Compare setting.)
PreviouslyFoundItem Optional. An ItemTypePropertyHandler expression. If Nothing (the default), Find returns the first ItemTypePropertyHandler whose name matches the given pattern. Otherwise, Find returns the next ItemTypePropertyHandler found after this one. Thus, you can call Find successively to find successive ItemTypePropertyHandlers whose names match the pattern.
PreviouslyFoundLevel Optional. A Level expression. If Nothing (the default), returns the first item matching the given criteria. Otherwise, returns the next item found after the given item. Thus, you can call Find to find successive items that match the criteria.
PreviouslyFoundLineStyle Optional. A LineStyle expression. If Nothing (the default), returns the first item matching the given criteria. Otherwise, returns the next item found after the given item. Thus, you can call Find to find successive items that match the criteria.
RasterColorModePattern A String expression. The pattern associated with the color mode.
StartingRasterColorMode Optional. A RasterColorMode expression. If Nothing (the default), returns the first item matching the given criteria. Otherwise, returns the next item found after the given item. Thus, you can call Find to find successive items that match the criteria.
RasterCompressionModePattern A String expression. The pattern associated with the compression mode.
StartingRasterCompressionMode Optional. A RasterCompressionMode expression. If Nothing (the default), returns the first item matching the given criteria. Otherwise, returns the next item found after the given item. Thus, you can call Find to find successive items that match the criteria.
FormatPattern A String expression. The pattern associated with the raster format.
StartingFormat Optional. A RasterFormat expression. If Nothing (the default), returns the first item matching the given criteria. Otherwise, returns the next item found after the given item. Thus, you can call Find to find successive items that match the criteria.
Path A String expression. The access path of a required raster.
StartingRaster Optional. A Raster expression. If Nothing (the default), returns the first item matching the given criteria. Otherwise, returns the next item found after the given item. Thus, you can call Find to find successive items that match the criteria.
PreviouslyFoundDimensionStyle Optional. A DimensionStyle expression.
PreviouslyFoundTagDefinition Optional. A TagDefinition expression.
PreviouslyFoundTextStyle Optional. A TextStyle expression.

Remarks

The following example finds and displays the names of all of the Windows TrueType fonts whose names contain the word "symbol" (case-insensitive): 

    Dim oFont As Font
    Do
        Set oFont = ActiveDesignFile.Fonts.Find(msdFontTypeWindowsTrueType, "*symbol*", oFont)
        If oFont Is Nothing Then Exit Do
        Debug.Print oFont.Name
    Loop
Sample output (appearing in the immediate window of the Visual Basic Editor): 
    Bookshelf Symbol 1
    Map Symbols
    Symbol
    WP TypographicSymbols

You can find one specific ItemTypeProperty by simply giving its name, without any wildcard characters.

You can successively find all ItemTypeProperties by giving the name pattern, "*".

You can find one specific ItemTypeLibrary by simply giving its name, without any wildcard characters.

You can successively find all ItemTypeLibraries by giving the name pattern, "*".

You can find one specific ItemType by simply giving its name, without any wildcard characters.

You can successively find all ItemTypes by giving the name pattern, "*".

You can find one specific ItemTypePropertyHandler by simply giving ItemTypeLibrary name and Itemtype name, without any wildcard characters.

You can successively find all ItemTypePropertyHandlers by giving the ItemTypeLibrary name and Itemtype name , "*".
