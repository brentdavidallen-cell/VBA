Width Property

                          

Read/Write Long for Application.

Read/Write Double for SheetDefinition, TagElement, TextStyle.

Syntax

object.Width [= Long]

The Width property syntax has these parts:

Part Description
object A valid object.
Long A Long expression.
Double A Double expression.
Width A Double expression.

Remarks

Application.Width gets or sets the width of MicroStation's main window. The value is measured in screen pixels.
