LowerSuffixCharSymbol Property

                          

Read/Write String.

Gets this object's lower suffix character symbol.

Syntax

object.LowerSuffixCharSymbol [= Char]

The LowerSuffixCharSymbol property syntax has these parts:

Part Description
object A valid object.
Char A String expression.

Remarks

This is used only for dual dimensions.
