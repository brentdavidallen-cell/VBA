RasterFormats Object

                                                     

A collection of RasterFormat objects. It gives access to all raster file formats available for export or import. It is used like this: 


    'With a variable:
    Dim theFormats As RasterFormats

    Set theFormats = RasterManager.RasterFormats
    theFormats.[method/property]

    'Without a variable:
    RasterManager.RasterFormats.[method/property]

Remarks

It is possible to step through the collection using the For Each statement like this: 


    Dim oFormat As RasterFormat
    Dim theFormats As RasterFormats

    Set theFormats = RasterManager.RasterFormats

    For Each oFormat In theFormats
        oFormat.[method/property]
    Next
