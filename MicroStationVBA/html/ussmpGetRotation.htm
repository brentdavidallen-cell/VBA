Extracting Rotation for any Element Type

The example shows how to get the rotation for any element type that supports the rotation method. 


Function GetRotation(ele As Element) As Matrix3d
    If ele.IsEllipticalElement Then
        '  Both Arc and Ellipse are elliptical
        GetRotation = ele.AsEllipticalElement
    ElseIf ele.IsCellElement Then
        GetRotation = ele.AsCellElement
    ElseIf ele.IsConeElement Then
        GetRotation = ele.AsConeElement.Rotation
    ElseIf ele.IsTextElement Then
        GetRotation = ele.AsTextElement.Rotation
    ElseIf ele.IsTextNodeElement Then
        GetRotation = ele.AsTextNodeElement.Rotation
    ElseIf ele.IsSharedCellElement Then
        GetRotation = ele.AsSharedCellElement.Rotation
    ElseIf ele.IsSharedCellDefinitionElement Then
        GetRotation = ele.AsSharedCellDefinitionElement.Rotation
    ElseIf ele.IsTagElement Then
        GetRotation = ele.AsTagElement.Rotation
    Else
        Err.Raise &H80004005, "GetRotation Method", "Wrong Element Type"
    End If
End Function
