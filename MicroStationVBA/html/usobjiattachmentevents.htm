IAttachmentEvents Object

                                                     

Lets programs monitor all of the changes to reference file attachments.

To declare that a class is an event handler for reference file attachment events, use the statement Implements IAttachmentEvents. To activate the event handler, the program must call AddAttachmentEventsHandler passing in an instance of the class that implements IAttachmentEvents. To deactivate the event handler, the program must call RemoveAttachmentEventsHandler , passing in the same object that was passed to AddAttachmentEventsHandler.

The event handler remains active until the program deactivates it using RemoveAttachmentEventsHandler, or until the the project is unloaded or is reset.
