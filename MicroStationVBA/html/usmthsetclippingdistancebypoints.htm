SetClippingDistanceByPoints Method

                          

Sets Origin and Extents.Z to the appropriate values so that the front clipping plane contains one of these points and the back clipping plane contains the other.

Syntax

object.SetClippingDistanceByPoints Point1, Point2

The SetClippingDistanceByPoints method syntax has these parts:

Part Description
object A valid object.
Point1 A Point3d expression.
Point2 A Point3d expression.

Remarks

Point1 may specify either the front or back clipping plane. SetClippingDistanceByPoints finds the planes that contain these points and are parallel to the view. Then it computes the display depths as the locations where the planes intersect on the View's Z-axis. It uses the smaller value as the back display depth and the larger value as the front display depth.

See View for information on the relationship of Origin, Extents, ActiveDepth, and Rotation.
