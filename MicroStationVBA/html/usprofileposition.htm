FilePosition Property

                          

Read-only Long.

Gets this element's file position.

Syntax

object.FilePosition

The FilePosition property syntax has these parts:

Part Description
object A valid object.

Remarks

This property is provided only to facilitate the conversion of legacy MicroStation BASIC applications to VBA. New VBA projects should not use file positions. (The "modern" way to uniquely identify an element is by its ID.)
