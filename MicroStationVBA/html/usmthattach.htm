Attach Method

                          

Attaches a raster and returns True if the operation succeeded. Only the file name is mandatory (file name = complete access path). For example: 


    Dim oRasterManager As RasterManager
    Dim oRaster As Raster
    Dim theViews as Byte

    Set oRasterManager = RasterManager
    theViews = msdView1+msdView4+msdView5

    'Here the raster note.bmp will be attached in views 1, 4, 5, its logical name will be:
    '"First" and its description will be:  "It's a test".
    'By default the access mode is MsdFileAccessModeRead/Write.
    oRasterManager.Rasters.Attach "C:\...\note.bmp", theViews, false, , "First", "It's a test"

    'Only the first parameter is mandatory.
    oRasterManager.Rasters.Attach "C:\...\pencil.bmp"

    'Get the newly attached raster.
    Set oRaster =  oRasterManager.Rasters.Attach("C:\...\note.bmp")

Syntax

Set Raster = object.Attach (FileName [, Views [, RasterAttachMode [, FileAccessMode [, LogicalName [, Description]]]]])

The Attach method syntax has these parts:

Part Description
object A valid object.
FileName A String expression. The access path.
Views Optional. A Byte expression. One or more views where the raster will be posted. Each bit of the byte represents a view. For easy use of this parameter, utilize the MsdViews enumeration which contains one item for each view. Add the items to MsdViews to pass more than one view in the parameter.
RasterAttachMode Optional. An MsdAttachMode expression. Indicates the type of raster attachment.
FileAccessMode Optional. An MsdFileAccessMode expression. Indicates if the access to the rasters is in read/write or read-only mode. If the file is already opened in read/write the mode will be changed to read-only.
LogicalName Optional. A String expression. The raster's logical name. A maximum of 23 characters is allowed. Extra characters will be truncated.
Description Optional. A String expression. The raster's description. A maximum of 41 characters is allowed. Extra characters will be truncated.
