Settings Object

                                                     

The Settings object is used to control MicroStation's active settings. Some examples of MicroStation's active settings are Active Angle, Active Cell, and Active Color.

Changes to a setting via the Settings object behave just like settings entered from MicroStation's user interface in that the changes take effect immediately, but they are not saved to design file until a Save Settings command is executed

There is just one Settings object. A program can access it via ActiveSettings of Application.

Users of the original MicroStation Basic will be accustomed to accessing these settings via MbeSettings.
