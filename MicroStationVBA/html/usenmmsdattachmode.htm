MsdAttachMode Enum

        

Defines constants that determine the type of raster attachment.

The MsdAttachMode enum has these members:

Name Description
msdAttachNone 1 (&H1) The raster is attached to the design file temporarily.
msdAttachReference 3 (&H3) The raster is attached to the design file using a raster reference element.
