IncludeOnlyCell Method

                          

Restricts the forthcoming scan to only those elements that are instances of the named cell.

Syntax

object.IncludeOnlyCell CellName

The IncludeOnlyCell method syntax has these parts:

Part Description
object A valid object.
CellName A String expression.

Remarks

Calling this method automatically restricts the scan to only graphical type elements.

The scanner looks for instances of regular cells and shared cells with the given name. A program could use the following function to decide if there are any instances of a given cell. 

Function AnyInstancesOfCell(Name As String, CheckForSharedCells As Boolean, CheckForRegularCells As Boolean) As Boolean
    Dim ee As ElementEnumerator
    Dim esc As New ElementScanCriteria

    esc.IncludeOnlyCell Name

    esc.ExcludeAllTypes
    If CheckForSharedCells Then esc.IncludeType msdElementTypeSharedCell
    If CheckForRegularCells Then esc.IncludeType msdElementTypeCellHeader

    Set ee = ActiveModelReference.Scan(esc)
    AnyInstancesOfCell = ee.MoveNext
End Function
