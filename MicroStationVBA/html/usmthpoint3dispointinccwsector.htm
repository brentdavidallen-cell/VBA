Point3dIsPointInCCWSector Method

                          

Test if a point is within the counter-clockwise sector defined by an origin and two boundary points, with an up vector to determine which direction is counter clockwise.

Syntax

Boolean = object.Point3dIsPointInCCWSector (TestPoint, Origin, Target0, Target1, UpVector)

The Point3dIsPointInCCWSector method syntax has these parts:

Part Description
object A valid object.
TestPoint A Point3d expression. The point to test.
Target0 A Point3d expression. The first target point.
Target1 A Point3d expression. The second target point.
Origin A Point3d expression.
UpVector A Point3d expression.
