Hiding an Element

This example shows how to hide a graphical element. After this, the element will no longer be visible. Subsequent view updates will not display the element unless IsHidden is set back to True. 


Sub HideElement(ele As Element)
    If ele.IsHidden Then Exit Sub

    ele.Redraw msdDrawingModeErase
    ele.IsHidden = True
    ele.Rewrite
End Sub
