AddViewUpdateEventsHandler Method

                          

Adds a view-update-events event handler to the set of active handlers.

Syntax

object.AddViewUpdateEventsHandler EventHandler

The AddViewUpdateEventsHandler method syntax has these parts:

Part Description
object A valid object.
EventHandler An IViewUpdateEvents expression.

Remarks

The event handler remains active until it is terminated with RemoveViewUpdateEventsHandler, or the object becomes invalid. If the event handler is implemented in VBA, the object becomes invalid if the project is reset manually or due to an error.
