DefineFromElement Method

                          

Syntax

object.DefineFromElement View, ClosedElement

The DefineFromElement method syntax has these parts:

Part Description
object A valid object.
View A View expression.
ClosedElement An Element expression.

Remarks

The element must be a closed element, i.e., shape, ellipse, complex shape, or closed B-spline curve. Non-linear segments of the closed element, if any, are stroked into multiple linear segments.

The following example shows how to use DefineFromElement: 

Private Sub fenceDraw(ByRef oElemID As DLong)
    Dim oFence As Fence
    Dim oElem As Element
    Dim myView As View

    Set oFence = ActiveDesignFile.Fence
    oFence.Undefine

    Set myView = CommandState.LastView

    Set oElem = ActiveDesignFile.GetElementByID(oElemID)

    oFence.DefineFromElement myView, oElem
    oFence.Draw
End Sub
