IsExportable Property

                          

Read-only Boolean.

Is this combination of format, color mode, compression mode and block type is supported in export ?

Syntax

object.IsExportable (FileTypeID, ColorModeID, CompressionModeID, BlockType)

The IsExportable property syntax has these parts:

Part Description
object A valid object.
FileTypeID A Long expression.
ColorModeID A Long expression.
CompressionModeID A Long expression.
BlockType An MsdRasterBlockType expression.
