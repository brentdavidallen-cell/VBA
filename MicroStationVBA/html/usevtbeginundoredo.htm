BeginUndoRedo Event

                          

Event handler called whenever an element is changed by either the Undo or Redo command in MicroStation

Use AddChangeTrackEventsHandler and RemoveChangeTrackEventsHandler to add and remove a Change Track events handler. A Change Track events handler is an instance of a class that implements the IChangeTrackEvents interface.

Syntax

object.BeginUndoRedo AfterUndoRedo, BeforeUndoRedo, Action, IsUndo

The BeginUndoRedo method syntax has these parts:

Part Description
object A valid object.
AfterUndoRedo An Element expression.
BeforeUndoRedo An Element expression.
Action An MsdChangeTrackAction expression.
IsUndo A Boolean expression.
