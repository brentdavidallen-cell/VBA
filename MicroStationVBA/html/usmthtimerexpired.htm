TimerExpired Method

                          

MicroStation calls the TimerExpired event handler when the requested timer interval has expired.

Syntax

object.TimerExpired RestartTimer

The TimerExpired method syntax has these parts:

Part Description
object A valid object.
RestartTimer A Boolean expression. Set this to True to cause MicroStation to restart the timer; otherwise MicroStation does not restart the timer. Instead, it calls RemoveTimerEventHandler for the handler.

Remarks

See ITimerEvent for more information.
