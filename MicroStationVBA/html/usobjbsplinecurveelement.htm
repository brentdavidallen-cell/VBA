BsplineCurveElement Object

                                                     

A kind of Element that corresponds to a MicroStation B-spline curve element.

Remarks

If you have a generic Element object (as the result of a Scan, for example), you can determine whether it's a BsplineCurveElement by examining the element's IsBsplineCurveElement property. If this property returns True, you can extract the element's implementation of BsplineCurveElement by calling the element's AsBsplineCurveElement property. (You can accomplish the same thing by assigning the element to a variable of the type BsplineCurveElement. But Element.AsBsplineCurveElement saves you a step.)

You cannot use the New keyword to create a BsplineCurveElement object. Rather, use New to create a BsplineCurve (or InterpolationCurve) object, populate its properties, then create the BsplineCurveElement using CreateBsplineCurveElement1 (or CreateBsplineCurveElement2).

The new element will not become part of the design file until it is added to a model (using AddElement on the appropriate ModelReference object). 


    Dim oBsplineCurve As New BsplineCurve
    With oBsplineCurve
        ' Set properties of oBsplineCurve here...
    End With

    ' Create the element
    Dim oBsplineCurveElement As BsplineCurveElement
    Set oBsplineCurveElement = CreateBsplineCurveElement1(Nothing, oBsplineCurve)

    ' Add the element to the active model (whether or not it's displayed)
    ActiveModelReference.AddElement oBsplineCurveElement

    ' Modify and update the element in its model
    With oBsplineCurve
        ' Modify properties of oBsplineCurve here...
    End With
    oBsplineCurveElement.UpdateFromBsplineCurve oBsplineCurve

    ' Display the element (whether or not it's in a model)
    oBsplineCurveElement.Redraw
