IncludeOnlyFilePositionRange Method

                          

Include elements within the specified Minimum and Maximum file position.

Syntax

object.IncludeOnlyFilePositionRange Min, Max

The IncludeOnlyFilePositionRange method syntax has these parts:

Part Description
object A valid object.
Min A Long expression. The starting file position.
Max A Long expression. The ending file position.

Remarks

The Min and Max parameter values are inclusive.
