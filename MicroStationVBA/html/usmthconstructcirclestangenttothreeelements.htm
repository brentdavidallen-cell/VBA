ConstructCirclesTangentToThreeElements Method

                          

Creates and returns an ElementEnumerator of new Elements that represent circles tangent to the three given coplanar Elements.

Syntax

Set ElementEnumerator = object.ConstructCirclesTangentToThreeElements (Element1, Element2, Element3, Template [, OutputType [, SamplesCount]])

The ConstructCirclesTangentToThreeElements method syntax has these parts:

Part Description
object A valid object.
Element1 An Element expression. The first of three elements to which the returned circles are tangent.
Element2 An Element expression. The second of three elements to which the returned circles are tangent.
Element3 An Element expression. The third of three elements to which the returned circles are tangent.
Template An Element expression. An existing element whose settings are used to initialize the new elements. If Nothing, the new elements' settings are initialized from MicroStation's active settings.
OutputType Optional. An MsdTangentElementOutputType expression. Determines the Type (EllipseElement, ArcElement or ShapeElement) of the Elements in the returned ElementEnumerator object. If omitted, a default value of msdTangentCircles is used.
SamplesCount Optional. A Long expression. Determines how many parameter space samples (default 10) to use in computing initial root approximations. A higher value will, up to a point, produce more returned circles at the cost of greater computation time.

Remarks

Not all possible circles may be returned by this method, and some circles may be identical.
