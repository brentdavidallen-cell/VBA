View Property

                          

Read-only View.

Syntax

object.View

The View property syntax has these parts:

Part Description
object A valid object.

Remarks

When called with a CadInputMessage object that represents a data point or unassigned cursor button, View represents the view where the user entered the point. Do not use this property of a CadInputMessage object unless the InputType indicates that the user entered a data point.
