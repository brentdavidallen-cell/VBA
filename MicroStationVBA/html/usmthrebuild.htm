Rebuild Method

                          

Rebuilds this object in place by approximating it with a smoother, better parametrized object.

Syntax

object.Rebuild [Order [, KeepEndTangents [, Parametrization [, ChordHeightTolerance [, ParametricTolerance [, PointCoincidenceTolerance]]]]]]

The Rebuild method syntax has these parts:

Part Description
object A valid object.
Order Optional. A Long expression. The Order of the rebuilt object. If omitted, a default order of 4 (for a cubic curve) is used.
KeepEndTangents Optional. A Boolean expression. Indicates whether the start and end tangents of this object are preserved in the rebuilt object. If omitted, a default value of True is used.
Parametrization Optional. An MsdBsplineParametrizationType expression. The knot parametrization of the rebuilt object. Only the values msdBsplineParametrizationChordLength, msdBsplineParametrizationCentripetal and msdBsplineParametrizationInherited are valid. If omitted or invalid, msdBsplineParametrizationChordLength is used.
ChordHeightTolerance Optional. A Double expression. The maximum perpendicular distance between original and rebuilt objects. If omitted or negative, a default tolerance of 1.0e-4 times the largest coordinate difference of the unweighted Poles of this object is used.
ParametricTolerance Optional. A Double expression. Unused.
PointCoincidenceTolerance Optional. A Double expression. Unused.

Remarks

For BsplineCurve objects, if this object is G1-continuous (continuity of position and tangent direction, if not tangent length) then the resulting object is at least C1-continuous (continuity of position, tangent direction and tangent length). This method is useful for producing a nonRational curve from a Rational one, and for approximate curve degree-reduction.
