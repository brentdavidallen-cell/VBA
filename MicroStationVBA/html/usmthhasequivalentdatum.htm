HasEquivalentDatum Method

                          

Returns True if this GeographicCoordinateSystem has Datum equivalent to that of the given GeographicCoordinateSystem

Syntax

Boolean = object.HasEquivalentDatum (Other)

The HasEquivalentDatum method syntax has these parts:

Part Description
object A valid object.
Other A GeographicCoordinateSystem expression. The GeographicCoordinateSystem to test for datum equivalence.
