GetLocatedElement Method

                          

GetLocatedElement attempts to read and return the last element that the user interactively located.

Syntax

Set Element = object.GetLocatedElement (WantComponentHeader)

The GetLocatedElement method syntax has these parts:

Part Description
object A valid object.
WantComponentHeader A Boolean expression. Relevant only if the selected element is a component of a ComplexElement. Indicates whether GetLocatedElement returns the header of the complex element, rather than the component.

Remarks

See Interaction of HitPoint, LocateElement, and GetLocatedElement for an example that illustrates how to use these methods together.
