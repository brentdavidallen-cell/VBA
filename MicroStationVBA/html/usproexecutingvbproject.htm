ExecutingVBProject Property

                          

Read-only Object.

Gets a VBProject object for the currently executing VBA project.

Syntax

object.ExecutingVBProject

The ExecutingVBProject property syntax has these parts:

Part Description
object A valid object.

Remarks

ExecutingVBProject is often different from VBE's ActiveVBProject. ActiveVBProject gets the VBProject object of the project most recently accessed from the VBA IDE.

You must add a reference to Microsoft Visual Basic for Applications Extensibility 5.3 to a project to be able to use ExecutingVBProject in that project.

The following example shows how to use ExecutingVBProject to get the file name of the currently executing project. 

Function GetVBAFileName() As String
    Dim oVB As VBProject

    Set oVB = ExecutingVBProject
    GetVBAFileName = oVB.FileName
End Function
