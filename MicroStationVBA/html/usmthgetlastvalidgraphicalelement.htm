GetLastValidGraphicalElement Method

                          

Returns the last Element in the ModelReference's GraphicalElementCache.

Syntax

Set Element = object.GetLastValidGraphicalElement

The GetLastValidGraphicalElement method syntax has these parts:

Part Description
object A valid object.

Remarks

This is a shortcut for ModelReference.GraphicalElementCache.GetLastValidElement.

It will not return a component or a deleted element. It returns the last accessible element that is not a component. It may be a complex element that contains components.

It raises an error if the cache does not contain any valid elements.
