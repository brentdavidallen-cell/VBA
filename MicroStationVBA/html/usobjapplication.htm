Application Object

                                                     

The top-level object in MicroStation's object model.

Remarks

All of the methods and properties of Application are in the global namespace of your VBA project. Therefore, you can access these methods and properties without having to explicitly reference the Application object. For example, the expression ActiveModelReference is equivalent to Application.ActiveModelReference.

See ApplicationObjectConnector for information about accessing the Application object from a separate process.
