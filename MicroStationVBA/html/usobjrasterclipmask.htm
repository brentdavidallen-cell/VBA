RasterClipMask Object

                                                     

A RasterClipMask is a closed, non-intersecting shape that creates a hole in a Raster. The part of the Raster inside the clip mask is invisible. It is also a VertexList and as such, it is possible to access, insert, remove and modify the RasterClipMask vertices.

Remarks

Use methods of RasterClipMasks to get a RasterClipMask object as follows: 


    'Instantiation with an Item :

    Dim theClipMasks As RasterClipMasks
    Dim oClipMask As RasterClipMask

        ...

    Set oClipMask = theClipMasks.Item(2)
