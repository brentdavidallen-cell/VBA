Closed Property

                          

Read/Write Boolean.

Gets or sets a value indicating whether this object is closed.

Syntax

object.Closed [= Boolean]

The Closed property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

For BsplineCurve objects, if this property is True, then the B-spline curve is physically closed (and may additionally be periodically defined); if this property is False, then the B-spline curve may still be physically closed, but never periodically defined.

For BsplineCurve objects, setting this property does not in general preserve the shape of the B-spline curve: the curve's Poles are preserved but its Knots are recomputed. To change the closure of a BsplineCurve while maintaining its shape, use the methods Close and Open.
