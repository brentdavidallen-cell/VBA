CommandKeyin Property

                          

Read-only String.

If the CadInputMessage object represents a command, then CommandKeyin represents the string that can be entered in MicroStation's key-in area to start the command.

Syntax

object.CommandKeyin

The CommandKeyin property syntax has these parts:

Part Description
object A valid object.

Remarks

Do not call this method unless InputType indicates that the user entered a command key-in.
