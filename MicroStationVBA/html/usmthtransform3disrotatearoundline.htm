Transform3dIsRotateAroundLine Method

                          

Returns true if the transform is a non-zero rotation around a line.

Syntax

Boolean = object.Transform3dIsRotateAroundLine (Transform, FixedPoint, DirectionVector, Radians)

The Transform3dIsRotateAroundLine method syntax has these parts:

Part Description
object A valid object.
Transform A Transform3d expression. IN The transformation to test
FixedPoint A Point3d expression. OUT a point on the line.
DirectionVector A Point3d expression. OUT vector in the line direction.
Radians A Double expression. OUT rotation angle in radians.

Remarks

Returns true if the transformation is a non-zero rotation.
