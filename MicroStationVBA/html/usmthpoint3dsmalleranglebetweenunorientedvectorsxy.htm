Point3dSmallerAngleBetweenUnorientedVectorsXY Method

                          

Returns the angle between two vectors, considering both the vectors and their negations and choosing the smaller. This angle is between 0 and pi/2.

Syntax

Double = object.Point3dSmallerAngleBetweenUnorientedVectorsXY (Vector1, Vector2)

The Point3dSmallerAngleBetweenUnorientedVectorsXY method syntax has these parts:

Part Description
object A valid object.
Vector1 A Point3d expression. The first vector
Vector2 A Point3d expression. The second vector
