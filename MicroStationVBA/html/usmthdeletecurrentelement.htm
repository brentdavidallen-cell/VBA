DeleteCurrentElement Method

                          

Deletes the current element from the celll.

Syntax

object.DeleteCurrentElement

The DeleteCurrentElement method syntax has these parts:

Part Description
object A valid object.

Remarks

DeleteCurrentElement sets the current-element marker to point to the element preceding the element that it deletes. Therefore, the next call to MoveToNextElement moves the current-element marker to the element following the deleted element.

This routine deletes all of the text elements from a cell. 

Sub DeleteText(ele As CellElement)
    ele.ResetElementEnumeration
    Do While ele.MoveToNextElement(True)
        Dim eleTemp As Element
        Set eleTemp = ele.CopyCurrentElement
        If eleTemp.IsTextElement Then
            eleTemp.Redraw msdDrawingModeErase
            ele.DeleteCurrentElement
        End If
    Loop
    ele.Rewrite
End Sub
