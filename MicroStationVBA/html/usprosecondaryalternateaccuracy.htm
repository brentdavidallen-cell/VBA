SecondaryAlternateAccuracy Property

                          

Read/Write MsdDimAccuracy.

Gets this object's secondary alternate accuracy.

Syntax

object.SecondaryAlternateAccuracy [= MsdDimAccuracy]

The SecondaryAlternateAccuracy property syntax has these parts:

Part Description
object A valid object.
MsdDimAccuracy An MsdDimAccuracy expression.

Remarks

See MsdDimAccuracy for possible valid values.
