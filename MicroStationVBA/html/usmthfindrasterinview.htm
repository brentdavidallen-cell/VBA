FindRasterInView Method

                          

This function return the raster displayed on the top according to view number and a localization point. The transparency does not change the behavior of this function. If a raster has a transparent part covering another raster, the selection of a localization point in the transparent part returns the raster displayed on the top (the one with the transparent part). Returns Null if no raster is found.

Syntax

Set Raster = object.FindRasterInView (InPoint [, ViewSpecifier])

The FindRasterInView method syntax has these parts:

Part Description
object A valid object.
InPoint A Point3d expression. A localization point where to search for the raster, in masters units.
ViewSpecifier Optional. A VARIANT expression. The index of the view (1 through 8) or a View object where to search for the raster. 

    Dim oPoint As Point3d
    Dim oRaster As Raster
    oPoint.X = 2.8
    oPoint.Y = 5.4
    Set oRaster = RasterManager.Rasters.FindRasterInView(oPoint, 1)
    MsgBox oRaster.RasterInformation.FileName
