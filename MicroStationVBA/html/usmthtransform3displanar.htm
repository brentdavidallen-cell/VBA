Transform3dIsPlanar Method

                          

Returns true if transformation effects are entirely within the plane with given normal.

Syntax

Boolean = object.Transform3dIsPlanar (Transform, Normal)

The Transform3dIsPlanar method syntax has these parts:

Part Description
object A valid object.
Transform A Transform3d expression. The transform to be tested for out-of-plane effects.
Normal A Point3d expression. The plane normal
