MaxRasterLogicalNameLength Property

                          

Read-only Long.

Gets the maximum possible length for a raster's ExtendedInformation.LogicalName. 


    Dim oName As String

    oName = RasterManager.Rasters.Item(1).ExtendedInformation.LogicalName
    oName = "New " + oName

    If (Len(oName) < RasterManager.MaxRasterLogicalNameLength) Then
        RasterManager.Rasters.Item(1).ExtendedInformation.LogicalName = oName
    End If

Syntax

object.MaxRasterLogicalNameLength

Part Description
object A valid object.
