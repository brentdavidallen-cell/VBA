CellElement Object

                                                     

A kind of Element that corresponds to a MicroStation cell element.

Remarks

You cannot use the New keyword to create a CellElement object. Rather, use one of the global functions CreateCellElement1 or CreateCellElement2.

The new element will not become part of the design file until it is added to a model (using AddElement on the appropriate ModelReference object).

    Dim oCell As CellElement
    Set oCell = CreateCellElement1(...)
    '
    '  ...Set other properties of oArc, if you wish...
    '
    ActiveModelReference.AddElement oCell
    oCell.Redraw ' Display the newly-added element.

CellElement has the methods AppendToCellOfCurrentElement, CopyCurrentElement, DeleteCurrentElement, GetMarkerFromCurrentElement, MarkCurrentElement, MoveToMarkedElement, MoveToNextElement, and ReplaceCurrentElement that provide a way to directly manipulate the contents of a CellElement.

These methods use an internal marker to select a current element. AppendToCellOfCurrentElement appends an element to the cell that contains the current element. CopyCurrentElement extracts a copy of the current element. ReplaceCurrentElement replaces the current element with the element passed to it. DeleteCurrentElement removes the current element from the cell. MarkCurrentElement stores a program-defined marker on the current element. GetMarkerFromCurrentElement retrieves the marker from the current element. MoveToMarkedElement searches for a component element that is marked with the specified marker. If it finds an element with that marker, it adjusts the internal marker to select that element as the current element. MoveToNextElement selects the next element as the current element. MoveToMarkedElement is very similar to MoveNext.

All of the markers including both the internal and program-defined markers are saved with the in-memory copy of the element. The markers cannot be saved back to the DesignFile. If the program retrieves the CellElement again, all of the markers will be gone.
