RotationAboutZ Property

                          

Read/Write Double.

Allows to obtain or to adjust the rotation applied to a raster.

Syntax

object.RotationAboutZ [= Double]

The RotationAboutZ property syntax has these parts:

Part Description
object A valid object.
Double A Double expression. Angle of the raster's rotation in radians.

Remarks

The rotation is expressed in radians [0...2PI[. Application.Radians can be used to convert the angle from degrees to radians. 


    Dim theGeo As GeoReferenceInformation

    Set theGeo = RasterManager.Rasters.Item(1). GeoReferenceInformation
    TheGeo.RotationAboutZ = 0.7854
    MsgBox "New rotation is: " & theGeo.RotationAboutZ
