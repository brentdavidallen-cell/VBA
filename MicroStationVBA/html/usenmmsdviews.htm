MsdViews Enum

        

Defines constants that represent one of the 8 views.

The MsdViews enum has these members:

Name Description
msdViewNone 0 (&H0) Represents no view.
msdView1 1 (&H1) Represents view 1.
msdView2 2 (&H2) Represents view 2.
msdView3 4 (&H4) Represents view 3.
msdView4 8 (&H8) Represents view 4.
msdView5 16 (&H10) Represents view 5.
msdView6 32 (&H20) Represents view 6.
msdView7 64 (&H40) Represents view 7.
msdView8 128 (&H80) Represents view 8.
msdViewAll 255 (&HFF) Represents all 8 views.
