ApplyRotation Property

                          

Read/Write Boolean.

Gets or sets the ApplyRotation state. 


    Dim rendInf As RenderingInformation

    Set rendInf = RasterManager.Rasters.Item(1).RenderingInformation

    If (rendInf.ApplyRotation = True) Then
        rendInf.ApplyRotation = False
    End If

Syntax

object.ApplyRotation [= Boolean]

The ApplyRotation property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression. Is the rotation applied?
