HasSettings Property

                          

Read-only Boolean.

Used to ask whether the object has settings.

Syntax

object.HasSettings

The HasSettings property syntax has these parts:

Part Description
object A valid object.

Remarks

The StandardsCheckerController uses HasSettings to determine if a Standards Checker handler has settings. If so, the StandardsCheckerController displays a Settings button on the handler's line in the Checkers portion of the main Standards Checker dialog box. If the user presses this button, the Standards Checker calls the handler's EditSettings method. The handler's EditSettings method should display a dialog box to let the user configure that portion of the Standards Checker.

Not all standard checkers have user-configurable settings. Some standards checkers have all of the standards hard-coded in the application. A checker that has all of the settings hard-coded in the application should return False for HasSettings.
