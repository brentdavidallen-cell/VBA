Segment3dPlane3dIntersect Method

                          

Compute the intersection of an unbounded segment and a plane. Return true if there is an intersection point. Rerturn false if segment is parallel or coincident to the plane.

Syntax

Boolean = object.Segment3dPlane3dIntersect (Segment, Plane, Point, Fraction)

The Segment3dPlane3dIntersect method syntax has these parts:

Part Description
object A valid object.
Segment A Segment3d expression. Segment
Plane A Plane3d expression. plane
Point A Point3d expression. Computed intersection point
Fraction A Double expression. Fractional parameter of intersection point on segment.
