CreateCellElement3 Method

                          

Creates a CellElement by searching the attached cell libraries for the specified name.

Syntax

Set CellElement = object.CreateCellElement3 (CellName, Origin, TrueScale)

The CreateCellElement3 method syntax has these parts:

Part Description
object A valid object.
CellName A String expression. The name of the cell.
Origin A Point3d expression. Where to place the cell.
TrueScale A Boolean expression. Indicates whether MicroStation automatically scales to convert between different working units of the design file and the cell library. It does this before scaling as directed by the Scale argument.

Remarks

Important: The geometric arguments are given in the coordinate system of the currently active model. Therefore, before you call this method, you should ensure that the model to which you intend to add the element is the active model. (You can make a model active by calling the Activate method of the appropriate ModelReference object.)
