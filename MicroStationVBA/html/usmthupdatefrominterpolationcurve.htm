UpdateFromInterpolationCurve Method

                          

Updates this object with the data from the given InterpolationCurve object.

Syntax

object.UpdateFromInterpolationCurve Curve

The UpdateFromInterpolationCurve method syntax has these parts:

Part Description
object A valid object.
Curve An InterpolationCurve expression. The interpolation curve with which to update this object.

Remarks

If this object has been written to the active model (typically, ActiveModelReference.AddElement), then this method updates the active model with the object's new state.
