Matrix3dRotationFromRowZ Method

                          

Returns a rotation matrix that preserves the direction of the given vector in the last row (RowZ).

Syntax

Matrix3d = object.Matrix3dRotationFromRowZ (Normal)

The Matrix3dRotationFromRowZ method syntax has these parts:

Part Description
object A valid object.
Normal A Point3d expression. A vector, not necessarily normalized. The zero vector yields the identity matrix.

Remarks

The returned Matrix3d represents a change of basis from world to local coordinates, where the local basis vectors (in world coordinates) are the matrix rows.

The returned Matrix3d is orthogonal: its inverse is its transpose.

If x is a vector in the plane that passes through the world origin and has the given Normal, and R is the Matrix3d returned by this method, then the product Rx is a vector in the xy-plane.
