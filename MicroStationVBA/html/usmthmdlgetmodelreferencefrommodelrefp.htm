MdlGetModelReferenceFromModelRefP Method

                          

Retrieves the associated ModelReference object. Most VB/VBA programs will not use this method. This is intended only for programs that use the MDL API. See Calling DLL Functions for more information.

Syntax

Set ModelReference = object.MdlGetModelReferenceFromModelRefP (ModelRefP)

The MdlGetModelReferenceFromModelRefP method syntax has these parts:

Part Description
object A valid object.
ModelRefP An Int64 expression.

Remarks

If ModelRefP is an attachment reference, MdlGetModelReferenceFromModelRefP returns an object that can also be used as an Attachment object.
