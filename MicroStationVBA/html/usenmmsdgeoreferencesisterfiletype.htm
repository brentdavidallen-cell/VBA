MsdGeoReferenceSisterFileType Enum

        

Defines constants that represent the type of sister file to use for georeference.

The MsdGeoReferenceSisterFileType enum has these members:

Name Description
msdGeoReferenceSisterFileTypeNone 0 (&H0) Represents no file being associated.
msdGeoReferenceSisterFileTypeHgr 1 (&H1) Represents an association with an HGR sister file.
msdGeoReferenceSisterFileTypeTwf 2 (&H2) Represents an association with a TIFF (ESRI) world file.
