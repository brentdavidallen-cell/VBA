IModelActivateEvents Object

                                                     

Defines the methods MicroStation uses to notify a program of model-activate events.

Remarks

To receive these events, a program must have a class that implements IModelActivateEvents. The program must create an instance of the class, and register it with MicroStation by calling AddModelActivateEventsHandler.

MicroStation calls the BeforeActivate event handlers prior to making another model the active model. MicroStation calls the AfterActivate event handlers after making another model the active model. Both of this event handlers get an argument named TheModel. When MicroStation calls a BeforeActivate event handler, TheModel is a reference to the model that is about to become the ActiveModelReference. When MicroStation calls an AfterActivate event handler, TheModel is a reference to the model that has become the ActiveModelReference. At that point, ActiveModelReference and TheModel are the same.
