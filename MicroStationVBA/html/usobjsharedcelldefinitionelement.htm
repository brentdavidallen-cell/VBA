SharedCellDefinitionElement Object

                                                     

A kind of Element that corresponds to a MicroStation shared cell definition element.

Remarks

See IncludeOnlyCell for information on how a program can find the instances of a shared cell.

When called to remove a shared cell definition, RemoveElement raises an error if the design file contains any elements that used the shared cell defintion. See IsUsed for a way to test whether the shared cell definition is used. In the XM and earlier releases, RemoveElement would remove a shared cell definition regardless of whether the definition was used. However, it left the shared cell instances in the file even though they would no longer display.
