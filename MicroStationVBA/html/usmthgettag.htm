GetTag Method

                          

Gets a particular tag association with this object. The tag is identified by its tag set and name.

For an element, calling GetTag raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.

Syntax

Set TagElement = object.GetTag (TagSet, TagName)

The GetTag method syntax has these parts:

Part Description
object A valid object.
TagSet A TagSet expression.
TagName A String expression.
