EndPrintSet Event

                          

This event occurs when MicroStation has finished printing a set. It is not invoked if not printing as a set.

Syntax

object.EndPrintSet isSuccess

The EndPrintSet method syntax has these parts:

Part Description
object A valid object.
isSuccess A Boolean expression. Indicates whether the print operation was successful.

Remarks

When printing from Print Organizer, the BeginPrintSet and EndPrintSet events are invoked in a different process than the per-page print events. Therefore, the BeginPrintSet and EndPrintSet event handlers must not share any global data with other print event handler methods.
