Segment3d Type

        

Represents a line segment in three-dimensional space.

The Segment3d type has these members:

Name Description
StartPoint A Point3d value that represents the start of the segment.
EndPoint A Point3d value that represents the end of the segment.

Remarks

For points in a ModelReference's geometry, the values are measured in the model's coordinate system, unless stated otherwise.
