MsdReferenceSystem Enum

        

Defines constants that represent the coordinate system used for vertex measurement

The MsdReferenceSystem enum has these members:

Name Description
msdReferenceSystemRaster 1 (&H1) The origin is the lower-left corner of the raster and the unit is a raster pixel. (Used for clip masks and boundaries)
msdReferenceSystemDgn 2 (&H2) The origin is the origin of the DGN file and the unit is the DGN file's master unit.
