MeasureResult1 Property

                          

Read-only Double.

Use the MeasureResult1 and MeasureResult2 properties to access the results of the last MicroStation measure command.

Syntax

object.MeasureResult1

The MeasureResult1 property syntax has these parts:

Part Description
object A valid object.
