ExtrudeClosedPlanarCurve Method

                          

This function generates a solid by extruding a given closed planar curve.

Syntax

Set SmartSolidElement = object.ExtrudeClosedPlanarCurve (planarClosedElement, forwardDist, backwardDist [, capSurface])

The ExtrudeClosedPlanarCurve method syntax has these parts:

Part Description
object A valid object.
planarClosedElement An Element expression. input closed planar curve.
forwardDist A Double expression. forward distance the curve extruding.
backwardDist A Double expression. backward distance the curve extruding.
capSurface Optional. A Boolean expression.
