ThousandsOpts Property

                          

Read/Write MsdDimThousandsOpts.

Gets this object's thousands place delimiter.

Syntax

object.ThousandsOpts [= MsdDimThousandsOpts]

The ThousandsOpts property syntax has these parts:

Part Description
object A valid object.
MsdDimThousandsOpts An MsdDimThousandsOpts expression.

Remarks

See MsdDimThousandsOpts for possible valid values.
