Transform3dIsMirrorAboutPlane Method

                          

Returns true if the transform is a mirror with respect to a plane.

Syntax

Boolean = object.Transform3dIsMirrorAboutPlane (Transform, PlanePoint, PlaneNormal)

The Transform3dIsMirrorAboutPlane method syntax has these parts:

Part Description
object A valid object.
Transform A Transform3d expression. IN The transformation to test
PlanePoint A Point3d expression. OUT Some point on the plane.
PlaneNormal A Point3d expression. OUT unit vector perpendicular to the plane.

Remarks

Returns true if the transformation is a mirror.
