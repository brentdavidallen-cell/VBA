SetFitPoints Method

                          

Sets the array of fit points of this object.

Syntax

object.SetFitPoints FitPoints

The SetFitPoints method syntax has these parts:

Part Description
object A valid object.
FitPoints A Point3d () expression. The array of fit points to set.

Remarks

For InterpolationCurve objects, see the FitPoint property for further description of this array.
