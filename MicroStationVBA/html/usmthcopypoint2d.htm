CopyPoint2d Method

                          

Copies aPoint2d to or from the DataBlock object.

Syntax

object.CopyPoint2d Point2dReference, CopyToDataBlock

The CopyPoint2d method syntax has these parts:

Part Description
object A valid object.
Point2dReference A Point2d expression.
CopyToDataBlock A Boolean expression.

Remarks

If CopyToDataBlock is True, it copies 16 bytes from Point2dReference to the data block. Otherwise, it copies 16 bytes from the data block to Point2dReference. It either case, it increments Offset by 16, the number of bytes transferred.
