DisplayFlag Property

                          

Read/Write Boolean.

Gets or set the flag that controls whether or not the object is displayed.

Syntax

object.DisplayFlag [= Boolean]

The DisplayFlag property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

Attachment.DisplayFlag corresponds to the Display column of the References dialog box. Setting Attachment.DisplayFlag to True does not guarantee that the Attachment will be displayed, since there are other reasons it may not be displayed. See ElementsVisible for more information. 


Sub FlipDisplayed(att As Attachment)
    att.DisplayFlag = Not att.DisplayFlag  ' Redraw is automatic
    att.Rewrite                            ' Save the change
End Sub
