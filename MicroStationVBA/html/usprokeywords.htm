Keywords Property

                          

Read/Write String.

Syntax

object.Keywords [= String]

The Keywords property syntax has these parts:

Part Description
object A valid object.
String A String expression.

Remarks

Keywords of a DesignFile is one of the standard Windows file properties. You can use the Windows Explorer to view this along with the other file properties. You can also use MicroStation's File Properties dialog box (available from the File pulldown) to view or edit the keyword list.
