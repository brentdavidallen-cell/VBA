UpdateFromBsplineSurface Method

                          

Updates this object with the data from the given BsplineSurface object.

Syntax

object.UpdateFromBsplineSurface Surface

The UpdateFromBsplineSurface method syntax has these parts:

Part Description
object A valid object.
Surface A BsplineSurface expression. The B-spline surface with which to update this object.

Remarks

If this object has been written to the active model (typically, ActiveModelReference.AddElement), then this method updates the active model with the object's new state.
