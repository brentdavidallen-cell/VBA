InsertArrayElements Method

                          

Insert no. of empty array elements at the starting index of specific index position for the specified array property

Syntax

Boolean = object.InsertArrayElements (AccessString, index, size)

The InsertArrayElements method syntax has these parts:

Part Description
object A valid object.
AccessString A String expression. The access string to the array property (shall not contain brackets)
index A Long expression. The starting index of the array at which to insert the new elements
size A Long expression. The number of empty array elements to insert

Remarks

InsertArrayElements inserts no. of empty array elements at the the starting index of specified array property ItemTypePropertyHandler
