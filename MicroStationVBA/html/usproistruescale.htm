IsTrueScale Property

                          

Read-only Boolean.

Syntax

object.IsTrueScale

The IsTrueScale property syntax has these parts:

Part Description
object A valid object.

Remarks

Attachment.IsTrueScale indicates whether the model was attached with True Scale turned on. If IsTrueScale is True, then units in the active model align one to one with units in the referenced model. For example, if the units in the active model are inches and the units in the referenced model are meters, meters are converted to inches so the elements are referenced in at the same relative size.
