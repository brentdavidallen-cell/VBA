Origin Property

                          

Read-only Point3d for ACSManager, AuxiliaryCoordinateSystemElement, CellElement, LineElement, PointElement, SharedCellDefinitionElement, SharedCellElement, SmartSolidElement, TagElement, TextNodeElement.

Read/Write Point3d for GeoReferenceInformation, TextElement, View.

Read/Write Point2d for SheetDefinition.

Gets or sets the object's origin.

Syntax

object.Origin [= Point3d]

The Origin property syntax has these parts:

Part Description
object A valid object.
Point3d A Point3d expression. The raster's origin.
Point2d A Point2d expression.

Remarks

See User Defined Types as Property Values for a description of a common mistake programmers make when trying to change just one coordinate of the origin.

ACSManager.Origin raises an error if there is no auxiliary coordinate system defined. A program can test for this using ACSManager.IsDefined.

For PointElement, the value is measured in the coordinate system of the containing model.

For GeoReferenceInformation, the origin is the lower-left corner expressed in master units. 


    Dim oPoint3d As Point3d
    Dim theGeo As GeoReferenceInformation

    oPoint3d.X = 1
    oPoint3d.Y = 2

    Set theGeo = RasterManager.Rasters.Item(1). GeoReferenceInformation

    theGeo.Origin = oPoint3d

    MsgBox "Origin.X: " & theGeo.Origin.X

A TagElement's origin is a point on the base element. The origin changes when the base element is moved.

TextElement.Origin is actually the User Origin value displayed in the Element Info dialog. TextElement does not provide a property that provides access to the value displayed as Origin. However, it is possible to get that value using a PropertyHandler: 


    Dim originValue As Point3d
    Dim oPH As PropertyHandler

    Set oPH = CreatePropertyHandler(eleText)

    oPH.SelectByAccessString "Origin"
    originValue = oPH.GetValueAsPoint3d
    Debug.Print "Origin is " & originValue.X; ", " & originValue.Y & ", " & originValue.Z

    oPH.SelectByAccessString "UserOrigin"
    originValue = oPH.GetValueAsPoint3d
    Debug.Print "UserOrigin is " & originValue.X; ", " & originValue.Y & ", " & originValue.Z

For View, the origin is the point in the model that is mapped to the back clipping plane's lower left corner. Changing this value does not cause an immediate change in what is displayed in the view. To show the change in the view, call the View.Redraw method. See View for information on the relationship of Origin, Extents, ActiveDepth, and Rotation.
