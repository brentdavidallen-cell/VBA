InsertVertex Method

                          

Inserts a vertex at the specified position.

Syntax

object.InsertVertex Index, Point

The InsertVertex method syntax has these parts:

Part Description
object A valid object.
Index A Long expression. The position at which to insert the new vertex. Must be in the range 0 (the first vertex) to VerticesCount (the next position after the last existing vertex).
Point A Point3d expression. The point to insert into the vertex list.

Remarks

In this object library, most index values are one-based. However, the index values that you give to InsertVertex, ModifyVertex and RemoveVertex are zero-based.

If Index is VerticesCount, the new vertex is appended to the end of the current list of vertices.
