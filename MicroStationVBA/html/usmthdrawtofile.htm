DrawToFile Method

                          

Draw the line style or element to the given enhanced metafile.

Syntax

object.DrawToFile FileName, Width, Height [, DrawBackGround]

The DrawToFile method syntax has these parts:

Part Description
object A valid object.
FileName A String expression. Enhanced metafile name to draw to.
Width A Long expression. The width of the rectangle in number of pixels.
Height A Long expression. The height of the rectangle in number of pixels.
DrawBackGround Optional. A Boolean expression. Draw back ground.

Remarks

Element.DrawToFile draws the element in the given rectangle and returns an IPicture interface.

For an element, calling DrawToFile method raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
