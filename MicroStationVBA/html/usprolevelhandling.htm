LevelHandling Property

                          

Sets the way in which levels are brought from the source model to the destination model. See MsdCopyContextLevelOption.

Syntax

object.LevelHandling [= LevelOption]

The LevelHandling property syntax has these parts:

Part Description
object A valid object.
LevelOption An MsdCopyContextLevelOption expression.

Remarks

Determines how CopyElement handles levels. The default is msdCopyContextLevelByUserPref when a CopyContext is created.
