StartPoint Property

                          

Read/Write Point3d for ArcElement.

Read-only Point3d for BsplineCurveElement, ChainableElement, ComplexStringElement, CurveElement, LineElement.

Gets the start point of the element.

Syntax

object.StartPoint [= Point3d]

The StartPoint property syntax has these parts:

Part Description
object A valid object.
Point3d A Point3d expression.

Remarks

The point's values are measured in the coordinate system of the containing model.
