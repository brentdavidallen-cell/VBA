MatchDimensionToDestination Property

                          

Set whether elements are converted to the same dimensionality as the destination model.

Syntax

object.MatchDimensionToDestination [= Boolean]

The MatchDimensionToDestination property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

Setting this option to True in the CopyContext causes CopyElement to convert the new element to the dimensionality (2D/3D) of the destination model. The value will always be treated as True if the elements are being written to the file. The default is True when a CopyContext is created.
