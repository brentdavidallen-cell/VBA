BsplineCurve Object

                                                     

Corresponds to the MDL B-spline curve structure, MSBsplineCurve.

Remarks

As per MDL analogue, most of the manipulation of a BsplineCurveElement in VBA is performed on a BsplineCurve or InterpolationCurve object, and only at the last step is an element created (cf. CreateBsplineCurveElement1, CreateBsplineCurveElement2) or updated (cf. UpdateFromBsplineCurve, UpdateFromInterpolationCurve) from this object.

The properties of a BsplineCurve object may be set directly, or they may be populated by various BsplineCurve methods. When a BsplineCurve property is changed, dependent properties are adjusted as necessary to maintain a valid object state. For example, one side effect of BsplineCurve.SetPoles is that the Order is set if it was previously invalid.

To get a "blank" BsplineCurve object, use the New keyword.
