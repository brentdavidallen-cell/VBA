MsdLevelElementAccess Enum

        

Defines constants that represent a level's element access mode, which applies to all elements on the level. These values are used by the ElementAccess property of the Level object.

The MsdLevelElementAccess enum has these members:

Name Description
msdLevelElementAccessAll 0 (&H0) All elements have full access, and new elements can be added to the level. The level can be set active.
msdLevelElementAccessLocked 1 (&H1) The elements cannot be modified or deleted, but they can be copied within the level or to another level. Elements can be added to the level, but, once added, become locked like the other elements. The level can be set active.
msdLevelElementAccessReadOnly 2 (&H2) The elements cannot be modified or deleted, but they can be copied to another level. No elements can be added to the level. The level cannot be set active.
msdLevelElementAccessViewOnly 3 (&H3) The elements cannot be modified, deleted, or copied. No elements can be added to the level. The level cannot be set active.
