RemoveFace Method

                          

Removes the input face from it's owning body

Syntax

Set SmartSolidElement = object.RemoveFace (solid, closestPointOnFace, healType, healLoop)

The RemoveFace method syntax has these parts:

Part Description
object A valid object.
solid A SmartSolidElement expression. input body on which face is to be removed
closestPointOnFace A Point3d expression. input point that lie on the face to be removed
healType A Long expression. heal option (one of HEAL_Cap = 0, HEAL_Child = 1, HEAL_Parent = 2, HEAL_Shrink = 3 or HEAL_No = 4)
healLoop A Boolean expression. TRUE to heal loops independently and FALSE to do otherwise
