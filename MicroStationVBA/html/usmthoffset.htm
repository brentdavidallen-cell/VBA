Offset Method

                          

Creates and returns an object that is an approximation of the offset from this object at the given distance.

Syntax

For the BsplineCurve object:

   Set BsplineCurve = object.Offset (SignedOffsetDistance, Rotation [, CuspType [, Tolerance]])

For the BsplineSurface object:

   Set BsplineSurface = object.Offset (SignedOffsetDistance [, Tolerance])

The Offset method syntax has these parts:

Part Description
object A valid object.
SignedOffsetDistance A Double expression. The distance to offset this object. The sign of SignedOffsetDistance determines the offset direction: positive values offset this object in the direction of the (main) normal.
Rotation A Matrix3d expression. The rotation matrix (e.g., View.Rotation, Matrix3dRotationFromRowZ) specifying the plane in which to compute the offset. Pass Matrix3dIdentity to compute the offset in the xy-plane. Pass Matrix3dZero to compute the offset in the average plane of this object.
CuspType Optional. An MsdBsplineCurveOffsetCuspType expression. The way in which the portions of the offset on either side of a cusp (corner) in this object are joined.
Tolerance Optional. A Double expression. The discrete distance tolerance of the offset approximation. If omitted or less than 1.0e-15, a default tolerance of 1.0e-3 times the largest coordinate difference of the unweighted Poles of this object is used.

Remarks

For BsplineCurve objects, the returned offset is a cubic composite Bezier curve with C1 continuity. Note that the offset curve returned by this method is always planar, even if this object is not. To compute a nonplanar offset curve, use BsplineCurve.Offset3d.

For BsplineSurface objects, the returned offset is a bicubic composite Bezier surface with tangent continuity across the patches.

For BsplineSurface objects, any boundaries in the object are ignored.
