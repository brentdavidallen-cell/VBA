GetSheetDefinition Method

                          

Creates a SheetDefintion object and initializes it with the sheet definition of the model used to call GetSheetDefinition.

Syntax

Set SheetDefinition = object.GetSheetDefinition

The GetSheetDefinition method syntax has these parts:

Part Description
object A valid object.

Remarks

If the program modifies the SheetDefinition, it must call SetSheetDefinition to apply the changes.

A program can apply the sheet definition to any model. It does not have to use the same model reference for the calls to GetSheetDefinition and SetSheetDefinition. A program can take advantage of this to copy a sheet definition to multiple models.
