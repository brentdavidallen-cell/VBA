RemoveDatabaseLink Method

                          

Removes all of an object's associated database links that meet the criteria specified in the arguments to the method.

Syntax

Long = object.RemoveDatabaseLink (DatabaseType [, MSLinkNumber [, EntityNumber]])

The RemoveDatabaseLink method syntax has these parts:

Part Description
object A valid object.
DatabaseType An MsdDatabaseLinkage expression. All links to be removed must match this.
MSLinkNumber Optional. A Long expression. If non-zero, then all links to be removed must match this. If this is zero, RemoveDatabaseLink does not examine the link number when deciding whether or not to remove the link.
EntityNumber Optional. A Long expression. If non-zero, then all links to be removed must match this. If this is zero, RemoveDatabaseLink does not examine the entity number when deciding whether or not to remove the link.

Remarks

This does not update the relational databases. It just modifies the MicroStation element. The return value is the number of linkages removed from the Element.

For an element, calling RemoveDatabaseLink will raise an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
