GetSharedCellDefinition Method

                          

Retrieves the SharedCellDefinitionElement for a SharedCellElement.

Syntax

Set SharedCellDefinitionElement = object.GetSharedCellDefinition

The GetSharedCellDefinition method syntax has these parts:

Part Description
object A valid object.

Remarks

To change the appearance of a SharedCellElement, change the definition. Do not change the element itself. For example, to change the colors: 


Sub ChangeLineColors(eleSC As SharedCellElement, newColor As Long)
    Dim eleSCDEF As SharedCellDefinitionElement
    Dim ee As ElementEnumerator

    Set eleSCDEF = eleSC.GetSharedCellDefinition
    Set ee = eleSCDEF.GetSubElements
    Do While ee.MoveNext
        If ee.Current.IsLineElement Then
            ee.Current.Color = newColor
            ee.Current.Rewrite
        End If
    Loop
    '  All instances should be updated.
    RedrawAllViews
End Sub
