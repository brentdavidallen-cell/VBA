Transform3dFromLineAndRotationAngle Method

                          

Returns a transformation of rotation about a specified line.

Syntax

Transform3d = object.Transform3dFromLineAndRotationAngle (Point0, Point1, Radians)

The Transform3dFromLineAndRotationAngle method syntax has these parts:

Part Description
object A valid object.
Point0 A Point3d expression. The start point of the line
Point1 A Point3d expression. The end point of the line
Radians A Double expression. The rotation angle
