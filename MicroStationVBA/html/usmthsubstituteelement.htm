SubstituteElement Method

                          

Specifies an cloned element to be printed instead of the original element.

Syntax

object.SubstituteElement newEle

The SubstituteElement method syntax has these parts:

Part Description
object A valid object.
newEle An Element expression.

Remarks

The input element *must* have been created using PrintBeginElementParam.CloneElement. It is not legal to call Element.Clone or Element.Copy while inside the printing process. All modifications to the element must be completed prior to calling this method. This method is inefficient and leads to increased memory consumption during printing. It should be avoided if possible.
