PatternApparentColor Method

                          

Returns the color of this object's pattern, as it appears within the specified view.

Syntax

Long = object.PatternApparentColor (View)

The PatternApparentColor method syntax has these parts:

Part Description
object A valid object.
View A View expression.

Remarks

The color is represented as an index into MicroStation's color table.

An error is raised if this object does not have an associated pattern. If you're in doubt, firstly check the value of HasPattern.
