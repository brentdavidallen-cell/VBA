RewriteLevels Method

                          

Commits any changes you've made to this object's Levels collection, and/or to any of its individual Levels.

Syntax

object.RewriteLevels

The RewriteLevels method syntax has these parts:

Part Description
object A valid object.

Remarks

Whenever you finish adding, deleting, and/or changing any of this object's Levels, you should call Levels.Rewrite. Otherwise, your changes will be discarded when the design file is closed.
