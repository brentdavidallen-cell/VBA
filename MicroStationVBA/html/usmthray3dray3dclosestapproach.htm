Ray3dRay3dClosestApproach Method

                          

Compute the closest approach of two lines in 3d, considering both lines unbounded. Returns true if the lines intersect, false if parallel or coincident.

Syntax

Boolean = object.Ray3dRay3dClosestApproach (Ray0, Ray1, Point0, Fraction0, Point1, Fraction1)

The Ray3dRay3dClosestApproach method syntax has these parts:

Part Description
object A valid object.
Ray0 A Ray3d expression. First ray
Ray1 A Ray3d expression. Second ray
Point0 A Point3d expression. Computed intersection, with Z part taken from ray0 at the (apparent) intersection.
Fraction0 A Double expression. The fractional position of the interesection point along Ray0
Point1 A Point3d expression. Computed intersection, with Z part taken from ray1 at the (apparent) intersection.
Fraction1 A Double expression. The fractional position of the interesection point along Ray1
