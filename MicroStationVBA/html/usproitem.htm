Item Property

                          

Read-only Attachment for Attachments.

Read-only DimensionStyle for DimensionStyles.

Read-only Font for Fonts.

Read-only Level for Levels.

Read-only LineStyle for LineStyles.

Read-only ModelReference for ModelReferences.

Read-only RasterBlockType for RasterBlockTypes.

Read-only RasterClipMask for RasterClipMasks.

Read-only RasterColorMode for RasterColorModes.

Read-only RasterCompressionMode for RasterCompressionModes.

Read-only RasterFormat for RasterFormats.

Read-only Raster for Rasters.

Read-only TagDefinition for TagDefinitions.

Read-only TagSet for TagSets.

Read-only TextStyle for TextStyles.

Read-only ViewGroup for ViewGroups.

Read-only View for Views.

Syntax

object.Item (IndexOrName)

The Item property syntax has these parts:

Part Description
object A valid object.
Index A VARIANT expression. Index of the element in the collection.
IndexOrName A VARIANT expression.
ViewIndex A Long expression.

Remarks

For each collection, the Item property is the default property. Thus, you never have to reference the Item property explicitly: 

    Set oFont = ActiveDesignFile.Fonts("Arial")

This is the same as:

    Set oFont = ActiveDesignFile.Fonts.Item("Arial")

For the Fonts, Levels, and TextStyles collections, you can select an item by its ID number, by its name, or by its index within the collection. To select an item by its ID number, pass a string comprised of "#" followed by the ID number. (Note that Visual Basic's string concatenation operator "&" will automatically convert a number to a string.)

For example:

    Dim nLevelID As Long
    Dim oLevel As Level
    nLevelID = 15
    Set oLevel = ActiveDesignFile.Levels("#" & nLevelID)
    ' Now oLevel is the level whose ID number is 15.

For the LineStyles collection, you can select an item by its line style number, by its name, or by its index within the collection. To select an item by its line style number, pass a string comprised of "#" followed by the line style number. (Note that Visual Basic's string concatenation operator "&" will automatically convert a number to a string.)

For example:

    Dim nLineStyleNumber As Long
    Dim oLineStyle As LineStyle
    nLineStyleNumber = 1
    Set oLineStyle = ActiveDesignFile.LineStyles("#" & nLineStyleNumber)
    ' Now oLineStyle is the LineStyle whose line style number is 1.

For a RasterBlockTypes object, the Index can be the index of the block type. 


    Dim theRasterBlockTypes As RasterBlockTypes
    Dim oRasterBlockType As RasterBlockType

    ...

    Set oRasterBlockType = theRasterBlockTypes(1)
    MsgBox oRasterBlockType.ToString

For a RasterClipMasks object, the Index can only be the index of the clip mask

For a RasterColorModes object, the IndexOrName can be the index of the color mode or a string representing the color mode. 


    Dim oRasterColorMode As RasterColorMode
    Dim oRasterColorMode2 As RasterColorMode
    Set oRasterColorMode = RasterManager.RasterFormats("TI~FF").RasterColorModes("~16 Colors")
    Set oRasterColorMode2 = RasterManager.RasterFormats(1).RasterColorModes(1)
    MsgBox oRasterColorMode.ToString
    MsgBox oRasterColorMode2.ToString

For a RasterCompressionModes object, the IndexOrName can be the index of the compression mode or a string representing the compression mode. 


    Dim theRasterCompressionModes As RasterCompressionModes
    Dim oRasterCompressionMode As RasterCompressionMode
    Dim oRasterCompressionMode2 As RasterCompressionMode

    ...

    Set oRasterCompressionMode = theRasterCompressionModes("~None")
    Set oRasterCompressionMode2 = theRasterCompressionModes(1)
    MsgBox oRasterCompressionMode.ToString
    MsgBox oRasterCompressionMode2.ToString

For a RasterFormats object, the IndexOrName can be the format's index or a string representing the raster file format. 


    Dim oFormat As RasterFormat
    Dim oFormat2 As RasterFormat
    Set oFormat = RasterManager.RasterFormats("*TIFF*")
    Set oFormat = RasterManager.RasterFormats(1)
    MsgBox oFormat.ToString
    MsgBox oFormat2.ToString

For a Rasters object, the IndexOrName can be the raster's index, its full path or its logical name.
