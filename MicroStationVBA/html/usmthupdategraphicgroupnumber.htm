UpdateGraphicGroupNumber Method

                          

Returns a graphic group that can be used for setting an element's GraphicGroup property.

Syntax

Long = object.UpdateGraphicGroupNumber

The UpdateGraphicGroupNumber method syntax has these parts:

Part Description
object A valid object.

Remarks

UpdateGraphicGroupNumber saves that value to the design file so MicroStation will know not to use it the next time it is started. It also increments the graphic group number, so the next time UpdateGraphicGroupNumber is called, it will return the next number.

The pair of statements

        ele.GraphicGroup = CurrentGraphicGroup
        UpdateGraphicGroupNumber

is equivalent to:

        ele.GraphicGroup = UpdateGraphicGroupNumber
