GetUserAttributeData Method

                          

Returns this object's associated user data.

Syntax

DataBlock () = object.GetUserAttributeData (AttributeID)

The GetUserAttributeData method syntax has these parts:

Part Description
object A valid object.
AttributeID A Long expression. A value that uniquely identifies the type of user data. Attribute ID's are also known as linkage ID's. See AddUserAttribute data for more information on attribute ID's.

Remarks

Each DataBlock object in the array that GetUserAttributeData returns contains all of the data from a single user attribute block. Use the methods of DataBlock to extract the data from a data block.

The array has an entry for each user attribute that has the specified AttributeID. If the object has 3 user attributes with that AttributeID, then the array has 3 entries indexed 0, 1, and 2. If the object does not have any user attributes wih the specified AttributeID, then the array has no entries.

For an element, calling GetUserAttributeData raises an error if the element is not graphical. If you're in doubt, firstly check the element's IsGraphical property.
