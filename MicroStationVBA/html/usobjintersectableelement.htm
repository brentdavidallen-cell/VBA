IntersectableElement Object

                                                     

Represents the aspects of a MicroStation element that pertain to only those element types that are intersectable. An element is intersectable if it can intersect with other elements at a finite number of points.

Remarks

A program does not explicitly create an IntersectableElement object. IntersectableElement is an interface that some of the element types implement.

You can use IsIntersectableElement to determine if an element is intersectable. You can use AsIntersectableElement to reference an element as an intersectable element. Alternatively, you can access the element as an intersectable element by assigning the Element reference to a variable that is declared as IntersectableElement.
