Hide and Show Level Example

This example illustrates how to determine if a view displays elements from the specified level. It also demonstrates how to cause the view to starting displaying or hiding elements from that level. 


Sub ToggleLevelDisplay(strLevelName As String, viewIndex As Integer)
    Dim oLevel As Level
    Dim oView As View

    Set oLevel = ActiveDesignFile.Levels(strLevelName)
    Set oView = ActiveDesignFile.Views(viewIndex)

    If oView.IsLevelShown(oLevel) Then
        oView.HideLevel oLevel
    Else
        oView.ShowLevel oLevel
    End If

    ' Now make the view reflect the change
    oView.Redraw
End Sub
