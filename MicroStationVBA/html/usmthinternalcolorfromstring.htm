InternalColorFromString Method

                          

Syntax

Long = object.InternalColorFromString (ColorDescription)

The InternalColorFromString method syntax has these parts:

Part Description
object A valid object.
ColorDescription A String expression. May be a color table index, an RGB triplet, or the name of an entry in the color book.

Remarks

To get the internal color corresponding to a color book value, use ActiveModelReference.InternalColorFromString("Standard:aquamarine").

To get the internal color corresponding to an RGB value, use ActiveModelReference.InternalColorFromString("40,69,102").

To get the internal color corresponding to a color table index, use ActiveModelReference.InternalColorFromString("40").
