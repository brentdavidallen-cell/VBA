DimWithLeaderAlignment Property

                          

Read/Write MsdDimBallAndChainAlignment.

Gets this object's dim_with_leader alignment.

Syntax

object.DimWithLeaderAlignment [= MsdDimBallAndChainAlignment]

The DimWithLeaderAlignment property syntax has these parts:

Part Description
object A valid object.
MsdDimBallAndChainAlignment An MsdDimBallAndChainAlignment expression.

Remarks

See MsdDimBallAndChainAlignment for possible valid values.
