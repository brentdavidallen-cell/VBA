MsdDimPlacementTextPosition Enum

        

Valid values for DimensionStyle text positioning. Used for the property TextPosition.

The MsdDimPlacementTextPosition enum has these members:

Name Description
msdDimPlacementTextPositionManual 0 (&H0) The user is always prompted to position text.
msdDimPlacementTextPositionSemiAuto 1 (&H1) The user is prompted to position text only if the text will not fit between the extension lines.
msdDimPlacementTextPositionAuto 2 (&H2) The user is never prompted to position text.
