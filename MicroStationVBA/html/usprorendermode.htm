RenderMode Property

                          

Read/Write MsdRenderingMode.

Gets or sets the Render Display Mode.

Syntax

object.RenderMode [= RenderingMode]

The RenderMode property syntax has these parts:

Part Description
object A valid object.
RenderingMode An MsdRenderingMode expression.

Remarks

This value is equivalent to the Display Mode combobox in the Set View Display Mode dialog box.

Changing this value does not cause an immediate change in what is displayed in the view. To show the change in the view, call the View.Redraw method.
