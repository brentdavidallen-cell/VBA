GetIntersectionShapesFromRegion Method

                          

Constructs and returns a list of elements that form the boolean intersection of the regions enclosed by this object and the (union of the) given coplanar elements.

Syntax

Set ElementEnumerator = object.GetIntersectionShapesFromRegion (Elements, Template [, FillMode])

The GetIntersectionShapesFromRegion method syntax has these parts:

Part Description
object A valid object.
Elements A Element () expression. An array of coplanar elements whose union is intersected with this object.
Template An Element expression. An existing element whose settings are used to initialize the new elements. If Nothing, the new elements' settings are initialized from MicroStation's active settings.
FillMode Optional. An MsdFillMode expression. Indicates whether the returned elements are filled. If the value is not specified or it is msdFillModeUseActive, MicroStation's active fill mode is used.

Remarks

The input geometry must be coplanar (any plane) with this object, or this method will fail.

On successful return, an empty ElementEnumerator means the boolean operation resulted in the empty set.
