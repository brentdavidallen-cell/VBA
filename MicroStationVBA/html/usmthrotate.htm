Rotate Method

                          

Rotates this object.

Syntax

For the TextNodeElement object, TextElement object, TagElement object, SmartSolidElement object, SharedCellElement object, SharedCellDefinitionElement object, ShapeElement object, SavedViewElement object, PointStringElement object, NamedGroupElement object, MultiLineElement object, LineElement object, EllipseElement object, Element object, DimensionElement object, CurveElement object, ConeElement object, ComplexStringElement object, ComplexShapeElement object, CellElement object, BsplineSurfaceElement object, BsplineCurveElement object, AuxiliaryCoordinateSystemElement object, ArcElement object and ApplicationElement object:

   object.Rotate Pivot, AboutX, AboutY, AboutZ

For the Attachment object:

   object.Rotate Pivot, AboutX, AboutY, AboutZ, ViewSpecifier

The Rotate method syntax has these parts:

Part Description
object A valid object.
AboutX A Double expression. An angle expressed in radians.
AboutY A Double expression. An angle expressed in radians.
AboutZ A Double expression. An angle expressed in radians.
ViewSpecifier A VARIANT expression. The view index (1 through 8) or View object. Attachment.Rotate ignores this when rotating an attachment to a 2D model.
Pivot A Point3d expression.

Remarks

Use Redraw to erase the object before rotating it and to display it again after rotating it. Use Rewrite to make the change permanent. Attachment and the element types all support Redraw and Rewrite.

Attachment.Rotate rotates the Attachment about the View's axes, not the model's axes. Use ViewSpecifier to select the view.

If the Attachment is clipped by an element, the clip element is rotated and immediately written to the file. In that case, the Attachment should always be written to the file to keep it in synch with its clipping element.
