FacetSolidAsShapes Method

                          

This function is used to polygonize (facet) a body.The faces of a body are polygonized and the output is a array of shapes.

Syntax

Set ElementEnumerator = object.FacetSolidAsShapes (maxNumEdges, maxEdgeLength, maxChordHeight, maxNormalAngleInRadians)

The FacetSolidAsShapes method syntax has these parts:

Part Description
object A valid object.
maxNumEdges A Long expression. maximum number of sides on a facet. For example, a value of three produces only triangular facets.
maxEdgeLength A Double expression. the maximum width of a facet.
maxChordHeight A Double expression. the maximum distance between a curve on the body and its facet edges.
maxNormalAngleInRadians A Double expression. the maximum angle in radians between a surface and its facet, i.e. the upper bound on the angle deviation between surfaces normals at any two positions under the facet.
