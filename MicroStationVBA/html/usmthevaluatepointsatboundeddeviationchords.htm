EvaluatePointsAtBoundedDeviationChords Method

                          

Returns points (and parameters) which determine chords of this object with bounded deviation from the object.

Syntax

Point3d () = object.EvaluatePointsAtBoundedDeviationChords (Parameters, MaxDeviation)

The EvaluatePointsAtBoundedDeviationChords method syntax has these parts:

Part Description
object A valid object.
Parameters A Double () expression. The returned parameters associated with the returned points.
MaxDeviation A Double expression. The maximum chord height of the chords to compute.
