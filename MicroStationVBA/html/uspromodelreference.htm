ModelReference Property

                          

Read-only ModelReference.

Gets a ModelReference object corresponding to the model in which this object resides.

Syntax

object.ModelReference

The ModelReference property syntax has these parts:

Part Description
object A valid object.

Remarks

For an ElementCache object, this property raises an error if the object is not associated with a model reference. If you're in doubt, firstly verify that the value of Purpose does not equal msdElementCachePurposeNonModel.
