Levels Object

                                                     

A collection of Level objects.

Remarks

Levels supports the standard collection interface, so it is possible to step through the collection using the For Each statement. It is also possible to retrieve a Level by indexing into the Levels collection with the level's name: 

    Dim oFurnitureLevel As Level
    Set oFurnitureLevel = ActiveDesignFile.Levels("Furniture")
