GetAsNormalCell Method

                          

Gets the equivalent normal cell.

Syntax

Set CellElement = object.GetAsNormalCell (ApplyByCellLevelAndSymbology)

The GetAsNormalCell method syntax has these parts:

Part Description
object A valid object.
ApplyByCellLevelAndSymbology A Boolean expression.

Remarks

Unlike DropToNormalCell, this does not delete the shared cell instance and does not add the new cell to the model. This is similar to Element.Drop except it creates a new cell element instead of a collection of elements that can be be used to generate the equivalent cell.
