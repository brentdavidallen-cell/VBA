GetLargestElementID64 Method

                          

This method cannot be used in VBA or Visual Basic 6.

Returns the largest element ID number that is used within this design file.

Syntax

Int64 = object.GetLargestElementID64

The GetLargestElementID64 method syntax has these parts:

Part Description
object A valid object.

Remarks

In VBA or Visual Basic 6, use the method, GetLargestElementID.
