Reset Method

                          

Reset this object to its original state.

Syntax

object.Reset

The Reset method syntax has these parts:

Part Description
object A valid object.

Remarks

It is redundant (but harmless) to call Reset on a newly created object.

TransientElementContainer.Reset resets the TransientElementContainer, erasing and removing all of the elements. Most VB/VBA programs will not use TransientElementContainer.Reset because MicroStation automatically resets the transient element container when there are no more references to the container object. It also resets the container when a different design file becomes the active design file. .NET programs need to use either Reset or the system method System.Runtime.InteropServices.Marshal.ReleaseComObject to cause MicroStation to reset the transient element container without waiting for the object to be garbage collected.
