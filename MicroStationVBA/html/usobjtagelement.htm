TagElement Object

                                                     

A kind of Element that represents a tag that has been attached to another MicroStation element.

Remarks

When a user attaches tags using the Attach Tags command, MicroStation creates tag elements to hold the tag information, and associates those tag elements with the tagged element. A TagElement object represents one of those tag elements.

A program can create tag elements by using AddTag or AddTags. There is no explicit CreateTagElement method.

The text attributes of a new text element are determined by the current text style.

It is possible for a program or a user to manipulate a tag element independently of the element it is attached to. However, the tag element responds to changes to the tagged element. When a tagged element moves, all of its attached tag elements move too. When a tagged element is deleted, all of the attached tag elements are also deleted.
