Configuration Variable Example

This example illustrates the methods for manipulating configuration variables. It uses several methods of the Workspace class to query and manage configuration variables. It uses IsConfigurationVariableDefined to determine if a configuration variable is defined. It uses ConfigurationVariableValue to get the value of a configuration variable. It uses RemoveConfigurationVariable to remove a configuration variable. It uses ExpandConfigurationVariable to expand a string that contains nested configuration variables. 


Sub ConfigVars()

    With ActiveWorkspace
        Debug.Print "IsConfigurationVariableDefined (MS_DEF)" & .IsConfigurationVariableDefined("MS_DEF")
        Debug.Print "IsConfigurationVariableDefined (MS_NOTDEFINED)" & .IsConfigurationVariableDefined("MS_NOTDEFINED")
        .AddConfigurationVariable "AddedByVba", "VBA DEFINITION"
        Debug.Print "The value of AddByVba is: " & .ConfigurationVariableValue("AddedByVba")
        .RemoveConfigurationVariable "AddedByVba"
        Debug.Print "IsConfigurationVariableDefined (AddedByVba)" & .IsConfigurationVariableDefined("AddedByVba")

        Dim expansion As String

        expansion = .ExpandConfigurationVariable("$(_USTN_USER)$(_USTN_USERNAME).txt")
        Debug.Print "The expansion is : " & expansion
    End With

End Sub
