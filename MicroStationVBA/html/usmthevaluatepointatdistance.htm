EvaluatePointAtDistance Method

                          

Returns the point (and parameter) at a given distance along this object.

Syntax

Point3d = object.EvaluatePointAtDistance (Parameter, DistanceFromStart [, StartParameter [, Tolerance]])

The EvaluatePointAtDistance method syntax has these parts:

Part Description
object A valid object.
Parameter A Double expression. The returned parameter associated with the returned point.
DistanceFromStart A Double expression. The valid distance along the object at which to compute the point, given as a signed distance from the point at StartParameter. A negative value indicates distance in the opposite direction of this object's parametrization.
StartParameter Optional. A Double expression. The valid parameter of the point from which to measure the DistanceFromStart distance. If omitted, 0.0 is used.
Tolerance Optional. A Double expression. The maximum relative integration error. If omitted or nonpositive, a default value of 1.0e-5 is used.

Remarks

For BsplineCurve objects, this method replaces the method EvaluatePointAtLength, which was deprecated after MicroStation V8 2004 Edition.
