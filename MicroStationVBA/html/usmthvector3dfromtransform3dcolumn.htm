Vector3dFromTransform3dColumn Method

                          

Extracts a column of the matrix part of a transform Note that this addresses only the matrix part.

Syntax

Vector3d = object.Vector3dFromTransform3dColumn (Transform, Col)

The Vector3dFromTransform3dColumn method syntax has these parts:

Part Description
object A valid object.
Transform A Transform3d expression. The transform
Col A Long expression. column index. Columns are numbered 0,1,2. Others indices are reduced cyclically.
