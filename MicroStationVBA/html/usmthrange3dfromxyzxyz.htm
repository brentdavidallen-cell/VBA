Range3dFromXYZXYZ Method

                          

Initializes the range to contain two points given as components. Minmax logic is applied to the given points.

Syntax

Range3d = object.Range3dFromXYZXYZ (X1, Y1, Z1, X2, Y2, Z2)

The Range3dFromXYZXYZ method syntax has these parts:

Part Description
object A valid object.
X1 A Double expression. first x
Y1 A Double expression. first y
Z1 A Double expression. first z
X2 A Double expression. second x
Y2 A Double expression. second y
Z2 A Double expression. second z
