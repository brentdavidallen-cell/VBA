SuffixSymbolType Property

                          

Read/Write MsdDimSymbolType.

Gets this object's type of suffix symbol.

Syntax

object.SuffixSymbolType [= MsdDimSymbolType]

The SuffixSymbolType property syntax has these parts:

Part Description
object A valid object.
MsdDimSymbolType An MsdDimSymbolType expression.

Remarks

See MsdDimSymbolType for possible valid values.
