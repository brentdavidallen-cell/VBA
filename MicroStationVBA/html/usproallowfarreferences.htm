AllowFarReferences Property

                          

Read/Write Boolean.

Gets/sets whether it is possible to make a NamedGroupElement contain a member from a directly attached Attachment.

Syntax

object.AllowFarReferences [= Boolean]

The AllowFarReferences property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

This property only controls whether the named group supports references to members of directly attached model. Members from nested references are not supported regardless of the value of AllowFarReferences.

Whenever you change any aspect of a NamedGroupElement, you should subsequently call Rewrite to make the change permanent.
