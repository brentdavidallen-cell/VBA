RemoveKnots Method

                          

Cleans up the Knot vector (and thins the Pole array) of this object.

Syntax

object.RemoveKnots [Tolerance [, KeepStartTangent [, KeepEndTangent [, FastRemove]]]]

The RemoveKnots method syntax has these parts:

Part Description
object A valid object.
Tolerance Optional. A Double expression. The Knot removal tolerance. If omitted or negative, a default tolerance of 1.0e-5 is used.
KeepStartTangent Optional. A Boolean expression. Indicates whether the start tangent of this object is preserved. If omitted, a default value of True is used.
KeepEndTangent Optional. A Boolean expression. Indicates whether the end tangent of this object is preserved. If omitted, a default value of True is used.
FastRemove Optional. A Boolean expression. Indicates whether a fast Knot removal algorithm (that removes less Knots/Poles) is used. If omitted, a default value of True is used.
