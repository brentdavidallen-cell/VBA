Moving the Closest Vertex

This example illustrates some of the methods of the VertexList interface. Given an Element that is a VertexList, it moves the vertex that is closest to the specified point. 


Sub ModifyClosestVertex(ele As Element, pnt As Point3d)
    Dim vertexIndex As Integer

    On Error GoTo done
    With ele.AsVertexList
        vertexIndex = .GetClosestVertex(pnt)

        ele.Redraw msdDrawingModeErase
        .ModifyVertex vertexIndex, pnt
        ele.Redraw msdDrawingModeNormal
        ele.Rewrite
    End With

done:
End Sub
