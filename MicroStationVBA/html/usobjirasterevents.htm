IRasterEvents Object

                                                     

Lets programs monitor all actions applied to rasters.

To declare that a class is an event handler for raster events, use the statement Implements IRasterEvents. To activate the event handler, the program must call RasterManager.AddRasterEventsHandler passing in an instance of the class that implements IRasterEvents. To deactivate the event handler, the program must call RasterManager.RemoveRasterEventsHandler, passing in the same object that was passed to AddRasterEventsHandler.

The event handler remains active until the program deactivates it using RemoveRasterEventsHandler, or until the the project is unloaded or is reset.
