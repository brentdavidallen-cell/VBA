FromNetworkWithIntersections Method

                          

Initializes this object with a Gordon surface that interpolates the network of curves given in the u- and v-parameter directions, and which intersect at the given uv-parameters.

Syntax

object.FromNetworkWithIntersections UCurves, VCurves, IntersectionParameters [, ULoftOrder [, VLoftOrder [, UTensorOrder [, VTensorOrder [, DerivativeContinuity [, Tolerance]]]]]]

The FromNetworkWithIntersections method syntax has these parts:

Part Description
object A valid object.
UCurves A BsplineCurve () expression. The nonrational curves in the u-parameter direction (each at a constant value of v). For best results, these curves should be oriented similarly.
VCurves A BsplineCurve () expression. The nonrational curves in the v-parameter direction (each at a constant value of u). For best results, these curves should be oriented similarly.
IntersectionParameters A Point2d () expression. The 2-dimensional array of intersection parameters, each a uv-parameter pair such that UCurves(i) evaluated at the u-parameter IntersectionParameters(i,j).X is the same point as VCurves(j) evaluated at the v-parameter IntersectionParameters(i,j).Y. The number of rows of this array must equal the number of entries in the UCurves array; the number of columns of this array must equal the number of entries in the VCurves array.
ULoftOrder Optional. A Long expression. The UOrder of the loft component of the resulting object. If omitted, a default order of 4 (cubic) is used. The maximum value of ULoftOrder is limited to the number of entries in the VCurves array.
VLoftOrder Optional. A Long expression. The VOrder of the loft component of the resulting object. If omitted, a default order of 4 (cubic) is used. The maximum value of VLoftOrder is limited to the number of entries in the UCurves array.
UTensorOrder Optional. A Long expression. The UOrder of the interpolating tensor product component of the resulting object. If omitted, a default order of 4 (cubic) is used. The maximum value of UTensorOrder is limited to the number of entries in the VCurves array.
VTensorOrder Optional. A Long expression. The VOrder of the interpolating tensor product component of the resulting object. If omitted, a default order of 4 (cubic) is used. The maximum value of VTensorOrder is limited to the number of entries in the UCurves array.
DerivativeContinuity Optional. A Boolean expression. Indicates whether the resulting object has C1 continuity (in which case its polynomial degree in each parameter direction is doubled). If omitted or False, the resulting object only has C0 continuity.
Tolerance Optional. A Double expression. The absolute maximum curve/surface deviation after knot removal. If omitted or nonpositive, a default value of 1.0e-5 times the largest of the largest coordinate differences of the unweighted Poles of each given curve is used.

Remarks

For BsplineSurface objects, the input BsplineCurve objects must be nonrational; see BsplineCurve.Rebuild to approximate a Rational B-spline curve with a nonrational one (this also improves the curve's parametrization, so be sure to recompute the IntersectionParameters before calling this method).
