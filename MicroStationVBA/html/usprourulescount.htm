URulesCount Property

                          

Read/Write Long.

Gets or sets the number of isoparametric curves in the u-parameter direction displayed for this object.

Syntax

object.URulesCount [= Long]

The URulesCount property syntax has these parts:

Part Description
object A valid object.
Long A Long expression.

Remarks

For BsplineSurface objects, points along an isoparametric curve in the u-parameter direction have constant v-parameter.
