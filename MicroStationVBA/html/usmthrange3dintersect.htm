Range3dIntersect Method

                          

Compute the intersection of two ranges. If any direction has no intersection, or if the intersection has zero thickness, the result range is initialized to a null range.

Syntax

Range3d = object.Range3dIntersect (Range1, Range2)

The Range3dIntersect method syntax has these parts:

Part Description
object A valid object.
Range1 A Range3d expression. first range
Range2 A Range3d expression. second range
