CopyByte Method

                          

Copies a Byte to or from the DataBlock object.

Syntax

object.CopyByte ByteReference, CopyToDataBlock

The CopyByte method syntax has these parts:

Part Description
object A valid object.
ByteReference A Byte expression.
CopyToDataBlock A Boolean expression.

Remarks

If CopyToDataBlock is True, it copies a byte from ByteReference to the data block. Otherwise, it copies a byte from the data block to ByteReference. It either case, it increments Offset by 1, the number of bytes transferred.
