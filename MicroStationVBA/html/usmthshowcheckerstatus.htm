ShowCheckerStatus Method

                          

Displays a message in the status area of the status area of the StandardsCheckerController's status dialog box. See ShowCheckerErrorWithFixOptions for an example.

Syntax

object.ShowCheckerStatus StatusMessage

The ShowCheckerStatus method syntax has these parts:

Part Description
object A valid object.
StatusMessage A String expression.

Remarks

If a standards checker does not call ShowCheckerStatus, then the StandardsCheckerController displays the standards checker's DialogString in the status area.

A standards checker that may run for a long time without reporting an error should periodically display an updated status message in the status area to let the user know it is still progressing.
