ACSType Property

                          

Read/Write MsdACSType.

Syntax

object.ACSType [= MsdACSType]

The ACSType property syntax has these parts:

Part Description
object A valid object.
MsdACSType An MsdACSType expression.

Remarks

A program can disable the auxiliary coordinate system by setting ACSManager.ACSType to msdACSTypeNone.
