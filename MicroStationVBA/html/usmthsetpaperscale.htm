SetPaperScale Method

                          

Sets the PaperScale for the GeographicCoordinateSystem

Syntax

object.SetPaperScale Value, Model, ReprojectData

The SetPaperScale method syntax has these parts:

Part Description
object A valid object.
Value A Double expression. The new value for PaperScale. Must be greater than 0.0. Only the value 1.0 allows accurate ground distance measurement in the design file. Larger values cause smaller Cartesian Coordinates.
Model A ModelReference expression. The Model from which the GeographicCoordinateSystem originated.
ReprojectData A Boolean expression. If true, the data in the design file is projected to take the new paper scale into account. If false, the data is left alone. Use false if you are correcting the paper scale.

Remarks

The changed GeographicCoordinateSystem is written immediately to the design file. For most purposes, the PaperScale should be set to 1.0. It is supported for use in legacy processes.
