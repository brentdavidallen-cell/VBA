LastView Method

                          

Gets the View object of the last view the user interacted with.

Syntax

Set View = object.LastView

The LastView method syntax has these parts:

Part Description
object A valid object.

Remarks

LastView returns the view where the user last entered a data point during the current command state. If an element creation command such as PLACE LINE is active and the user has entered a data point for that command, then LastView returns the view where the user entered the datapoint even if the view has subsequently been closed. If a view control command such as WINDOW AREA is active and the user has entered a data point for that command, then LastView returns the view where the user entered the data point. If the user starts an element placement command, enters a data point, starts a view control command, enters a data point, and then exits the view control command resuming the element placement command, then LastView returns the view of the data point that was entered during the element placement command. It does not return the view of the data point entered during the view control command, even though that was the most recent data point.

CursorInformation has more information on the location of the cursor.
