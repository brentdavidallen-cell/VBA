GetContainingNamedGroups Method

                          

Gets an array that is a list of NamedGroupElements for named group that directly contain this element as a member.

Syntax

NamedGroupElement () = object.GetContainingNamedGroups

The GetContainingNamedGroups method syntax has these parts:

Part Description
object A valid object.

Remarks

If the element is not a member of any named groups, GetContainingNamedGroups returns a zero-length array.
