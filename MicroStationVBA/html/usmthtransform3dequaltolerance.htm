Transform3dEqualTolerance Method

                          

Returns true if two transformations are equal within tolerance, using separate tolerances for the matrix and point parts.

Syntax

Boolean = object.Transform3dEqualTolerance (Transform1, Transform2, MatrixTolerance, PointTolerance)

The Transform3dEqualTolerance method syntax has these parts:

Part Description
object A valid object.
Transform1 A Transform3d expression.
Transform2 A Transform3d expression.
MatrixTolerance A Double expression.
PointTolerance A Double expression.
