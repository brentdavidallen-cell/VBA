TerminatorArrowhead Property

                          

Read/Write MsdDimTerminatorArrowhead.

Gets this object's arrow head terminator appearance.

Syntax

object.TerminatorArrowhead [= MsdDimTerminatorArrowhead]

The TerminatorArrowhead property syntax has these parts:

Part Description
object A valid object.
MsdDimTerminatorArrowhead An MsdDimTerminatorArrowhead expression.

Remarks

See MsdDimTerminatorArrowhead for possible valid values. Applicable only when terminator type is arrow.
