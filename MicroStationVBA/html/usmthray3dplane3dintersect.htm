Ray3dPlane3dIntersect Method

                          

Compute the intersection of an unbounded ray and a plane. Return true if there is an intersection point. Rerturn false if ray is parallel or coincident to the plane.

Syntax

Boolean = object.Ray3dPlane3dIntersect (Ray, Plane, Point, Fraction)

The Ray3dPlane3dIntersect method syntax has these parts:

Part Description
object A valid object.
Ray A Ray3d expression. Ray
Plane A Plane3d expression. plane
Point A Point3d expression. Computed intersection point
Fraction A Double expression. Fractional parameter of intersection point on ray.
