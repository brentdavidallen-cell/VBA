DLongComp Method

                          

Compares two DLong expressions and returns a Long value indicating the result of the comparison:
 If  DLongComp returns

 Value1 is less than Value2

  -1

 Value1 is equal to Value2

  0

 Value1 is greater than Value2

  1

Syntax

Long = object.DLongComp (Value1, Value2)

The DLongComp method syntax has these parts:

Part Description
object A valid object.
Value1 A DLong expression.
Value2 A DLong expression.

Remarks

Example: 

    Dim N1 As DLong
    Dim N2 As DLong
    N1 = DLongFromString("1234567890987654321")
    N2 = DLongFromString("9876543210123456789")
    Select Case DLongComp(N1, N2)
        Case -1
                Debug.Print "N1 is less than N2"
        Case 0
                Debug.Print "N1 is equal to N2"
        Case 1
                Debug.Print "N1 is greater than N2"
    End Select
