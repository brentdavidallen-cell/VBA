Rasters Object

                                                     

A collection of Raster objects. It gives access to all rasters attached to a design file. It is used like this: 


    'With a variable:
    Dim theRasters As Rasters

    Set theRasters = RasterManager.Rasters
    theRasters.[method/property]

    'Without a variable:
    RasterManager.Rasters.[method/property]

Remarks

It is possible to step through the collection using the For Each statement like this: 


    Dim oRaster As Raster
    Dim theRasters As Rasters

    Set theRasters = RasterManager.Rasters

    For Each oRaster In theRasters
    oRaster.[method/property]
    Next
