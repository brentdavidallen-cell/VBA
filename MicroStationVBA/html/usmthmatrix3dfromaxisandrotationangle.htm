Matrix3dFromAxisAndRotationAngle Method

                          

Returns a matrix of rotation about the x,y, or z axis (indicated by axis = 0,1, or 2) by an angle in radians.

Syntax

Matrix3d = object.Matrix3dFromAxisAndRotationAngle (Axis, Radians)

The Matrix3dFromAxisAndRotationAngle method syntax has these parts:

Part Description
object A valid object.
Axis A Long expression. The axis index 0=x, 1=y, 2=z
Radians A Double expression. The rotation angle in radians
