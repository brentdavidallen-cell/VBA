GetParameters Method

                          

Returns the array of interpolation parameters of this object.

Syntax

Double () = object.GetParameters

The GetParameters method syntax has these parts:

Part Description
object A valid object.

Remarks

For InterpolationCurve objects, the represented curve passes through the ith FitPoint at the ith parameter returned by this method.

For InterpolationCurve objects, the parameters returned by this method are normalized between 0.0 and 1.0, inclusive, and are computed from the FitPoints using the chord length parametrization.
