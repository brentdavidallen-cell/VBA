Vector3dFromXYZ Method

                          

Sets the x,y, and z components of a vector

Syntax

Vector3d = object.Vector3dFromXYZ (Ax, Ay, Az)

The Vector3dFromXYZ method syntax has these parts:

Part Description
object A valid object.
Ax A Double expression. The x component.
Ay A Double expression. The y component.
Az A Double expression. The z component.
