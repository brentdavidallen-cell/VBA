Matrix3dIsXYRotationSkewAndScale Method

                          

Reinterprets the matrix as a combination of XY rotation, skew and scaling and Z scaling.

Syntax

Boolean = object.Matrix3dIsXYRotationSkewAndScale (Matrix, XAxisAngle, YAxisSkewAngle, XScale, YScale, ZScale)

The Matrix3dIsXYRotationSkewAndScale method syntax has these parts:

Part Description
object A valid object.
Matrix A Matrix3d expression. matrix being analyzed.
XAxisAngle A Double expression. The angle, in radians counterclockwise, from the global x axis to the x axis (first column) of this matrix.
YAxisSkewAngle A Double expression. The angle, in radians counterclockwise, by which the Y axis of the matrix is skewed away from the usual position of 90 degrees counterclockwise from X.
XScale A Double expression. The length of the x axis vector of this matrix. This scale factor is always positive.
YScale A Double expression. The length of the y axis vector of this matrix. This scale factor may be positive or negative. If negative, the matrix operation includes mirroring (reflection).
ZScale A Double expression. The (signed) scale factor which the matrix applies to z data. This is typically 1 or -1.

Remarks

Returns true if the matrix can be decomposed in this manner. false if there are out-of-plane effects indicated by values in the z parts of the matrix.

If the matrix is "just" rotation around z, the two rotation angles will be identical and both scale factors are 1.0.

If the matrix is "just" rotation around z and uniform scaling, the two rotation angles will be identical and the scale factors will be identical to each other but different from 1.

In the non-mirrored case, the x direction is given by the first column of the matrix; the nominal y direction is 90 degrees counter clockwise, and the returned skew angle is the angle from that nominal y axis direction to the second column of the matrix. In the mirrored case, the mirroring is indicated by a negated y scale factor, and the skew angle is measured from the same nominal y axis (90 degrees counterclockwise from first column) to the NEGATED second column. That is, the returned numbers always represent a skewed-but-righthanded system, with reflection rolled into the y scale factor.
