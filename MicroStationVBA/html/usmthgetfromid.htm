GetFromID Method

                          

Gets the collection element associated with the identification number.

Syntax

For the RasterBlockTypes object:

   Set RasterBlockType = object.GetFromID (Type)

For the RasterColorModes object:

   Set RasterColorMode = object.GetFromID (Id)

For the RasterCompressionModes object:

   Set RasterCompressionMode = object.GetFromID (Id)

For the RasterFormats object:

   Set RasterFormat = object.GetFromID (Id)

For the Rasters object:

   Set Raster = object.GetFromID (RasterId)

The GetFromID method syntax has these parts:

Part Description
object A valid object.
Id A Long expression. The identification number to search for.
Type An MsdRasterBlockType expression.
RasterId An Int64 expression.

Remarks

This method returns Nothing if no RasterBlockType has the requested identification number. This method can be used to determine a Raster's current block type as follows: 


    Dim oRaster As Raster
    Dim oFormat As RasterFormat
    Dim oRasterColorMode As RasterColorMode
    Dim oRasterCompressionMode As RasterCompressionMode
    Dim oRasterBlockType As RasterBlockType

    ...

    Set oFormat = RasterManager.RasterFormats.GetFromId(oRaster.RasterInformation.FormatID)
    Set oRasterColorMode = oFormat.GetFromID(oRaster.RasterInformation.ColorModeID)
    Set oRasterCompressionMode = oRasterColorMode.GetFromID(oRaster.RasterInformation.CompressionModeID)
    Set oRasterBlockType = oRasterCompressionMode.GetFromID(oRaster.RasterInformation.BlockTypeID)
    MsgBox oRasterBlockType.ToString

This method returns Nothing if no RasterColorMode has the requested identification number. This method can be used to determine a Raster's current color mode as follows: 


    Dim oRaster As Raster
    Dim oFormat As RasterFormat
    Dim oRasterColorMode As RasterColorMode

    ...

    Set oFormat = RasterManager.RasterFormats.GetFromId(oRaster.RasterInformation.FormatID)
    Set oRasterColorMode = oFormat.GetFromID(oRaster.RasterInformation.ColorModeID)
    MsgBox oRasterColorMode.ToString

This method returns Nothing if no RasterCompressionMode has the requested identification number. This method can be used to determine a Raster's current compression mode as follows:
