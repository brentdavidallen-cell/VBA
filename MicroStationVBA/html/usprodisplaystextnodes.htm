DisplaysTextNodes Property

                          

Read/Write Boolean.

Gets or sets a value indicating whether the view displays text nodes.

Syntax

object.DisplaysTextNodes [= Boolean]

The DisplaysTextNodes property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

This value is equivalent to the Text Nodes checkbox in the View Attributes dialog box. When DisplaysTextNodes is True, empty text nodes displayed as small crosses with numeric identifiers. However, in most cases displaying text nodes is unnecessary.

Changing this value does not cause an immediate change in what is displayed in the view. To show the change in the view, call the View.Redraw method.
