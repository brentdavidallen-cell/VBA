PartialDelete Method

                          

Returns the portions of the element that are not between the projection of Point1 and Point2 on the element.

Syntax

object.PartialDelete Partial1, Partial2, Point1, Point2, Selector, ViewSpecifier

The PartialDelete method syntax has these parts:

Part Description
object A valid object.
Partial1 An Element expression. This is an output argument. PartialDelete sets it to a reference to the first element it creates. PartialDelete may set this to Nothing. See the remarks for more information.
Partial2 An Element expression. This is an output argument. PartialDelete sets it to a reference to the second element it creates. PartialDelete may set this to Nothing. See the remarks for more information.
Point1 A Point3d expression. First point.
Point2 A Point3d expression. Second point.
Selector A Point3d expression. If the element is closed, PartialDelete uses Selector to select the portion of the element to delete.
ViewSpecifier A VARIANT expression. A View object, or an index in the range 1 through 8.

Remarks

This is the same process performed by MicroStation's DELETE PARTIAL command.

For closed elements, or open elements where the deleted portion includes the beginning or end of the element, PartialDelete only returns a single element. It sets Partial2 to Nothing.

If the entire input element is deleted (perhaps because Point1 and Point2 are the endpoints of the element) PartialDelete sets both Partial1 and Partial2 to Nothing.
