Remove Method

                          

Syntax

For the Attachments object:

   object.Remove AttachmentSpecifier

For the RasterClipMasks object:

   Boolean = object.Remove (ClipMaskDescriptor)

For the TagSets object and TagDefinitions object:

   object.Remove IndexOrName

The Remove method syntax has these parts:

Part Description
object A valid object.
AttachmentSpecifier A VARIANT expression. May be the index into the Attachments collection, the name of the attached model, or the Attachment object. If an Attachment object, it must be a member of the Attachments collection used to call Remove.
ClipMaskDescriptor A VARIANT expression. Index or a RasterClipMask object to remove.
IndexOrName A VARIANT expression.

Remarks

Remove generates an error if the IndexOrName does not refer to an object in the collection.

RasterClipMasks.Remove removes the clip mask from the list at the position given by Index.

TagDefinitions.Remove removes the TagDefinition from the collection. It also deletes every TagElement that is based on that TagDefinition.

TagSets.Remove removes all of the TagSet's TagDefinitions and every TagElement that is based on one of the deleted TagDefinitions.
