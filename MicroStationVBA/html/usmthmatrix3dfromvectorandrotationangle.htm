Matrix3dFromVectorAndRotationAngle Method

                          

Returns a matrix representing rotation around a vector.

Syntax

Matrix3d = object.Matrix3dFromVectorAndRotationAngle (Axis, Radians)

The Matrix3dFromVectorAndRotationAngle method syntax has these parts:

Part Description
object A valid object.
Axis A Point3d expression. The axis of rotation
Radians A Double expression. The rotation angle
