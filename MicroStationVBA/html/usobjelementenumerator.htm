ElementEnumerator Object

                                                     

Many methods that need to return a set of elements return an ElementEnumerator object that can be used to access all of the elements. Before a program can retrieve an element from an ElementEnumerator object, the program must first call MoveNext. MoveNext positions the enumerator at the next element. Initially, the enumerator is positioned before the first element, so it is necessary to call MoveNext once before retrieving the first element. MoveNext returns False if it was unable to advance, i.e., if there are no more elements. It is possible to use Current to retrieve the current element and Reset to restart the process. After calling Reset, the program must call MoveNext again to position the enumerator at the first element.
