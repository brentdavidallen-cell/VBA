SetCExpressionValueAsDLong Method

                          

Assigns the value to the location designated by the expression.

Syntax

object.SetCExpressionValueAsDLong CExpression, NewValue [, MdlApplicationName]

The SetCExpressionValueAsDLong method syntax has these parts:

Part Description
object A valid object.
CExpression A String expression. Where the value is saved.
NewValue A DLong expression. The value to be stored.
MdlApplicationName Optional. A String expression. The MDL application that published the variable referenced by the CExpression argument. This is important only if more than 1 MDL application has published a variable with that name.

Remarks

The only difference between this and SetCExpressionValue is that NewValue is a DLong. Do not use this method unless the CExpression refers to a 64-bit integer.
