Plane3d Type

        

Represents a plane in three-dimensional space.

The Plane3d type has these members:

Name Description
Origin A Point3d value that represents the origin point of the plane.
Normal A Point3d value that represents the normal vector of the plane.

Remarks

For points in a ModelReference's geometry, the values are measured in the model's coordinate system, unless stated otherwise.

The Normal vector does not have to be normalized.

A few functions that return or operate on Plane3d values are listed below.

  • Returns the intersection between two planes: Plane3dIntersectsPlane3d.
  • Returns the intersection between a plane and a line: Plane3dIntersectsRay3d.
  • Returns the intersection between a B-spline curve and a plane: BsplineCurve.IntersectPlane3d.
  • Returns the projection of a point onto a plane (optionally within a View): Point3dProjectToPlane3d.
  • Returns the front clipping plane of a View: View.ExtractFrontClippingPlane.
  • Returns the back clipping plane of a View: View.ExtractBackClippingPlane.
  • Returns the world-to-plane transform: Transform3dFromWorldToPlane3d.
  • Returns the plane-to-world transform: Transform3dFromPlane3dToWorld.
