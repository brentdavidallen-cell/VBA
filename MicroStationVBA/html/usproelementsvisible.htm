ElementsVisible Property

                          

Read/Write Boolean.

Gets or sets a value indicating whether elements within this object should be visible.

Syntax

object.ElementsVisible [= Boolean]

The ElementsVisible property syntax has these parts:

Part Description
object A valid object.
Boolean A Boolean expression.

Remarks

Getting ElementsVisible returns a value that indicates whether the Attachment is displayed. It can return False if DisplayFlag is False, if the Attachment's IsMissingModel is True, or if the Attachment is on a frozen Level.
