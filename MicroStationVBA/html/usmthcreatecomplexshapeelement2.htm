CreateComplexShapeElement2 Method

                          

Creates and returns a new ComplexShapeElement object.

Syntax

Set ComplexShapeElement = object.CreateComplexShapeElement2 (ChainableElements [, FillMode [, GapTolerance]])

The CreateComplexShapeElement2 method syntax has these parts:

Part Description
object A valid object.
FillMode Optional. An MsdFillMode expression. Indicates whether the returned ComplexShapeElement is filled. If the value is not specified or it is msdFillModeUseActive, MicroStation's active fill mode is used.
GapTolerance Optional. A Double expression. When GapTolerance is omitted or negative, this method is equivalent to CreateComplexShapeElement1: elements are added to the complex shape with no check for continuity. When nonnegative, an element is added only if either endpoint lies within a distance of GapTolerance to an endpoint of another element.
ChainableElements A ChainableElement () expression.

Remarks

Important: The geometric arguments are given in the coordinate system of the currently active model. Therefore, before you call this method, you should ensure that the model to which you intend to add the element is the active model. (You can make a model active by calling the Activate method of the appropriate ModelReference object.)

If GapTolerance is nonnegative, elements may be added to the ComplexShapeElement in reversed orientation to preserve continuity. If several ComplexShapeElements would be accreted in this fashion, only the first is returned; use AssembleComplexStringsAndShapes to return all such chains.
