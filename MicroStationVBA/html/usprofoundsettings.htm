FoundSettings Property

                          

Read-only Boolean.

Used to ask if an object has found its settings.

Syntax

object.FoundSettings

The FoundSettings property syntax has these parts:

Part Description
object A valid object.

Remarks

The StandardsCheckerController disables all Standards Checker handlers that have not found their settings. It indicates this to the user by graying out the handler's name in the StandardsCheckerController's main dialog box.

A Standards Checker handler should return True if it is able to operate. That is, it should return True if it has found the settings or if it is able to operate using the default settings without any additional information from the user.

If a checker can represent its settings in an XML DOM, then it can save its settings in the settings DgnLib file using SetApplicationSettings, can retrieve the settings using GetApplicationSettings, and can delete the settings using DeleteApplicationSettings. This approach is recommended but is not a requirement.
