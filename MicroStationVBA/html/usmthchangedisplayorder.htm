ChangeDisplayOrder Method

                          

Rasters.ChangeDisplayOrder moves a raster bringing it in front or back of other rasters.

Syntax

Boolean = object.ChangeDisplayOrder (RasterSpecifier, Command)

The ChangeDisplayOrder method syntax has these parts:

Part Description
object A valid object.
RasterSpecifier A VARIANT expression. A raster identification number, a Raster object or a RasterEnumerator listing rasters to move.
Command An MsdRasterDisplayOrderCommand expression. Where to move the raster: to front, to back, forward or backward
