DLongToHexString Method

                          

Returns the String representation of a DLong expression. The string is expressed in hexadecimal (base 16) digits.

Syntax

String = object.DLongToHexString (Value)

The DLongToHexString method syntax has these parts:

Part Description
object A valid object.
Value A DLong expression.

Remarks

Example: 

    Dim N As DLong
    Dim S As String
    N = DLongFromLong(123456789)
    S = DLongToHexString(N)
                    ' Now S contains the string value "75BCD15".
                    ' (&H75BCD15 = 123456789)

If you want to convert the value to a string of decimal (base 10) digits, use DLongToString.
