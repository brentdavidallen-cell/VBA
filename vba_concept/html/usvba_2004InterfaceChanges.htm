Change History

To get a list of methods, properties, and types added in a specific version select the search pane of MicroStation CONNECT Edition Visual Basic for Applications Help and search for a version number. Some example search patterns are 08.09*, 08.09.04, 08.11.05 and 10.00.

The documentation for each method, property, and type contains a Version section that identifies when that method, property, and type was added to the object model.

The version numbers that appear in the documentation are: 08.00.00, 08.00.01, 08.00.02, 08.00.04, 08.01.00, 08.01.01, 08.01.02, 08.01.03, 08.05.00, 08.05.01, 08.05.02, 08.09.02, 08.09.03, 08.09.04, 08.11.05, 08.11.07, 08.11.09 and 10.00. You can search for any of these numbers to find what was added in that version.
